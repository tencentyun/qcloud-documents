## 1. 接口描述
本接口(InquiryCdbPrice)用于查询某个云数据库实例类型的价格。
接口请求域名：<font style="color:red">cdb.api.qcloud.com</font>

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见公共请求参数页面。其中，此接口的Action字段为InquiryCdbPrice。

| 参数名称 | 必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| cdbType | 是 | String | 实例类型，支持固定规格类型和自定义规格类型。传 'custom' 代表自定义规格； 固定规格的取值，请使用[查询可创建规格](/doc/api/253/1333)接口获取实例类型。<font style='color:red'> 固定规格类型将会下线，推荐使用自定义规格类型。</font>|
| period | 是 | Int | 实例时长，单位：月，最小值1，最大值为36，请使用[查询可创建规格（支持可用区、配置自定义）](/doc/api/253/6109)接口获取可创建的实例时长，该接口返回字段timeSpan表示时长可选值|
| goodsNum | 是 | Int | 实例数量，默认为1, 最小值1，最大值为10，请使用[查询可创建规格（支持可用区、配置自定义）](/doc/api/253/6109)接口获取可创建的实例数量|
| memory | 否 | Int | 内存大小，cdbType=custom时，memory为必选 ，请使用[查询可创建规格（支持可用区、配置自定义）](/doc/api/253/6109)接口获取可创建的内存规格|
| volume | 否 | Int | 硬盘大小，cdbType=custom时，volume为必选 ，请使用[查询可创建规格（支持可用区、配置自定义）](/doc/api/253/6109)接口获取可创建的硬盘范围|
| zoneId | 否 | Int | 可用区ID，该参数缺省时，系统会自动选择一个可用区，请使用[查询可创建规格（支持可用区、配置自定义）](/doc/api/253/6109)接口获取可创建的可用区|

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码, 0表示成功，其他值表示失败。详见错误码页面的<a href='https://www.qcloud.com/doc/api/372/%E9%94%99%E8%AF%AF%E7%A0%81#1.E3.80.81.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81' title='公共错误码'>公共错误码</a>。|
| message | String | 错误信息|
| originalPrice | Int | 原价，单位：分（人民币） |
| price | Int | 实际价格，单位：分（人民币） |

## 4. 示例
固定规格：

输入
```
https://cdb.api.qcloud.com/v2/index.php?Action=InquiryCdbPrice
&<公共请求参数>
&cdbType=19
&period=1
&goodsNum=1
```

输出
```
{
	"code": 0,
	"message": "",
	"originalPrice": 2100,
	"price": 2100
}
```


自定义规格：

输入
```
https://cdb.api.qcloud.com/v2/index.php?Action=InquiryCdbPrice
&<公共请求参数>
&cdbType=custom
&memory=1000
&volume=25
&period=1
&goodsNum=1
```

输出
```
{
    "code": 0,
    "message": "",
    "originalPrice": 13200,
    "price": 12804
}
```
