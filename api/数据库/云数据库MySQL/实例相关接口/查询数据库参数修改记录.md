## 1. 接口描述
本接口(GetCdbParamsModifyHistory)用于查询数据库参数修改记录。
接口请求域名：<font style='color:red'>cdb.api.qcloud.com </font>



## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href='/doc/api/253/1739' title='公共请求参数'>公共请求参数</a>页面。其中，此接口的Action字段为GetCdbParamsModifyHistory。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| cdbInstanceId | 是 | String | 实例ID|
| page | 是 | Int | 拉取第几页|
| pageSize | 是 | Int | 每一页的数量|


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码, 0表示成功，其他值表示失败。详见错误码页面的<a href='https://www.qcloud.com/doc/api/372/%E9%94%99%E8%AF%AF%E7%A0%81#1.E3.80.81.E5.85.AC.E5.85.B1.E9.94.99.E8.AF.AF.E7.A0.81' title='公共错误码'>公共错误码</a>。|
| message | String | 模块错误信息描述，与接口相关。|
| codeDesc | String | 英文错误描述 |
| data | Array | 返回的结果数组 |
| data.totalNum | Int | 总的修改记录数 |
| data.curPageParamList | Array | 修改历史 | 
| data.curPageParamList.paramName | String | 参数名 | 
| data.curPageParamList.oldValue | String | 旧值 | 
| data.curPageParamList.newValue | String | 新值 | 
| data.curPageParamList.isSucess | Int | 修改是否成功1成功0失败 | 
| data.curPageParamList.modifyTime | String | 修改时间 | 


## 4. 示例

