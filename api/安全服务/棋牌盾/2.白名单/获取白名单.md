## 接口描述
用于获取棋牌盾 IP 白名单列表的接口。
协议：`HTTPS`
域名：`shield.api.qcloud.com`
接口名：`ShieldWhitelistGet`

## 请求参数
无。

## 响应参数

| 参数      | 示例        | 类型    | 描述                                       |
| --------- | ---------- | ----- | ---------------------------------------- |
| whitelist | [“域名列表”,…] | Array | 域名列表：<pre>"whitelist": [</br>"http://www.website1.com/",</br>"http://www.website2.com/"</br>]</pre> |
