## 接口描述
用于获取某 BGP 高防 IP 的源 IP 白名单列表的接口。
协议：`HTTPS`
域名：`bgpip.api.qcloud.com`
接口名：`GetSrcWhiteIP`

## 请求参数

| 参数 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| bgpId | 是 | String | BGP 高防 IP 的资源 ID |
