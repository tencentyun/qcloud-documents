一个典型的接口请求如下, Action=DescribeInstance表示查询云服务器实例的详情。

```
https://domain/v2/index.php?Action=DescribeInstances
&SecretId=xxxxxxx
&Region=gz
&Timestamp=1402992826
&Nonce=345122
&Signature=mysignature
&instanceId=101
```
其中instanceId为指令参数，其余为通用参数。
