
## 功能说明

返回结果中的错误码表示了用户调用云 API 的结果。其中，code 为公共错误码，适用于所有腾讯云产品的 API 接口。若 code 为0，表示调用成功，否则表示调用失败。若调用失败，用户可以根据公共错误码列表确定错误原因并采取相应措施。
示例如下：

```
{
    "code": "5100"
}
```
