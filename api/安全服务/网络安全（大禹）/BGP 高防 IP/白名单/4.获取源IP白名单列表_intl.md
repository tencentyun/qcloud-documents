## API Description
This API (GetSrcWhiteIP) is used to obtain the source IP whitelist under a High Defense IP.
Protocol: `HTTPS`
Domain name: `bgpip.api.qcloud.com`
 

## Request Parameters

| Parameter | Required | Type | Description |
|---------|---------|---------|---------|
| bgpId | Yes | String | Resource ID of High Defense IP |
