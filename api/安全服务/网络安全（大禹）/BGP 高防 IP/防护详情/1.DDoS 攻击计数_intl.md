## API Description
This API (BGPIPDDoSGetCounter) is used to obtain the number of DDoS attacks on High Defense IP, attack peak, and number of times the elastic defense is enabled.
Protocol: `HTTPS`
Domain name: `bgpip.api.qcloud.com`
 

## Request Parameters
The following request parameter list only provides the API request parameters. Common request parameters are required when the API is called. For more information, please see the [Common Request Parameters](https://cloud.tencent.com/document/api/213/6976) page. The Action field for this API is `BGPIPDDoSGetCounter`.

| Parameter | Required | Type | Description |
|---------|---------|---------|---------|
| id | Yes | String | Resource ID of High Defense IP |
| beginDate | Yes | String | Start time of the data collection. It should take the format of YYYY-MM-DD, such as 2016-11-10. |
| endDate | Yes | String | End time of the data collection. It should take the format of YYYY-MM-DD, such as 2016-11-11. |

## Response Parameters

| Parameter | Unit | Type | Description |
|---------|---------|---------|---------|
| attacks | time | Integer | Number of DDoS attacks |
| attackPeak | Mbps | Integer | DDoS attack peak |
| overload | time | Integer | Number of times the elastic defense is enabled |
| bandwidth | Mbps | Integer | Defense bandwidth of High Defense IP |
