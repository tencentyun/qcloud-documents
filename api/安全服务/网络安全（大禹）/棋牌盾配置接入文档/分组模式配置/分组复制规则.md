### 1. 接口描述
复制棋牌盾A分组的转发规则到B分组。
协议：HTTPS 
接口名：ShieldGroupCopyTransRules

### 2.输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，详见[公共参数说明](https://cloud.tencent.com/document/product/295/7279)页面。其中，此接口的Action字段为ShieldGroupCopyTransRules。

| 参数名称 | 是否必须 | 类型 | 描述 |
|---------|---------|---------|---------|
| id | <font color=red> 必选 </font color=red> | String |目标棋牌盾分组的资源ID，格式是grp-XXXXXXX，复制完成B分组中的规则全部删除 |
| cypId | <font color=red> 必选 </font color=red> | String |A分组即规则来源分组ID |
