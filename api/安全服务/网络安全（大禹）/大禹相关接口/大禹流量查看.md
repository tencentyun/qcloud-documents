## 1. 接口描述
本接口(getDayuStatPoints)用于根据用户输入，展示一段时间内的大禹流量。
接口请求域名：dayu.api.qcloud.com

## 2. 输入参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见<a href="https://www.qcloud.com/document/product/297/7291" title="公共请求参数">公共请求参数</a>
页面。其中，此接口的Action字段为getDayuStatPoints。

| 参数名称 | 是否必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| startDate | 是 | date | 流量查看开始时间 |
| endDate | 是 | date | 流量查看结束时间 |
| statType | 是 | String | 可选值：flux（流量）|
| hosts.n  | 否 | String | 域名。此处可以填写域名数组，如果不填则表示所有域名 |
| projectIds.n | 否 | String | 域名对应项目的项目ID。此处可以填写项目ID数组 |

## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码，0表示成功，其他值表示失败。详见错误码页面的<a href="https://www.qcloud.com/document/product/297/7297" title="公共错误码">公共错误码</a>。 
| message | String | 模块错误信息描述，与接口相关。|
| data | Array | 返回的数组 |

其中，data参数的构成如下：

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| name | String | 返回的数据名称 |
| desc | String | 返回的数据描述 |
| unit | String | 单位 |
| period | Int | 统计周期，例如为5时表示每5分钟进行一次数据统计 |
| points | Array | 开始时间到结束时间段内返回的数据 |
| hosts_count | Int | 域名个数 |
