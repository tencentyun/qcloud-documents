## 接口描述
通过 HTTPS Post 方式提交请求，腾讯云收到请求后，返回调用呼叫转接接口后的信息。

## 请求
### 请求地址
```
/ipcc/call/transfer
```

### 请求方式
Post 请求

### 请求包体
| 属性       | 类型     | 必选   | 说明       |
| -------- | ------ | ---- | -------- |
| appId    | String | 是   | 应用 ID    |
| callId   | String | 是   | 呼叫 ID    |
| called   | String | 是   | 固话或手机    |
| fileName | String | 否   | 放音文件名    |
| data     | String | 否   | 要进行透传的数据 |

## 响应

### 响应包体

| 属性     | 类型     | 必选  | 说明                                       |
| ------ | ------ | ---- | ---------------------------------------- |
| code   | String | 是   | 响应码<br>000000 表示成功<br>108100 解析失败-入参错误<br>108101 应用 ID 不存在<br>108102 消息队列已满<br>108105 callId 不存在 |
| appId  | String | 否   | 应用 ID                                    |
| callId | String | 否   | 呼叫号 ID                                   |
