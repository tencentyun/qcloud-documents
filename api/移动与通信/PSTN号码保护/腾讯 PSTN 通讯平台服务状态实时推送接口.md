## 1. 接口描述

功能： 腾讯 PSTN 通讯平台服务状态实时推送接口    
接口地址： 双方协商好  
请求方式：POST  

## 2. 参数说明 
| 参数名 | 要求 | 备注 | 
|---------|---------|------------|
| appId | 必选 | xxx | 
| cityId | 必选 | 主被叫号码归属地，详见《腾讯-中间号-城市 id.xlsx》 | 
| level | 必选 | 故障级别（0：正常  5：告警  6：错误） | 
| status | 必选 | 0：后端服务状态正常  501：呼入呼出线路负载过高  502：平台接入负载过高（包括接入、cache、DB 等）  503：（ASR）接听率指标超过告警阈值  504：（ACD）平均通话时长指标超过告警阈值  601：线路呼入呼出不可用需降级服务  602：平台绑定异常不可用需降级服务  603：（ASR） 接听率指标超过降级阈值  604：（ACD）平均通话时长指标超过降级阈值 | 

**确认响应包必填字段**  
```
HTTP/1.1 200 OK
Content-Type: text/plain;charset=utf-8
Content-Length: 0
```
