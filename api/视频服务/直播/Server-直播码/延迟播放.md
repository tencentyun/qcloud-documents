## 1.接口描述

- **接口**
  **Live_Set_Delay**：延迟播放直播流。
- **地址**
  API 调用地址为：` http://<font color='red'>fcgi.</font>video.qcloud.com/common_access`。
- **用途**
  可以延迟播放直播活动，延迟后，播放的是当前推流内容。

## 2.输入参数

| 参数名 | 参数含义 | 类型 | 备注 | 是否必需 |
|---------|---------|---------|---------|---------|
| cmd                        | 客户ID     | int       | 即直播 APPID，用于区分不同客户的身份 |  Y          | 
| interface                 | 接口名称   | string |  如：Get_LivePushStat  |  Y          | 
| t | [有效时间](https://cloud.tencent.com/document/product/267/32744#.E4.BA.8B.E4.BB.B6.E6.B6.88.E6.81.AF.E9.80.9A.E7.9F.A5.E5.8F.82.E6.95.B0.E8.AF.B4.E6.98.8E) | int  | UNIX 时间戳（十进制） |  Y | 
| sign | [安全签名](https://cloud.tencent.com/document/product/267/32744#.E4.BA.8B.E4.BB.B6.E6.B6.88.E6.81.AF.E9.80.9A.E7.9F.A5.E5.8F.82.E6.95.B0.E8.AF.B4.E6.98.8E) | string | MD5（key+t） | Y | 
| Param.s.channel_id | 直播码 | string | -| Y|
| Param.n.bizid | bizi | int | 控制台可查看 bizid | Y|
|Param.n.time_length|延播时间|int|单位是秒 |Y|



## 3.输出结果
| 参数名 | 参数含义 | 类型 | 备注            |
|---------|---------|---------|------------------|
| ret      | 返回码 |   int  |  0：成功；其他值：失败|
| message | 错误信息 |   string  |  错误信息|

|sign | b17971b51ba0fe5916ddcd96692e9fb3 |
