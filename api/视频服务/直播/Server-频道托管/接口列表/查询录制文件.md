## 1. 接口描述
域名：live.api.qcloud.com
接口名: GetVodRecordFiles

查询频道录制文件结果信息，包含文件名，任务id，文件id， 录制消息。

<td> <font color"red">是</font>
<td> Int
<td> 频道的ID号
<tr>
