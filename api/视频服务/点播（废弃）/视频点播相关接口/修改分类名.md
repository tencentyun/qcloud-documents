## 1. 接口描述
 
域名: vod.api.qcloud.com
接口名: ModifyClass

修改分类名

 

## 2. 输入参数
 

<table class="t"><tbody><tr>
<th><b>参数名称</b></th>
<th><b>必选</b></th>
<th><b>类型</b></th>
<th><b>描述</b></th>
<tr>
<td> classId <td> 是 <td> Int <td> 待修改的分类id
<tr>
<td> className <td> 是 <td> String <td> 新的分类名
</tbody></table>

 

## 3. 输出参数
 

<table class="t"><tbody><tr>
<th><b>参数名称</b></th>
<th><b>类型</b></th>
<th><b>描述</b></th>
<tr>
<td> code <td> Int <td> 错误码, 0: 成功, 其他值: 失败
<tr>
<td> message <td> String <td> 错误信息
</tbody></table>

输入

    "code": 0,
    "message": ""
