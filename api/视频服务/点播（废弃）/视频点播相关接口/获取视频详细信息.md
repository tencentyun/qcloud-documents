## 1. 接口描述
 
域名：vod.api.qcloud.com
接口名: DescribeVodPlayUrls

获取当前视频所有播放地址、格式、码率、高度、宽度信息

 

## 2. 输入参数
 
<table class="t"><tbody><tr>
<th><b>参数名称</b></th>
<th><b>必选</b></th>
<th><b>类型</b></th>
<th><b>描述</b></th>
<tr>
<td> fileId
<td> 是
<td> String
<td> 希望获取的视频的ID
</tbody></table>
