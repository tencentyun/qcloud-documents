## 1. 接口描述
 
本接口 (DescribeBmLoadBalancersTaskResult) 用于查询负载均衡实例异步任务的执行情况。

接口请求域名：<font style="color:red">lb.api.qcloud.com</font>


## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见[公共请求参数](/doc/api/456/6718)页面。

| 参数名称 | 是否必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| requestId | 是 | Int | 任务ID。由具体的异步操作接口提供。|




## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码。0表示成功，其他值表示失败。详见错误码页面的[公共错误码](/doc/api/456/6725)。|
| message | String | 模块错误信息描述，与接口相关。|
| codeDesc | String | 返回码信息描述。|
| data | Obj | 返回对象。|

data描述当前任务的状态信息，包含字段如下

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| status | Int | 任务当前状态。0：成功，1：失败，2：进行中。|
