## 1. 接口描述
 InquiryBmLBPrice 接口用来查询黑石负载均衡的价格。黑石负载均衡实例的具体价格可以参考产品说明的价格总览。

接口访问域名：lb.api.qcloud.com

## 2. 请求参数
以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，见[公共请求参数](/doc/api/456/6718)页面。其中，此接口的Action字段为 InquiryBmLBPrice。
<table class="t"><tbody><tr>
<th><b>参数名称</b></th>
<th><b>必选</b></th>
<th><b>类型</b></th>
<th><b>描述</b></th>
<tr>
<td> loadBalancerType
<td> 是
<td> Int
<td> 负载均衡的类型，取值为2和3。2表示公网类型，3表示内网类型。
</tbody></table>

## 3. 返回参数
 
<table class="t"><tbody><tr>
<th><b>参数名称</b></th>
<th><b>类型</b></th>
<th><b>描述</b></th>
<tr>
<td> code
<td> Int
<td> 公共错误码，0表示成功，其他值表示失败。详见错误码页面的<a href="" title="公共错误码">公共错误码</a>。
<tr>
<td> message
<td> String
<td> 模块错误信息描述，与接口相关。
<tr>
<td> price
<td> Int
<td>黑石负载均衡实例的价格，单位为“分/天”。
</tbody></table>

模块错误码

| code | 英文提示 | 错误描述 |
|------|------|------|
| 9006 | InternalError.CCDBAbnormal | CCDB 服务异常 |
| 9016 | InternalError.DESAbnormal | DES 服务异常 |
| 12003 | IncorrectStatus.LBWrongStatus | 该负载均衡状态不正确,无法执行当前操作 |
| 12012 | InvalidResource.NotExist | 该负载均衡实例不存在 |
| 12016 | InternalError.LBAbnormal | LB 服务异常|
