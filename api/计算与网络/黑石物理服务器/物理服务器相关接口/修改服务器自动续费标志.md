## 功能描述

ModifyDeviceAutoRenewFlag 接口用来修改服务器自动续费标志位。

接口访问域名：bm.api.qcloud.com

## 请求

### 请求示例

```
https://bm.api.qcloud.com/v2/index.php?
	Action=ModifyDeviceAutoRenewFlag
	&<公共请求参数>
	&instanceIds.0=<设备ID0>
	&instanceIds.1=<设备ID1>
	&autoRenewFlag=<自动续费>
```

### 请求参数

以下请求参数列表仅列出了接口请求参数，正式调用时需要加上公共请求参数，其它参数参见 [公共请求参数](/doc/api/456/6718)页面。其中，此接口的 Action 字段为 ModifyDeviceAutoRenewFlag。

| 参数名称          | 必选   | 类型           | 描述                       |
| ------------- | ---- | ------------ | ------------------------ |
| instanceIds   | 是    | Array(String) | 设备 ID 列表。                   |
| autoRenewFlag | 是    | Int          | 自动续费标志位。0：不自动续费；1：自动续费。 |
