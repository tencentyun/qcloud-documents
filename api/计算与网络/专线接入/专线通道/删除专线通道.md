## 功能描述
用于删除专用通道的接口。
域名：`dc.api.qcloud.com`
接口名：`DeleteDirectConnectTunnel`

##  请求参数
| 参数 | 类型  | 必选 | 描述 |
| --------------------| ------| --------| -----------  | 
| directConnectTunnelId | String | 是 | 专用通道 ID</br>例如 dcx-abcdefgh | 

## 响应参数
| 参数 | 类型 | 描述|
|---------|---------|---------|
| code| Int | 错误码</br>0：成功</br>其他值： 失败 |
| message |  String | 错误信息 |
 
## 代码示例
### 请求示例
```
  GET https://dc.api.qcloud.com/v2/index.php?Action=DeleteDirectConnectTunnel
  &<<a href="https://cloud.tencent.com/doc/api/229/6976">公共请求参数</a>>
  &directConnectTunnelId=dcx-abcdefgh
```

### 响应示例
```
{
    "code": 0,
    "message": ""
}
```
