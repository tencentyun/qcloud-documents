响应包体内的返回码(code), 反映了腾讯云API调用和执行的概要结果.
当返回码不为 0 时, 表示请求未正常执行, 返回码也称为错误码, 错误描述(message)对该结果进行了细化补充，用户可根据错误码判断API的执行情况。
message在部分终端（例如浏览器），中文会显示成 unicode 编码，需要解码。
