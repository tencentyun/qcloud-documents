## 1. 接口描述
接口请求域名：`ckafka.api.qcloud.com`。
本接口（DeleteTopic）用于在消息队列 CKafka 实例下删除主题。


## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见 [公共请求参数](https://cloud.tencent.com/document/product/597/10084) 文档。

| 参数名称 | 必选 | 类型 | 描述 |
| --- | --- | --- | --- |
| instanceId | 是 | String | 实例 ID |
| topicName | 是 | String | 主题名称 |


## 3. 示例

**输入：**

```http
 https://domain/v2/index.php?Action=DeleteTopic&instanceId=topic-xxxxxx&topicName=tinatest<公共请求参数>
```

**输出：**

```
{
	"code": 0,
	"codeDesc": "Success",
	"message": "ok"
}
```
