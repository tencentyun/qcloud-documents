## 功能说明
若 API 调用失败，则最终返回结果中的错误码 code 不为0，message 字段会显示详细错误信息。用户可以根据 code 和 message 在错误码列表查询具体的错误信息。
示例如下：
```shell
{
    "code": 5100,
    "message": "(100004)projectId不正确"
}
```
