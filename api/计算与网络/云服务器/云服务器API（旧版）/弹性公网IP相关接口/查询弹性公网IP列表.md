## 1. 接口描述
 
域名: eip.api.qcloud.com
接口名: DescribeEip

查询弹性公网 IP。

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 公共错误码。0表示成功，其他值表示失败。 |
| message | String | 模块错误信息描述，与接口相关。 |
|  totalCount  |  Int |  返回符合过滤条件的 EIP 数量；假如指定 limit，offset，该值有可能大于 data 列表中的数量 |
| data |   Array | 返回列表 |
