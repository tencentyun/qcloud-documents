## 1. 接口描述
域名:cvm.api.qcloud.com
接口名:DescribeInstanceMainInfo

获取虚拟子机主要信息列表

## 2. 输入参数
| 参数名称 | 必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| offset | 否 | Int | 偏移量,默认为0|
| limit | 否 | Int | 默认20|
| projectId | 否 | Int | 项目ID|
| vpcId | 否 | Int | vpcId|
| zoneId | 否 | Int | zoneId|
| lanIps.n (lanIps 为列表，此处入参需要填写列表元素 ) | 否 | String | ip列表|
| instanceName | 否 | String | 主机名称|
| searchWord | 否 | String | 只支持IP段搜索|
| uInstanceIds.n (uInstanceIds 为列表，此处入参需要填写列表元素 ) | 否 | String | 只支持统一资源ID|
