
## 1. 接口描述

本接口（SyncImages）用于将自定义镜像同步到其它地区。

* 该接口每次调用只支持同步一个镜像。
* 该接口支持多个同步地域。
* 单个帐号在每个地域最多支持存在10个自定义镜像。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见[公共请求参数](/document/api/213/15692)。

| 参数名称 | 是否必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：SyncImages |
| Version | 是 | String | 公共参数，本接口取值：2017-03-12 |
| ImageIds.N | 是 | Array of String | 镜像ID列表 ，镜像ID可以通过如下方式获取：<li>通过[DescribeImages](https://cloud.tencent.com/document/api/213/15715)接口返回的`ImageId`获取。</li><li>通过[镜像控制台](https://console.cloud.tencent.com/cvm/image)获取。<br>镜像ID必须满足限制：<li>镜像ID对应的镜像状态必须为`NORMAL`。</li><li>镜像大小小于50GB。</li>镜像状态请参考[镜像数据表](/document/api/213/9452#image_state)。 |
| DestinationRegions.N | 是 | Array of String | 目的同步地域列表；必须满足限制：<li>不能为源地域。</li><li>必须是一个合法的Region。</li><li>暂不支持部分地域同步。</li>具体地域参数请参考[Region](https://cloud.tencent.com/document/product/213/6091)。 |

## 3. 输出参数



| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| RequestId | String | 唯一请求ID，每次请求都会返回。定位问题时需要提供该次请求的RequestId。 |

## 4. 错误码



| 错误码 | 描述 |
|---------|---------|
| InvalidImageId.IncorrectState | 镜像状态不合法。 |
| InvalidImageId.NotFound | 未找到该镜像。 |
| InvalidImageId.TooLarge | 镜像大小超过限制。 |
| InvalidRegion.NotFound | 未找到该区域。 |
| InvalidRegion.Unavailable | 该区域目前不支持同步镜像。 |

## 5. 示例

## 示例1 同步镜像

### 场景描述

同步镜像img-o3ycss2p到广州

                
### 请求参数

```
https://cvm.tencentcloudapi.com/?Action=SyncImages
&ImageIds.0=img-o3ycss2p
&DestinationRegions.0=ap-guangzhou
&<公共请求参数>
```
### 返回参数

```
{
  "Response": {
    "RequestId": "71e69b56-32be-4412-ab45-49eded6a87be"
  }
}
```
