## 功能说明
若 API 调用失败，则最终返回结果中的错误码 code 不为 0，message 字段会显示详细错误信息。用户可以根据 code 和 message 在错误码列表查询具体的错误信息，并进行相应的处理。
示例如下：
```
{
    "code": 5100,
    "message": "(10010)projectId不正确"
}
```
