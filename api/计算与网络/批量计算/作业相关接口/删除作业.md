## 1. 接口描述
本接口（DeleteJob ）用于删除作业记录。
删除作业的效果相当于删除作业相关的所有信息。删除成功后，作业相关的所有信息都无法查询。
待删除的作业必须处于完结状态，且其内部包含的所有任务实例也必须处于完结状态，否则会禁止操作。完结状态，是指处于 SUCCEED 或 FAILED 状态。

接口请求域名：batch.api.qcloud.com

## 2. 输入参数
名称 | 类型  | 是否必选 | 描述
-----|------|-----|------
Version | String | 是 | API版本号
JobId | String | 是 | 作业ID

## 3. 输出参数
参数名称 | 类型  | 描述
-----|------|------
RequestId | String | 唯一请求ID。每次请求都会返回RequestId。当用户调用接口失败找后台研发人员处理时需提供该RequestId。


## 4. 错误码
以下错误码表仅列出了该接口的业务逻辑错误码。

错误码 | 描述
-----|------
JobNotFound | 指定作业ID不存在。
InvalidBatchJobId.Malformed | 非法的作业ID格式。
UnsupportedOperation | 禁止该操作。
InternalServerError | 内部服务错误。
