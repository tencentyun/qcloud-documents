## 1. 接口描述

本接口 (DeleteTopic) 用于在 IoT MQ 实例下删除主题。

接口请求域名：<font style="color:red">mqiot.api.qcloud.com</font>

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数，其它参数见 [公共请求参数](https://cloud.tencent.com/doc/api/431/5883) 页面。

| 参数名称 | 必选 | 类型 | 描述 |
| --- | --- | --- | --- |
| instanceId | 是 | String | 实例 id |
| topicName | 是 | String | 主题名称 |


## 3. 示例

输入：

```
 https://domain/v2/index.php?Action=DeleteTopic&instanceId=mqtt-xxxxxxxx&topicName=topictest&<公共请求参数>
```

输出：

```
  {
      "code" : 0,
      "message" : "ok",
  }

```
