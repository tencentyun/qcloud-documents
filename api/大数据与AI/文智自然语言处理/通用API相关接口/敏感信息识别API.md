## 1. 接口描述
域名：wenzhi.api.qcloud.com
接口名: TextSensitivity <br />
识别信息的色情、政治等敏感程度。可用于敏感信息过滤，舆情监控等。
>?敏感词库自动更新，暂不支持客户自定义。
