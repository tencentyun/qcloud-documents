## 接口描述

本接口（RemoveUserFromGroup）可用于从用户组删除用户。

请求域名:

```
cam.api.qcloud.com
```

## 输入参数

以下请求参数列表仅列出了接口请求参数，未列出 [公共请求参数](https://cloud.tencent.com/document/api/213/6976)。

| 参数名称 | 类型  | 必选 | 描述                                                         |
| -------- | ----- | ---- | ------------------------------------------------------------ |
| info     | array | 是   | 索引数组。成员是关联数组，有 uid 和 groupId 两个字段，分别是用户 ID 和用户组 ID |

## 输出参数

无。
