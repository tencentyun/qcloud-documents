## 1. 接口描述

删除一个已创建的自定义命名空间。若命名空间下仍有指标则无法删除,请先删除该命名空间下的 <a href="https://cloud.tencent.com/document/api/397/4241">指标</a>。

域名：monitor.api.qcloud.com
接口名: DeleteNamespace
