## 请求
### 请求包体
| 属性        | 类型     |  必选  | 说明                      |
| --------- | ------ | ---- | ----------------------- |
| appId     | String | 是   | 应用 ID                   |
| callId    | String | 是   | 呼叫 ID                   |
| event     | String | 是   | 通知事件类型（incomingcallack） |
| timeStamp | String | 否   |                          |

## 响应
### 响应包体
| 属性   | 类型     |  必选  | 说明                 |
| ---- | ------ | ---- | ------------------ |
| code | String | 是   | 返回结果，0 表示成功，1 表示失败 |
| msg  | String | 否   | 失败原因               |

## 实际案例

```
{

```

### JSON 响应示例
```
{
    "code": "0",
    "msg": "0"	
}
```
