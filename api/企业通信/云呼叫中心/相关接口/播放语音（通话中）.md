## 接口描述
给在通话过程中的双方，播放语音。

## 请求
### 请求地址
```
/ipcc/call/playIncall
```

### 请求方式
Post 请求

### 请求包体

| 属性     | 类型   | 必选 | 说明                                                         |
| -------- | ------ | ---- | ------------------------------------------------------------ |
| appId    | String | 是   | 应用 ID                                                      |
|playFlag	|Int	|是|	放音标签，0表示 TTS，1表示 VOICE |
|playChannel|String|否|Called 的具体号码（例如 188xxxxxxxx），或者设置为`both`表示给双方播放|
|playTime	|Int|	否|	播放次数，默认播放1次|
|voiceStr	|String	|是	|Playflag 设置为0，表示转换的 TTS 文本（800字节以内，时间较长）<br>Playflag设置为1，表示播放的语音文件|

## 响应

### 响应包体

| 属性   | 类型   | 约束 | 说明                                                         |
| ------ | ------ | ---- | ------------------------------------------------------------ |
| code   | String | 是   | 响应码<br>000000表示成功<br>108100表示解析失败 - 入参错误<br>108101表示应用 ID 不存在<br>108102表示消息队列已满<br>108103表示呼叫资源不足<br>108104表示其他错误 |
| appId  | String | 是   | 应用 ID                                                       |
| callId | String | 是   | 呼叫号 ID                                                     |

## 实际示例
