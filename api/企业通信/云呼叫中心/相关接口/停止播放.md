## 接口描述
停止正在播放的语音。
## 请求
### 请求地址

```
/ipcc/call/stoPlay
```

### 请求方式

```
Post请求
```

### 请求包体

| 属性        | 类型     | 约束   | 说明                   |
| --------- | ------ | ---- | -------------------- |
| appId     | String | 必选   | 应用 ID                |
| callId    | String | 必选   | 呼叫 ID

### JSON请求示例

```
{
     "appId":"247e35ff320a4142a105024055c367cf",
     "callId":"88-2018031000182918181271218-108bc381ca0f42d8-out"
}
```

## 响应
### 响应包体
| 属性        | 类型     | 约束   | 说明                   |
| --------- | ------ | ---- | -------------------- |
|code     |String | 必选 |响应码，000000表示成功  失败：108100 解析失败-入参错 108101 应用Id不存在 108102 消息队列已满 108105 Callid不存在  108104 其他错误 |
| appId     | String | 必选   | 应用 ID              |
| callId    | String | 必选   | 呼叫 ID             |

### JSON响应示例

```
{
     "code": "000000",
     "appId":"247e35ff320a4142a105024055c367cf",
     "callId":"88-2018031000182918181271218-108bc381ca0f42d8-out"
}
```
