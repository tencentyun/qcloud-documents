## 请求

### 请求方式

Post 请求

### 请求包体

| 属性            | 类型     | 必选   | 说明                |
| ------------- | ------ | ---- | ----------------- |
| appId         | String | 是   | 应用 ID             |
| callId        | String | 是   | 呼叫 ID             |
| fileName      | String | 是   | 留言文件云存储位置         |
| data          | String | 否   | 用户透传数据            |
| event         | String | 是   | 通知事件类型（callleaveendrpt）            |
| timeStamp     | String | 否   | 时间戳               |

## 响应

### 响应包体

| 属性   | 类型     | 必选   | 说明                 |
| ---- | ------ | ---- | ------------------ |
| code | String | 是   | 返回结果，0表示成功，1表示失败 |
| msg  | String | 否   | 失败原因               |
