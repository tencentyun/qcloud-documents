## 功能描述
通过线下对接配置回调业务 URL 的方式，通知业务方事件下发的状态。

## 接口描述
用户提前取消，电话未接听。

## 请求
### 请求包体
| 属性        | 类型     |  必选  | 说明                      |
| --------- | ------ | ---- | ----------------------- |
| appId     | String | 是   | 应用 ID                   |
| callId    | String | 是   | 呼叫 ID                   |
| event     | String | 是   | 通知事件类型（incomingcallcancel） |
| timeStamp | String | 否   | -           |

## 响应
需用户方响应，采用通用响应规则，如不响应，重试三次。
