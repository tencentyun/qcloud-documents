## 功能描述

Delete Object请求可以将一个文件（Object）删除。

## 请求

### 请求语法

```Http
DELETE /ObjectName HTTP/1.1
Host: <BucketName>-<UID>.<Region>.myqcloud.com
Date: date
Content-Length:length
Authorization: authorization string
```
