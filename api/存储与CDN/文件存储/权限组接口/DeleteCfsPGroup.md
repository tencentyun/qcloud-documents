## 接口描述

本接口（DeleteCfsPGroup）用于删除权限组。
接口请求域名：`cfs.api.qcloud.com`

## 输入参数

|       参数      | 必填 |  类型  |                               描述                           |
|-----------------|------|--------|--------------------------------------------------------------|
| PGroupId        | 是   | string | 权限组 ID|


## 输出参数

| 参数名称 |  类型 | 描述 |
|----------|----- | ---- |
|AppId     | int    |用户 ID   |
|PGroupId   | string |权限组 ID |
