## 1. 接口描述

域名：`cvm.api.qcloud.com`
接口名：`DescribeAvailabilityZones`
查询可用区信息。

<th><b>参数名称</b></th>

<tr>
<td> zoneId
<td> 否
<td> Int
<td> 可用区 ID
</tbody></table>
