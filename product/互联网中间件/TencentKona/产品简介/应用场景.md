## Spring MVC

基于 Spring MVC 开发的 Web 应用程序使用腾讯 Kona，可以享受便捷丰富的监控工具以及调试工具，方便用户快速对性能进行监控和诊断。

## 大数据场景

在大数据领域，Elasticsearch 和 Hbase 等软件都基于 Java 构建，通过使用腾讯 Kona 可以保证软件运行在安全的环境中，同时带来更高的启动速度和性能。
