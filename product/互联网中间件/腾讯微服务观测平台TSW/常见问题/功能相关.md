### TSW 是否支持观测实例级调用？
TSW 支持实例级调用观测，将于2021年5月后推出对 Java 应用的 JVM 监控能力。

### TSW 是否支持接口级监控？			
TSW 支持接口级监控，支持查看接口被哪些服务调用的监控视图、统计信息和调用详情。

### TSW 是否支持前端应用的链路追踪与 App端应用的链路追踪？		
TSW 暂不支持前端与 App 端，目前专注于 Server 端的监控。但您可以尝试通过 Skywalking 等开源客户端方法，将应用接入调用链中，将前端、App 端视为一个后端服务处理。

TSW将联合QAPM（客户端 APM）产品在未来推出前后端全链路观测方案，预计将于2021年7月上线。


### TSW 是否支持方法级调用链？
TSW 支持方法级调用链。目前，TSW 服务端不接受 Local Span 的上报，只接受远程调用的 Span 上报，预计将于2021年6月支持 Local Span 的上报。


### 在 SQL/NoSQL 调用中，为何会出现组件调用次数大于 SQL/NoSQL 操作数的情况？
数据库类组件调用中，若您的业务有 Commit（提交）操作，则不会计入 SQL/NoSQL 调用次数，但会计入组件的调用次数。
