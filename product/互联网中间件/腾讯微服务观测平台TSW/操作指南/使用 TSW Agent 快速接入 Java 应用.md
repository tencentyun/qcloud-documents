## 操作场景
使用官方提供的 TSW Agent 可快速接入 Java 应用。TSW Agent 提供三种安装方式，您可依据自身需求及应用环境选择适合的接入形式。

|   安装方式  |  适用场景 |  预计耗时 | 
| ----- |---- |--- |
| [脚本接入](#脚本接入) |  新手用户 | 1分钟 |
| [简单手动接入](#简单手动接入) |  本机网络与公网不通的新手用户 | 3分钟  |
| [手动安装](https://cloud.tencent.com/document/product/1311/51600) |  对链路追踪、Opentracing 类链路追踪产品有一定理解的用户 | 5分钟 | 

## 脚本接入[](id:脚本接入)
通过该接入方式，您可在1分钟内完成一个 Java 应用/服务实例的接入。该接入方式适合新手用户，通过脚本接入，您无需修改 Agent 配置，脚本可自动化完成 Agent 的下载、解压，配置修改与应用的启动。

### 前提条件
- 确保您的服务器能够访问公网。
- 已安装 JDK，建议升级至1.8.X最新版本使用。

### 操作流程
1. 登录 [TSW 控制台](https://console.cloud.tencent.com/tsw)。
2. 在左侧导航栏中，单击【服务监控】>【服务列表】。
3. 在服务列表页，单击列表上方的【接入服务】。
4. 在接入服务界面中，选择接入语言为 Java，选择数据采集方式为 TSW。
5. 输入服务名，服务名称请与您的应用程序名保持一致。
例如：您的应用程序包名为 Order-Service，则需要在服务名处填写一致的名称。
6. 复制右下方脚本接入中自动生成的脚本，在应用程序包所在的目录下执行该脚本。
![](https://main.qcloudimg.com/raw/6eba42d70ec673353074fd1ff0503c2f.png)



### 接入验证
向应用发送请求，在收到响应后，在 TSW 控制台查看调用数据。
您可以在1分钟内通过【链路追踪】>【调用链查询】>【[Span查询](https://console.cloud.tencent.com/tsw/trace?rid=1&tab=span)】查找调用详情。监控曲线与统计数据将在1分钟后开始正常显示。

## 简单手动接入[](id:简单手动接入)
通过该接入方式，您可在3分钟内完成一个 Java 应用/服务实例的接入。
本接入流程中，用户无需修改配置，主要操作依然通过脚本完成。该接入方式适合本机网络与公网不通场景下的新手用户操作接入，通过简单手动接入，您无需修改 Agent 配置，脚本可自动化完成 Agent 的解压，配置修改与应用的启动。仅 Agent 的下载无法通过脚本完成，其余工作依然由脚本自动化托管。

### 前提条件
已安装 JDK，建议升级至1.8.X最新版本使用。

### 操作流程
1. 登录 [TSW 控制台](https://console.cloud.tencent.com/tsw)。
2. 在左侧导航栏中，单击【服务监控】>【服务列表】。
3. 在服务列表页，单击列表上方的【接入服务】。
4. 在接入服务界面中，选择接入语言为 Java，选择数据采集方式为 TSW。
5. 输入服务名，服务名称请与您的应用程序名保持一致。（例：若您的应用程序包名为 Order-Service，则需要在服务名处填写一致的名称）
6. 下载 Agent 包（[下载链接](https://tsw-agent-1300555551.cos.ap-guangzhou.myqcloud.com/tsw-client-package.zip)），无需解压 Agent 包，直接复制/SCP 到主机应用目录下。
7. 复制左下方脚本接入中自动生成的脚本，在应用程序包所在的目录下执行该脚本。
![](https://main.qcloudimg.com/raw/9d75f77607ec2f28e90ed44c519ecc41.png)

### 接入验证
向应用发送请求，在收到响应后，在 TSW 控制台查看调用数据。
您可以在1分钟内通过【链路追踪】>【调用链查询】>【[Span查询](https://console.cloud.tencent.com/tsw/trace?rid=1&tab=span)】查找调用详情。监控曲线与统计数据将在1分钟后开始正常显示。
