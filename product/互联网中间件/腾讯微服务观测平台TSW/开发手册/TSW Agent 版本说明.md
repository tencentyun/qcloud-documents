TSW Agent 是一款基于字节码增强技术研发的 Java 应用探针，支持自动埋点完成请求上报。TSW Agent 采集的数据遵循 [Opentracing 规范](https://github.com/opentracing-contrib/opentracing-specification-zh/blob/master/semantic_conventions.md)，TSW Agent 包含（并二次分发）Apache SkyWalking 开源代码，按照 Apache License 2.0开源协议，在 TSW Agent 包中对 SkyWalking License 进行了引用。

>?[SkyWalking](https://github.com/apache/skywalking) 是 Apache 基金会的顶级项目，其核心是一个分布式追踪系统。Skywalking 社区活跃，技术更迭迅速，广泛兼容主流编程语言、组件与框架，为云原生微服务以及容器架构的链路追踪能力广受欢迎。
