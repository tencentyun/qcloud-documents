## 操作场景

腾讯云微服务引擎 TSE 注册中心对运行的实例、接口提供了多项监控指标，用以监测注册中心节点及服务的运行情况，例如：副本数、服务数等业务指标，请求失败总数、请求延迟等**请求指标**和 CPU、内存、网络等**系统指标**。

您可以根据这些指标实时了解注册中心的运行状况，针对可能存在的风险及时处理，保障系统的稳定运行。本文为您介绍通过 TSE 控制台查看注册中心监控的操作。

## 支持的监控指标含义

- 业务指标

| 指标名称                    | 指标含义                            |
| --------------------------- | ----------------------------------- |
| Alive 状态副本数(Count)     | 注册中心集群内 Alive 状态副本数量   |
| Failed 状态副本数量(Count)  | 注册中心集群内 Failed 状态副本数量  |
| 键值对总数(Count)           | 注册中心集群内键值对总数量          |
| Leaving 状态副本数量(Count) | 注册中心集群内 Leaving 状态副本数量 |
| Left 状态副本数(Count)      | 注册中心集群内 Left 状态副本数量    |
| 微服务总数(Count)           | 注册中心集群内微服务总数量          |
| 异常服务数(Count)           | 注册中心集群内异常服务数量          |
| 健康服务数(Count)           | 注册中心集群内健康服务数量          |


- 请求指标

| 指标名             | 指标含义                                  |
| ------------------ | ----------------------------------------- |
| 请求平均延时(ms)   | 注册中心集群内各节点/接口的请求平均延时   |
| 请求最大延时(ms)   | 注册中心集群内各节点/接口的最大请求延时   |
| 请求最小延时(ms)   | 注册中心集群内各节点/接口的请求最小延时   |
| 请求失败数(次)     | 注册中心集群内各节点/接口的请求失败数     |
| 请求成功数(次)     | 注册中心集群内各节点/接口的请求成功数     |
| 请求总数(次)       | 注册中心集群内各节点/接口的请求总次数     |
| 网络错误请求数(次) | 注册中心集群内各节点/接口的网络错误请求数 |
| 其他错误请求(次)   | 注册中心集群内各节点/接口的其他错误请求   |
| 系统错误请求数(次) | 注册中心集群内各节点/接口的系统错误请求数 |


- 系统指标

| 指标名                   | 指标含义                                   |
| ------------------------ | ------------------------------------------ |
| QPS（次/s）              | 注册中心集群内各节点每秒能够响应的查询次数 |
| TPS（次/s）              | 注册中心集群内各节点每秒处理的事务次数     |
| 请求平均延迟（ms）       | 注册中心集群内各节点的请求平均延迟         |
| 写接口平均请求耗时（ms） | 注册中心集群内各节点的写接口平均请求耗时   |
| 读接口平均请求耗时（ms） | 注册中心集群内各节点的读接口平均请求耗时   |
| CPU使用率（%）           | 注册中心集群内各节点的 CPU 使用率          |
| 内存使用率（%）          | 注册中心集群内各节点的内存使用率           |
| 网络入包量（个/s）       | 注册中心集群内各节点的网络入包量           |
| 网络入流量（MBytes）     | 注册中心集群内各节点的网络入流量           |
| 网络出包量（个/s）       | 注册中心集群内各节点的网络出包量           |
| 网络出流量（MBytes）     | 注册中心集群内各节点的网络出流量           |
| 磁盘读取次数（次）       | 注册中心集群内各节点的磁盘读取次数         |
| 磁盘读取带宽（MBps）     | 注册中心集群内各节点的磁盘读取带宽         |
| 磁盘写入次数（次）       | 注册中心集群内各节点的磁盘写入次数         |
| 磁盘写入带宽（MBps）     | 注册中心集群内各节点的磁盘写入带宽         |



## 查看监控数据

1. 登录 [TSE 控制台](https://console.cloud.tencent.com/tse)。
2. 在左侧导航栏，单击 **consul**，进入 Consul 注册中心页面。
3. 单击目标实例的“ID/名称”，进入实例详情页面。
4. 单击**运行监控**，即可查看以下监控指标信息。
