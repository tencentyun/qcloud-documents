### 商业版 SDK 额外支持的功能列表
| 功能               | 说明                                                         |
| ------------------ | ------------------------------------------------------------ |
| 标签过滤           | 可以为消息附上 Tag 属性，消息支持绑定多个标签，订阅者可按照标签筛选觉得是否处理消息。这样可以利用标签优化业务架构，节省 Topic 资源消耗。 |
| 支持退避式延时消息 | 对于失败超时重试场景，并不需要在短时间内大量重试，因为很可能还是失败，依次扩大时间间隔进行重试是比较合理的。 |
| 事务消息           | 实现本地操作和消息发送的事务一致性（预计2020年10月支持）。         |
