## 系统架构
![](https://main.qcloudimg.com/raw/44f1ffefb3cf13078c49f14d60b8d22e.png)
**主要特性**：
- 满足标准的高并发，低延迟直播要求。
- 腾讯云直播借助超过1100+节点的直播发布平台，游戏视频门户可具备和腾讯视频底层服务水平一致的发布能力，全面快速覆盖各地用户。
- 系统可轻松接入专业视频录制设备、播控平台等任意标准 HLS/RTMP 直播源，无论是人气游戏主播，还是竞技赛事，均可快速发布。
- 提供灵活的播放器 SDK，帮助游戏视频门户快速通过移动应用、网页等各种展示方式全方位覆盖玩家。
- 详细的统计分析更可帮助游戏平台精准把握玩家行为特征。
- 可实时录制并对接云点播服务，提供完善视频录制回看服务。
- 即时通信 SDK 功能可集成在移动端和网页应用中，提供基于文本、图片的即时通讯功能，丰富直播交互场景。

### 互动直播方案 + 云直播
![](https://main.qcloudimg.com/raw/e2a20d8152ac9625e96a719c318bdcb6.svg)
**主要特性**：
- 腾讯云互动直播服务专为交互类游戏直播设计。实时的音视频连麦互动，直播服务覆盖全球，可快速搭建一对一，一对多，多对多的直播互动场景。提供充分的玩家和游戏主播之间的实时交互。
- 只需集成移动直播 SDK，就可实现音视频的采集、编解码、传输、渲染播放等工作，业界领先的云端流控策，卓越的音视频技术，将保障您的服务在各平台与不同终端机型上都有着优异的直播效果，拥有与 QQ 音视频、腾讯课堂一样的服务质量。
- 可和云直播服务结合，普通观看者可不安装 SDK 直接收看 HLS 直播视频。
- 借助云点播服务能力，实现游戏直播交互场景的录制功能。
- 即时通信 SDK 功能可集成在移动端和网页应用中，提供基于文本、图片的即时通讯功能，丰富直播交互场景。

### 云点播
![](https://mc.qcloudimg.com/static/img/fc43e00b3b950221afea8f4e625c7025/image.png)
**主要特性**：
- 腾讯云点播服务为游戏门户提供完善的视频服务。
- 各类直播录制内容和优质视频均可通过点播服务进行播放。
- 覆盖全国的1100+ CDN 下发节点确保任意时间、任意地点的玩家均可流畅观看视频。完善 API 接口，全方位对接用户自有 CMS 媒体内容管理平台。
