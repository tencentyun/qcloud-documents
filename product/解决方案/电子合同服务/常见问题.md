### 个人用户可以使用电子合同吗？
电子合同服务为企业级服务，用户必须要先完成 [企业实名认证](https://console.cloud.tencent.com/developer/auth)，才可使用电子合同服务。

### 签章是什么？签章图片有什么要求？
签章用于签署合同，有企业签章和个人签章，由签署方自己制作。个人签章包括签名图片和个人印章。
签章图片只对大小有要求，最大80KB的 png 图片，对分辨率没有要求。

### 每个账户能添加多少个签章？
每个账户限制只能有三个签章，已有三个签章时再次调用新增签章接口会报错。

### 按坐标签署合同与按关键字签署合同有什么区别？
- 按坐标签署合同，是以文档的左下角为原点，设置签署的位置。
- 按关键字签署合同，是以关键字中心为坐标原点，由偏移量得到签章图片中心的坐标。
两种签署方式的坐标单位均为磅，1mm ≈ 2.83磅，一般A4纸的尺寸是210mm * 297mm，通过换算，即可得出相关坐标。

### 按坐标签署合同的 API 中，Position 参数具体是传什么 IP 地址？
签署用户的 IP 地址。

### 按坐标签署合同时，如何设置坐标？
左下角坐标：&SignLocations.0.SignLocationLBX，&SignLocations.0.SignLocationLBY。
右上角坐标：&SignLocations.0.SignLocationRUX，&SignLocations.0.SignLocationRUY。
>!右上角的 x 和 y 要大于左下角的，签章图片将填充在两个点划定的矩形内。

### 创建合同时签署人信息填写谁的信息？
假设合同由甲乙双方签署，就需要在该数组中添加甲乙双方的信息。
>!在发起方发起合同签约的场景下，此发起方有可能不是签署方本人，而是第三方业务平台。

### 签署前不发送验证码可以吗？
不可以，验证码是合作伙伴验证签署用户否是本人签署的重要依据，需要验证码验证。
