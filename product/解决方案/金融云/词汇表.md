<span id="A-E"></span>
## A-E 
### CAM
访问管理（Cloud Access Management，CAM）是腾讯云提供的一套 Web 服务，用于帮助客户安全管理腾讯云账户的访问权限、资源和使用权限。

### 大数据处理套件 TBDS
基于腾讯多年海量数据处理的经验，对外提供可靠、安全、易用的大数据处理平台。您可以借助 TBDS 在公有云、私有云、非云化环境，根据不同数据处理需求选择合适的大数据分析引擎和相应的实时数据开发、离线数据开发以及算法开发服务，来构建您的数据仓库、用户画像、精准推荐、风险管控等大数据应用服务。    

### 对等连接 PC
是一种大带宽、高质量的云上资源互通服务，可以帮助您打通腾讯云上的资源通信链路。 对等连接具有多区域、多账户、多种网络异构互通等特点。

<span id="F-J"></span>
## F-J 
### 分布式数据库 TDSQL
是一种兼容 MySQL 协议和语法，支持自动水平拆分的高性能分布式数据库。即业务显示为完整的逻辑表，数据却均匀的拆分到多个分片中。

<span id="K-O"></span>
## K-O 
### NAT 网关
是一种支持 IP 地址转换的网络云服务。通过 NAT 网关，在腾讯云上的资源可以更加安全的访问 Internet，保护私有网络信息不直接在公网上暴露。您也可以通过 NAT 网关实现海量的公网访问，最大支持 1000 万以上的并发连接数。

<span id="P-T"></span>
## P-T 
### PV
即 page view。指页面浏览量或点击量。

### 批量计算 Batch
批量计算 Batch 可以根据用户提供的批处理规模，智能地管理作业和调动其所需的最佳资源。为支持大数据计算业务的企业、科研单位等提供高性价比且易用的计算服务。

### ROI
即 return on investment，投资回报率。在平台场景下指，用户成本=消耗/人数。

### 容器服务 TKE
基于原生 kubernetes 提供以容器为核心的、高度可扩展的高性能容器管理服务。腾讯云容器服务完全兼容原生 kubernetes API ，扩展了腾讯云的 CBS、CLB 等 kubernetes 插件，为容器化的应用提供高效部署、资源调度、服务发现和动态伸缩等一系列完整功能，解决用户开发、测试及运维过程的环境一致性问题。

### SaaS
SaaS 是 Software-as-a-service（软件即服务）的简称，指为企业用户搭建信息化所需要的所有网络基础设施及软件、硬件运作平台，并负责所有前期的实施、后期的维护等一系列服务，企业用户无需购买软硬件、建设机房、招聘 IT 人员，即可通过互联网使用信息系统。

### 私有网络 VPC
基于腾讯云构建的专属云上网络空间，为您在腾讯云上的资源提供网络服务，不同私有网络间完全逻辑隔离。

### 时序数据库 CTSDB
提供安全、高效、易用的时序数据存储服务。 通过倒排索引加速任意维度查询，使用 rollup 降低数据存储粒度，提高存储压缩比。

### 数据加密服务
数据加密服务是采用国密局认证的物理加密机，利用虚拟化技术，提供弹性，高可用，高性能的数据加解密、密钥管理等云上数据安全服务，符合国家监管合规要求，满足金融，互联网等行业加密需求。

### 数据安全审计
是一款基于人工智能的数据库安全审计系统，可挖掘数据库运行过程中各类潜在风险和隐患，为数据库安全运行保驾护航。

### 弹性伸缩
是一种自动化、具备计划性、高效且高容错性的低成本计算资源管理策略。您可设定时间，周期性地执行管理策略或创建实时监控策略，来管理 CVM 实例数量，并完成对实例的环境部署，保证业务平稳顺利运行。

### 弹性网卡 ENI
是绑定私有网络内云服务器的一种弹性网络接口，可在多个云服务器间自由迁移。您可以在云服务器上绑定多个弹性网卡，实现高可用的网络方案；也可以在弹性网卡上绑定多个内网 IP，实现单主机多 IP 部署。

### 态势感知（私有云）
以大数据为基础，安全检测为核心、事件关联分析和威胁情报为重点，3D 可视化为特色的智能态势感知解决方案。是腾讯安全面向政府、军队、金融、制造业、医疗、教育等大型企事业单位推出的一款安全大数据分析及可视化平台。

<span id="U-Z"></span>
## U-Z
### UV
即 unique visitor。指通过互联网访问、浏览页面的自然人。

### VPN 连接
是一种基于网络隧道技术，实现本地数据中心与腾讯云上资源连通的传输服务，它能帮您在 Internet 上快速构建一条安全、可靠的加密通道。VPN 连接具有配置简单，云端配置实时生效、可靠性高等特点，其网关可用性达到99.95%，保证稳定、持续的业务连接，帮您轻松实现异地容灾、混合云部署等复杂业务场景。

### 网络安全-网站管家
帮助腾讯云内及云外用户应对 Web 攻击、入侵、漏洞利用、挂马、篡改、后门、爬虫、域名劫持等网站及 Web 业务安全防护问题。

### 无服务器云函数 SCF
腾讯云为企业和开发者们提供的无服务器执行环境，帮助您在无需购买和管理服务器的情况下运行代码。SCF 是实时文件处理和数据处理等场景下理想的计算平台。

### 文件存储 CFS
文件存储符合标准的 NFS 文件系统访问协议，为多个计算节点提供共享的数据源，支持弹性容量和性能的扩展，现有应用无需修改即可挂载使用，是一种高可用、高可靠的分布式文件系统，适合于大数据分析、媒体处理和内容管理等场景。

### 云数据迁移 CDM
腾讯云提供的 TB-PB 级别的数据迁移上云服务。本服务为您提供安全可靠的离线迁移专用设备，满足本地数据中心进行大规模数据迁移上云的需求，解决本地数据中心通过网络传输时间长、成本高、安全性低的问题。

### 云数据库 MongoDB
是腾讯云基于全球广受欢迎的 MongoDB 打造的高性能 NoSQL 数据库，100%完全兼容 MongoDB 协议，同时高度兼容 DynamoDB 协议，提供稳定丰富的监控管理，弹性可扩展、自动容灾，适用于文档型数据库场景，使您无需自建灾备体系以及控制管理系统。

### 云服务器
是一种稳定、安全、弹性、高性能的云端计算服务。在云中提供可扩展的计算服务，避免了使用传统服务器时需要预估资源用量以及前期投入的情况。

### 云硬盘
是一台稳定可靠、低延迟、可扩展的持久性块存储设备，可作为 CVM 的独立硬盘使用。它提供数据块级别的数据存储，采用三副本的分布式机制，为 CVM 提供数据可靠性保证。

### 阈值
即临界值。指某一场景下能够产生的最低值或最高值。

### 资源
作为营销奖品发放的线上物料。

### 资源与策略复用至
可将配置好的渠道策略与资源一键复制，与其他渠道共享。

### 总库存
平台上配置的所有资源库存。

### 自有库存
已从总库存中购买的资源、已自行录入的资源。
