收银台支持基础设置、收银设置、打印机设置、快捷键设置和发送消息设置。
- 基础设置包括商户设置和密钥设置。
- 收银设置包括终端设置、使用设置和惠顾用语。
- 快捷键设置包括显示/隐藏主窗口快捷键，唤其/隐藏收银窗口快捷键、截取金额位置快捷键。
- 发送消息设置包含支付完成之后触发的键盘消息设置。

管理员可查看所有设置选项并重新设置；店员仅能看到打印机设置、快键键设置和发送消息设置，且没有修改设置的权限。

## 前提条件
1. 已配置好服务商并激活服务商管理员权限，具体操作请参考 [配置服务商](https://cloud.tencent.com/document/product/569/9796)。
2. 已配置好子商户，具体操作请参考 [配置子商户](https://cloud.tencent.com/document/product/569/9795)，**注意一定要保存认证密钥和签名私钥**，设置收银台时需使用。 
>?如果之前没有保存好认证密钥和签名私钥，可以在子商户详情重置认证密钥和签名私钥（重置密钥会导致正在使用云支付的子商户支付失败），单击【重置密钥】后，保存认证密钥和签名私钥（重置之后，以前配置好客户端的认证密钥和签名私钥需要重新配置）。    
3. 已配置门店，添加并激活店员，如使用员工账户密码登录收银台，需要设置员工收银终端密码；添加收银设备，设备类型选择刷卡支付，具体操作请参考 [门店配置](https://cloud.tencent.com/document/product/569/9797)。
4. 已在微信云支付控制台的【子商户管理】页获取云支付子商户帐号。
5. 安装 Microsoft .NET Framework，收银台运行需安装不低于 Microsoft .NET Framework 4的版本，[下载地址](https://www.microsoft.com/zh-cn/download/details.aspx?id=17718)。

## 操作步骤
1. 输入云支付子商户帐号并单击【生成二维码登录】，弹出二维码后，使用服务商或子商户管理员微信扫码登录（服务商或子商户管理员微信需扫码激活权限）。![](https://main.qcloudimg.com/raw/b04901d50cb5a463d5dae14f2c29e459.jpg)

2. 如商户的收银台在局域网中，通过代理访问互联网，需要单击【设置网络代理】设置IP地址、端口、用户名和密码，其中如果没有用户名和密码可以不填。

>?代理搭建参考 [搭建 HTTP 代理](https://cloud.tencent.com/document/product/569/12641)。
