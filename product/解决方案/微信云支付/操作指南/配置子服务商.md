## 1. 准备

要添加子服务商，需要先完成以下步骤：

1. 为子服务商[注册一个腾讯云账号](https://cloud.tencent.com/document/product/378/17985)
   * 如果已经有腾讯云账号，可以跳过这一步
   * 子服务商不需要微信支付服务商商户号和支付宝的服务商应用ID
2. 查看子服务商的腾讯云账号ID，记录下来，后面会用到；具体查看方式如下：

![](https://main.qcloudimg.com/raw/79f4855a8e4d317a48a283a2654acc11.png)
![](https://main.qcloudimg.com/raw/c51a7afbab09f4260d76fc0de10e4147.png)

## 2. 添加子服务商

### 2.1 登录云支付进行添加

**服务商**登录[云支付](https://console.cloud.tencent.com/cpay)，在“子服务商管理”页面点击“添加子服务商”

![](https://main.qcloudimg.com/raw/8bd75b645c3e5b79a7c13d610959fcd5.png)

### 2.2 填写子服务基本信息

输入子服务商基本信息，然后点击“确定”按钮完成添加。 子服务基本信息包括：

| 参数类型 | 参数说明 | 备注 |
|---------|---------|---------|
| 云支付服务商账号 | 选择要创建子服务商的服务商 | - |
| 子服务商腾讯云账号ID| 子服务商的腾讯云账号ID | 在第1节有介绍查看方式 |
| 子服务商名称| 子服务商名称| - |
| 子服务商管理员姓名| 子服务商管理员姓名| - |
| 子服务商联系电话| 子服务商联系电话| - |
| 微信返佣比例 | 微信支付相关订单，服务商给子服务商的返佣比例 | 用于统计返佣金额; 统计结果仅用于参考 |
| 支付宝返佣比例 | 支付宝相关订单，服务商给子服务商的返佣比例 | 用于统计返佣金额; 统计结果仅用于参考 |
| 备注 | 子服务商的描述信息等 | - |

![](https://main.qcloudimg.com/raw/523c1c0dd0ddc0b5377232fdf6bc20a3.png)

### 2.3 激活第一个子服务商管理员

添加子服务商成功后，在子服务商列表中找到刚添加的子服务商；点击"管理二维码"按钮，可得到子服务商管理员身份激活二维码和子服务商管理二维码:

![](https://main.qcloudimg.com/raw/33c04832a840f72d3892ad64ddc5736e.png)

---

用子服务商管理员的微信扫**左边**的管理员激活码来激活管理员：

![](https://main.qcloudimg.com/raw/84703ad0462af8b93267cee3f3e0d517.png)

可能会出现下面的错误：

- 手机出现如下提示，则说明服务商填的AppID对应的微信公众号没有通过认证，请到微信公众平台认证：

![](https://mc.qcloudimg.com/static/img/610a55beb2c1add93b3d0fb827ba38e6/image.png)

- 手机出现如下提示，则说明服务商的微信公众号未将网页服务权限授权给腾讯云支付，请完成授权后，再扫码激活管理员身份：

![](https://mc.qcloudimg.com/static/img/65505957a2cfbc0bade6b3afba1e7115/image.png)

- 手机出现如下提示，是浏览器兼容性问题，实际已经激活成功，忽略这种错误即可：

![](https://mc.qcloudimg.com/static/img/4c8b31a331bb6c26c68e630fa9b4e1f6/image.png)

## 3. 进入子服务商管理后台

子服务商管理员身份激活成功后，可通过云支付公众号进入子服务管理后台界面，如下所示：

![](https://main.qcloudimg.com/raw/c19c69cb277ce510aec858b40c78b331.png)

![](https://main.qcloudimg.com/raw/8d86e885770baab7231daea7bb8ff6ee.png)

## 4. 添加新的子服务商管理员

### 4.1 进入添加子服务商管理员界面

![](https://main.qcloudimg.com/raw/8a699ff05ec36334fa166b7935d01745.png)

![](https://main.qcloudimg.com/raw/2989f1750df280e7c81ef0ff7bca2850.png)

### 4.2 输入子服务商管理员信息

![](https://main.qcloudimg.com/raw/6dd20e8fb35682c99eaa696f5e885e23.png)

注：根据需要选择要给子服务商管理员赋予的权限

### 4.3 激活子服务商管理员

点击弹出子服务商管理员激活码，然后用微信扫码激活管理员：

![](https://main.qcloudimg.com/raw/1b946d9e8255c663b590e2e077e1361e.png)

![](https://main.qcloudimg.com/raw/ef73cab1a851c43524270cdc257fc0f8.png)

## 5. 编辑和删除管理员
1. 单击管理员列表页面底部的【编辑管理员】。
![](https://main.qcloudimg.com/raw/9306b8022e4c95fdad87587cfff4f8b6.png)
2. 编辑管理员信息并单击【保存】；如果是删除管理员，则单击【删除】。
![](https://main.qcloudimg.com/raw/9982e5b788eb34096425e58a46948bf0.png)
>!云支付不支持管理员删除自己；如果需要将自己从管理员中删除，则需执行以下操作：
> 1. 添加另外一个人为管理员并授权其 “管理员添加和删除权限”。
> 2. 让新添加的管理员将您从管理员中删除。
