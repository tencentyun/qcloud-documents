为方便开发者快速验证方案，我们在体验开发阶段向每位用户提供**5个开通了全部服务的测试账号**， 用于开发者快速体验、验证方案，而正式开发阶段为用户确认使用腾讯互动客户端解决方案之后的正式流程。
![](https://main.qcloudimg.com/raw/f920ddaf065b4594cef548b12a79e721.png)
## 体验开发阶段
体验开发阶段无需开通任何服务，只需申请测试账号即可。

### 申请测试账号
请按以下格式发送邮件至 ticsupport@qq.com 申请测试账号，账号信息会以邮件形式回复（测试账号仅限于开发者快速上手使用，有效期15天）。

```
标题：申请互动课堂测试账号

内容：
    公司名：
    账号 ID：
    联系人姓名：
    联系电话：
    行业类型：（学前教育，K12，高等教育，语言类，职业教育，O2O）
```

账号 ID 为腾讯云账号唯一标识，在腾讯云官网注册登录之后可从 [账号中心](https://console.cloud.tencent.com/developer) 查看。


## 正式开发阶段

>! 开通正式服务之前，需要在 [账号中心](https://console.cloud.tencent.com/developer) 确认腾讯云账号的行业属性是否为【教育-在线教育】，如果不是则需修改为【教育-在线教育】，如下图所示：
>![](https://main.qcloudimg.com/raw/1b223601008876a3fcc1b9658535c7f6.png)

正式开发阶段需要用户开通以下服务：

### 实时音视频服务

开通 **实时音视频服务** 请参考 [开通服务](https://cloud.tencent.com/document/product/647/17195)。

>!开通服务需先选购好9688元及以上的预付费套餐包；如需单独的技术支持群，帮助您更快速稳定接入，则需购买73888的的预付费套餐包。

### 白板服务
开通白板服务请按以下格式发送邮件至 ticsupport@qq.com，审核通过会邮件通知（开通白板服务后才有多端互通、展示课件等功能）。
**审核条件为是否选购了实时音视频9688元及以上的预付费套餐包。**

```
标题：申请开通白板服务

内容：
    公司名：
    账号 ID：
    SDKAPPID：
    联系人姓名：
    联系电话：
    行业类型：（学前教育，K12，高等教育，语言类，职业教育，O2O）
    业务阶段：(未开发，开发中，测试中，已上线，大规模应用中)
```
