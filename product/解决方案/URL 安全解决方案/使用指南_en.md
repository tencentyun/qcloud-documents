The specific steps for developers to activate and use the Tencent Cloud URL Security Solution are as follows:

![Flow Chart](https://mc.qcloudimg.com/static/img/8945a66e719958117b0824d51febb59b/image.png)

## Applying for Activating Permissions
Go to the [Tencent Cloud official website](https://cloud.tencent.com), and select **Solutions** -> **Security and OPS Solutions** -> **URL Security**, or directly go to the overview page of [URL Security](https://cloud.tencent.com/solution/wechat-url).

![Security](https://mc.qcloudimg.com/static/img/0d661836902f4f429ccf5c40e7bfb7f8/image.png)

Click **Apply Now** button on the URL Security page.

![Apply Now](https://mc.qcloudimg.com/static/img/bc84f60d7bf898ec61d2fa1badfd5504/image.png)

After entering the [Console](https://console.cloud.tencent.com/tianyu/overview), locate the URL Security and click **Free Trial**, and then we will contact you and activate permissions for you.

![Free Trial](https://mc.qcloudimg.com/static/img/0150e712896f274dda375d7508b81823/image.png)

## Registered Domain Name
You can click **Service Monitoring** -> **URL Security** on the [BSP Console](https://console.cloud.tencent.com/tianyu/overview) page, or directly go to the [URL Security Console](https://console.cloud.tencent.com/tianyu/service/url) page.

![Service Monitoring](https://mc.qcloudimg.com/static/img/74af9255b8e7f9febc64dc69f229f935/image.png)

Click **Permission Verification** -> **Add Site**, and place the verification file in the root directory of the registered domain name (Based on this verification file, BSP needs to determine the customer's ownership to the registered domain name).

![Add Site](https://mc.qcloudimg.com/static/img/79e9ce3b682a2d26a7281bd096736355/image.png)

After the verification file is deployed, click the **Click to Verify** button.

![Verify](https://mc.qcloudimg.com/static/img/f2d1b8540c160a3af369e91827c4c0fb/image.png)

## Quick Unblocking Verification
After the domain name is successfully registered, click **Detailed Report** to view the blocked URLs under the registered domain name, and then click **Accelerate Unblocking** to perform quick unblocking.

![Unblocking](https://mc.qcloudimg.com/static/img/4bcbecabd0644ed6c2a48b5ed20a912c/image.png)
