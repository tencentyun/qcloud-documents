【报表】丰富多维度的统计报表，覆盖缺陷，需求，进度，工时，需求关联，项目报告，缺陷定时报告等。
![](https://main.qcloudimg.com/raw/13617b3b74a749577b4b69719b47b827.jpg)

1. BUG 统计-- 观察 BUG 的数据，走势图，定量度量，帮助项目团队进行质量控制，从而判断项目健康情况
我们以 BUG 分布图为例；水平轴系统：当前处理人；纵向系统：状态
 ![](https://main.qcloudimg.com/raw/a255a9a81ef21d39fc7f91b42eea898e.jpg)

点生成报表后，可预览该分布图效果。另存新报表，自定义标题，比如以【6 月的 BUG 当前处理人/优先级的分布图】做为标题。可保存过滤条件，同时也可用于定时报告中。

2. 需求分布统计--查看需求分布图，即时了解需求进度实现情况
![](https://main.qcloudimg.com/raw/cff908b286472406c5d3c17d23c83982.jpg)

3. 项目报告--统合项目各类转测试，进度报告，测试报告，以邮件形式知会团队成员。
![](https://main.qcloudimg.com/raw/d1995a1c0fb3c18163cc3b18f58371de.jpg)

项目报告类型有 3 种：项目进度，转测试，测试报告。用户可根据需要加入数据列表（文本，需求，BUG，附件等），人工触发邮件。

4. 定时报告--可配置按月，周，天频率，发送 BUG 数据邮件功能，数据可包含 BUG 视图，BUG 报表，文本，附件等。
配置方法：管理员可以在报表--新增缺陷定时报告
![](https://main.qcloudimg.com/raw/a52d48eaeceb3a50867e8ef2bbb98d8e.jpg)

新增定时报告，填写基本信息，发送频率，发送人，接收人等，提交。
![](https://main.qcloudimg.com/raw/cbd198a2a02d35b9b0fa64f2cb69f0e0.jpg)

编辑该定时报告，配置报告内容（文本内容，缺陷报表，缺陷视图等）。
![](https://main.qcloudimg.com/raw/4b6d405a6e3cd43be1bdd7c24b5eacb4.jpg)

如果没有满足用户需求的报表，可通过自定义 BUG 报表；配置自定义报表操作：【报表】>【BUG 统计】>【生成报表】>【另存为报表】。再返回到定时报表内容配置页面，添加该自定义 BUG 报表，配置好所有内容项目后，可预览该报告效果。
