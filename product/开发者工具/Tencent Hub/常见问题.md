### Tencent Hub 中可在哪里创建仓库？
Tencent Hub 默认在自己账号内创建仓库，也可以创建组织，在相应的组织内创建需要的仓库。


### 如何搭建 Tencent Hub 中 DevOps 工作流中的各个服务？
Tencent Hub 中的 DevOps 工作流支持3种方式搭建服务流程。

- 调用给定的 DevOps 组件。DevOps 中集成了腾讯内部构建，自动化测试等多种优秀开发服务能力，并封装成容器镜像组件。您只需根据自己的业务需求调用设计即可。
- 用户自定义组件。支持用户将自己的服务编写成镜像组件上传到 Tencent Hub，并在 DevOps 工作流中调用运行。（即将上线）
- 支持 Service API。Tencent Tub 支持 API 形式调用腾讯内部服务 API，搭建 DevOps 中的服务。


### Tencent Hub 如何收费？
内测申请期间可免费使用。内测结束后会按照使用情况分项付费，具体计费标准即将上线。
