最终的请求URL由以下几部分组成：
1) 请求域名：负载均衡相关的API访问域名统一使用 lb.api.qcloud.com。
2) 请求路径: 云API的请求路径固定为/v2/index.php。
3) 最终请求参数串: 包括公共请求参数和接口请求参数。

最终的请求 URL的拼接规则为:
> https:// + 请求域名 +请求路径 + ? +最终请求参数串

```
https://domain/v2/index.php?
Action=DescribeLoadBalancers
&SecretId=xxxxxxx
&Region=gz

&Signature=mysignature
&loadBalancerType=2
```
