## 1. 接口描述
域名:lb.api.qcloud.com
接口名:InquiryBmLBPrice

查询黑石负载均衡价格

## 2. 输入参数
| 参数名称 | 必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| loadBalancerType | 是 | Int |负载均衡的类型 : 2.公网有日租，3.内网|


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 错误码, 0: 成功, 其他值: 失败|
| message | String | 错误信息|
| price | Int |价格 |


## 4. 示例
输入
```
https://lb.api.qcloud.com/v2/index.php?Action=InquiryBmLBPrice
&loadBalancerType=3
&COMMON_PARAMS
```
输出
```
{
    "code":"0",
    "message":"",
    "price":"0"
}
```
