## 1. 接口描述
域名:lb.api.qcloud.com
接口名:DeleteBmLoadBalancers

删除黑石负载均衡

## 2. 输入参数
| 参数名称 | 必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| loadBalancerIds.n  | 是 | String | 负载均衡器ID|


## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 错误码, 0: 成功, 其他值: 失败|
| message | String | 错误信息|
| requestId | Int | 任务Id |


## 4. 示例
输入
```
https://lb.api.qcloud.com/v2/index.php?Action=DeleteBmLoadBalancers
&loadBalancerIds.1=lb-fi018waq
&COMMON_PARAMS
```
输出
```
{
    "code":"0",
    "message":"",
    "requestId":"16916"
}
```
