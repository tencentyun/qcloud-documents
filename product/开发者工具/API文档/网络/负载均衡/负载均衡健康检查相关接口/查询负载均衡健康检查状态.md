## 1. 接口描述
 DescribeLBHealthStatus 接口用来查询负载均衡实例的默认健康检查相关参数。
 
接口访问域名：lb.api.qcloud.com

## 2. 请求参数

<td>是
<td> String

## 4. 示例

输入：
```
https://lb.api.qcloud.com/v2/index.php?Action=DescribeLBHealthStatus
&<公共请求参数>
&loadBalanceId=lb-abcdefgh
```
输出：
```
{
"code":0,
"msg":"ok",
"data":[
	     {
			"ip":"10.2.3.0",
			"protocol":"TCP",
			"port":8001,
			"vport":8001,
			"healthStatus":0
	     }
	]
}
```
