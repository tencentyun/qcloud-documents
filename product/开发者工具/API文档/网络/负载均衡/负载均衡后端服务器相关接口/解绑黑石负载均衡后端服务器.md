## 1. 接口描述
域名:lb.api.qcloud.com
接口名:DeregisterInstancesFromBmLoadBalancer

解绑黑石负载均衡后端服务器

## 2. 输入参数
| 参数名称 | 必选  | 类型 | 描述 |
|---------|---------|---------|---------|
| loadBalancerId | 是 | String | 负载均衡器ID,|
| backends.n.instanceId | 是 | String | 映射的后端机器Id|




## 3. 输出参数
| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| code | Int | 错误码, 0: 成功, 其他值: 失败|
| message | String | 错误信息|
| requestId | Int | 任务Id |


## 4. 示例
输入
```
https://lb.api.qcloud.com/v2/index.php?Action=DeregisterInstancesFromBmLoadBalancer
&loadBalancerId=lb-51jf1v2l
&backends.0.instanceId=cpm-owlr8otf
&COMMON_PARAMS
```
输出
```
{
    "code":"0",
    "message":"",
    "requestId":"2375531"
}
```
