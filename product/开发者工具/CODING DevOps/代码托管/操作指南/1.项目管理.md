本文为您详细介绍项目的管理与使用。

## 创建项目

1. 登录 [CODING 控制台](https://console.cloud.tencent.com/coding)，单击团队域名进入 CODING 使用页面。
2. 鼠标移动至右上角的<img src ="https://main.qcloudimg.com/raw/7531b01c25014beb2754277107fc4ab1.png" style ="margin:0">，单击浮现出来的【创建项目】。
   ![](https://main.qcloudimg.com/raw/213ecdb0e276db77093c93ea5eb84136.png)
3. 选择项目模板。CODING 提供的项目模板类型分别如下：
   - 代码托管项目：默认包含代码仓库和文档管理功能。
   - 项目管理项目：默认包含项目协同和文档管理功能。
   - DevOps 项目：默认包含项目协同、代码仓库、构建与部署、制品库、测试管理和文档管理功能。提供从需求收集、规划迭代、任务分配到代码提交、构建、测试到部署的一站式研发流程。
   - 范例项目：含有可实践的 Demo 数据项目，这些数据包括项目配置、项目协同 Demo 事项、测试管理 Demo 测试用例、代码脚手架和范例 CI/CD 配置等。
4. 选择项目模板后，填写项目地址、项目名称即可创建项目，同时您还可以填写项目描述并添加项目成员。建议您勾选“启用 README.md”文件初始化项目。
   - 项目名称：必填项，支持1 - 31位的中英文、数字、下划线(_)、中划线(-)、点(.)和空格的组合。
   - 项目标识：必填项，支持1 - 31位的字母、数字、下划线(_)、中划线(-)和点(.)的组合。
   - 项目描述：选填项，最多100个字符。
   - 仓库类型：可选择 Git 仓库或 SVN 仓库。
5. 单击【完成创建】，完成项目创建。

## 更新项目信息

1. 团队所有者 / 管理员进入首页左侧菜单栏的项目，单击特定项目右侧的菜单按钮，接着轻点浮出来的【项目设置】。
![](https://main.qcloudimg.com/raw/99db5d2f45366e52b9c2a718fbf74da1.png)
2. 在项目设置弹窗中，可更新项目地址、项目名称、项目描述及项目时间。
![](https://main.qcloudimg.com/raw/1234e4c655004e62b63d0220a4ba0048.png)

> ! 修改项目名称将导致项目的访问 URL（包含 Git 仓库的 URL）改变，在此之前的 URL 将失效。Git 仓库地址修改方法：`git remote set-url origin [NEW_URL] `。

## 成员管理

1. 团队所有者 / 管理员将鼠标移动至页面右上角的头像图标处，单击浮出来的【团队管理】。
![](https://main.qcloudimg.com/raw/bb974aa24f95fa1bc553861c511c34a5.png)
2. 进入团队概览页面，单击左侧菜单栏中的【团队成员】。在成员管理弹窗中可 [添加](https://cloud.tencent.com/document/product/1112/37250)/移除项目成员、修改项目成员用户组。
![](https://main.qcloudimg.com/raw/2ac5ab9a965956192efc180d9904f2b6.png)

## 删除 / 归档项目

1. 团队所有者 / 管理员进入首页左侧菜单栏的项目，单击特定项目右侧的菜单按钮，接着轻点浮出来的【项目设置】。
2. 选择对应的项目单击最右端【…】，可进行【归档项目】或【删除项目】操作。
   - 项目归档后，项目内所有成员将无法访问该项目，且不能对该项目进行任何操作。
   - 删除项目会删除项目内的所有数据，且删除操作无法恢复。若项目内存有重要数据，建议归档项目。
  ![](https://main.qcloudimg.com/raw/aa3ac35d2bff54125fb78bdd57a13bf2.png)
3. 被归档的项目收纳至【已归档】管理页面，单击接触归档按钮后即可解除归档。
![](https://main.qcloudimg.com/raw/3660d7b24654b9ba78409f1ce7dbb119.png)
