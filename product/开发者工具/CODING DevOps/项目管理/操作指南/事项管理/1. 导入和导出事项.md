本文为您详细介绍如何在 CODING 项目管理中导入与导出事项。

## 进入项目
1. 登录 [CODING 控制台](https://console.cloud.tencent.com/coding)，单击**立即使用**进入 CODING 使用页面。
2. 单击团队首页左侧的**项目**，进入项目列表页，选择目标项目。
3. 单击进入左侧菜单栏中的**项目协同**功能。

[](id:intro)
## 功能介绍
史诗、需求、任务、缺陷统称为**事项**，支持批量导入导出。可以在事项列表右上角菜单内选择**导入事项**、**导出当前筛选事项**、**导出所有事项**，并按照 CODING 提供的模板进行操作。下文将以**需求**为例，演示如何进行事项的导入与导出。
![](https://qcloudimg.tencent-cloud.cn/raw/0a31c579726c20d7d3dbcd75117f11e6.png)

[](id:import)
## 导入事项
1.  在事项列表页右上角菜单内选择**导入需求**。
![](https://qcloudimg.tencent-cloud.cn/raw/35e3332c8b0008cf36881849d1e16f1f.png)
2.  单击**下载模板**，请根据如图所示导入要求进行填写。
![](https://qcloudimg.tencent-cloud.cn/raw/4d1eb5c9d8223d2a8115be3db643b352.png)
3.  模板包含**标题**（必填）、**描述**、**处理人**、**状态**、**优先级**、**截止时间**，填写完成后，上传 Excel/CSV 文件。
![](https://qcloudimg.tencent-cloud.cn/raw/a3eb5dc263e32582bf40fb95533d2a55.png)
>!事项属性用于快速说明此事项目前的状态、截止日期、处理人、优先级等信息。若事先在项目内设置了必填属性，在导入事项时将忽略此规则；若事先在项目内设置了属性默认值，导入事项时将自动填充默认值。详情请参见 [自定义事项属性](https://cloud.tencent.com/document/product/1113/58821)。
4.  查看导入结果。
导入完成后，导入界面将提示或通过系统通知告知。若导入失败，可根据指引下载错误记录，对照修改后重新导入。<br>

## 导出事项
1.  在事项列表页右上角菜单内，可以选择导出全部需求，或者根据需要进行筛选后，导出当前筛选需求。
>!该功能不会导出需求下的子任务。
>
![](https://qcloudimg.tencent-cloud.cn/raw/deaa64472e57db90e6a72ff8027823ab.png)
2.  导出成功后即可单击**下载文件**，并查看导出结果。
![](https://qcloudimg.tencent-cloud.cn/raw/7cc2e6fa32e54a94f358cdcd851c1baa.png)
![](https://qcloudimg.tencent-cloud.cn/raw/541d2644ee7ffb466c4fb63f6691f3e3.png)
