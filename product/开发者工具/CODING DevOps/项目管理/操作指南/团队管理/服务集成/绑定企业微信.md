本文为您介绍如何绑定企业微信。

## 进入团队管理页

1. 登录 [CODING 控制台](https://console.cloud.tencent.com/coding)，单击**立即使用**进入 CODING 使用页面。
2. 将鼠标移至页面右上角的头像处，会出现下拉菜单，单击进入团队管理界面。

## 功能介绍

您可以在**团队管理** > **服务集成**中绑定企业微信、[企业微信小程序](https://cloud.tencent.com/document/product/1113/49922)、[私有 GitLab](https://cloud.tencent.com/document/product/1113/49923)。绑定企业微信账号后，您可以使用企业微信扫码登录 CODING，免除了登录时需要输入账号密码的繁琐操作；绑定 CODING DevOps+ 小程序后，团队成员便可以在小程序内实时接受通知，查看任务详情并进行评论、查看代码合并请求、查看构建记录等，方便随时随地管理项目；绑定私有 GitLab 后，即可在 CODING 的持续集成、制品库等模块中使用外部仓库。具体使用步骤如下：

## 团队帐户绑定企业微信

1.  团队管理员在**团队管理** > **服务集成** > **企业微信选项**中单击**绑定**按钮。
![](https://main.qcloudimg.com/raw/71cfa7cf57d7be19e92b17bc89b92a88.png)
2.  跳转至企业微信二维码页面后使用**企业微信管理员账号**扫码，企业微信 App 将出现**安装扣钉 CODING 的应用**弹窗，单击**确定**。
![](https://main.qcloudimg.com/raw/5ac7fe2c2998dcf6789ea167109ccc6e.png)
3.  设置应用可见范围。选择可见的部门和人员，并单击**确认**。
![](https://main.qcloudimg.com/raw/29f50dbbfe14532325744360fd53bb19.png)
4.  单击**同意以上授权，并添加**，同时您可以选择是否添加 CODING 客服为外部联系人选项。
![](https://main.qcloudimg.com/raw/a9d12852121cdde8dbc93a516a67b4fe.png)
5.  绑定企业微信成功后，将可在**服务集成** > **企业微信选项**中查看到**已绑定**字眼。
![](https://main.qcloudimg.com/raw/2438972759a33c573f8267a1ce6604d5.png)

## 个人帐户绑定企业微信

### 单击通知消息自动绑定

CODING 企业版管理员账号与企业微信绑定成功后，可见应用的用户将会收到一条绑定成功的消息通知。单击该消息即可进行自动绑定。
![](https://main.qcloudimg.com/raw/18f1c82a26d9ac0b3d4f7ffefb56d722.png)

### 手动绑定

1.  团队普通成员在**个人账户** > **绑定设置** > **企业微信选项**中单击**绑定**按钮。
![](https://main.qcloudimg.com/raw/65b1607d4d26ec23ceb2eae28fdb81ef.png)
2.  使用企业微信扫码登录后即可完成绑定操作。
![](https://main.qcloudimg.com/raw/bcdcd1a0ea6f6ab3b03416e17e6f9237.png)
3.  绑定企业微信成功后，将可在**绑定设置** > **企业微信选项**中查看到**已绑定**字眼。
![](https://main.qcloudimg.com/raw/1623e946a0e3f512049dc9b5530cc6f9.png)
