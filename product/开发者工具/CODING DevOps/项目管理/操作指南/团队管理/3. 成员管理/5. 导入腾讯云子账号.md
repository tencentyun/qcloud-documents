本文为您介绍如何导入腾讯云子账号至 CODING 中。

[](id:introduction)
## 导入子账号指引
CODING 支持从腾讯云导入腾讯云子账号列表，但是暂不支持导入腾讯云账号中的组织结构。您可以在导入腾讯云子账号列表后，再前往**成员管理**中手动移动子账号至相对应的部门。
![](https://qcloudimg.tencent-cloud.cn/raw/cb3cfef8c3bbea06465de0325b4586c9.png)
若未绑定腾讯云账号，团队管理员依据弹窗提示前往团队设置中心绑定腾讯云账号、配置账号源。例如腾讯云授权登录态已过期，将弹出腾讯云登录弹窗，支持微信扫码和账号密码授权。授权完成后即可完成与腾讯云的关联。
![](https://qcloudimg.tencent-cloud.cn/raw/a461396be2581181f5be6af94755a2fa.png)
导入完成后，需对成员进行授权操作，为对应成员授予访问 CODING 服务的团队权限。
![](https://qcloudimg.tencent-cloud.cn/raw/5259a7a8ddbbc4f2b8f3b6a1e12b2b0d.png)

## 子账号登录指引
CODING 支持在登录页面使用腾讯云账号类型便捷登录。在登录页面输入主账号 ID、子用户名与密码完成登录。
**子账号登录条件：**
-   主账号已在腾讯云 [CODING DevOps 控制台](https://console.cloud.tencent.com/coding) 开通服务。
-   对应子账号存在 QcloudCODINGFullAccess 策略。
