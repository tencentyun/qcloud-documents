本文为您介绍如何管理团队设置。

## 进入团队管理页

1. 登录 [CODING 控制台](https://console.cloud.tencent.com/coding)，单击**立即使用**进入 CODING 使用页面。
2. 将鼠标移至页面右上角的头像处，会出现下拉菜单，单击进入团队管理界面。

### 更新团队信息

团队所有者/管理员可以在**基本设置**中进行团队头像、团队名称的修改。

![](https://main.qcloudimg.com/raw/cf48dc3d4669392747b62b6b7df72756.png)

### 示例项目设置

CODING 为每个团队准备了一个示例项目，若团队所有者/管理员选**将现有成员加入示例项目**选项后，新成员加入团队后便会自动加入该项目，帮助快速熟悉使用产品功能。示例项目将会在团队注册完成后自动开启。单击**重置示例项目**按钮即可将示例项目的数据更新为 CODING 最新内置的数据。

>! 若未勾选**将现有成员加入示例项目**选项，示例项目重置后，成员列表和示例数据将只包含当前操作。

![](https://main.qcloudimg.com/raw/e88066263181175022edbe289ae49ca1.png)

### 实名认证

1. 团队所有者在高级设置中可以进行团队实名认证。实名认证后，团队将可以使用 Pages 部署等产品功能。
![](https://main.qcloudimg.com/raw/355957a38f4b607b354c42531d4a85e9.png)
2. 填写认证成员的姓名与身份证号，单击**下一步**。
![](https://main.qcloudimg.com/raw/fc9334ca22bef978b79e818b3a317643.png)
3. 获取验证码，填写验证码后，单击**提交验证**即可。
![](https://main.qcloudimg.com/raw/3734866b2c88a94fb735e6dc4cd6edd4.png)
4. 实名认证成功后，即可看到认证成功消息提示。接下来您就可以正常使用需实名认证通过的功能特性了。
![](https://main.qcloudimg.com/raw/7d5d0abad536344a4fbc0b177f53beb8.png)

>?
> - 团队实名认证、变更团队所有者等高级设置为团队所有者独有权限。
> - 高级设置页面对非团队所有者不可见。
