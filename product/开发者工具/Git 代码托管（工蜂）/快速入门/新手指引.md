本文档主要介绍如何快速使用腾讯 Git 代码托管（工蜂） 。

## 设置帐号以及新建项目

1. 登录腾讯云账号。
若是新用户需在腾讯云官网进行[【注册】](https://cloud.tencent.com/register?s_url=https%3A%2F%2Fcloud.tencent.com%2Fdocument%2Fproduct%2F213)，注册指引可参考 [如何注册腾讯云](https://cloud.tencent.com/doc/product/378/9603) 。

2. 到 [腾讯工蜂](https://cloud.tencent.com/product/TGit) 产品介绍页上单击【开通 DevMaster】，然后跳转到 DevMaster页。
![img](https://main.qcloudimg.com/raw/09ac82bf77543388d7ed8640cc2dc5d2.png)

3. 单击【立即体验】，然后页面跳转至【开通服务】页
![img](https://main.qcloudimg.com/raw/e844ab5eace7556bd3d50ea8041163b4.png)

4. 单击【开通服务】
![img](https://main.qcloudimg.com/raw/b874f20e3fb5ee48a388565dfab6ee4c.png)

5.设置公司/团队标识、用户账号和密码。
![img](https://main.qcloudimg.com/raw/5642a5474a6a418b72b7c6aa7bcf5eb6.png)

6.点击下一步，页面跳转到新建项目，在这里我们需要输入项目名称以及git代码库路径
![img](https://main.qcloudimg.com/raw/c48dfa65b93a467ebea6a78af3d0fd81.png)

7.完成上述步骤后，就可以开始体验腾讯工蜂平台了。

## 登录腾讯工蜂

1.登录了腾讯云账号后，输入链接 `http://git.qcloud.com`。
![img](https://main.qcloudimg.com/raw/4ab7fc94feef33350a446f214aaf76e7.png)
![img](https://mc.qcloudimg.com/static/img/141e536aa5429f9604a776be2af0ee60/image.png)

2.扫码登录即可。
