本文档主要介绍成员管理的一些基本操作。

## 添加项目成员
### 该成员已注册且登录过腾讯工蜂
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com) 后，进入具体的项目。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 单击左侧栏中的【成员】。
![img](https://main.qcloudimg.com/raw/466982b47270f160280001476cfa03cb.png)
3. 单击【添加成员】并输入该成员完整的 Git 用户名，然后为这个用户选择【项目访问角色】，确认后单击【将用户添加到项目】即可添加成功。
![img](https://main.qcloudimg.com/raw/a0cb6052ef197ef24a1038d4cf956682.png)

### 从另一个项目中导入成员
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com) 后，进入具体的项目。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 单击左侧栏中的【成员】。
![img](https://main.qcloudimg.com/raw/466982b47270f160280001476cfa03cb.png)
3. 单击右上角的【导入成员】。
![img](https://main.qcloudimg.com/raw/ae34f4f6ff1f37622c0d5585e2e4e9d4.png)
4. 单击项目下拉框，选择需要导入的项目，然后单击【导入成员】。
![img](https://main.qcloudimg.com/raw/87b7c6ec43c789edf472319675a0198b.png)

## 项目权限管理
### 修改成员权限
1. 完成上述添加成员的步骤后，还可修改成员的 [权限](https://cloud.tencent.com/document/product/612/11344)，在当前界面可以看到刚才添加的成员，单击右边修改权限的按钮。
![img](https://main.qcloudimg.com/raw/2da2ecf898bea9ba39334a61951e0b38.png)
2. 然后选择要修改的权限后，单击【保存变更】按钮即可。
![img](https://main.qcloudimg.com/raw/b4c5494006a6d9fb447b8171f7f75479.png)


### 删除成员权限
1. 完成上述添加成员的步骤后,在当前界面可以看到刚才添加的成员，单击右边对应的【-】移除按钮。
![img](https://main.qcloudimg.com/raw/c530a1df9918d760916d603e61f9a30e.png)
2. 在弹出的对话框中，单击【确定】即可。
![img](https://main.qcloudimg.com/raw/5df5fd2aaa48eca127d69cb7e0e8e0c6.png)

## 项目组权限管理
### 修改成员权限
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com) 后，单击左侧栏的项目组。
![img](https://main.qcloudimg.com/raw/d56485cc98bb96184eb63f34e78b6537.png)
2. 进入具体的项目组。
![simg](https://main.qcloudimg.com/raw/968f9ac27c3811525c38db2f76410d53.png)
3. 单击左侧栏的成员，之后步骤可参考在项目中修改成员权限。
![img](https://main.qcloudimg.com/raw/a5ee908f1a6db02772a5dcb66b2cf340.png)
