本文档主要介绍如何在腾讯 Git 代码托管（工蜂）平台下切换中英文以及设置高亮语法。

## 中英文切换
1. 首先在 [腾讯工蜂](https://git.code.tencent.com) 平台上，单击【个人设置】。
![img](https://main.qcloudimg.com/raw/e583fd886f24288e341a126299e7b07b.png)
2. 在左边的导航栏中单击【偏好设置】。
![img](https://main.qcloudimg.com/raw/00c5444d83f0a2e1238214334debf36b.png)
3.在设置语言界面选择中文或英文，单击即可。
![img](https://main.qcloudimg.com/raw/eed468aa22a53ec86872b1712cee0ccb.png)

   
## 语法高亮选择
根据中英文切换的步骤导航之【偏好设置】处，然后在【语法高亮主题】处，您可以选择语法高亮显示，单击您想选择的模式即可。 
![img](https://main.qcloudimg.com/raw/4dd2eec9f4d2b06273b75aed8c523381.png)
