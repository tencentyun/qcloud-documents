本文档主要介绍的是如何在腾讯 Git 代码托管（工蜂）平台上进行分支管理。

## 创建分支
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com/) 后，单击项目名，进入项目的管理界面。（空项目不能创建分支）。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 进入项目后直接单击项目首页路径下的分支进入到分支管理页面；
![img](https://main.qcloudimg.com/raw/cafa5969fc5bb40042551580c4c44897.png)
 或者先单击左边的【提交】，再单击右边的【版本库分支】，然后单击【+创建分支】。
![img](https://main.qcloudimg.com/raw/8379d52ffb5a68f6a5c3b1308031ff00.png)
3. 在创建分支的界面，填写信息后单击【创建分支】即可。
![img](https://main.qcloudimg.com/raw/f43129112d3399f56ab9d250212f7730.png)

## 删除分支
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com/) 后，单击项目名，进入项目的管理界面。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 直接单击项目首页路径下的分支进入到新建分支页面；
![img](https://main.qcloudimg.com/raw/cafa5969fc5bb40042551580c4c44897.png)
 或者先单击左边的【提交】，再单击右边的【版本库分支】，
 ![img](https://main.qcloudimg.com/raw/c49ce74df4d8c4a97b048199c48e8aa0.png)
3. 找到需要删除的分支，单击删除图标。
![img](https://main.qcloudimg.com/raw/970d150c73c5e590c1dbfbe8fd0fed8f.png)
4. 在弹出的对话框中，单击【确定】即可。
![img](https://main.qcloudimg.com/raw/7d22effa2080f68ba54e9ef31f39bd5f.png)

## 比较分支
### 第一种方式：
1. 进入到分支的管理界面，具体步骤可参考删除分支的前两步。
2. 找到需要比较的分支，单击【对比差异】。
![img](https://main.qcloudimg.com/raw/ece82c5a846db58566ac8f8985240012.png)
3. 跳转到比较的界面。输入两个比较的分支，单击【对比】即可。
![img](https://main.qcloudimg.com/raw/f3a0da8e6d0eed9d95b37d1b0408f864.png)

### 第二种方式：
1. 同样进入到分支的管理界面，具体步骤可参考删除分支的前两步。
2. 单击上面导航的【代码对比】，直接输入两个比较的分支，单击【对比】即可。
![img](https://main.qcloudimg.com/raw/f3a0da8e6d0eed9d95b37d1b0408f864.png)

## 合并分支
### 第一种方式：
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com/) 后，单击项目名，进入项目的管理界面。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 单击左边导航栏的【合并请求】，再单击右边【+创建合并请求】。
![img](https://main.qcloudimg.com/raw/1a8ec56dd5976f46d2ac5d4b0d8c9abd.png)
  
3. 跳转页面后，选择好【源分支】 和【目标分支】，单击【比较两分支】。
![img](https://main.qcloudimg.com/raw/19b03979a44e5210cc40d33b014eb145.png)
4. 根据自身情况填写好信息，最后单击【提交】即可。
![img](https://main.qcloudimg.com/raw/e530d7f948bc851691a4d97abc094d0a.png)

### 第二种方式：
1. 进入到分支的管理界面，具体步骤可参考删除分支的前两步。
2. 找到需要合并的分支，单击【+合并请求】。
![img](https://main.qcloudimg.com/raw/899ea6539c7f27cf66b0d500b4e2ebc7.png)
3. 根据自身情况填写好信息，最后单击【提交】即可。
![img](https://main.qcloudimg.com/raw/e530d7f948bc851691a4d97abc094d0a.png)

## 保护分支
1. 登录 [腾讯工蜂平台](https://git.code.tencent.com/) 后，单击项目名，进入项目的管理界面。
![img](https://main.qcloudimg.com/raw/55fef40ccaca75443eb78c25c58ba742.png)
2. 单击左边导航栏的【设置】。
![img](https://main.qcloudimg.com/raw/ed6a13e406c9b4a868015941790b96a1.png)
3. 单击左边导航栏的【保护分支】项，选好需要保护的分支，单击【保存变更】即可。单击已经受保护的分支后面的设置，还可以设置这个保护分支的评审以及成员。
![img](https://main.qcloudimg.com/raw/dfaa3b410f716e92676d6ebc5bd803d3.png)
