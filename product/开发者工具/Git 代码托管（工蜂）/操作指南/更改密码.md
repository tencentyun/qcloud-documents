本文档主要介绍如何在腾讯 Git 代码托管（工蜂）平台下更改密码。

>!
> 1. 输入新密码时，密码不能少于八个字符。
> 2. 此处修改的是用于 Clone 或 Push 等操作的密码。

## 操作步骤
1. 首先在腾讯工蜂平台上，单击【Profile Settings】。
![img](https://main.qcloudimg.com/raw/1ec1095347512f2ab9a6a297ddf0a3de.png)
2. 在左边的导航栏中单击【Password】。
![img](https://main.qcloudimg.com/raw/e4d54e69944b0d10742ed74a4ee69049.png)
3. 在更改密码页面，输入新密码和重复一次新密码，单击【Save password】后即可更改您的密码。
![img](https://main.qcloudimg.com/raw/90ebebf8de54d1e96c81b3fb5112dc5e.png)
