# wx.removeTabBarBadge
### wx.removeTabBarBadge(Object object)

移除 tabBar 某一项右上角的文本

#### 参数

##### Object object

属性       | 类型       | 默认值 | 必填 | 说明                      
-------- | -------- | --- | -- | ------------------------
index    | number   |     | 是  | tabBar 的哪一项，从左边算起       
success  | function |     | 否  | 接口调用成功的回调函数             
fail     | function |     | 否  | 接口调用失败的回调函数             
complete | function |     | 否  | 接口调用结束的回调函数（调用成功、失败都会执行）
