## 1. 术语和定义
1.1 腾讯云提供的云端开发环境 Cloud Studio 服务是基于浏览器的集成式开发环境（IDE），为开发者提供了一个不间断的云端工作站，提供完整的云端编程体验；无需安装，随时随地用浏览器打开云端开发环境 Cloud Studio 就可以进行代码编写，同时云端开发环境 Cloud Studio 还自带多种预置环境，一键切换或分享；云端开发环境 Cloud Studio 已完整集成 Git，支持提交、推拉、解决冲突等各种操作。云端开发环境 Cloud Studio 采用容器技术，让您的工作空间直接连到容器，可以实现环境的快速切换和分享；云端开发环境 Cloud Studio 自带常用环境，包括 Python、Java、Node.js 等。具体以您购买的服务及腾讯云提供的服务内容为准。目前云端开发环境 Cloud Studio 供用户免费使用，最多可以创建5个工作空间。

1.2 服务不可用：云端开发环境 Cloud Studio 服务不可用是指无法打开在线编辑器和终端无法使用。在连续10分钟或更长时间内，腾讯云云端开发环境 Cloud Studio 服务均持续不可用才计入服务不可用时间，低于连续10分钟不计入服务不可用时间。
以下情况不纳入不可用时间计算：
（1）不可使用的连续时间低于10分钟；
（2）日常系统维护时间、隐式依赖的云服务器服务、云数据盘服务、负载均衡服务、私有网络服务、云监控服务的不可用时间；
（3）由于用户原因、第三方原因或不可抗力导致的不可用时间。

1.3  服务不可用时间：服务月度内云端开发环境 Cloud Studio 服务不可用的分钟数之和。
1.4  服务月度：服务月度是指您购买本服务的服务期限所包含的各个自然月度，如您购买3个月本服务，服务开通之日为3月17日，则包含4个服务月度，其中第1个服务月度是指3月17日到3月31日，第2个服务月度是指4月1日到4月30日，第3个服务月度是指5月1日到5月31日，第4个服务月度是指6月1日到6月16日。服务可用性按服务月度单独核算。
