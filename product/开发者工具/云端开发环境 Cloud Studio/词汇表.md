# 词汇表
**IDE**
IDE，即集成式开发环境（Integrated Development Environment），是一种辅助程序开发人员开发软件的应用软件，在开发工具内部就可以辅助编写源代码文本、并编译打包成为可用的程序。

**工作空间**
一个工作空间是一个虚拟计算单元，我们为每个工作空间提供了独立的存储、计算资源以及开发环境。在一个工作空间内您可以存放自己的项目代码，安装所需要的软件环境，运行或编译自己的项目。

**SSH**
SSH，即安全外壳协议（Secure Shell），是一种加密的网络传输协议。在 Cloud Studio 里面指的是通过在目标 Git 仓库添加 Cloud Studio 的 SSH 公钥，来免除推拉代码时输入用户名密码。

**Git**
Git 是一个分布式版本控制软件。在 Cloud Studio 里面指的是通过可视化界面来进行代码版本控制的功能。

**运行环境**
运行环境（英语：Runtime environment）又称“运行时系统”（run-time system），指一种把半编译的运行码在目标机器上运行的环境。
