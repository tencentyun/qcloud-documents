## 什么是 Cloud Studio
腾讯云云端开发环境（ Cloud Studio）是基于浏览器的集成式开发环境（IDE），为开发者提供了一个永不间断的云端工作站。用户在使用 Cloud Studio 时无需安装，随时随地打开浏览器就能使用。

Cloud Studio 包含代码高亮、自动补全、Git 集成、终端等 IDE 的基础功能，同时支持实时调试、插件扩展等，可以帮助开发者快速完成各种应用的开发、编译与部署工作。

## 与本地 IDE 的区别

| 功能 | 云端开发环境 Cloud Studio  | 本地 IDE |
|---------|---------|---------|
| 安装配置 | 无需安装，在浏览器输入地址就能使用，对设备配置要求低。 | 需要下载安装，设备配置要求稍高。 |
| 文件系统 | 存储于云端容器中，文件操作都是通过网络进行的，可随时随地访问。 | 本地文件系统，仅单一设备可访问。 |
| 环境 | 云端环境，无需繁琐配置，可以保证多用户环境统一。 | 本地环境需要安装配置，不同设备难以统一。 |
| 分享 | 基于 Web，一个链接即可分享。 | 分享不方便。 |

## 产品优势
### 免安装，跨平台
无需安装，不限操作系统，在浏览器输入地址就能使用。
### 预置环境，免除繁琐配置
自带多种语言环境，无需繁琐配置，即刻快速启动应用。
### 功能丰富
Git 集成、代码提示、样式主题、多功能终端等功能应有尽有。
### 支持云服务器连接
您可以将 Cloud Studio 连接至自己的云服务器，进行开发或部署工作。
### 强大的插件系统
支持安装插件，满足不同使用需求。

## 应用场景
## 快速启动项目
使用 Cloud Studio 的预置环境，您可以直接创建对应类型的工作空间，快速启动项目进入开发状态，而无需进行繁琐的环境配置。

## 实时调试网页
Cloud Studio 内置预览插件，可以实时显示网页应用。当您的代码发生改变之后，预览窗口会自动刷新，这样您就可以在 Cloud Studio 内实时开发调试网页了。

## 远程访问云服务器
Cloud Studio 支持您连接自己的云服务器，这样就可以在编辑器中查看云服务器上的文件，进行在线开发部署工作。
