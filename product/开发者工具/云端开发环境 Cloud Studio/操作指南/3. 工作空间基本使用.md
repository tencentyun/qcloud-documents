# 工作空间基本使用

## 工作空间是什么？
一个工作空间是一个虚拟计算单元，它包含独立的存储、计算资源以及开发环境。您可以在工作空间内存放自己的项目代码，安装所需要的软件环境，运行或编译项目。

## 如何创建工作空间？
前往腾讯云[云端开发环境 Cloud Studio 控制台](https://cloudstudio.net/dashboard)，点击「创建工作空间」的按钮，就会进入创建工作空间的流程。

![控制台](https://main.qcloudimg.com/raw/cf49873f8b890dd3a0172a3827eb8333.jpg)

您需要填写工作空间名、描述，并选择运行环境、代码来源，来创建一个工作空间。

![创建工作空间](https://main.qcloudimg.com/raw/dde46b5474a13890585f67d5a9a6aa1f.jpg)

对应的各个表单项解释如下：
- **工作空间名**：您的工作空间的唯一标识。
- **描述**：对该工作空间的作用描述。
- **运行环境**：工作空间内代码运行的环境，您可以选择预置环境，包含 Ubuntu、Python（内置 2.7 和 3.6，默认为 Python 3.6）、Java 和 Node.js 四种；也可以选择将其连接到自己的云主机上，详情参见[连接到自己的云主机](/)。
- **代码来源**：工作空间内的代码来源，您可以选择[从 Git 仓库导入](/)，也可以选择“空”，即不添加任何代码。

## 如何运行、停止、删除工作空间？
### 运行
想要运行某个工作空间，直接点击对应卡片就会在新页面中打开并运行该空间，此时该工作空间卡片上会显示“运行中”状态。

### 停止
对于处于运行中状态的工作空间，会在卡片右下角显示一个“停止”操作，点击它就可以停止运行该工作空间。

![停止工作空间](https://main.qcloudimg.com/raw/8aa6662ff5a35ef1cf86cc63ca392ff0.jpg)

### 删除
不需要的工作空间可以删除，已删除的工作空间会展示在下方「已删除工作空间」列表中。

![删除工作空间](https://main.qcloudimg.com/raw/1acc0dea53f662fd8d38a302bb34ba08.jpg)

删除一个工作空间后，他还会被保留 24 小时，在此之前您可以随时恢复，超过 24 小时将被永远销毁。

![恢复工作空间](https://main.qcloudimg.com/raw/13431b9dea407f0fefc61dc916003567.jpg)

## 工作空间有何限制？
目前每个用户最多有 5 个工作空间，并且只能同时运行一个工作空间，如果您需要打开另一个工作空间需要先关闭当前运行中的工作空间。

## 工作空间界面
工作空间是我们主要的工作区域，它主要由顶部菜单栏、左侧操作面板、右侧代码编辑区和底部状态栏组成。

![工作空间界面](https://main.qcloudimg.com/raw/ca6747a3e7dc514770b313efb76164c0.png)

我们可以根据自己的习惯设置界面外观、偏好，安装自己需要的插件。需要注意的是，**您的偏好设置和插件在每个工作空间中是互相隔离的**，也就是说您可以给不同的工作空间设置不同的偏好，安装不同的插件。
