# 开通 Cloud Studio 服务
登录腾讯云之后，进入 [Cloud Studio 控制台](https://console.cloud.tencent.com/cloudstudio)。如果您还未开通 Cloud Studio 服务，会看到如下开通页面。

![开通页面](https://main.qcloudimg.com/raw/06df185bbba193247113e81b7b7c852c.jpg)

点击立即开通按钮，会跳转到角色管理的授权页面，点击同意授权会回到 Cloud Studio 控制台。

![同意授权](https://main.qcloudimg.com/raw/5c4f974088964144799863b92b866736.jpg)

此时，开通按钮变成了“立即使用”，点击该按钮就会打开到 Cloud Studio 工作空间列表页面，此时您就可以使用 Cloud Studio 了。
