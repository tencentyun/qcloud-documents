## 功能描述

本接口用于进行文本内容的直接审核，请求方式为同步请求。

该接口的支持情况如下：

>?
>
>- 直接审核纯文本信息，需要先经过 base64 编码（仅支持 UTF8 编码）。
>- 支持识别多种违规场景，包括：色情、违法、广告等场景。
>- 支持根据不同的业务场景 [配置自定义的审核策略](https://cloud.tencent.com/document/product/460/56345)。

## 费用说明

- 每个审核场景单独计费，例如您选择审核色情、广告两种场景，则审核**1条文本**，**计2次**审核费用。

## 限制说明

- 长度限制：当审核的内容为纯文本信息，需要先经过 base64 编码，文本编码前的原文长度不能超过10000个 UTF8 编码字符。
- 文本审核语言支持：目前支持中文、中英混合、阿拉伯数字的检测。

## 推荐使用 SDK

数据万象 SDK 提供了完整的 Demo、自动集成、计算签名等能力。您可通过 SDK 方便快捷地调用接口。[点此查看 SDK 文档](https://cloud.tencent.com/document/product/460/47727)。
