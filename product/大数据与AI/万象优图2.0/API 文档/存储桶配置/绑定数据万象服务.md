## 请求
#### 请求示例

```
PUT / HTTP/1.1
Host: <BucketName-APPID>.pic.<Region>.myqcloud.com
Date: GMT Date
Authorization: Auth String
```

>? Authorization：Auth String （详情参阅 [请求签名](https://cloud.tencent.com/document/product/436/7778) 文档）。
>
