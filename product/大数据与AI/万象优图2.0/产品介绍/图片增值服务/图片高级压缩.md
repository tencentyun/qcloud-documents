## 简介

数据万象提供图片高级压缩功能，可以更加高效地将图片转换为 TPG 或 HEIF 等高压缩比格式，有效降低图片传输链路及加载耗时，降低带宽及流量成本。

## 适用场景

#### 电商平台

针对商品图、用户头像、图片评论等进行区分压缩，高清场景、可损场景使用不同压缩功能支持，最大化压缩收益，并提升用户体验。

#### 社交软件

社交软件可使用万象提供的 WEBP 转码、HEIF 转码等压缩功能，适配不同操作系统的使用环境；客户也可将 TPG 的编解码能力集成至自己软件中，实现软件内通讯的最大化压缩。

#### 在线教育

客户可根据不同场景需求使用不同压缩功能，平衡压缩收益与视觉体验。利用完善的图片解决方案，开发者既能快速地对外提供图片服务，也能基于云服务的弹性调度能力快速调整资源，适应骤变的业务场景。


## 使用方法

#### 开启/关闭服务

您可在控制台开启或关闭图片高级压缩功能，操作详情请见对应 [图片高级压缩](https://cloud.tencent.com/document/product/460/46821) 控制台文档。

#### 使用服务

您可通过对图片访问链接添加参数的形式，在访问图片时直接压缩，详情请见 [图片高级压缩](https://cloud.tencent.com/document/product/460/43680) API 文档。
