内容审核费用指使用内容审核功能而产生的费用，内容审核功能包括 [图片审核](https://cloud.tencent.com/document/product/460/46493)、[视频审核](https://cloud.tencent.com/document/product/460/46494)、[音频审核](https://cloud.tencent.com/document/product/460/47120)、[文本审核](https://cloud.tencent.com/document/product/460/52712)。

内容审核按照审核类型方式分为图片审核费用、视频审核费用、音频审核费用和文本审核费用。不同的审核类型、审核方式，对应的价格不同。

审核方式如下：
- **增量审核**指用户预先配置内容审核功能，当用户将数据上传至存储桶后自动进行审核。
- **存量审核**为批量任务，用户创建存量审核任务后，后台会对存储桶内指定的数据进行批量审核。存量审核会自动调度后台的审核情况，后台审核资源会优先保证增量审核服务，存量审核主要利用后台闲时资源，所以价格上相对**增量审核**会更低。
- **人工审核**区别于机器审核，由天御专业团队提供，对机器审核判定为敏感的内容进行人工二次复检。



## 图片审核费用

图片审核费用指使用图片审核功能而产生的费用。图片审核按照图片审核次数，梯度收费，计费周期为**月**。

- **图片审核（增量）定价**

| 月审核次数（N）      | 确定部分    | 不确定部分  |
| :-------------- | :---------- | :---------- |
| N ≤ 300万          | 1.50元/千次 | 0.38元/千次 |
| 300万 ＜ N ≤ 1500万  | 1.35元/千次 | 0.34元/千次 |
| 1500万 ＜ N ≤ 3000万 | 1.30元/千次 | 0.33元/千次 |
| N ＞ 3000万      | 1.10元/千次 | 0.28元/千次 |

>!
> - 调用 API 进行审核为实时处理，按增量审核进行收费。
> - 图片审核为打分制，对于每一张图片均有一个0 - 100的分值。
> - **确定部分**指确定是审核出的违规图片和确定正常的图片，分值在60分以下及90分以上，无需人工干预。
> - **不确定部分**指疑似违规的图片，系统无法区分是否违规，分值在60 - 90分，建议用户通过人工审核的方式确定。
> - 图片审核每日有2000次的免费审核额度，未用完的额度不累计至下一天，计费会扣除免费的2000次（先扣费用比较高的确定部分）。
> - 图片审核可审核多种场景，**每种场景单独计费**。如您勾选了鉴黄、广告两种场景，则审核1张图片，会算为审核2次。
>

- **图片审核（存量）定价**

| 月审核次数（N）      | 价格         |
| :-------------- | :----------- |
| N ≤ 300万          | 0.45元/千次  |
| 300万 ＜ N ≤ 1500万  | 0.405元/千次 |
| 1500万 ＜ N ≤ 3000万 | 0.39元/千次  |

>! 存量的图片审核不区分**确定部分**和**不确定部分**。
