## 简介

智能识别为您提供图片内容识别和理解能力，集成了腾讯云 AI 的多种强大功能，对存储于对象存储 COS 的数据提供**二维码识别**、**图片标签识别**等增值服务，广泛应用于电商网站、社交软件、相册应用、视频网站等多种场景，方便您对图片进行内容管理。 

>?目前数据万象控制台仅提供智能识别服务展示，后续将持续更新。

### 二维码识别

数据万象二维码识别功能可识别图片中有效二维码的位置及内容，输出图像中二维码包含的文本信息（每个二维码对应的 URL 或文本），并可对识别出的二维码添加马赛克。

>?
>
>- 二维码识别费用指对图片进行二维码识别而产生的费用，根据识别次数收费，计费周期为月。计费详情请查看 [计费文档](https://cloud.tencent.com/document/product/1247/45320#.E5.9B.BE.E7.89.87.E6.A0.87.E7.AD.BE.E8.B4.B9.E7.94.A8)。
>- 目前仅提供 API 使用方式，使用详情请查看 [二维码识别](https://cloud.tencent.com/document/product/460/37513) API 文档。

### 图片标签识别

图片标签可以识别图片中的**场景**、**物品**、**人物**等信息，包含八个大类、六十多个二级分类、数千个标签，例如自然风光（山、海、天空、夕阳等）、人造环境（建筑、游乐场、会议室等）、人像（男性、女性、自拍、合照等）、物品（美食、穿着、日常用品等）、动物（猫、狗、鸟类、哺乳动物、海洋生物等各种宠物或野生动物）。

图像标签可以用于相册分类、信息流内容推荐、广告推荐、图库分类检索、视频内容理解、拍照识图等各种场景。

>?
>
>- 当前内容识别的图片限制为：图片格式支持 PNG、JPEG、JPG，图片大小不超过3MB，图片宽高大于50像素 x 50像素。
>- 标签识别计费隶属于数据万象的**内容识别费用**，指对图片进行二维码识别而产生的费用，根据识别次数收费，计费周期为月。计费详情请查看 [计费文档](https://cloud.tencent.com/document/product/460/6970#.E4.BA.8C.E7.BB.B4.E7.A0.81.E8.AF.86.E5.88.AB.E8.B4.B9.E7.94.A8)。
>- 目前仅提供 API 使用方式，使用详情请查看 [图片标签识别](https://cloud.tencent.com/document/product/460/39082) API 文档。
