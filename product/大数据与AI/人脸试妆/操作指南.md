### 注册登录
登录腾讯云官网。如果没有账号，请参考 [账号注册教程](https://cloud.tencent.com/document/product/378/17985)。

### 开通服务
登录 [人脸试妆控制台](https://console.cloud.tencent.com/fmu)，勾选同意《腾讯云服务协议》后，单击即可开通服务。

### 通过 API/SDK/命令行来使用服务
- 我们为您提供了丰富多样的 API 接口，您可以查看 [人脸试妆 API 文档](https://cloud.tencent.com/document/product/1172/40697)。
- 建议您使用 [API 3.0 Explorer](https://console.cloud.tencent.com/api/explorer?Product=fmu&Version=2019-12-13&Action=TryLipstickPic&SignVersion=) 工具进行在线测试，该工具提供了在线调用、签名验证、SDK 代码生成和快速检索接口等能力，能显著降低使用云 API 3.0的难度。
- 建议您通过 API 3.0 Explorer 生成服务端 SDK 代码，结合腾讯云的已编写好的开发工具集（SDK），可以快速调用人脸特效服务。SDK 已支持多种语言，包括 Python、Java、PHP、Go、NodeJS、.Net 等，您可以在每个服务的文档中下载对应的 SDK，也可以通过 [SDK 中心](https://cloud.tencent.com/document/sdk) 获取。
- 通过 [腾讯云命令行工具](https://cloud.tencent.com/document/product/440/6176) 来调用腾讯云 API ，管理您的腾讯云资源，您还可以基于腾讯云 CLI 来做自动化和脚本处理，从而简化一些具有复杂 API 的服务。

### 调用成功
当请求返回结果图片的 Base64 信息，表示接口调用成功，您可以查看具体的人脸试妆效果。

### 查看调用
您可以登录 [人脸试妆控制台](https://console.cloud.tencent.com/fmu)，查看各服务的具体调用情况。
