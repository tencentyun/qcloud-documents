### SPARK-SparkJobHistory 

| 指标名称 | 指标单位 | 指标含义                              |
| -------- | -------- | ------------------------------------- |
| YGC      | 次       | Young GC 次数                         |
| FGC      | 次       | Full GC 次数                          |
| FGCT     | s        | Full GC 消耗时间                      |
| GCT      | s        | 垃圾回收时间消耗                      |
| YGCT     | s        | Young GC 消耗时间                     |
| S0       | %        | Survivor 0区内存使用占比             |
| E        | %        | Eden 区内存使用占比                   |
| CCS      | %        | Compressed class space 区内存使用占比 |
| S1       | %        | Survivor 1区内存使用占比             |
| O        | %        | Old 区内存使用占比                    |
| M        | %        | Metaspace 区内存使用占比              |
