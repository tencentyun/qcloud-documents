## 集群创建
集群创建参见 [创建一个 EMR 集群](https://cloud.tencent.com/document/product/589/10981)。
## 集群管理
集群创建成功后，登录弹性 MapReduce [控制台](https://console.cloud.tencent.com/emr)，进入集群列表页面。
![](https://main.qcloudimg.com/raw/0e9c67a528fd4d828bacae908cc9340c.jpg)
* 单击 “项目下拉列表” 可以按项目浏览集群。
* 单击 “地域下拉列表” 可以按地域浏览集群。
* 勾选 EMR 集群实例，单击【分配至项目】按钮，可以分配集群至项目。
* 单击 “实例 ID” 或者【详情】按钮，可以查看实例详情。
* 单击 “更多下拉列表”，可以重启组件或者销毁集群。


### 集群详情查看
在集群列表页面，单击相应集群的实例 ID 或者【详情】按钮后进入集群管理页。
用户可以在实例详情页查看集群的基本信息、软件配置信息和硬件信息。
**基本信息：**
![](https://main.qcloudimg.com/raw/9389aebca2bd7e735e6a85d214754ea9.jpg)
其中，Master 公网 IP 是快捷入口页面的地址入口 IP。

**软件配置信息和硬件信息：**
![](https://main.qcloudimg.com/raw/144227756d9ac2ed9ea9ec79d1d0aa09.jpg)
* 软件配置信息显示了本集群安装的软件和版本号。
* 硬件信息展示了各节点硬件配置的详细信息。

### 节点详情查看
在集群管理页面单击【节点信息】，可以查看当前集群节点信息。
![](https://main.qcloudimg.com/raw/a07ab3f8b6e45483a8505a484268f339.jpg)
* 可以通过切换子 tab 查看不同节点信息。
* 节点信息展示了集群内私有网络的 IP、部署的进程以及硬件配置。
* 通过单击 “CVM ID” 可以打开该节点对应的云服务器控制台界面。

### 节点扩缩容
在节点详情页面切换到 Core 节点或者 Task 节点会出现【增加节点】按钮，用户可进行节点扩容。当前只支持 Core 节点和 Task 节点扩容。扩容的节点规格需要和之前的节点规格一致。
在选择好要进行的扩容操作之后，系统会根据扩容节点给出扩容费用。
![](https://main.qcloudimg.com/raw/d603c98fe7429f32a2df289805643c92.jpg)
在Task节点可以对其进行缩容操作，即减少Task节点的个数，当前只有Task节点支持该操作。

### 快捷入口
在集群（实例）管理页单击【快捷入口】按钮可以进入到快捷入口页面。快捷入口集成了各组件（Software）的 Web UI 访问入口(地址入口)，登录这些组件需要进行用户名和密码验证，用户名默认为 root，密码为用户在创建集群时设置的密码。
单击【修改密码】按钮，能够修改登录快捷入口的密码。
![](https://main.qcloudimg.com/raw/9f0924959ebd94e98be15251fd1ea659.jpg)
