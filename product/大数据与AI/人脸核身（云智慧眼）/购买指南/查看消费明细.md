您可以在腾讯云控制台的**费用中心**，查看您的账户使用人脸核身服务所产生的费用信息。如需查询资源包的使用情况，例如每日的使用量，您可以在 [人脸核身控制台](https://console.cloud.tencent.com/faceid) 进行查询，详情请参见 [查询资源包使用量](#JYB)。

>!
- 在人脸核身服务账单结算时，系统将按照**免费额度 > 预付费资源包 > 按量计费**的顺序进行结算，接入渠道与计费标签对应关系请参阅 [计费概述](https://cloud.tencent.com/document/product/1007/56803)。
- 默认情况下系统采用按量计费方式进行结算。对于享受免费额度的账户，先抵扣免费额度，再按量计费。
- 购买资源包的情况下，在资源包生效期，系统将优先抵扣免费额度和资源包，超出免费额度和资源包部分将采用按量计费方式结算，从您的账户中扣除相应金额。

## 查询费用账单

1. 登录 [腾讯云控制台](https://console.cloud.tencent.com)。
2. 在右上方导航栏**费用**中，单击**费用中心**，进入费用中心总览页面。
3. 在左侧菜单栏中，单击**费用账单**。
4. 在下拉子菜单中，单击**账单详情**，然后选择人脸核身产品，即可在线查看人脸核身服务的资源 ID 账单和明细账单。
>?选择**账单概览**可查看账单费用趋势和账单汇总，选择**账单下载**可进行已出账月份的账单下载。

### 资源 ID 账单 
资源账单根据资源 ID 维度汇总展示账单金额，可以查看对应资源的费用汇总。
![](https://qcloudimg.tencent-cloud.cn/raw/6c5bc3cc11395ab7c3a1226fe80bafe4.png)

#### 操作指引
1. 进入账单详情页面，在**资源 ID 账单**选项卡页面，下拉列表中选择**人脸核身·云智慧眼**，可以查看您的人脸核身服务的消费情况。
![](https://qcloudimg.tencent-cloud.cn/raw/ab2f80edf32ebdeb03c152ec1db977dd.png)
2. 在账单页面右上方，单击<img src="https://qcloudimg.tencent-cloud.cn/raw/7c772a621e15e02b78eb1445b55b565a.png" style="margin:-3px 0px">，可筛选账单显示字段。
3. 在账单页面右上方，单击<img src="https://main.qcloudimg.com/raw/e421450264489d44d20f11a44e15dfaa.png" style="margin:-3px 0px">，可下载您的月度人脸核身服务的消费明细。

 - 支持根据产品、项目、地域、计费模式、交易类型等条件进行筛选查看。
 - 支持输入资源 ID 或实例别名进行筛选。
 - 支持自定义设置列表展示字段。
 - 包年包月的明细支持单击订单 ID，跳转订单管理页面查看订单详情。
 - 明细最右侧可单击**账单详情**，进入该资源的账单详情。
