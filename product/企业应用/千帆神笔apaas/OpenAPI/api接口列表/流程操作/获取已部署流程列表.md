1. **接口说明**

接口请求地址：/flow/WorkFlowOpenApi/ListDeployedFlows

该接口用于获取已部署流程列表。

2. **输入参数**

| 参数名称 | 必选 | 类型 | 描述         |
| -------- | ---- | ---- | ------------ |
| AppCode  | 否   | Long | 应用唯一编码 |



3. **输出参数**

| 参数名称 | 类型   | 描述     |
| -------- | ------ | -------- |
| Data     | Object | 返回参数 |

**Data**

| 参数名称   | 类型            | 描述         |
| ---------- | --------------- | ------------ |
| TotalCount | Integer         | 计数总数     |
| FlowSet    | Array of Object | 流程设置参数 |

**Object**

| 参数名称    | 类型   | 描述     |
| ----------- | ------ | -------- |
| FlowCode    | Long   | 流程编码 |
| Version     | String | 版本号   |
| Name        | String | 返回名称 |
| AppCode     | String | 应用编码 |
| Description | String | 描述     |




4. **示例**

```json
{
   "AppCode":1381826670134000000
}
```

> 返回结果

```json
{
    "Response": {
        "Data": {
            "TotalCount": 2,
            "FlowSet": [
                {
                    "FlowCode": 123456,
                    "Version": "1.0.0",
                    "Name": "namemocka",
                    "AppCode": "123456",
                    "Description": "descriptionmocka"
                },
                {
                    "FlowCode": 23456,
                    "Version": "1.0.1",
                    "Name": "namemockb",
                    "AppCode": "123456",
                    "Description": "descriptionmockb"
                }
            ]
        }
    }
}
```
