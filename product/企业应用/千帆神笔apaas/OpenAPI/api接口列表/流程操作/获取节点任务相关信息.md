1. **接口说明**

接口请求地址：/flow/WorkFlowOpenApi/GetFlowInstanceTasks

该接口用于获取节点任务的相关信息。

2. **输入参数**

| 参数名称       | 必选 | 类型            | 描述                          |
| -------------- | ---- | --------------- | ----------------------------- |
| TenantId       | 是   | Long            | 租户ID                        |
| InstanceId     | 否   | Long            | 流程实例ID                    |
| TaskIds        | 否   | Array of Long   | 节点任务ID列表                |
| TaskType       | 否   | Integer         | 1-审批节点 2-处理节点         |
| ElementIds     | 否   | Array of String | 节点ID列表                    |
| AppCode        | 否   | Long            | 应用编码                      |
| FlowCode       | 否   | Long            | 流程编码                      |
| TaskStatus     | 否   | Integer         | 1-待办 2-已办 3-无效 4-草稿箱 |
| TaskAssigneeId | 否   | Long            | 处理人ID                      |
| StartTime      | 否   | Date            | 启动时间                      |
| EndTime        | 否   | Date            | 结束时间                      |
| Limit          | 否   | Integer         | 分页数                        |

3. **输出参数**

| 参数名称 | 类型   | 描述     |
| -------- | ------ | -------- |
| Data     | Object | 返回参数 |

**Data**

| 参数名称   | 类型            | 描述     |
| ---------- | --------------- | -------- |
| Result     | Array of Object | 返回参数 |
| TotalCount | Integer         | 返回合计 |

**Result**

| 参数名称        | 类型           | 描述                                     |
| --------------- | -------------- | ---------------------------------------- |
| InstanceId      | Long           | 流程实例ID                               |
| TaskId          | Long           | 节点任务ID                               |
| FlowName        | String         | 流程名称                                 |
| FlowCode        | String         | 流程编码                                 |
| FlowStartUser   | TenantUserInfo | 流程发起人                               |
| ElementName     | String         | 节点名称                                 |
| AppCode         | Long           | 应用编码                                 |
| TaskAssignee    | TenantUserInfo | 任务处理人                               |
| StartTime       | Date           | 启动时间                                 |
| EndTime         | Date           | 结束时间                                 |
| TaskStatus      | Integer        | 1-待办 2-已办 3-无效 4-草稿箱            |
| OperationResult | Integer        | 执行结果：1-提交；2-同意；3-拒绝；4-转办 |

**TenantUserInfo**

| 参数名称  | 类型   | 描述          |
| --------- | ------ | ------------- |
| SourceUid | String | idaas的用户ID |
| TenantUid | Long   | 用户ID        |

** 正常返回 **

```json
{
  "Response": {
    "Data": {
    }
  }
}
```

** 异常返回 **

```json
{
  "Response": {
    "Error": {
      "Code": "XXXXX",
      "Message": "XXXX"
    }
  }
}
```
