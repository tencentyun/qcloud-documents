### 数据模型

对象: 学生

> ![img](https://qcloudimg.tencent-cloud.cn/raw/91e23880bb52fff0ab656fd848d1a04c.jpg)
| 字段 ID        | 字段名称 |
| -------------- | -------- |
| iD             | 编号     |
| name           | 学生姓名 |
| sex            | 学生性别 |
| age            | 学生年龄 |
| admission_date | 入学时间 |
| class          | 班级     |
| 录入员         | recorder |
