本文档主要介绍企业第一次正式接入腾讯电子签集成版的操作流程。

- 开放平台总体接入流程如下：

## 步骤1：创建腾讯云账号[](id:step1)

- 如果您还没有腾讯云账号，您需要 [注册腾讯云](https://cloud.tencent.com/document/product/378/17985) 账号，认证方式参见 [企业对公打款认证](https://cloud.tencent.com/document/product/378/10496#.E6.AD.A5.E9.AA.A43.EF.BC.9A.E9.80.89.E6.8B.A9.E8.AE.A4.E8.AF.81.E6.96.B9.E5.BC.8F)，完成 [企业实名认证](https://cloud.tencent.com/document/product/378/10496)。
- 如果您有完成对公打款实名认证的腾讯云企业账号可跳过此步骤。

## 步骤2：补充实名认证[](id:step2)
完成腾讯云对公打款认证后，还需在腾讯电子签完成补充实名认证。您可登录 [腾讯电子签](https://ess.tencent.com/) 官网线上补充实名或将相关材料通过邮件发送到 e-contract@tencent.com 进行线下补充实名 。我们将在1-3个工作日内完成审核，审核结果将通过邮件告知您。目前腾讯电子签支持 [线上](#online) 及 [线下](#offline) 两种不同的企业实名方式：
### 线上补充实名[](id:online)
访问 [腾讯电子签](https://ess.tencent.com/) 官网，通过注册的腾讯云企业账号登录。
![](https://main.qcloudimg.com/raw/229b065fcc307bd4b0370bfeb3bbcc19.png)
登录成功后请前往 [企业信息](https://ess.tencent.com/company-info) 单击【立即认证】，填写补充实名信息，提交审核。

### 线下补充实名[](id:offline)
如果您不方便进行线上实名，您可将对应材料通过邮件发送到 [e-contract@tencent.com](e-contract@tencent.com) 或联系您的客户经理进行线下补充实名，以下是您需要准备的材料：

## 步骤3：申请应用号以及接口白名单[](id:step3)

实名认证后，将包含以下信息的 Excel 通过邮件发送到 e-contract@tencent.com，资质审核通过之后将会给到您对应的应用号。如您需用到白名单接口相关能力，请在邮件中附上白名单接口名。白名单接口请参见 [白名单接口列表](https://cloud.tencent.com/document/product/1420/53858) 文档。

| 字段     | 描述                                                         |
| :------- | :----------------------------------------------------------- |
| Uin      | 腾讯云主账号的 UIN，如：100016315204                          |
| 企业名称 | 企业的名称，如：北京XXXX网络技术有限公司                     |
| 应用名称 | 应用的名称，如：微信                                         |
| 应用类型 | 应用类型，如：PC Web、APP（IOS） 、APP（Android） 、微信小程序、H5、其他 |
| 应用简介 | 应用的介绍，100个字符以内                                    |

## 步骤4：获取 SecretId 和 SecretKey[](id:step4)

您可选择使用主账号获取 API 密钥或者使用绑定了电子签策略的子账号来获取 API 密钥。

### 使用主账号获取 API 密钥进行调用

使用企业账号登录腾讯云【访问管理】 > 【访问密钥】 > 【[API 密钥管理](https://console.cloud.tencent.com/cam/capi)】单击【新建密钥】获得 SecretId 和 SecretKey。
![](https://main.qcloudimg.com/raw/3b6003b54700ed6772e5b8b2882786e2.png)

### 使用子账号获取 API 密钥进行调用

企业可使用子账号绑定电子签策略，来调用 API 使用电子签服务。
1. 使用主账号登录腾讯云访问管理，在用户列表中选中子账号，单击【授权】。如无子账号可参见 [自定义创建子用户](https://cloud.tencent.com/document/product/598/13674) 创建子账号。
![](https://main.qcloudimg.com/raw/d18fcc7f82623c713e8dc96b77d28067.png)
2. 在关联策略搜索 QcloudESSBASICFullAccess 选中并保存即完成了授权。
![](https://main.qcloudimg.com/raw/1284f0f842bf81e1234c6e24976fc78a.png)
3. 使用子账号登录腾讯云【访问管理】 > 【访问密钥】 > 【[API 密钥管理](https://console.cloud.tencent.com/cam/capi)】。
![](https://main.qcloudimg.com/raw/2b9d2334ec9c2f8103822a0597d05bb0.png)
4. 单击【新建密钥】获得 SecretId 和 SecretKey。
![](https://main.qcloudimg.com/raw/350c040c132985119849caad7f47e9eb.png)
通过 SecretId 、 SecretKey、应用号和用户 ID（主账号或绑定电子签策略的子账号）可以使用 API 接口调用电子签服务。至此接入流程已全部结束。
