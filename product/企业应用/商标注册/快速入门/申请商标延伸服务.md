您可前往 [商标延伸](https://console.cloud.tencent.com/trademark/extended) 中查看、管理您在腾讯云申请的商标延伸服务。

## 应用场景

商标延伸共有以下13个应用场景，您可根据您的业务需求，申请对应的延伸服务。

| 序号   | 服务名称       | 目标人群                                     |
| ---- | ---------- | ---------------------------------------- |
| 1    | 商标变更申请     | 变更注册人名义、登记地址等注册事项（代理人、文件接收人）的商标，向商标局提交申请。 |
| 2    | 商标转让申请     | 转让商标（商标权属发生改变），依照商标法规定向商标局提出申请。          |
| 3    | 商标驳回复审申请   | 商标实质审查未通过，在规定时效内向商评委提交复审申请，再次争取。         |
| 4    | 商标撤三申请     | 他人已经注册满三年的商标，长期不使用，向商标局要求撤销该商标。          |
| 5    | 商标撤三答辩申请   | 商标被他人申请撤销后，及时提交答辩申请以保留商标的使用权。            |
| 6    | 商标异议申请     | 他人申请注册的商标，商标局初审公告期间，提出异议申请阻止注册。          |
| 7    | 商标异议答辩申请   | 公示期被他人提出异议，在规定时效内申请答辩，挽回注册。              |
| 8    | 商标宣告无效申请   | 他人已注册成功的商标侵犯您商标的在先权或违反《商标法》相关规定，请求宣告该商标无效。 |
| 9    | 商标宣告无效答辩申请 | 注册商标被他人宣告无效后，及时提出答辩，维护自身商标权益。            |
| 10   | 商标使用许可备案申请 | 授权他人使用商标，许可备案并报送备案材料，有备无患。               |
| 11   | 补发商标注册证申请  | 《商标注册证》遗失或者破损，商标注册人可提出补发证书申请。            |
| 12   | 不予注册复审申请   | 商标被异议、异议答辩失败，可向商标评审委员会提不予注册复审。           |
| 13   | 撤回商标注册申请   | 办理商标注册等商标事宜，可在商标局做出决定前撤回申请。              |

## 申请周期

| 序号   | 商标服务名称     | 申请周期说明                                   |
| ---- | ---------- | ---------------------------------------- |
| 1    | 商标变更申请     | 商标变更需要2个月左右，下发变更证明，无受理通知书。               |
| 2    | 商标转让申请     | 商标转让受理需要1个月左右，转让完成需要4个月左右。           |
| 3    | 商标驳回复审申请   | 目前商标驳回复审需要6个月左右。  根据商标法第三十四条规定：对驳回申请、不予公告的商标，商标局应当书面通知商标注册申请人。商标注册申请人不服的，可以自收到通知之日起十五日内向商标评审委员会申请复审。商标评审委员会应当自收到申请之日起九个月内做出决定，并书面通知申请人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长三个月 |
| 4    | 商标撤三申请     | 商标撤三申请需要9个月左右。  根据商标法第四十九条第二款规定：注册商标成为其核定使用的商品的通用名称或者没有正当理由连续三年不使用的，任何单位或者个人可以向商标局申请撤销该注册商标。商标局应当自收到申请之日起九个月内做出决定。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长三个月。 |
| 5    | 商标撤三答辩申请   | 商标撤三答辩需要2个月左右，无受理环节。                     |
| 6    | 商标异议申请     | 商标异议申请需要12个月左右。  商标法第三十五条规定：对初步审定公告的商标提出异议的，商标局应当听取异议人和被异议人陈述事实和理由，经调查核实后，自公告期满之日起十二个月内做出是否准予注册的决定，并书面通知异议人和被异议人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长六个月。 |
| 7    | 商标异议答辩申请   | 商标异议答辩需要9个月左右，无受理环节。                     |
| 8    | 商标无效宣告申请   | 商标无效宣告申请需要12个月左右。  根据商标法第四十四条第三款规定：商标评审委员会应当自收到申请之日起9个月内做出维持注册商标或者宣告注册商标无效的裁定，并书面通知当事人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长三个月。 |
| 9    | 商标无效宣告答辩申请 | 商标无效宣告答辩需要6个月左右，无受理环节。                   |
| 10   | 商标使用许可备案申请 | 商标使用许可备案需要4个月左右。  根据商标法第四十三条第三款规定：许可他人使用其注册商标的，许可人应当将其商标使用许可报商标局备案，由商标局公告。商标使用许可未经备案不得对抗善意第三人。 |
| 11   | 补发商标注册证申请  | 补发商标注册证需要2个月左右，无受理通知书。                   |
| 12   | 商标不予注册复审申请 | 目前商标不予注册复审需要6个月左右。  根据商标法第三十五条第三款规定：商标局做出不予注册决定，被异议人不服的，可以自收到通知之日起十五日内向商标评审委员会申请复审。商标评审委员会应当自收到申请之日起十二个月内做出复审决定，并书面通知异议人和被异议人。有特殊情况需要延长的，经国务院工商行政管理部门批准，可以延长六个月。 |
| 13   | 撤回商标注册申请   | 撤回商标注册申请需要2个月左右。无受理通知书。                  |
