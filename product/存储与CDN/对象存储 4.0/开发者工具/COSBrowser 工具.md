## 功能概述
COSBrowser 工具即 COS 桌面版工具，用户可以使用该工具进行可视化、方便的数据上传、下载等操作。

>!
>- COS Browser 会使用系统配置的代理来尝试网络连接，请确保您的代理配置正常或请停用无法连接互联网的代理配置。Windows 用户可在操作系统的【Internet 选项】中查询，macOS 用户可在【网络偏好设置】中查询。
>- 使用该工具上传同名文件，会覆盖较旧的同名文件，不支持校对是否存在同名文件的功能。

## 使用方法

用户可使用云 API 密钥 SecretId 和 SecretKey （不支持项目密钥）进行登录，该密钥可在 [访问管理控制台](https://console.cloud.tencent.com/cam/capi) 获取，登录后会保留登录信息。
目前支持批量上传、下载、删除功能。拖拽上传下载暂不支持。
