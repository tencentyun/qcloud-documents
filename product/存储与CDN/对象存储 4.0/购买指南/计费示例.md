### 案例 1：个人用户且享有免费额度

#### 背景

个人用户 A 运营着一个网站，经常发布一些包含有图片、背景音乐、视频的文章，同时还有软件下载区等。用户 A 将网站内容存储在**广州地域**的**标准存储**服务中，并开启了 CDN 加速服务。

#### 需求

每日图片预计占用标准存储容量100GB，产生 CDN 回源流量100GB，总请求100万次。

| ---------------- | -------- | ------------- | ------------ | ---------------------- |
| 存储容量费用     | 50GB       | 0.118元/GB/月 / 30   | (100 - 50)GB          |   0.19666667元                  |
| 请求费用         | 0               | 0.01元/万次      | 100万次                 |               1元                    |
| CDN 回源流量费用 | 0         | 0.15元/GB     | 100GB                      | 15元                   |
| 日费用           | -                 | -                  | -                                      | **16.1966667元**             |

用户 A 享有的免费额度过期后，每日的费用分析如下：

#### 需求
