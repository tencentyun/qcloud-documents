## 简介

对象存储 COS 的费用组成为：[存储容量费用](#jf1)、[请求费用](#jf2)、[数据取回费用](#jf3)、[流量费用](#jf4) 和 [管理功能费用](#jf5)。下文将对各个计费项进行详细介绍。

>?
>- 费用是否产生，取决于用户如何去使用 COS 服务。除了存储容量费用、请求费用、流量费用这些基本费用外，其他费用（例如数据取回费用和管理功能费用），用户可按实际业务需求进行选择。
>- 用户在学习并了解以下计费项后，可根据自身业务需求，评估所需的资源量（例如每月存储量、请求次数、流量），然后使用 [COS 价格计算器](https://buy.cloud.tencent.com/price/cos/calculator) 进行计算，并导出预算清单。
>- 各计费项的定价，请参见 [产品定价](https://cloud.tencent.com/document/product/436/6239) 文档。

## 存储容量费用

存储容量费用的计量项为存储容量，存储容量指用户数据占用的实际存储空间。存储容量费用根据用户数据占用的存储容量大小进行计算。

根据访问频度的高低，对象存储 COS 提供四种 [存储类型](https://cloud.tencent.com/document/product/436/33417)：标准存储（多AZ）、标准存储、低频存储、归档存储，分别对应标准存储（多 AZ）容量费用、标准存储容量费用、低频存储容量费用和归档存储容量费用。用户可根据自身业务场景，将数据以不同的存储类型存储到 COS，以节省存储成本。

### 标准存储（多 AZ）容量费用

指用户选择**标准存储（多 AZ）**类型将数据上传到 COS 进行保存所产生的存储容量费用。

### 标准存储容量费用
指用户选择**标准存储**类型将数据上传到 COS 进行保存所产生的存储容量费用。


### 低频存储容量费用
指用户选择**低频存储**类型将数据上传到 COS 进行保存所产生的存储容量费用。

>?
>- 对于低频存储类型的数据，在存储时间和文件大小上有要求：存储时间不足30天，按30天计算；单个存储文件不足64KB，按64KB计算，大于或等于64KB，按实际大小计算。
>- 如果您在未开启版本控制功能的情况下，上传一份**同名**的低频或者归档存储类型对象，COS 将会在您上传成功后删除原来的同名对象，此时同样会收取**因提前删除对象**而产生的存储费用。

### 归档存储容量费用
指用户选择**归档存储**类型将数据上传到 COS 进行保存所产生的存储容量费用。

>?归档存储类型的数据仅支持存储到公有云地域（南京地域暂不支持归档），并且在存储时间和文件大小上有要求：存储时间不足90天，按90天计算；单个存储文件不足64KB，按64KB计算，大于或等于64KB，按实际大小计算。

**计费说明**
以上存储容量费用均按**月**结算，计算公式如下所示：

- 存储容量费用 = 存储容量单价 x 月存储容量
- 月存储容量 = 当月“日存储容量”之和 / 当月天数
- 日存储容量 = 当日“每5分钟存储容量”之和 / 288（采样点数）


**适用场景**
用户选择以不同的存储类型将数据上传到 COS 进行存储。

**适用的存储类型**
- 标准存储（多 AZ）容量费用适用于标准存储（多 AZ）类型。
- 标准存储容量费用适用于标准存储类型。
- 低频存储容量费用适用于低频存储类型。
- 归档存储容量费用适用于归档频存储类型。

## 请求费用

请求费用的计量项为请求次数，请求次数包括**用户请求次数**和用户配置功能后产生的**后端请求次数**。请求费用按照发送到 COS 的请求指令次数进行计算。

- 用户请求次数：指用户使用 API、SDK 或控制台等方式进行数据上传、下载、查询、删除等操作，这些操作实际上是通过对腾讯云对象存储发送请求指令来实现的。
- 后端请求次数：指生命周期沉降、归档回热到期后标准存储副本的删除，跨地域复制的读取和写入请求、清单报告投递等请求次数。


>?
>- 请求数无论请求成功或失败均计算。
>- 请求费用以**万次**为最小计数单位，因此月累计请求次数不足1万次，按1万次计算。
>- 对于归档存储类型的数据，不可读取或下载，但您可以将数据先取回（或解冻）到标准存储，再从标准存储中读取，因此请求次数统计到标准存储中。

**计费说明**
请求费用按**月**结算，计算公式为：请求费用 = 每万次请求单价 x 月累计请求次数 / 10000。

**适用场景**
用户对数据主动发起请求操作或者用户配置功能后产生了后端请求次数。


**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才可请求）。

## 数据取回费用

数据取回费用的计量项为数据取回量，数据取回量根据读取数据的大小进行计算。该项费用仅适用于冷数据存储类型，例如低频存储和归档存储。

此外，对于归档存储类型的数据取回，COS 按照不同的数据取回（解冻）时间，提供三种可选模式：极速模式（Expedited）、标准模式（Standard）和批量模式（Bulk），分别对应极速取回费用、标准取回费用和批量取回费用。

- 极速模式：时间最短，只需要1到5分钟，支持256MB以下文件快速取回。
- 标准模式：使用标准模式，一般可以在3到5小时完成数据取回。
- 批量模式：成本最低，一般在5到12小时可完成数据取回。

>?**低频存储**和**归档存储**类型为冷数据存储类型。对于低频数据来说，当您读取或下载该数据时，后台需要先取回数据后才可以读取或下载。对于归档数据来说，不可读取和下载，此时数据取回也可以称之为数据解冻（即把归档数据恢复至标准数据这一过程）。

**计费说明**
数据取回费用按**月**结算，计算公式为：数据取回费用 = 每 GB 单价 x 月数据取回量。

**适用场景**
用户读取和下载低频存储类型的数据，或者对归档存储类型的数据进行取回（解冻）。


**适用的存储类型**
低频存储、归档存储。

## 流量费用

流量费用的计量项为流量，流量是用户读取或下载数据时数据流的累计值，流量费用根据流量的累计值进行计算。

按照不同的网络环境，流量分为内网流量、外网流量、CDN 回源流量、跨地域复制流量和全球加速流量。

### 内网流量 

内网流量分为内网上行流量和内网下行流量。

- 内网上行流量：数据通过腾讯云内网从客户端传输到 COS 产生的流量。
- 内网下行流量：数据通过腾讯云内网从 COS 传输到客户端产生的流量。

>?相同地域内，腾讯云产品之间会自动使用内网访问，不产生流量费用。关于如何判断是否为腾讯云内网访问，请参见 [内网与外网访问](https://cloud.tencent.com/document/product/436/31315#.E5.86.85.E7.BD.91.E4.B8.8E.E5.A4.96.E7.BD.91.E8.AE.BF.E9.97.AE)。


**计费说明**
内网上行流量和内网下行流量都为**免费**。

**适用场景**
例如在内网环境下通过部分腾讯云产品访问 COS。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才适用）。

### 外网流量
外网流量分为外网上行流量和外网下行流量。

- 外网上行流量：数据通过互联网从客户端传输到 COS 产生的流量，该流量不收费。
- 外网下行流量：数据通过互联网从  COS 传输到客户端产生的流量。

>?
>- 对于归档存储类型的数据，数据不可读取和下载，但您可以将数据恢复（或解冻）到标准存储类型，再进行读取，因此流量统计到标准存储中。
>- 用户直接通过**对象链接**下载对象或通过**静态网站源站**浏览对象产生的流量属于外网下行流量。


**计费说明**
外网上行流量为**免费**。外网下行流量按**日**结算，计算公式为：外网下行流量费用 = 每 GB 单价 x 日累计外网下行流量。

**适用场景**
在外网环境下，通过接口访问和下载 COS 数据。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才适用）。

### CDN 回源流量
数据从 COS 传输到腾讯云 CDN 边缘节点产生的流量。            

>!
>- 如果使用第三方 CDN 回源到腾讯云 COS，将产生**外网下行流量**。
>- 开启腾讯云 CDN 加速后，通过 **腾讯云 CDN 加速域名**从客户端浏览或下载 COS 的数据产生的流量属于 CDN 回源流量。
>- 下图展示了用户在开启或未开启腾讯云 CDN 加速域名的情况下，数据从 COS 到用户这一过程所产生的流量费用。
![](https://main.qcloudimg.com/raw/2dbe3e63ad529331ffddb0af5822ceb5.png)


**计费说明**
CDN 回源流量按**日**结算，计算公式为：CDN 回源流量费用 = 每 GB 单价 x 日累计 CDN 回源流量。

**适用场景**
用户开启了腾讯云 CDN 加速域名，并使用该域名进行数据传输。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才适用）。

### 跨地域复制流量
数据从一个地域的存储桶传输到另一个地域的存储桶产生的流量。  

>?
>- 用户通过调用拷贝接口或使用跨地域复制功能从一个地域的存储桶将数据传输到另一个地域的存储桶产生的流量属于跨地域复制流量。
>- 跨地域复制流量费用按照源存储桶所属地域的单价进行计算。

**计费说明**
跨地域复制流量按**日**结算，计算公式为：跨地域复制流量费用 = 每 GB 单价 x 日累计跨地域复制流量。

**适用场景**
用户通过调用拷贝接口或使用跨地域复制功能进行数据复制。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才适用）。

### 全球加速流量
当用户开启了全球加速功能后，使用传输加速域名进行数据传输产生的流量。传输加速流量分为上行流量和下行流量。同时按照不同的**通用地域**之间的传输方向，分为**中国大陆**地域之间相互传输（不支持金融云地域）、**中国大陆**与**中国香港及海外地域**相互传输、**中国香港及海外地域**之间相互传输。

- 上行流量指用户通过传输加速域名将本地数据上传到 COS 产生的流量。
- 下行流量指用户通过传输加速域名查看或下载数据到本地所产生的流量。

**计费说明**
全球加速流量按**日**结算，计算公式为：全球加速流量费用 = 每 GB  单价 x 日累计全球加速流量。

**适用场景**
用户开启了全球加速功能，使用加速传输域名对存储桶内的数据进行加速访问。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储（需恢复才适用）。

## 管理功能费用

管理功能费用指用户开启并使用了管理功能后所产生的费用。按照不同的管理功能，对应划分有不同的管理功能费用：清单功能费用、检索功能费用、批量处理费用和对象标签费用。

### 清单功能费用
当用户开启了 [清单功能](https://cloud.tencent.com/document/product/436/33703) 后，列出存储桶对象列表时产生的费用。

**计费说明**
清单功能费用按**日**结算，按照每列出百万个对象进行收费。

**适用场景**
用户为存储桶开启并使用了该清单功能。

**适用的存储类型**
该功能不区分存储类型。

### 检索功能费用
当用户开启了 [检索功能](https://cloud.tencent.com/document/product/436/37635) 后，扫描对象时产生的费用。

**计费说明**
检索功能费用按**日**结算，按照扫描的数据大小进行计算。

**适用场景**
用户使用了检索功能对存储桶内的对象进行扫描。

**适用的存储类型**
目前仅支持标准存储。

### 批量处理费用
当用户开启了 [批量处理功能](https://cloud.tencent.com/document/product/436/38601) 后，COS 会按创建的任务数和对象处理量进行收费。

**计费说明**
批量处理费用按**日**结算，按照创建的任务数和对象处理量进行计算。

**适用场景**
用户使用了批量处理功能，对存储桶内的对象进行批量处理操作。

**适用的存储类型**
该功能不区分存储类型。

### 对象标签费用
当用户开启了 [对象标签功能](https://cloud.tencent.com/document/product/436/42993) 后，COS 会按对象标签数量进行收费。

**计费说明**
对象标签费用按**日**结算，按照设置的对象标签数量进行计算。

**适用场景**
用户使用了对象标签功能，为对象添加标签。

**适用的存储类型**
标准存储（多 AZ）、标准存储、低频存储、归档存储。
