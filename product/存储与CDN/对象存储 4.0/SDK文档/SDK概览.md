## 升级到 XML 版本 SDK

如果您之前使用的是历史版本（JSON 版本），建议您查看对应 SDK 的升级文档，升级到 XML SDK。XML 版本不仅在架构、可用性和安全性上有了非常大的提升，而且在易用性、健壮性和传输性能上也做了非常大的改进。
