## 标准存储免费额度

开通 COS 服务的用户，标准存储部分每月可享受一定量的免费存储空间、免费流量、免费请求数目。

每月计费时，会先抵扣免费额度，超出部分再进行后付费结算。

| 资源类型 | 资源子类型        | 每月免费额度 |
| ---- | :----------- | ------ |
| 存储空间 | 存储空间         | 50 GB  |
| 流量   | 外网访问流量       | 10GB   |
|      | 腾讯云 CDN 回源流量 | 10GB   |
| 请求   | 读请求          | 100 万次 |
|      | 写请求          | 10 万次  |

*注：低频存储、近线存储暂无免费额度*
