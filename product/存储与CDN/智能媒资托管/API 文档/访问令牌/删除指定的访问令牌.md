## 接口说明
删除指定访问令牌无需校验媒体库密钥，故可在客户端调用该接口。

## 请求
#### 请求示例

```plaintext
DELETE /api/v1/token/`{LibraryId}`/`{AccessToken}`
```

#### 请求参数
| 参数名称    | 描述      | 是否必选 |
| ----------- | --------- | -------- |
| LibraryId   | 媒体库 ID | 必选参数 |
| AccessToken | 访问令牌  | 必选参数 |



## 响应
#### 响应码
删除成功，返回 HTTP 204 No Content。
#### 响应格式
空，无响应体。
