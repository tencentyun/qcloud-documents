在使用智能媒资托管服务之前，需要先创建媒体库。智能媒资托管依赖对象存储服务，您可以参阅 [对象存储 - 产品概述](https://cloud.tencent.com/document/product/436/6222) 文档了解对象存储服务。

## 创建存储桶

1. 登录 [对象存储控制台](https://console.cloud.tencent.com/cos5) 创建存储桶，存储桶的所属地域建议与您的主要用户或您业务后端的部署地域接近，存储桶的访问权限设置为“私有读写”。您可以参阅 [对象存储 - 创建存储桶](https://cloud.tencent.com/document/product/436/13309) 文档进一步了解相关操作。
2. 进入新创建的存储桶，单击左侧菜单栏中的**概览**，查看概览中的基本信息，记录存储桶名称和所属地域（英文部分）。
![](https://main.qcloudimg.com/raw/202cdbf8e9e8d31fcd54f4ba5737575f.png)
3. 在左侧菜单中，选择**数据处理 > 媒体处理**，单击**编辑**，开启媒体处理功能。
媒体处理目前主要用于视频文件的封面提取，如果不开启，相关功能将无法使用。
![](https://qcloudimg.tencent-cloud.cn/raw/8e607ba876520f7608f01dd7a10f951b.png)
4. 单击**保存**。


## 创建媒体库

1. 登录 [智能媒资托管控制台](https://console.cloud.tencent.com/smh)，单击**媒体库管理**。
2. 单击**新建媒体库**。
![](https://qcloudimg.tencent-cloud.cn/raw/15521d0e9484fb973ff61059f882c914.png)
3. 在弹出的对话框中，根据实际需求进行设置。请注意，这些开关一旦设置将无法修改。
![](https://qcloudimg.tencent-cloud.cn/raw/a1b540e55fc7fc2f3429d8a3edd76e06.png)
配置说明如下：
   - **媒体库名称**：用于命名当前创建的媒体库。
   - **备注**：可对当前媒体库进行文字备注。
   - **多租户模式**：开启多租户模式，可以为您的产品和服务动态创建一些隔离的存储空间，例如独立的栏目或文章，或为产品的每个最终用户。
   - **媒体库类型**：分为文件类型、媒体类型。一旦选定创建后则不可更改。
     - 文件类型：用户可上传任意格式文件。
     - 媒体类型：用户仅可上传图片及视频格式。若您选择媒体类型，则可以选择多相簿模式。开启多相簿模式后，照片和视频等媒体资源只能上传到相簿中而不允许直接上传到租户空间中，且相簿不支持多级，即相簿中不能再创建子相簿。
   - **存储桶**：存储桶（Bucket）是对象的载体，可理解为存放对象的“容器”。用户可以通过腾讯云控制台、API、SDK 等多种方式管理存储桶以及配置属性。您可选择已创建的存储桶作为媒体库的存储源。
   - **存储类型**：可设置数据的存储类型，用于降低存储成本。
     - 标准存储：对象存储服务的基本存储格式，拥有低访问时延、高吞吐量的性能，可为用户提供高可靠性、高可用性、高性能的对象存储服务。详情可参考 [标准存储](https://cloud.tencent.com/document/product/436/6222#.E6.A0.87.E5.87.86.E5.AD.98.E5.82.A8.EF.BC.88.E5.A4.9A-az.EF.BC.89.2F.E6.A0.87.E5.87.86.E5.AD.98.E5.82.A8)。
     - 低频存储：冷数据类型，可为用户提供高可靠性、较低存储成本和较低访问时延的对象存储服务。详情可参考 [低频存储](https://cloud.tencent.com/document/product/436/6222#.E4.BD.8E.E9.A2.91.E5.AD.98.E5.82.A8.EF.BC.88.E5.A4.9A-az.EF.BC.89.2F.E4.BD.8E.E9.A2.91.E5.AD.98.E5.82.A8)。
     - 智能分层存储：可存放在标准存储层和低频存储层两个存储层，COS 可根据对象的访问频次自动在对应的两个存储层之间变换，无数据取回费用，降低用户的存储成本。详情可参考 [智能分层存储](https://cloud.tencent.com/document/product/436/6222#.E6.99.BA.E8.83.BD.E5.88.86.E5.B1.82.E5.AD.98.E5.82.A8.EF.BC.88.E5.A4.9A-az.EF.BC.89.2F.E6.99.BA.E8.83.BD.E5.88.86.E5.B1.82.E5.AD.98.E5.82.A8)。需注意，使用智能分层存储需您在 COS 对应存储桶开启智能分层存储配置。
4. 以上配置完成后，单击**确定**即可创建完成。
![](https://qcloudimg.tencent-cloud.cn/raw/ffb23fd7ea7e0ec2e889c11350f4b5e4.png)
5. 之后您可通过单击“配置”按钮对媒体库进行相关功能的管理与配置。
![](https://qcloudimg.tencent-cloud.cn/raw/6827e44ad965a8c4e59a5a9339816bb5.png)
至此，您已完成媒体库的创建，针对媒体库的其他操作可使用 API 接口来完成。目前智能媒资托管服务尚在公测中，控制台功能较为基础，如果您有其他需求，欢迎向我们 [提交工单](https://console.cloud.tencent.com/workorder/category)，我们将及时与您联系处理您的需求。
