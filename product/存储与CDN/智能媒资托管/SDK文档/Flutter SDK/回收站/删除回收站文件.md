## 删除回收站项目

#### 功能说明

用于批量删除回收站项目。

#### 示例代码

```
SMHResponse? response;
    try {
    response = await SMHAPIRecycledApis.deleteRecycleItem(
        libraryId: libraryId,
        spaceId: spaceId,
        recycledItemId: recycledItemId
    );
} catch (e) {
    print(e);
}
```

## 清空回收站

#### 功能说明

用于清空回收站。

#### 示例代码

```
SMHResponse? response;
    try {
    response = await SMHAPIRecycledApis.clearRecycle(
        libraryId: libraryId,
        spaceId: spaceId,
    );
} catch (e) {
    print(e);
}
```
