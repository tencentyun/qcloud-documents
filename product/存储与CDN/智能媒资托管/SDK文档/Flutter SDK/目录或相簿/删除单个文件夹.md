## 删除目录或相簿

#### 功能说明

用于删除目录或相簿。

#### 示例代码

```
SMHResponse? response;
    try {
    response = await SMHAPIDirectoryApis.deleteDirectory(
        libraryId: libraryId,
        spaceId: spaceId,
        dirPath:"dirPath"
    );
} catch (e) {
    print(e);
}
```
