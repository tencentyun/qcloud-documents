## 获取文件信息

#### 功能说明

用于获取文件信息。

#### 示例代码

```
SMHResponse? response;
try {
    response = await SMHAPIFileApis.getDownloadUrl(
        libraryId: libraryId,
        spaceId: spaceId,
        filePath: '00000/test.pdf',
    );
} catch (e) {
    print(e);
}
```
