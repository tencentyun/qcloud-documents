## 移动或重命名文件

#### 功能说明

用于移动或重命名文件。

#### 示例代码

```
SMHResponse? response;
    try {
        response = await SMHAPIFileApis.moveOrRenameFile(
            libraryId: libraryId,
            spaceId: spaceId,
            filePath: '00000/test_rename.pdf',
            from: '00000/test.pdf',
    );
} catch (e) {
    print(e);
}
```
