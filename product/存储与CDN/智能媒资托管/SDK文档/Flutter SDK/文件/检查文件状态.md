## 检查文件状态

#### 功能说明

用于检查文件状态。

#### 示例代码

```
SMHResponse? response;
try {
    response = await SMHAPIFileApis.headFile(
        libraryId: libraryId,
        spaceId: spaceId,
        filePath: '00000/test_rename.pdf',
    );
} catch (e) {
    print(e);
}
```
