## 批量删除回收站项目

#### 功能说明

用于批量删除回收站项目。

#### 示例代码

```
SMHResponse? response;
SMHRecycleInputItemEntity item = SMHRecycleInputItemEntity();
item.spaceId = spaceId;
item.recycledItemId = 1;
    try {
    response = await SMHUserRecycledApis.batchDeleteRecycleFile(
        organizationId: orgId,

        recycledItems: [item],
        withAllGroups: true,
    );
} catch (e) {
    print(e);
}
```
