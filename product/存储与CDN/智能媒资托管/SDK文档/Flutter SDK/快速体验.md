## 如何运行我们的 Demo
本仓库包含了 Flutter(支持 Android、iOS) 版本的腾讯云媒资托管 Flutter SDK 的 Demo，您可以下载 Demo 来进行体验。
* 使用 Android Studio 打开项目根目录运行 Android 或 iOS，
* 使用 Android Studio 打开./android 运行 Android 版本 demo。
* 使用 xcode 打开./ios 运行 iOS 版本 demo。
