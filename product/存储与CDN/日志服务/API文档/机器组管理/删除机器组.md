## 删除机器组

**描述**

删除机器组

**请求行**

```
DELETE /machinegroup
```

**请求示例**

```
DELETE /machinegroup?group_id=xxxx-xx-xx-xx-xxxxxxxx HTTP/1.1
Host: <Region>.cls.myqcloud.com
Authorization: <AuthorizationString>

```

**请求头**

除公共响应头部外，无特殊响应头部。

**请求参数**

| 字段名        |  类型  | 位置  | 是否必须 |      含义                       |
|--------------|--------|------|---------|--------------------------------|
| group_id     | string | query| 是      |要删除的机器组的 id                |


**返回示例**

```
HTTP/1.1 200 OK
Content-Length: 0

```

**响应头**

除公共响应头部外，无特殊响应头部。

**返回内容说明**

无

**错误码**

见错误码说明
