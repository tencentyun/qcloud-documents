## 操作场景

针对一些复杂的查询场景，一层 SQL 无法满足需求，通过 SQL 嵌套查询可以满足复杂的需求。

嵌套子查询和无嵌套查询的区别在于，要在 SQL 中指定 from 条件。在查询中要指定 from log 这个关键字，表示从日志中读取原始数据。


## 操作示例

```
* | select sum(pv) from 
(
select count(1) as pv from log group by status 
)
```
