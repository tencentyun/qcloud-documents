为了能更充分地使用 BRI，您需要了解一些相关的基本概念和术语。

### APK
Android应用程序包（Android Application Package，APK）是 Android 操作系统使用的一种应用程序包文件格式。

### URL
统一资源定位符（Uniform Resource Locator，URL），俗称网址。

### IMEI
国际移动设备识别码（International Mobile Equipment Identity，IMEI），即通常所说的手机序列号、手机“串号”，用于在移动电话网络中识别每一部独立的手机等移动通信设备，相当于移动电话的“身份证”。

### AppID
AppID 是腾讯云账户的账户标识之一，用于关联云资源。在用户成功申请腾讯云账户后，系统自动为用户分配一个 AppID。可通过 [腾讯云控制台](https://console.cloud.tencent.com/developer) 【账号信息】查看 AppID。

### API 密钥
API 密钥是用户访问腾讯云 API 进行身份验证时需要用到的安全凭证，由 SecretId 和 SecretKey 一起组成。一个用户账号可以创建多个云 API 密钥，若用户还没有云 API 密钥，则需要在 [云 API 密钥控制台](https://console.cloud.tencent.com/capi) 新建，否则就无法调用云 API 接口。

### SecretId
云 API 密钥的一部分，SecretId 用于标识 API 调用者身份。

### SecretKey
云 API 密钥的一部分，SecretKey 是用于加密签名字符串和服务器端验证签名字符串的密钥。
