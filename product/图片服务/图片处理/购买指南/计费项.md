图片处理 IP 计费项包括：基础图片处理费用、Guetzli 图片压缩费用、图片高级压缩费用、盲水印费用、智能封面费用。下面是各计费项的解释说明及定价详情。

## 基础图片处理费用

基础图片处理费用指用户在下载或做持久化操作时，对图片进行处理所产生的费用，具体功能请参见 [产品功能](https://cloud.tencent.com/document/product/1246/45270#.E4.BA.A7.E5.93.81.E5.8A.9F.E8.83.BD)。

- 基础图片处理费用采用按量计费方式，计费周期为**月**。
- 基础图片处理功能按照图片原文件大小进行数据处理量计算，如原图为4MB大小进行转码处理，则按照4MB进行计量。


基础图片处理定价参见下表：

| 数据处理量 | 价格       |
| ---------- | ---------- |
| ≤10TB      | 免费       |
| >10TB      | 0.025元/GB |


## Guetzli 图片压缩费用

Guetzli 图片压缩费用指对 JPG 图片进行 Guetzli 画质无损压缩而产生的费用，根据压缩次数收费，计费周期为**月**。
定价参见下表：

|功能| 价格| 
|---------|---------|
| Guetzli 图片压缩| 1元/千次| 

## 图片高级压缩费用

图片高级压缩费用指使用图片高级压缩功能而产生的费用，图片高级压缩包含 TPG 图片压缩与 HEIF 图片压缩。图片高级压缩按照**使用次数**进行计费，计费周期为**月**。定价参见下表：

| 功能           | 价格       |
| -------------- | ---------- |
| 图片高级压缩      | 0.1元/千次 |

## 盲水印费用

盲水印费用指对图片进行盲水印操作产生的费用。盲水印操作包括**添加**和**提取**两种形式，按照处理次数收费，计费周期为**月**。

| 操作类型    | 价格     |
| ----------- | -------- |
| 盲水印-添加 | 1元/千次 |
| 盲水印-提取 | 1元/千次 |

## 智能封面费用

智能封面费用指通过对视频文件内容的理解，提取最优片段生成图片产生的费用。智能封面按照**视频时长**进行计费，计费周期为**月**。

| 功能           | 价格       |
| -------------- | ---------- |
| 智能封面      | 0.15元/分钟 |
