## 功能概述

图片处理服务由腾讯云 [数据万象 CI](https://cloud.tencent.com/document/product/460/6962) 提供，数据万象通过 **imageMogr2** 接口对图片进行锐化处理。

## 接口形式

```plaintext
download_url?imageMogr2/sharpen/<value>
```
## 参数说明

操作名称：sharpen。

| 参数             | 含义                                                         |
| ---------------- | ------------------------------------------------------------ |
| download_url | 文件的访问链接，具体构成为`<BucketName-APPID>.cos.<picture region>.<domain>.com/<picture name>`，<br>例如`examplebucket-1250000000.cos.ap-shanghai.myqcloud.com/picture.jpeg`。 |
| /sharpen/&lt;value> | 图片锐化功能，value 为锐化参数值，取值范围为10 - 300间的整数。参数值越大，锐化效果越明显。（推荐使用70） |
