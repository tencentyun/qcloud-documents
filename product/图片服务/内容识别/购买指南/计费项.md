内容识别（Content Recognition，CR）计费项包括：图片标签费用、二维码识别费用和人脸智能裁剪费用。下面是各计费项的解释说明及定价详情。


## 图片标签费用

图片标签计费隶属于 [数据万象（Cloud Infinite，CI）](https://cloud.tencent.com/document/product/460) 的内容识别费用。内容识别按照**每月**鉴别次数，梯度收费，计费周期为**月**。

内容识别定价参见下表：

| 月鉴图总量    | 确定部分    | 
| ------------- | ----------- | 
| 0 - 300万       | 1.50元/千张 | 
| 300万 - 1500万  | 1.35元/千张 | 
| 1500万 - 3000万 | 1.30元/千张 |
| 3000万以上    | 1.10元/千张 | 



## 二维码识别费用

二维码识别计费隶属于 [数据万象](https://cloud.tencent.com/document/product/460) 的 二维码识别费用，指对图片进行二维码识别而产生的费用，根据识别次数收费，计费周期为**月**。定价参见下表：

| 功能 | 价格 | 
|---------|---------|
|二维码识别  | 1元/千次 | 



## 人脸智能裁剪费用
人脸智能裁剪计费隶属于 [数据万象](https://cloud.tencent.com/document/product/460) 的基础图片处理费用。基础图片处理费用采用按量计费方式，计费周期为月。基础图片处理功能按照图片原文件大小进行数据处理量计算，如原图为4MB大小进行转码处理，则按照4MB进行计量。

基础图片处理定价参见下表：

| 数据处理量 | 价格 | 
|---------|---------|
| ≤10TB| 免费 |
| >10TB| 0.025元/GB |
