您可以单击会议分享链接或者输入会议号或者扫描特邀会议二维码加入会议。
## 通过链接加入会议
1. 如果您本地已安装腾讯会议 App，当您收到的邀请信息为链接形式，则可以单击邀请链接，验证身份后即可直接进入会议。
![](https://main.qcloudimg.com/raw/da72278ac77515b9dbc6d2a05e65abd4.jpg)
2. 单击链接进入会议详情页面后，如果会议创建者创建的会议类型为预定会议，则下方会显示【添加到我的会议】，单击该项，可将该项会议添加到您的会议列表，防止您后续入会时忘记会议号。
![](https://main.qcloudimg.com/raw/7544f63a04cb10ded60b3aa0019ea620.png)

## 通过会议号加入会议
打开腾讯会议 App，在腾讯会议主面板，选择【加入会议】，输入9-12位会议号，以及您希望在会议中显示的名字（默认使用您个人资料页的昵称），并勾选相应的入会前设置项，单击【加入会议】即可成功入会。
 - 开启麦克风：勾选后，您加入会议后，会为您自动开启麦克风。
 - 开启扬声器：勾选后，您加入会议后，会为您自动开启扬声器。
 - 开启摄像头：勾选后，您加入会议后，会为您自动开启摄像头。
![](https://main.qcloudimg.com/raw/2974218869eac556241ea7a271ca8b22.png)

## 通过拨号加入会议
#### 手机一键拨号入会
您可以通过手机一键拨号，根据语音提示输入参会者 ID 入会。
1. 单击邀请信息中“手机一键拨号入会”下方的号码，并进行呼叫；
2. 根据语音提示，输入参会者 ID 按“#”号键确认，如您不知道参会者 ID，可以直接按“#”号键入会；
3. 当语音提示“已加入会议，当前 x 人在会议中”时，则代表您已入会成功。

#### 根据您的位置拨号
您也可以根据您的位置拨号，并通过语音提示输入会议号及参会者 ID 入会。
1. 根据您的位置拨号：+8675536550000（中国大陆）、+85230018898（中国香港）；
2. 输入会议号并按“#”号键确认；
3. 输入参会者 ID 按“#”号键确认，如您不知道参会者 ID，可以直接按“#”号键入会；
4. 当语音提示“已加入会议，当前 x 人在会议中”，则代表您已入会成功。
![](https://main.qcloudimg.com/raw/83438bef6fcdd81316409072f93cbd0f.jpg)



## 通过扫描特邀会议二维码加入会议
目前移动端支持扫描特邀会议二维码加入会议。
1. 打开腾讯会议 App，在腾讯会议主面板，选择右上角的扫描图标；
2. 扫描桌面端特邀会议的二维码；
3. 调起微信，选择要邀请的对象，单击【发送】即可。
![](https://main.qcloudimg.com/raw/d1c405c683bf759cde3e316874119874.jpg)
