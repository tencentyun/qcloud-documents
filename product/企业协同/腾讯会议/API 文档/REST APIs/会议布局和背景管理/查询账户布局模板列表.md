## 接口描述
**描述**：查询企业下所有的布局模板列表，目前暂不支持 OAuth2.0 鉴权访问。
**调用方式**：GET
**接口请求域名**：
```Plaintext
https://api.meeting.qq.com/v1/layout-templates
```




## 输入参数
HTTP 请求头公共参数请参见签名验证章节的 [公共参数说明](https://cloud.tencent.com/document/product/1095/42413#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0)。

无额外请求参数。



## 输出参数

| 参数名称               | 必选 | 参数类型         | 参数描述           |
| ---------------------- | ---- | ---------------- | ------------------ |
| layout_template_number | 是   | Integer          | 布局模板数量。     |
| layout_template_list   | 否   | 布局模板对象数组 | 布局模板对象列表。 |

**布局模板对象**

| 参数名称           | 必选 | 参数类型 | 参数描述                                                     |
| ------------------ | ---- | -------- | ------------------------------------------------------------ |
| layout_template_id | 是   | String   | 布局模板 ID。                                                 |
| thumbnail_url      | 是   | String   | 缩略图 URL。                                                  |
| picture_url        | 是   | String   | 布局图 URL（布局图中包含宫格 ID，该 ID 用于确定布局宫格和用户座次的映射关系，设置和修改会议布局时需要当做入参传递）。 |
| render_rule        | 是   | String   | 渲染规则（提供用户个人渲染布局图片的规则说明）。        |
