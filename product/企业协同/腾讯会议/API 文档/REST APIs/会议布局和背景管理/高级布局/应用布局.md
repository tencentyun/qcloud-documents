## 接口描述
- **描述：**
 - 将会议中的高级自定义布局应用到指定成员或者整个会议。
 - 恢复指定成员或整个会议的默认布局。
 - 目前暂不支持 OAuth2.0 鉴权访问。
 - 目前仅会应用于 H.323/SIP 终端。
- **调用方式：** PUT
- **接口请求域名：** 
```josn
https://api.meeting.qq.com/v1/meetings/{meeting_id}/applying-layout
```
