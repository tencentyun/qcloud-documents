## 接口描述
- **描述：**根据布局 ID 批量删除布局。
 - 正在被应用的布局无法删除，请先设置成其他布局或恢复成默认原始布局后再行删除。
 - 接口不做布局是否存在的校验，删除不存在的布局不会有提示。
- **调用方式：** POST
- **接口请求域名：** 
```josn
https://api.meeting.qq.com/v1/meetings/{meeting_id}/delete-advanced-layouts
```



## 输入参数
| 参数名称   | 示例                | 备注   |
| ---------- | ------------------- | ------ |
| meeting_id | 1205455724644967815 | 会议ID |
