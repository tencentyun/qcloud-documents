## 接口描述
- **描述**：会议可以通过会议室 ID 呼叫会议室邀请其入会。
- **权限：**会议创建者所在企业的管理员和会议参会者可呼叫与自己同企业下的会议室入会，若使用会议室呼叫地址，需主持人或联席主持人身份。
- **调用方式**：PUT
- **接口请求域名**：
```Plaintext
https://api.meeting.qq.com/v1/meeting-rooms/room-call
```
