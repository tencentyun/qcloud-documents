## API 调用接口方法：
>!在使用 API 之前，需要接入腾讯云智绘鉴权体系，获得 access_token，在请求头 Headers 中传入。请参见 [腾讯云智绘 API 使用方法](https://cloud.tencent.com/document/product/1351/77404) 。

- 调用地址：
```plaintext
https://api.zhihui.qq.com/bll-smart-image/atom/image-render
```
- 请求方式：POST
- 返回类型：JSON

## 请求头 Headers 说明
| 参数名称 | 示例 | 
|---------|---------|
| Authorization | Bearer (access_token string) | 

	
	
## 请求 Body 说明

| 参数名称 | 参数类型 | 是否必须 |参数描述 |
|---------|---------|---------|------ |
| 标准生成结构 | JSON | 是 |具体样式请通过 [售后在线服务](https://cloud.tencent.com/online-service?from=connect-us) 联系，以下为案例示意。<b>注意：</b>参数中包含 url 链接的输入，需要找售后在线服务添加白名单。|
