Web 应用托管提供两种创建应用的方式：**从代码仓库导入**和**从模板克隆**。

## 从代码仓库导入

Web 应用托管目前支持从以下平台导入，创建应用：

- [Github](https://github.com/)
- [Gitlab](https://gitlab.com/)
- [Gitee 码云](https://gitee.com/)

您可以在 [创建应用页面](https://console.cloud.tencent.com/webify/new)，进行 Git 平台的授权，选择一个代码仓库并导入应用。仓库后续的每次更新，都将触发自动部署。

## 从模板克隆

Web 应用托管提供一系列现有模板，您可以在 [创建应用页面](https://console.cloud.tencent.com/webify/new) 任意选择一个模板，然后使用此模板创建一个新的代码仓库。

>?您可以将变更推送至代码仓库，触发应用的自动部署。
