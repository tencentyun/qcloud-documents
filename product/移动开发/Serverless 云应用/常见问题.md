### Serverless 云应用和云开发的关系是什么？
Serverless 云应用是云开发提供的一种无服务器容器托管平台，您可不使用云开发的其他资源，单独使用 Serverless 云应用，也可集成云开发的 SDK，联合使用其他函数、数据库等资源。


### Serverless 云应用支持什么语言？
Serverless 云应用不限制语言与框架，开发者选择任意方式进行代码编写，包括但不限于 JAVA、PHP、Go。


### 是否需要改造为云函数才能使用 Serverless 云应用？
面向代码或镜像提供 Serverless 服务， 无需做云函数改造。


### 在开通 Serverless 云应用前需要开通云开发环境吗？
需要先开通云开发“按量计费”环境，如有多个环境，则每个环境可分别选择是否开通 Serverless 云应用，详情请参考文档 [开通环境](https://cloud.tencent.com/document/product/876/41391)。



### Serverless 云应用和云函数的区别是什么？

|                 |                    云函数                    | Serverless云应用                                  |
| :-------------: | :------------------------------------------: | ------------------------------------------------- |
|    请求并发     |  单实例单并发，多并发时需要拉起多个实例处理  | 单实例多并发                                      |
|    语言和框架    |            开发语言和框架支持有限            | 兼容已有框架                                      |
| Troubleshooting |                   容易定位                   | 相对灵活，依赖自定义                              |
|    常驻运行     |                    不支持                    | 可常驻运行                                        |
|    日志监控     |                   基于函数                   | 基于服务                                          |
|    版本灰度     |                支持按流量灰度                | 支持按流量灰度                                    |
|   弹性扩缩容    |                     支持                     | 支持                                              |
|    对外服务     |             提供默认 URL 和 SDK              | 提供默认 URL                                      |
|     跨平台      |         函数规则不同，很难跨平台部署         | 可跨平台部署                                      |
|    私有部署     |                    不支持                    | 可迁移私有化/混合部署                             |
|    上手难度     |                     简单                     | 中等                                              |
|    计费方式     | 按请求量计费、按请求次数和每次调用产生的 GBs | 按请求量计费、按请求次数和容器运行消耗的CPU、内存 |
