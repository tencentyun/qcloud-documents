本文档主要指导您如何购买云开发服务。

## 按量计费
1. 登录 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击【新建环境】或者直接登录 [腾讯云购买页](https://buy.cloud.tencent.com/tcb)。
2. 填写环境名称、环境 ID ，计费模式选择【按量计费】。
3. 确认订单后开通，我们会在1 - 5分钟左右为您发货。
4. 环境开通成功后，进入环境，选择静态网站托管进行开通。


## 资源包购买

1. 登录 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击【新建环境】或者直接登录 [腾讯云购买页](https://buy.cloud.tencent.com/tcb)。
2. 填写环境名称、环境 ID ，计费模式选择【按量计费】。
3. 环境开通后，单击开通后的环境，选择左侧菜单栏【环境设置】，进入环境设置页面。
4. 单击【资源包管理】-【购买资源包】，选择需要购买的资源包后单击【立即购买】。
5. 按页面提示支付成功后，我们会在1 - 5分钟左右为您发货。
