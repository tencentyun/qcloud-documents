云开发数据库提供以下几种数据类型：

- String：字符串。
- Number：数字。
- Object：对象。
- Array：数组。
- Bool：布尔值。
- GeoPoint：地理位置点。
- Date：时间。
- Null

下面对几个需要额外说明的字段做下补充说明。

### Date
Date 类型用于创建客户端时间，精确到毫秒，可以用 JavaScript 内置 Date 对象创建。如需使用服务端时间，则使用 API 中提供的 ServerDate 对象，创建一个服务端当前时间的标记。

我们的数据库有针对日期类型的优化，建议大家使用时使用 Date 或 [ServerDate](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/reference-client-api/database/db.serverDate.html) 构造时间对象。

### GeoPoint
GeoPoint 类型用于表示地理位置点，用经纬度唯一标记一个点，这是一个特殊的数据存储类型。如需对类型为地理位置的字段进行查找，必须建立地理位置索引。

具体的地理位置 API 可参考 [Geo API](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/reference-client-api/database/db.geo.html) 文档。
