## 操作场景
本文档主要指导您如何在网页端及服务端删除文件。

## 操作步骤


### 网页端（Web 端）
示例代码如下：
```javascript
tcb.init({
    env: 'xxxx'
})
tcb.deleteFile({
  fileList: ['a7xzcb'] // 文件 ID 数组
}).then((res) => {{
  console.log(res.fileList)
}}).catch(console.error);
```

### 服务端
示例代码如下：
```javascript
const app = require('tcb-admin-node');
app.init();

app.deleteFile({
    fileList: ['a7xzcb+yilTJE4NPSQQW5EYks']
}).then((res) => {
    // handle success
    console.log(res.fileList)
}).catch(console.error);
```
