## 操作场景
本文档主要指导您如何通过云存储在网页端及服务端下载文件。

## 操作步骤

### 网页端（Web 端）
由于跨域等原因，网页端暂时不支持下载文件。

### 服务端
示例代码如下：
```javascript
const app = require("tcb-admin-node");
app.init();
app.downloadFile({
    fileID: "cloud://aa-99j9f/my-photo.png",
    // tempFilePath: '/tmp/test/storage/my-photo.png'
}).then((res) => {
    // 返回临时文件路径
    console.log(res.tempFilePath)
}).catch(console.error);
```
