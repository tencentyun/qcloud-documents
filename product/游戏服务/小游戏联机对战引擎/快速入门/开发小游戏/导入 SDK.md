1. 将 SDK 文件中的 MGOBE.js 拷贝到项目的 js/libs 文件夹下。单击进入 [SDK 下载](https://cloud.tencent.com/document/product/1038/33406) 页面。
2. 参考以下示例代码，将 MGOBE.js 导入至 game.js。
```
// 导入 MGOBE.js
import "./js/libs/MGOBE.js";
// 获取 Room、Listener 对象
const { Room, Listener } = MGOBE;
```
