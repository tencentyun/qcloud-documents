### 探测器
探测器指本产品中探测规则集合。用户在使用本产品时，必须授权系统读取和分析用户的云审计数据，系统便会为用户创建专属探测规则。

### 可信 IP 集合
可信 IP 集合指已经列入白名单的 IP 地址列表，可通过集合中的 IP 地址安全地与用户的腾讯云账户进行通信。威胁发现不会根据可信 IP 列表生成结果。

### 威胁 IP 集合
威胁 IP 集合指已知的恶意 IP 地址列表，只要威胁发现监控到集合中的 IP 与用户的腾讯云账户进行通信，便会生成结果列表。

### 云审计
云审计指腾讯云的一款记录用户在腾讯云上操作行为的产品。它会记录用户在控制台和云 API 调用的操作记录。

### 威胁列表
威胁列表指本产品通过内置算法和用户自定义的安全算法，分析用户云审计数据之后，生成的潜在威胁报告列表。
