## 监控数据存储时长

目前云监控数据的最长保留时间为半年，只支持半年内的监控数据查询。

| 监控粒度     | 存储时长 |
| ------------ | -------- |
| 秒级         | 1天      |
| 1分钟、5分钟 | 31天     |
| 1小时        | 93天     |
| 1天          | 186天    |

## 权限控制

任何具有腾讯云账号的用户均可以使用云监控服务。
云监控有独立的权限控制，主账号可以查看所有服务的监控以及告警信息、子账号默认没有任何权限，如需了解更多权限信息请参见 [访问管理](https://cloud.tencent.com/document/product/248/45428)。
