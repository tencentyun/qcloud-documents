## 各维度对应参数总览

| 参数名称                       | 维度名称    | 维度解释                     | 格式                                  |
| ------------------------------ | ----------- | ---------------------------- | ------------------------------------- |
| Instances.N.Dimensions.0.Name  | InstanceId | 轻量应用服务器 ID 的维度名称 | 输入 String 类型维度名称：InstanceId |
| Instances.N.Dimensions.0.Value | InstanceId| 轻量应用服务器实例的具体 ID  | 输入具体实例 ID，例如：lhins-rusd2345、lhins-2zymro91 |



## 入参说明

**查询云服务器监控数据，入参取值如下：**
&Namespace = QCE/LIGHTHOUSE
&Instances.N.Dimensions.0.Name = InstanceId
&Instances.N.Dimensions.0.Value = 轻量应用服务器实例的具体 ID
