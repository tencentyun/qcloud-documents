Tencent Cloud Monitor provides the following monitoring metrics for direct connect gateway:

| Indicator Name | Description         | Unit   | Dimension     |
| ----- | ------------ | ---- | ------ |
| Network inbound bandwidth | Inbound traffic per second of direct connect gateway | Mbps | Direct connect gateway |
| Network outbound bandwidth | Outbound traffic per second of direct connect gateway | Mbps | Direct connect gateway |
| Outbound packets   | Outbound packets per second of direct connect gateway | pck/sec  | Direct connect gateway |
| Inbound packets   | Inbound packets per second of direct connect gateway | pck/sec  | Direct connect gateway |

For more information about how to use the monitoring metrics of direct connect gateway, please see the API [Read Monitoring Data from Message Queue](https://cloud.tencent.com/document/product/248/11013) in the Cloud Monitor API.
