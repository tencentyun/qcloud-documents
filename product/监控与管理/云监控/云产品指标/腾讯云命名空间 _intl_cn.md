命名空间是专属于某产品的指标容器，可以参考[云监控基本概念](/doc/product/248/968)的更多内容。腾讯云提供以下命名空间，用户在使用 API 时需要指定需要的值：

|产品类型|	命名空间|
|--|--|
|云服务器	|qce/cvm|
|物理专线	|qce/dc_line|
|专线通道	|qce/dc_channel|
|专线网关	|qce/vpg|
|负载均衡	|qce/lb|
|CDN	|qce/cdn|
|云数据库My SQL	|qce/cdb|
|云数据库TD SQL	|qce/tdsql|
|云数据库SQL Server	|qce/sqlserver|
|云存储Redis	|qce/redis|

有关更多如何使用云监控命名空间的内容，可以查看云监控 API 中的[读取监控数据接口](https://cloud.tencent.com/doc/api/405/4667)。
