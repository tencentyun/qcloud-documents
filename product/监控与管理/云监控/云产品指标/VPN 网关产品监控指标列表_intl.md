Tencent Cloud Monitor provides the following monitoring indicators for VPN gateway:

| Indicator's Name | Description | Unit | Dimension |
| ----- | ------------ | ---- | ------ |
| Public network outbound bandwidth | Outbound traffic per second of VPN gateway | Mbps | VPN gateway |
| Public network inbound bandwidth | Outbound traffic per second of VPN gateway | Mbps | VPN gateway |
| Outbound packets | Outbound traffic per second of VPN gateway | PCS/s | VPN gateway |
| Inbound packets | Inbound traffic per second of VPN gateway | PCS/s | VPN gateway |

For more information about how to use the monitoring indicators of VPN gateway, please see [Read Monitoring Data API from the Message Queue](https://cloud.tencent.com/document/product/248/11013) in the Cloud Monitor API.
