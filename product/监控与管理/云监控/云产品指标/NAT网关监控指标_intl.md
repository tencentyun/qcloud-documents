Tencent Cloud Monitor provides the following monitoring indicators for NAT gateway:

| Indicator's Name | Description | Unit | Dimension |
| ----- | ------------ | ---- | ----- |
| Public network outbound bandwidth | Outbound traffic per second of NAT gateway | Mbps | NAT gateway |
| Public network inbound bandwidth | Outbound traffic per second of NAT gateway | Mbps | NAT gateway |
| Outbound packets | Outbound traffic per second of NAT gateway | PCS/s | NAT gateway |
| Inbound packets | Inbound traffic per second of NAT gateway | PCS/s | NAT gateway |
| Number of connections | Number of real-time concurrent connections of NAT gateway | PCS/s | Number of NAT gateway |

For more information about how to use the monitoring indicators of NAT gateway, please see [Read Monitoring Data API from the Message Queue](https://cloud.tencent.com/document/product/248/11013) in the Cloud Monitor API.
