

本文将为您介绍如何复制 Dashboard。

## 操作步骤

1. 登录 [云监控控制台](https://console.cloud.tencent.com/monitor)。
2. 在左侧导航栏中单击 **Dashboard 列表**，进入 Dashboard 列表页。
3. 在 Dashboard 列表中找到您所需要复制的 Dashboard，单击**复制**
4. 在弹框中重新定义 Dashboard 名称，单击**保存**即可。
