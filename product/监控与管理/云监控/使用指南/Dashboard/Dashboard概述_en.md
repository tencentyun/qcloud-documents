Dashboard provides a custom monitoring dashboard service for users to customize key metrics across products and instances to centrally display resources of one business or cluster.

Dashboard provides flexible custom view features applicable to multiple monitoring scenarios, such as cross-instance data aggregation, real-time/historical data display, similar metric comparison, and linkage charts.

### Application Scenarios

- Subscribing key metric-based monitoring charts across products and instances for an overall display
- Aggregation, display and monitoring of mass instance metric data
- Key metric comparison of various instances for troubleshooting
- Flexible adjustment of chart order, display area, refresh frequency and granularity based on custom view needs
