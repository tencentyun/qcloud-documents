Tencent Cloud Monitor provides the following monitoring metrics for physical direct connect:

| Indicator Name | Description        | Unit   | Dimension   |
| ----- | ----------- | ---- | ---- |
| Network outbound bandwidth | Outbound traffic per second of physical direct connect | Mbps | Physical direct connect |
| Network inbound bandwidth | Inbound traffic per second of physical direct connect | Mbps | Physical direct connect |

For more information about how to use the monitoring metrics of physical direct connect, please see the API [Read Monitoring Data from Message Queue](https://cloud.tencent.com/document/product/248/11013) in the Cloud Monitor API.
