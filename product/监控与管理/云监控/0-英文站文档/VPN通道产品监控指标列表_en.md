Tencent Cloud Monitor provides the following monitoring indicators for VPN channel:

| Indicator Name | Description | Unit | Dimension |
| ----- | ------------ | ---- | ----- |
| Network outbound bandwidth | Outbound traffic per second of VPN channel | Mbps | VPN channel |
| Network inbound bandwidth | Outbound traffic per second of VPN channel | Mbps | VPN channel |
| Outbound packets | Outbound traffic per second of VPN channel | PCS/s | VPN channel |
| Inbound packets | Inbound traffic per second of VPN channel | PCS/s | VPN channel |
| Packet loss rate | Packet loss rate monitored by VPN in one minute | % | VPN channel |
| Latency | Average latency monitored by VPN in one minute | ms | VPN channel |
| Disconnected channel | Real-time connection status of VPN channel | 0/1 | VPN channel |

For more information about how to use the monitoring indicators of VPN channel, please see [Read Monitoring Data API from the Message Queue](https://cloud.tencent.com/document/product/248/11013) in the Cloud Monitor API.
