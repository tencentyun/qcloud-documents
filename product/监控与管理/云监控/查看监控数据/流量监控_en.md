Cloud Monitor allows you to monitor the usage of public network and CDN.

You can check the real-time usage and the usage for the last 7 days, 15 days and 30 days, or any day.

* Click the corresponding URL, you can export the usage report of outbound bandwidth in public network for different IP for the last three months. 
* Click the corresponding URL, you can export the usage report of total outbound bandwidth in public network based on the users' dimensions for the last six months.
