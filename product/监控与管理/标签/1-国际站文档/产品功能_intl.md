This product is mainly used to query resources and create tags.
## Querying Resources
You can query and tag your existing resources on Tencent Cloud by the region, type and instance. The found resources are displayed in a list. You can also specify a tag key for these resources according to your needs for classification and management.
## Creating a Tag
Creating a tag is equivalent to creating a database locally. It stores all of your resource tag information in Tencent Cloud and helps you manage your resources in the future. Creating a tag includes creating a tag key and a tag value.
