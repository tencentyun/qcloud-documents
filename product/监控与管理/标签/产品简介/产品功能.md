您可以通过标签控制台从标签和资源两个维度进行可视化管理，也可以通过标签 API 对资源标签进行管理。


## 标签控制台 - 资源标签
在 [资源标签](https://console.cloud.tencent.com/tag) 页面您可以查看每个资源的标签绑定情况，也可以灵活的为资源绑定、解绑标签。具体如下：
- 支持地域、资源类型、标签、所属项目组合查询资源。
- 支持多标签键值模糊查询。
- 支持查看资源的标签总数。
- 支持资源批量编辑。
- 支持资源数据自定义导出。


## 标签控制台 - 标签列表
在 [标签列表](https://console.cloud.tencent.com/tag/taglist) 页面您可以查看每个标签的资源绑定情况，也可以灵活的为标签绑定、解绑资源。具体如下：
- 支持批量创建、删除标签。
- 支持查看标签下的资源数量。
- 支持查看标签绑定的资源。
- 支持为标签绑定、解绑资源。


## 标签 API
除了控制台可视化操作外，您还可以调用 [标签 API](https://cloud.tencent.com/document/product/651/35307) 对标签进行增删改查操作，以及对资源标签进行绑定、解绑等。
- 通过 [API 概览](https://cloud.tencent.com/document/product/651/35307) 查找标签提供的 API，并在您的程序、应用中使用。
- 通过 API Explorer 在线调用标签提供的 API，可在线生成多语言 SDK。
