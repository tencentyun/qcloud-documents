 
You can work with the following two lists on the tag console.

## Resource List  

### Features
-  Modify resource tags in batches.
-  Add/Delete key values in batches.



### List display details  
You can see all the tag key-value pairs you added in the list.

## Tag Key List  

In the tag key list, you can view, add, or delete tag keys and names. At most three tag names display in the list. To view other tag names, click **More**.
