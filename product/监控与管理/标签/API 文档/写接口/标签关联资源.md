
## 接口描述
标签关联资源 AddResourceTag 用于对腾讯云上资源进行关联操作。

请求域名：`tag.api.qcloud.com `

## 输入参数
以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](https://cloud.tencent.com/document/product/651/18354)。

|参数名称|必选|类型|描述|
|---------|---------|---------|--------|
|tagKey|	是|	String	|标签键|
|tagValue|	是|	String	|标签值|
|resource|	是|	String	|[资源六段式标记](https://cloud.tencent.com/document/product/598/10606)|

## 输出参数

无

## 错误码

| 错误码 | 描述 |
|---------|---------|
|30001|	标签键不存在|
|30002|	系统预留标签键  `qcloud:`  和 `tencent:` 禁止创建|
|30004|	主账号 Uin 不存在|
|30016|	创建者 Uin 不存在|
|30003|	资源六段式标记参数错误|
|30010|	用户创建标签键达到上限数1000|
|30020|	单个标签键对应标签值达到上限数1000|
|30005|	用户资源标签已存在不能重复添加|

## 示例
### 输入示例

```
https://domain/v2/index.php?Action=AddResourceTag&tagKey=t1&tagValue=v1&resource=qcs::cvm:ap-beijing:uin/1234567:instance/ins-123&<公共请求参数>
```
### 输出示例

```
{
    "code": 0,
    "message": "",
    "codeDesc": "Success",
    "data": []
}
```
