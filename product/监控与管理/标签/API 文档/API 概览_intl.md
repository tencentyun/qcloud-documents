## Write APIs

| API Name | Feature | 
|---------|---------|
| [CreateTag](https://cloud.tencent.com/document/product/651/13563) | Creates tags | 
| [DeleteTag](https://cloud.tencent.com/document/product/651/13568) | Deletes tags | 
| [UpdateResourceTagValue](https://cloud.tencent.com/document/product/651/13564) | Modifies tag resource values | 
| [ModifyResourceTags](https://cloud.tencent.com/document/product/651/13565) | Works with all resource tags in batches | 
| [AddResourceTag](https://cloud.tencent.com/document/product/651/13566) | Associates tags with resources | 
| [DeleteResourceTag](https://cloud.tencent.com/document/product/651/13567) | Disassociates resources from tags |
