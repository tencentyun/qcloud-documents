## 接口描述
 查询标签键 GetTagKeys 用于查询已建立的标签列表中的标签键。

 请求域名：`tag.api.qcloud.com`

## 输入参数
以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](https://cloud.tencent.com/document/product/651/18354)。

|参数名称|必选|类型|描述|
|---------|----------|---------|--------|
|createUin|	否	|Int|	创建者用户 Uin，不传或为空只将 Uin 作为条件查询|
|page|	否	|	Int	|页码，默认为 1|
|rp|	否	|	Int	|每页大小，默认为 15|


## 输出参数

| 参数名称 | 类型 | 描述 |
|---------|----------|---------|
| total | Int | 结果总数|
| page | Int| 当前页码 |
| rp | Int | 每页大小|
| rows | Array | 标签列表 |

其中 rows 数组的参数如下：

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
|tagKey|String	|标签键|


## 错误码

| 错误码 | 描述 |
|---------|---------|
|30004|	主账号 Uin 不存在|

## 示例
### 输入示例

```
https://domain/v2/index.php?Action=GetTagKeys&<公共请求参数>
```
### 输出示例

```
{
    "total": 1,
    "page": "1",
    "rp": "15",
    "rows": [
        {
            "tagKey": "testTagKey"
        }
    ]
}
```
