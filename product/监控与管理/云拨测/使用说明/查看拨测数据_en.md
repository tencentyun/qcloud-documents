In the "Automated Testing Task List", click the corresponding task name to view the automated testing data of this task.
The data that can be viewed is: availability data and request latency data.

## 1. Availability
Availability ratio calculation method: (Number of automated testing - Number of failures)/Number of automated testing x 100%.
Overall availability ratio: The average value of the availability ratios of all automated testing points.
![](//mccdn.qcloud.com/img5694f5d64354c.png)

## 2. Latency
Automated testing latency: The time period from the automated testing point initiating an automated testing request to completely receiving the data returned by server.
