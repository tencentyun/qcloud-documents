The registered users of Tencent Cloud are free to use the Cloud Automated Testing. For free users, they can use the following features:

1) Fourteen automated testing points can be chosen from, covering first-tier cities
2) Up to 10 automated testing tasks can be configured
3) Up to 5 automated testing points can be configured for each task
4) The highest frequency for each automated testing task is once every 5 minutes
5) Support all basic automated testing functions
6) Support free SMS alarms
