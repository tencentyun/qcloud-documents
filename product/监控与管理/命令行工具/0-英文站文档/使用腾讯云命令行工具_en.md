See the following steps for how to use Tencent Cloud CLI.

### Installing Tencent Cloud CLI
Tencent CLI can be installed and used on Windows, Mac OS and Linux/Unix to meet the requirements of different developers. Its commands can be automatically completed under Linux/Unix environment.

For more information, see:

[Install Tencent CLI on Linux/Mac OS](https://cloud.tencent.com/doc/product/440/6182)

[Install Tencent Cloud CLI on Windows](https://cloud.tencent.com/doc/product/440/6183)

### Configuring CLI
You can set multiple accounts for Tencent Cloud CLI and rapidly switch between accounts. For more configurations, please see:

[Configure Tencent Cloud CLI](https://cloud.tencent.com/doc/product/440/6184)

### Using CLI
Tencent Cloud CLI can be used for rapid API debugging, non-graphical interface operation, operation integration, and secondary development. For more information about how to use it, please see:

[Get Started with Tencent Cloud CLI](https://cloud.tencent.com/doc/product/440/6185)
