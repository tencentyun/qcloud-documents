您可以输入如下内容查看tccli的帮助信息
>查看支持的产品。您也可以在 [API中心](https://cloud.tencent.com/document/api) 的文档中查看。
```bash
tccli help
```
>以 CVM 为例，查看产品支持的接口。
```bash
tccli cvm help
```
>以 CBS 中的 DescribeDisks 接口为例，查看接口支持的参数。具体的参数说明和 API 的相关信息可以在腾讯云官网查看对应的 API 文档。
```bash
tccli cbs DescribeDisks help
```


TCCLI默认显示简化版帮助信息，如果想查看详细信息，可以使用在help后面加上 `--detail` 选项，如：
>查看支持的产品的详细信息。
```bash
tccli help --detail
```
>以 CVM 为例，查看产品支持的接口的详细信息。
```bash
tccli cvm help --detail
```
>以 CBS 中的 DescribeDisks 接口为例，查看接口的入参、出参的详细信息，以及使用示例。
```bash
tccli cbs DescribeDisks help --detail
```
