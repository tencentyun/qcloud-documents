## 1. Error Code 10060

```Socket error 10060 - Connection timed out ```

10060 is returned via Socket programming API, which means network connection timeout. This error occurs when your server cannot be connected to Tencen Cloud API server. Check the followings for troubleshooting:

* Whether a proxy is required when you access the public network locally
* Whether a rule used to limit external access is specified for the firewall of local server.
* Whether a rule used to limit external access is specified for the router.
