### 资源池（管理）和空闲机池的区别？
资源池（管理）：机器导入（或从腾讯云项目同步过来）后，机器不属于任何一个业务，会被默认放到资源池管理下，资源池管理里面的机器可以分配到任意开发商下的业务空闲机池：属于某一个业务，但还没有分配到具体的集群或者模块下的机器。

### 为什么外网 IP 导入后不显示？
外网 IP 属于用户自定义字段，因此需要映射到 [配置平台](http://o.qcloud.com/console?app=cc-new) 支持的字段：

### 已经购买了腾讯云服务器，但是在配置平台中同步不过来怎么处理？
已经购买了腾讯云服务器，但是在配置平台中同步不过来，提示“也许您还没来得及在腾讯云上购买主机，去购买”。如下图所示：
![](https://main.qcloudimg.com/raw/38a3914e7e208c5f6123bfba49fab0c3.png)
蓝鲸的配置平台导入腾讯云服务器前，需要用户在【云 API 密钥】- 【API 密钥管理】创建 secretId 和 key 才能获取到腾讯云的项目列表。
![](https://main.qcloudimg.com/raw/14961e5abe173d17ef4767a798405615.png)
