## 上报接口说明

### 1. 接口描述

| 接口描述             | 接口说明                                                     |
| -------------------- | ------------------------------------------------------------ |
| 接口请求域名         | monitor.tencentcloudapi.com                                  |
| 默认接口请求频率限制 | 50次/秒                                                      |
| 上报指标限制         | 单次上报最多 30 个指标/值对，请求返回错误时，请求中所有的指标/值均不会被保存。 |
| 默认指标总数         | 100个/租户                                                   |
| 上报的时间戳         | 期望保存的时间戳，建议构造整数分钟时刻的时间戳。 时间戳时间范围必须为当前时间到 300 秒前之间。 同一 IP 指标对的数 据需按分钟先后顺序上报。 |

### 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](https://cloud.tencent.com/document/product/628/18814)。

| 参数名称          | 必选 | 类型                 | 描述                                                         |
| ----------------- | ---- | -------------------- | :----------------------------------------------------------- |
| Action            | 是   | String               | 公共参数，本接口取值：PutMonitorData                         |
| Version           | 是   | String               | 公共参数，本接口取值：2018-07-24                             |
| Region            | 是   | String               | 公共参数，腾讯云设备报的地域信息(可以通过http://metadata.tencentyun.com/latest/meta-data/placement/region获取)，查询可用地域列表：https://cloud.tencent.com/document/api/213/15692#.E5.9C.B0.E5.9F.9F.E5.88.97.E8.A1.A8 |
| Metrics.N         | 是   | Array of MetricDatum | 一组指标和数据                                               |
| AnnounceInstance  | 否   | String               | 上报实例名或IP(IP通过http://metadata.tencentyun.com/latest/meta-data/local-ipv4获取)，最大长度128字节 |
| AnnounceTimestamp | 否   | Integer              | 上报时自行指定的时间戳，例如 1572438660                      |

其中，MetricDatum参数说明如下：

| 参数名称   | 必选 | 类型    | 描述                                                 |
| ---------- | ---- | ------- | ---------------------------------------------------- |
| MetricName | 是   | String  | 指标名称,最大长度255字节, 只允许A-Za-z0-9\-_字符匹配 |
| Value      | 是   | Integer | 指标值，整数，不得超过32字节，超过32字节会被截断     |

### 3. 输出参数

| 参数名称  | 类型   | 描述参数名称                                                 |
| --------- | ------ | ------------------------------------------------------------ |
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId |
