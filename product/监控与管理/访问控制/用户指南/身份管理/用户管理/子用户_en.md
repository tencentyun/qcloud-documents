## Sub-User Overview
Sub-users are entities created by the root account. A sub-user is assigned an ID and credentials, therefore it can log in to console and configure independently. A sub-user also has the access to APIs.

## Sub-User Management

### Create a sub-user
1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam), click "Create User", and then select "Sub-user", as shown below:
![](https://mc.qcloudimg.com/static/img/021cb0d60e0e519e831774aa94bc6f0f/1.png)

3. Fill in the user information. During this process, you can create sub-users in batch, set access type and console password or perform other operations, as shown below:
![](https://mc.qcloudimg.com/static/img/08e4a1d45664ed62a08c34c47ea4f75f/2.png)

4. Set permissions.
(1) Add the sub-user to an existing user group or create a new user group (required), as shown below:
![](https://mc.qcloudimg.com/static/img/ddabc0c473c34e613d34c89d182acd52/3.png)
(2) Copy the permissions and associated policy (which describes permissions) of the existing user for the sub-user. After being associated with the policy, the sub-user obtains the permissions described by the policy (optional).

5. You can email the complete information to the mailbox, or save some of the information to a local Excel file, as shown below:
![](https://mc.qcloudimg.com/static/img/49c58132ecb4df058af6d7f5c25ff9ae/6.png)

6. Click "Complete".


### Delete a sub-user

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam) to locate the sub-user you want to delete, then click "Delete" in the "Operation" column, as shown below:
![](https://mc.qcloudimg.com/static/img/6ffb754fb8180bfa59a3142e38845599/8.png)

2. Click "Delete" to delete the sub-user.
