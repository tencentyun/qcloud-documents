## 操作场景
本文档介绍如何删除单个或者多个子用户，删除之后，子用户将不再拥有该主账号的管理权限。
## 前提条件
已登录访问管理控制台，进入 [用户列表](https://console.cloud.tencent.com/cam) 管理页面。
## 操作步骤

#### 删除单个子用户
1. 在用户列表管理页面，找到需要删除的子用户。
2. 单击右侧操作列的【更多】>【删除】。
3. 在弹出的删除用户窗口，确认当前子用户下的 API 密钥已禁用且删除，详细请参考 [访问密钥](https://cloud.tencent.com/document/product/598/37140#.E5.88.A0.E9.99.A4.E5.AD.90.E8.B4.A6.E5.8F.B7-api-.E5.AF.86.E9.92.A5)。
4. 单击【确认删除】，完成删除单个子用户操作。

#### 删除多个子用户
1. 在用户列表管理页面，左侧勾选需删除的子用户。
2. 单击左上方的【更多操作】>【删除】。
3. 在弹出的删除用户窗口，确认已勾选子用户下的 API 密钥已禁用且删除，详细请参考 [访问密钥](https://cloud.tencent.com/document/product/598/37140#.E5.88.A0.E9.99.A4.E5.AD.90.E8.B4.A6.E5.8F.B7-api-.E5.AF.86.E9.92.A5)。
4. 单击【确认删除】，完成删除多个子用户操作。
