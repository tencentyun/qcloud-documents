## Collaborator Overview
A collaborator can log in to Tencent Cloud to manage cloud resources or receive message notifications.

## Collaborator Management

### Create a collaborator

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam), click "Create User", and then select "Collaborator", as shown below:
![](https://mc.qcloudimg.com/static/img/0ac627689fbd652ae4ee694164857f61/1.png)

2. Fill in the user information, as shown below:
![](https://mc.qcloudimg.com/static/img/02946deaa136594ad4aee17bf9985778/image.png)

3. Set permissions for the collaborator, as shown below:
(1) Add the collaborator to an existing user group or create a new user group (required), as shown below:
![](https://mc.qcloudimg.com/static/img/ddabc0c473c34e613d34c89d182acd52/3.png)
(2) Copy the permissions and associated policy (which describes permissions) of the existing user for the collaborator. After being associated with the policy, the collaborator obtains the permissions described by the policy (optional).

4. Click "Complete".

### Subscribe to messages for a collaborator
1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam) to locate the collaborator for whom you want to subscribe to messages, and then click "Subscribe to Messages" in the "Operation" column, as shown below:
![]( https://mc.qcloudimg.com/static/img/10065d2765f82b63faa044168fffe408/1.png)

2. Select the message type you want to subscribe to and click "OK".

### Delete a collaborator

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam) to locate the collaborator you want to delete, then click "Delete" in the "Operation" column, as shown below:
![](https://mc.qcloudimg.com/static/img/253a9e3a11eb252a123dfb73f712cc9d/2.png)

2. Click "Delete" to delete the collaborator.
