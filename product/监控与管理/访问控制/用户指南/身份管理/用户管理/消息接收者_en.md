## Message Receiver Overview
A message receiver can only log in to Tencent Cloud to receive message notifications.

## Message Receiver Management

### Create a message receiver

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam), click "Create User" and then select "Message Receiver", as shown below:
![](https://mc.qcloudimg.com/static/img/5a68d6324df8a8e7b8bbfd742c02ee8c/2.png)

2. Fill in the user information, as shown below:
![](https://mc.qcloudimg.com/static/img/d178db04af592d2b69281053fd5dfcee/image.png)

3. Click "Complete".

### Subscribe to messages for message receiver

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam) to locate the message receiver for whom you want to subscribe to messages, and then click "Subscribe to Messages" in the "Operation" column, as shown below:
![]( https://mc.qcloudimg.com/static/img/53debfc51ecc729a94a48b496ef18ff0/3.png)

2. Select the message type you want to subscribe to and click "OK".

### Delete a message receiver

1. Log in to the Tencent Cloud Console, go to [User Management](https://console.cloud.tencent.com/cam) to locate the message receiver you want to delete, then click "Delete" in the "Operation" column, as shown below:
![](https://mc.qcloudimg.com/static/img/6ee0615caeaccd947e57fac7d76d9746/4.png)

2. Click "Delete" to delete the message receiver.
