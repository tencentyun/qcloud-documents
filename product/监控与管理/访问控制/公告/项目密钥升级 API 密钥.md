## 升级说明
为了给您提供更完善的服务，腾讯云计划对**项目密钥**进行升级优化，升级完成后，项目密钥将下线（下线时间为2022年12月31日），由 **API 密钥替代项目密钥**。由于项目密钥仅支持访问对象存储和数据万象（原万象优图），且项目密钥的安全性没有 API 密钥高，目前**已经不再支持新建项目密钥**，同时也不再推荐使用项目密钥进行资源访问，请您尽快将仍在使用中的项目密钥升级为 API 密钥。

 
## API 密钥介绍
API 密钥即访问密钥，是用户访问腾讯云 API 进行身份验证时需要用到的安全凭证，由 SecretId 和 SecretKey 一起组成。API 密钥分为持久密钥和临时密钥，其中：
- **持久密钥**：主账号和子账号所拥有的密钥。一个账号最多可以创建2个 API 密钥，主账号的 API 密钥代表您的主账号身份并拥有主账号的全部权限；子账号的 API 密钥代表所属子账号的身份并拥有该子账号关联策略所获得的权限。
- **临时密钥**：为角色临时申请的密钥。临时密钥代表一个临时身份并拥有其关联策略所获得的权限，默认会话保持时间为2小时，可设置10分钟 - 12小时。

 

## 安全使用建议
- 尽量使用临时密钥访问云资源，并按权限最小化原则授权。
- 必须使用持久密钥访问云资源时，建议使用子账号持久密钥而非主账号持久密钥。
- 持久密钥集中管理，并进行定期轮换。

 

## 升级操作指引

### 步骤1：创建 API 密钥
1. 创建一个子用户，具体创建指引参见 [新建子用户](https://cloud.tencent.com/document/product/598/13674)。
2. 在子用户详情页面，单击 **API 密钥**，进入 API 密钥管理页面。
3. 在 API 密钥管理页面，单击**新建 API 密钥**，完成创建 API 密钥操作
![](https://qcloudimg.tencent-cloud.cn/raw/57d1292ce65ab79539e85a7902048cf6.png)


### 步骤2：替换 API 密钥

1. 为上述创建的子用户授予与项目密钥相同或更大的权限。
2. 在所有使用了项目密钥的业务程序中，替换为新的 API 密钥。
>!如果您正在使用项目密钥调用 V2 版本（[GetFederationToken](https://cloud.tencent.com/document/product/598/13896)）接口生成临时密钥，请升级后使用 API 密钥调用 V3 版本（[GetFederationToken](https://cloud.tencent.com/document/product/1312/48195)）接口生成临时密钥。

 
### 步骤3：禁用项目密钥

1. 在访问管理控制台，单击左侧菜单栏的 [项目密钥管理](https://console.cloud.tencent.com//cam/capi/project)，进入项目密钥管理页面。
2. 在要禁用的项目密钥操作栏，单击**禁用**即可禁用该项目密钥。
![](https://qcloudimg.tencent-cloud.cn/raw/fb27a61f97ec91f0c4c3c0ef759f59cf.png)


### 步骤4：删除项目密钥
>!项目密钥删除后不可恢复。
>
1. 禁用项目密钥后，持续监控业务是否受影响，若有影响可重新启用密钥。
2. 若无影响，可在项目密钥管理页面，单击操作栏的**删除**即可删除项目密钥。
![](https://qcloudimg.tencent-cloud.cn/raw/259d22033b7a5764d6562163f3c73fe8.png)
