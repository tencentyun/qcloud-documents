
企业帐号 CompanyExample 下有一个子账号 Developer，该子账号需要拥有对企业帐号 CompanyExample 名下的所有资源都有完全访问权限。

方案A：

企业帐号 CompanyExample 直接将预设策略 AdministratorAccess 授权给子账号 Developer。授权方式请参考 [授权管理](https://cloud.tencent.com/document/product/378/8961)。

方案B：

步骤1：通过策略语法方式创建以下策略。
```
{
    "version": "2.0",
    "statement": [
        {
            "effect": "allow",
            "action": "*",
            "resource": "*"
        }
    ]
}
```
步骤2：将该策略授权给子账号。授权方式请参考 [授权管理](https://cloud.tencent.com/document/product/378/8961)。
