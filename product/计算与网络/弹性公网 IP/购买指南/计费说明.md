弹性公网 IP（EIP）的费用根据传统账户类型和标准账户类型这两种不同类型的账户收费不同。本文介绍不同类型账户的 EIP 计费详情。

## 背景信息

- **账户类型介绍**
目前腾讯云账户分为标准账户类型和传统账户类型，2020年6月17日零点后注册的腾讯云账户均为标准账户类型，该时间点前注册的账户请在控制台查看您的账户类型，具体操作请参见以下的 [账户类型区分](#judge)。
二者区别如下：
 - 传统账户类型（原“带宽非上移账户”）：在 CVM 上管理带宽/流量。传统账户类型的 IP 和 CLB 没有网络带宽或流量属性，需要在 CVM 上购买和管理。
 - 标准账户类型（原“带宽上移账户”）：在 IP 和 CLB 上管理带宽/流量。标准账户类型购买的 CVM 不再保留外网带宽或流量资源，而公网 CLB/IP 端具备外网带宽或流量资源。
- <span id="judge" />**账户类型区分**：登录 <a href="https://console.cloud.tencent.com/cvm/eip">EIP 控制台</a>，在“弹性公网IP”实例列表页面顶部，查看是否有提示信息。
 - 若无提示信息，则说明您的账户类型为标准账户类型。
 - 若有以下提示信息，则说明您的账户类型为传统账户类型。
![](https://main.qcloudimg.com/raw/c94611f1e508d484c40789010409c478.png)

## 费用组成

EIP 的费用由 **IP 资源费用**和**公网网络费用**组成。传统账户类型和标准账户类型的账户收费介绍如下：

### 传统账户类型

传统账户类型仅涉及 IP 资源费用，公网网络费用在 CVM 实例上收取。

- 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费。
- 当 EIP 绑定云资源时：EIP 自身不收取任何费用，仅在 CVM 实例上收取 <a href="https://cloud.tencent.com/document/product/213/10578">公网网络费用</a>。
  ![](https://main.qcloudimg.com/raw/2f1174acba496fb43e1a82c959c58683.png)

### 标准账户类型

标准账户类型有四种计费模式，费用情况如下：

- 按流量：收取公网网络费用和 IP 资源费用。
 - 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费，不收取公网网络费用。
 - 当 EIP 绑定云资源时：仅收取 <a href="#net">公网网络费用</a>。
- 包月带宽：无论是否绑定云资源，仅收取 <a href="#net">公网网络费用</a>。
- 按小时带宽：无论是否绑定云资源，仅收取 <a href="#net">公网网络费用</a>。
- 共享带宽包：收取公网网络费用和 IP 资源费用。
 - 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费，不收取公网网络费用。
 - 当 EIP 绑定云资源时：仅收取 <a href="#net">公网网络费用</a>。
   ![](https://main.qcloudimg.com/raw/2f2834eced22b7fc16fe64e6431ffc47.png)

## IP 资源费用

### 计费周期

IP 资源费用的计费周期为按小时收费，付费模式为后付费。
IP 资源费用从申请 EIP 开始计费，绑定云资源时暂停计费，解绑云资源时恢复计费，释放 EIP 时停止计费。计费时长按秒级单位统计，每小时对上一个小时产生的费用进行一次结算并扣费。若在同一计费周期内，多次解绑和绑定云资源，则计费时长为累计未绑定云资源的时长。
![](https://main.qcloudimg.com/raw/a52c516727c2a0028dcd80b56e1e1ab8.png)

### 计费公式

IP 资源费用 = 弹性公网 IP 所在地域闲置价格 × 计费时长

### 计费示例

若某传统账户类型用户在 09:00:00 - 09:59:59 时间内，申请了广州地域的 EIP，闲置 15 分钟（900秒）后绑定 CVM，则产生的 IP 资源费用为：0.2元/小时 ×（900 / 3600）小时 = 0.05元，于下一个小时（即 10:00:00 - 10:59:59）进行结算并扣费。

## 公网网络费用

标准账户类型使用 EIP 绑定云资源后产生的公网网络流量将在 EIP 端收取公网网络费用。

### 入网带宽上限（上行带宽）
公网的入网带宽是指从 Internet 流入 EIP 的带宽，详情请参考 [带宽上限](https://cloud.tencent.com/document/product/1199/48333)。



### 计费模式

公网网络费用分为按流量、包月带宽、按小时带宽三种计费模式。按流量和按小时带宽计费模式的 EIP 可以加入共享带宽包中，加入后，EIP 原本的计费模式将变更为共享带宽包模式，不再额外收取公网网络费用。

### <span id="1"></span>按流量

根据使用的公网流量计费，付费模式为后付费，每小时结算一次。适用于业务流量峰值在不同时间段波动较大的场景。

#### 计费示例

若某标准账户类型用户购买了新加坡地域的 EIP，选择按流量计费模式。该用户在07:00:00 - 07:59:59内共计使用流量10GB，则08:00:00时刻产生的费用为：0.8元/GB × 10GB = 8元

> ?公网流量是根据下行字节数统计而来的流量数据，是应用层数据。在实际网络传输中，产生的网络流量要比纯应用层流量多5% - 15%，故腾讯云统计的流量可能会比用户在服务器自行统计的流量多10%左右。
>
> - TCP/IP 包头消耗：基于 TCP/IP 协议的 HTTP 请求，每一个包的大小最大是1500个字节，包含了 TCP 和 IP 协议的40个字节的包头，包头部分会产生流量，但是无法被应用层统计到，这部分的开销大致为3%左右。
> - TCP 重传：正常网络传输过程中，发送的网络包会有3% - 10%左右会被互联网丢掉，丢掉后服务器会对丢弃的部分进行重传，此部分流量应用层也无法统计，占比约为3% - 7%。
