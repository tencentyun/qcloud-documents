弹性公网 IP（EIP）的费用根据非带宽上移和带宽上移不同类型的账户收费不同。本文介绍不同类型账户的 EIP 计费详情。

## 背景信息
- **账户类型介绍**
 - 非带宽上移：在 CVM 上管理带宽/流量。非带宽上移账户的 IP 和 CLB 没有网络带宽或流量属性，需要在 CVM 上购买和管理。
 - 带宽上移：在 IP 和 CLB 上管理带宽/流量。带宽上移账户购买的 CVM 不再保留外网带宽或流量资源，而公网 CLB/IP 端具备外网带宽或流量资源。
- <span id="judge" />**账户类型区分**：登录 <a href="https://console.cloud.tencent.com/cvm/eip">EIP 控制台</a>，在“弹性公网 IP”页面查看列表字段。
  - 若列表中没有“带宽上限”字段，则为非带宽上移账户。
    ![](https://main.qcloudimg.com/raw/cedd98f5e9bf8afcf0984cda822e5c12.png)
  - 若列表中有“带宽上限”字段，则为带宽上移账户。
    ![](https://main.qcloudimg.com/raw/6fdd79572d9c7d5cf1a7d88aea40f9b0.png)

## 费用组成
EIP 的费用由 **IP 资源费用**和**公网网络费用**组成。非带宽上移和带宽上移类型的账户收费介绍如下：

### 非带宽上移账户
非带宽上移账户仅涉及 IP 资源费用，公网网络费用在 CVM 实例上收取。
- 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费。
- 当 EIP 绑定云资源时：EIP 自身不收取任何费用，仅在 CVM 实例上收取 <a href="https://cloud.tencent.com/document/product/213/10578">公网网络费用</a>。
  ![](https://main.qcloudimg.com/raw/2f1174acba496fb43e1a82c959c58683.png)

### 带宽上移账户
带宽上移账户有四种计费模式，费用情况如下：
- 按流量：收取公网网络费用和 IP 资源费用。
 - 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费，不收取公网网络费用。
 - 当 EIP 绑定云资源时：仅收取 <a href="#net">公网网络费用</a>。
- 包月带宽：无论是否绑定云资源，仅收取 <a href="#net">公网网络费用</a>。
- 按小时带宽：无论是否绑定云资源，仅收取 <a href="#net">公网网络费用</a>。
- 共享带宽包：收取公网网络费用和 IP 资源费用。
 - 当 EIP 未绑定云资源时：EIP 仅收取 <a href="#ip">IP 资源费用</a>，按小时计费，不收取公网网络费用。
 - 当 EIP 绑定云资源时：仅收取 <a href="#net">公网网络费用</a>。
	![](https://main.qcloudimg.com/raw/2f2834eced22b7fc16fe64e6431ffc47.png)

## IP 资源费用

### 计费周期

IP 资源费用的计费周期为按小时收费，付费模式为后付费。
IP 资源费用从申请 EIP 开始计费，绑定云资源时暂停计费，解绑云资源时恢复计费，释放 EIP 时停止计费。计费时长按秒级单位统计，每小时对上一个小时产生的费用进行一次结算并扣费。若在同一计费周期内，多次解绑和绑定云资源，则计费时长为累计未绑定云资源的时长。
![](https://main.qcloudimg.com/raw/a52c516727c2a0028dcd80b56e1e1ab8.png)

### 计费公式
IP 资源费用 = 弹性公网 IP 所在地域闲置价格 × 计费时长

### 计费示例
若某非带宽上移用户在 09:00:00 - 09:59:59 时间内，申请了广州地域的 EIP，闲置 15 分钟（900秒）后绑定 CVM，则产生的 IP 资源费用为：0.2元/小时 ×（900 / 3600）小时 = 0.05元，于下一个小时（即 10:00:00 - 10:59:59）进行结算并扣费。

> ! 为避免产生不必要的 IP 资源费，请您申请 EIP 后及时绑定云资源，解绑云资源后及时释放不再使用的 EIP。

## 公网网络费用
带宽上移账户使用 EIP 绑定云资源后产生的公网网络流量将在 EIP 端收取公网网络费用。

### 计费模式
公网网络费用分为按流量、包月带宽、按小时带宽三种计费模式。按流量和按小时带宽计费模式的 EIP 可以加入共享带宽包中，加入后，EIP 原本的计费模式将变更为共享带宽包模式，不再额外收取公网网络费用。

### 共享带宽包
共享带宽包计费模式的 EIP 为按月后付费，适用于大规模业务，且使用公网的不同实例间可形成流量错峰的场景。您可以使用如下两种方式创建：
1. 创建申请 EIP 时，您可以直接创建共享带宽包模式的 EIP。
2. 将按流量和按小时带宽计费模式的 EIP 可以加入共享带宽包中，加入后，EIP 原本的计费模式将变更为共享带宽包模式，

| 地域     | 价格（单位：元/Mbps/月） |
| -------- | ------------------------ |
| 全部地域 | 108                      |

> ?每月1日结算，若使用不足1月则按有效天数比例计费，详情请参见 [共享带宽包计费说明](https://cloud.tencent.com/document/product/684/15255) 。

#### 计费示例

若某带宽上移用户购买了新加坡地域的 EIP，选择按小时带宽计费模式，并将其加入共享带宽包。1月削峰后的计费带宽（详情请参见 [削峰规则](https://cloud.tencent.com/document/product/684/15255#.E6.9C.88-top5-.E8.AE.A1.E8.B4.B9)）为10Mbps，则总费用为：108元/Mbps/月 × 10Mbps = 1080元
