Tencent Cloud provides a variety of persistent storage devices, allowing the user to select an appropriate type of disk to store files, build databases, etc.

- 5 types of disks available: Local HDD, SSD local disk, HDD cloud storage, SSD cloud storage, and premium cloud storage.
- Elastic mounting/unmounting: Ordinary cloud disk supports elastic mounting and unmounting,  allowing you to build a large-capacity file system for the cloud server.
- Elastic expansion: A disk supports a maximum volume size of 4TB, and you can expand it at any time.
- Snapshot backup: It supports snapshot creation and rollback, timely backup of key data, and creating disks using snapshots to realize fast business deployment.
