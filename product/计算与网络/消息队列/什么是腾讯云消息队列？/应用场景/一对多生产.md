CMQ 的 Topic 主题订阅模型，支持生产者向多个订阅者，同时异步投递消息的能力。您可将消息投递到不同 HTTP 终端，或投递到 Queue 队列内。CMQ 还提供消息、订阅 TAG，提供消息过滤能力。
>?Topic 不支持投递到 VPC 下的 HTTP 终端。
