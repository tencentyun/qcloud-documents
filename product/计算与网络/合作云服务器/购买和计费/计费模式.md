腾讯云目前开放了合作云服务器按量计费的模式。

## 按量计费

- 计费方式
  合作云服务器计费内容包括主机费用和网络流量费用两部分，按天按量计费。
  - 主机费用根据配置规格按天计算，不足一天按一天计算。
  - 网络流量费用按照当天实际消耗流量计算，以GB为单位每天计算。
  
 例如，您选购以下1台合作云服务器：
	
| 参数类型 | 参数值 | 
|---------|---------|
| 地域 | 悉尼 | 
| CPU  | 2核 |
| 内存 | 2GB | 
|操作系统 |Windows | 
| 系统盘 | 100GB | 
| 公网带宽上限 | 100M  | 

 **主机按天计算29.28元/天。宽带按量计算0.9元/GB。2019年5月6日当天您消耗网络流量300GB，则网络流量费用为300GB \* 0.9元/G = 270元，主机费用为29.28元。
	当天合计费用：主机费用29.28元 + 当天网络流量费用270元 = 299.28元**
- 配置变更
	发生变更时，按照当天最高的配置规格计费，第二天开始按照新配置计费。
- 销毁
	支持随时销毁，第二天即停止计费。
