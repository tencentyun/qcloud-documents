**1. Serverless Framework SSR目前支持哪些SSR框架?**

目前，serverless framework已支持基于React的next.js框架，与基于Vue的nuxt.js 框架，除此之外，您还可以通过egg.js. express等组件轻松搭建您自己的SSR框架项目

**2. 什么是SSR？与传统CSR相比优势是什么?**

SSR中文为服务端渲染，指在服务器上渲染出HTML，发送到浏览器，浏览器无需等待JavaScript解析，直接渲染HTML后即可生成首页。SSR框架可以：1. 拥有更好的 SEO，由于html已在服务器完成渲染，搜索引擎爬虫工具可以更好地抓取到页面的信息。 2. 缩短首页加载时间，服务端完成渲染返回给浏览器，无需等待所有的 JavaScript 都完成下载并执行，用户将会更快速地看到完整渲染的页面。

**3. Serverless Framework SSR 有哪些优势？**

Serverless Framework SSR 具有标准化、0配置、组件化开发、弹性扩展、安全可靠、智能监控&实时日志的优势。

**4. Serverless Framework SSR 收费吗？**

Serverless Framework SSR 暂时免费，但该产品所用到的相关腾讯云产品将按照资源使用量进行收费（遵循各产品的计费规则）。
