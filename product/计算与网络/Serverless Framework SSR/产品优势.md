- **低延迟**
SSR 框架下，服务器返回的是 HTML 文件，通过解析 HTML 文件，浏览器就能渲染出页面，使用户更快的看到页面的内容，大大缩短了首页的白屏等待时间。

- **SEO 友好**
SSR 框架下，SEO 的关键信息直接在后台就渲染成 HTML，保证搜索引擎的爬虫可以准确搜索到关键数据。

- **"0"配置**
用户无需进行复杂的配置，只需要关心业务逻辑项目代码，即可高效、快速进行部署，Serverless 会帮助用户完成全部配置。

- **应用监控**
Serverless SSR 提供 Dashboard 监控能力，可通过 Dashboard 实时监控项目状态，方便进行业务排障。

- **降低成本**
Serverless SSR 按照用户请求的使用量进行收费，没有请求时无需付费，降低开发成本。

- **组件化开发**
Serverless SSR 提供组件化的开发和集成，便于用户修改和资源复用，使用更加灵活。
