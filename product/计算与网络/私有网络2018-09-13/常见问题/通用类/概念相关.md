### VPC 有哪些组成部分？
腾讯云 VPC 由多个不同的服务组成：
- **软件定义网络**
 - **私有网络 ( VPC )** 是腾讯云中逻辑隔离的虚拟网络。从所选的范围内定义 VPC 的 IP 地址空间。
 - **子网**是 VPC 的 IP 地址范围内的一个区段，其中可放入各组隔离的资源。
 - **路由表**用于定义私有网络内每个子网的网络流量走向，包含一系列路由策略。

- **Internet 连接**
 - **NAT 网关** 是一项高度可用的托管网络地址转换服务，可用于私有子网中的资源访问 Internet。
 - **弹性 IP**是可以独立申请的公网 IP 地址，用于公网访问，支持与实例（如：云服务器、NAT 网关）的动态绑定和解绑，主要用于屏蔽实例故障。
 - **公网网关**是一种云服务器，具备转发 Internet 和 VPC 间流量的能力，没有外网 IP 的云服务器可通过公网网关来访问 Internet。

- **部署混合云**
 - **VPN 连接**是一种通过公网加密通道连接您 IDC 和 VPC 的方式，包括三个组成部分：VPN 网关、对端网关、VPN 通道。
 - **专线接入**是一种快速连接腾讯云与本地数据中心的方法，您可以通过一条物理专线，一次性打通位于多地域的腾讯云计算资源，包括三个组成部分：物理专线、专用通道和专线网关。

- **云上资源互通**
 - **对等连接**是一种连接两个 VPC 的服务，支持跨账户和跨地域 VPC 之间的流量互通，两端的云服务器和云数据库等资源可以相互访问。
 - **基础网络互通**：是将基础网络内的云服务器关联至指定 VPC 的服务，可以打通基础网络中的云服务器与 VPC 之间的网络通信。

- **安全控制**
 - **安全组**是一种有状态的包过滤虚拟防火墙，它用于控制单台或多台云服务器的出入流量，可以精确协议和端口维度。
 - **网络 ACL**是一个子网级别的无状态可选包过滤虚拟防火墙，用于控制进出子网的数据流，可以精确到协议和端口维度。

### 云服务器、内网负载均衡、云数据库能否修改内网 IP？
云服务器主网卡的主内网 IP 支持修改，辅助网卡的主内网 IP 不支持修改。内网负载均衡和云数据库不支持修改内网 IP。

### VPC 内的云服务器或数据库能否切换到其它 VPC？
暂不支持切换功能，请您提前做好网络规划。

### VPC 有哪些使用限制？
VPC 在配额方面有限制，详情请参见 [使用限制](https://cloud.tencent.com/document/product/215/20093)。

### 能否将主机部署在同一 VPC的不同可用区？
可以。VPC 主机可用区属性是继承自子网的，在创建子网时需要选择可用区信息。不同子网可以在不同可用区，在不同可用区的子网下购买主机，即可实现将主机部署在不同可用区。

### 腾讯云上资源能否互通？
您可以通过对等连接和基础网络互通实现私有网络内的资源与其它云资源的互通。详情请参见 [主要功能 - 灵活互通的云上资源](https://cloud.tencent.com/document/product/215/20055#.E7.81.B5.E6.B4.BB.E4.BA.92.E9.80.9A.E7.9A.84.E8.85.BE.E8.AE.AF.E4.BA.91.E4.B8.8A.E8.B5.84.E6.BA.90)。

### 我能创建多少个 VPC、子网、路由表、公网网关和网络 ACL？
VPC 在配额方面有限制，详情请参见 [使用限制](https://cloud.tencent.com/document/product/215/20093)。

### VPC 网段能否更改？
不支持更改，但您可删除当前 VPC，并重新创建一个 VPC。
