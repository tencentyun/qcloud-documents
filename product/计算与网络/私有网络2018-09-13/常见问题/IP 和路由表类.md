### VPC 和子网中可以使用的 IP 地址范围是什么？
- 私有网络支持三个网段的内网 IP：`10.a.0.0/8`（a 属于 0 - 255）、`172.b.0.0/16`（b 属于 0 - 31）、`192.168.0.0/16` 。私有网络的 CIDR 可以为以上三个网段，或者是网段中的一部分。
- 网络块包括的 IP 数 =` 2^(32-掩码)`，因而 `10.1.0.0/16` 网络块最多包含 65536 个 IP 地址。

### 什么是 CIDR？
CIDR 是由用户指定的独立网络空间地址块，通过 IP 和掩码结合，实现对网络的整体划分。以 10.1.0.0/16 为例，其中10.1.0.0 为网络块的 IP，16 为网络块的掩码。通过设定掩码的大小，可以调整网络块的大小设定。网络块包括的 IP 数 = 2^(32-掩码)，因而 10.1.0.0/16 网络块最多包含 65536 个 IP 地址。

### VPC 的 CIDR 有什么限制？
- VPC 的网络空间掩码支持在 /28 至 /16 之间，即您的私有网络空间最少包含 16、最大包含 65536 个 IP 地址。
- VPC 目前支持三个网段的内网 IP：`10.a.0.0/8`（a 属于 0 - 255）、`172.b.0.0/16`（b 属于 0 - 31）、`192.168.0.0/16`。VPC 的 CIDR 可以为以上三个网段，或者是网段中的一部分。
- 子网的 CIDR 必须是所在 VPC 组成的一部分。

### 如何修改云服务器内网 IP？
云服务器主网卡的主内网 IP 支持修改，辅助网卡的主内网 IP 不支持修改，具体操作步骤，请参见 [操作指南](https://cloud.tencent.com/document/product/576/18541)。

### 为什么广播包或组播包会部分丢失？
为了保证云服务器业务正常通信，在子网 / 私有网络内云服务器数量较大、 发包频率较高的情况下，平台会随机丢弃部分广播包或组播包。
