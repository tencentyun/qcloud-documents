[弹性公网 IP 地址（EIP）](https://cloud.tencent.com/document/product/213/5733)，简称弹性 IP 地址或弹性 IP。它是专为动态云计算设计的静态 IP 地址，是某地域下一个固定不变的公网 IP 地址。借助弹性公网 IP 地址，您可以快速将地址重新映射到账户中的另一个实例或 NAT 网关实例，从而屏蔽实例故障。

关于弹性公网 IP 的操作，请参见 [弹性公网 IP-操作步骤](https://cloud.tencent.com/document/product/213/16586#.E6.93.8D.E4.BD.9C.E6.AD.A5.E9.AA.A4)。
