1. 登录 [腾讯云控制台](https://console.cloud.tencent.com/) ，选择【云产品】>【网络】>【私有网络】，进入私有网络控制台。
2. 单击左侧目录中的【安全】>【参数模板】，进入管理页面。
3. 在【IP 地址】页面，单击【+新建】。
 ![1](https://main.qcloudimg.com/raw/1179f299586a1869e0eeb68c2bb55ed3.png)
4. 在弹出框中，填写名称和 IP 地址。
 ![](https://main.qcloudimg.com/raw/c1f2b4a0efa205df503b2431fdf27415.png)
4. 单击【提交】即可。
