## 通用限制
1. 共享带宽包目前处于内测阶段，需要提交 [内测申请](https://cloud.tencent.com/apply/p/8o8lmsr5nj8)。
2. 一个账户只能使用一种类型的带宽包。腾讯云账户分为标准账户类型和传统账户类型，2020年6月17日零点后注册的账户均为标准账户类型，该时间点前注册的账户请在控制台查看您的账户类型，具体操作请参见 [账户类型区分](https://cloud.tencent.com/document/product/1199/41692#.E8.83.8C.E6.99.AF.E4.BF.A1.E6.81.AF)。
  - 标准账户类型仅支持使用 IP 带宽包。
  - 传统账户类型中，除 <a href="https://cloud.tencent.com/document/product/1142">弹性公网 IPv6</a>、<a href="https://tcloud-doc.isd.com/document/product/214/8848">IPv6 负载均衡</a> 和 <a href="https://cloud.tencent.com/document/product/1199/41646">弹性公网 IP</a> 中的静态单线 IP 支持使用 IP 带宽包外，其余情况仅支持使用设备带宽包。

## 设备带宽包限制
1. 每个地域仅可开通1个带宽包。
2. 单个地域共享带宽包与其他计费模式不可共存，开通带宽包后该地域所有云服务器、负载均衡均自动变为按带宽包计费，已付费的带宽费用将按小时带宽折算后退还。

## IP 带宽包限制
1. 单个地域可开通多个带宽包，单个地域最高不得超过20个。
2. 单个地域共享带宽包与其他网络计费模式（包年包月带宽、小时带宽和流量计费）可共存。
3. 暂不支持包年包月的普通公网 IP 和 EIP 加入 IP 带宽包。
4. 每个带宽包最多可加入100个资源（包括同地域的 EIP、普通公网 IP 和 CLB）。
