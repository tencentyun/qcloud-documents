腾讯云提供了两种购买负载均衡的方式：官网购买和 API 购买。本小节将详细介绍两种购买方式。

### 官网购买
所有用户均可通过 [腾讯云官方网站](https://buy.cloud.tencent.com/lb) 购买负载均衡。其中，内网负载均衡免费，公网负载均衡仅收取实例费。公网负载均衡的计费模式是按量计费（以小时结算），公网带宽请在云服务器上购买，详见负载均衡 [计费说明](https://cloud.tencent.com/document/product/214/8848)。

在官网购买这两种服务器的具体操作如下：
1. 登录腾讯云 [负载均衡服务购买页](https://buy.cloud.tencent.com/lb)。
2. 在实例类型的选择上，推荐您选择“应用型 CLB”。
3. 根据实际情况和需求选择网络、所属项目，设置实例名称。
4. 确认购买后并支付完成后即开通负载均衡服务，您可进行负载均衡配置使用。

### API 购买
欲通过 API 购买负载均衡的用户，请参阅 [负载均衡API](http://cloud.tencent.com/doc/api/244/%E7%AE%80%E4%BB%8B) 创建实例。
