>- 为提供更优质的产品服务，容器服务已于2019年6月17日起默认采用兼容原生 kubernetes 特性的新版控制台，原TKE控制台将于近期下线。

容器服务新版控制台 TKE2 对比 TKE，进行了一系列功能模块的调整，为您提供一个原生、更易用的平台。新版控制台不仅可以帮助您按需构建托管/独立集群、自定义节点镜像、设置负载亲和性调度等基本操作，也提供了事件指标监控、对比监控、独立集群 Master&Etcd 监控等监控服务。具体可参考下文新旧版本功能对比。

## 新旧控制台功能对比
| 功能点                         | 新版控制台 TKE2（推荐）| 旧版控制台 TKE |
| --------------------------- | ------------- | -------- |
| 集群基本增删改查                    | 支持            | 支持       |
| 托管/独立集群模式                   | 支持            | 不支持      |
| 节点基本增删改查                    | 支持            | 支持       |
| 节点的驱逐和封锁                    | 支持            | 支持       |
| 自定义节点镜像                     | 支持            | 不支持      |
| 集群伸缩组管理                     | 支持            | 支持       |
| 集群内工作负载管理                   | 支持            | 有限支持     |
| 设置负载亲和性调度                   | 支持            | 不支持      |
| 集群内 Service&ingress 管理        | 支持            | 有限支持     |
| Service 支持使用已有 LB             | 支持            | 不支持      |
| Service LoadBalancer 只绑定部分节点 | 支持            | 有限支持     |
| 集群内配置管理（configmap，secret）   | 支持            | 不支持      |
| 集群存储管理                      | 支持            | 不支持      |
| 集群内 K8S 资源日志查看                | 支持            | 有限支持     |
| 集群内 K8S 资源事件查看                | 支持            | 有限支持     |
| 镜像仓库                        | 支持            | 支持       |
| 应用管理（应用模板+配置项）              | 推荐使用 Helm      | 简单支持     |
| 模板市场                        | 推荐使用 Helm      | 简单支持     |
| 日志采集功能                      | 支持            | 有限支持     |
| 告警设置功能                      | 支持            | 不支持      |
| 事件持久化                       | 支持            | 不支持      |
| 使用已有 CVM 创建集群                 | 支持            | 不支持      |
| 查看节点 K8S 属性                   | 支持            | 不支持      |
| 事件指标监控                      | 支持            | 有限支持     |
| 对比监控                        | 支持            | 不支持      |
| 独立集群 Master、Etcd 监控           | 支持            | 不支持      |
