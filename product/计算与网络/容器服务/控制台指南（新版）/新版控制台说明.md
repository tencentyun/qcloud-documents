新版控制台目前已全地域支持 ，如果您需要使用， 可以从下列方法进入新版控制台。
1.通过 [新版控制台](https://console.qcloud.com/tke2) 直接访问。
2.登录 [腾讯云控制台](https://console.cloud.tencent.com)，选择 “云产品 > 计算 > 容器服务”，进入“集群”页面，单击右上角“前往新版控制台”。
![](https://main.qcloudimg.com/raw/e6b78a08cab94e7093d5f8610a5d4a8d.png)
>?
>1. 于2019年6月17日起，容器服务将默认采用新版TKE2控制台，同时 [原TKE控制台](https://console.qcloud.com/tke) 仍可正常访问。
>2. 我们建议您使用新版的TKE2控制台，如果您对新版控制台有任何疑问，欢迎提工单给我们。
>
容器服务新版控制台 TKE2 对比 TKE，进行了一系列功能模块的调整，为您提供一个原生、更易用的平台。新版控制台不仅可以帮助您按需构建托管/独立集群、自定义节点镜像、设置负载亲和性调度等基本操作，也提供了事件指标监控、对比监控、独立集群 Master&Etcd 监控等监控服务。具体可参考下文新旧版本功能对比。

## 新旧控制台功能对比
| 功能点                         | 新版控制台TKE2（推荐）| 旧版控制台TKE |
| --------------------------- | ------------- | -------- |
| 集群基本增删改查                    | 支持            | 支持       |
| 托管/独立集群模式                   | 支持            | 不支持      |
| 节点基本增删改查                    | 支持            | 支持       |
| 节点的驱逐和封锁                    | 支持            | 支持       |
| 自定义节点镜像                     | 支持            | 不支持      |
| 集群伸缩组管理                     | 支持            | 支持       |
| 集群内工作负载管理                   | 支持            | 有限支持     |
| 设置负载亲和性调度                   | 支持            | 不支持      |
| 集群内Service&ingress管理        | 支持            | 有限支持     |
| Service支持使用已有LB             | 支持            | 不支持      |
| Service LoadBalancer只绑定部分节点 | 支持            | 有限支持     |
| 集群内配置管理（configmap,secret）   | 支持            | 不支持      |
| 集群存储管理                      | 支持            | 不支持      |
| 集群内K8S资源日志查看                | 支持            | 有限支持     |
| 集群内K8S资源事件查看                | 支持            | 有限支持     |
| 镜像仓库                        | 支持            | 支持       |
| 应用管理（应用模板+配置项）              | 推荐使用Helm      | 简单支持     |
| 模板市场                        | 推荐使用Helm      | 简单支持     |
| 日志采集功能                      | 支持            | 有限支持     |
| 告警设置功能                      | 支持            | 不支持      |
| 事件持久化                       | 支持            | 不支持      |
| 使用已有CVM创建集群                 | 支持            | 不支持      |
| 查看节点K8S属性                   | 支持            | 不支持      |
| 事件指标监控                      | 支持            | 有限支持     |
| 对比监控                        | 支持            | 不支持      |
| 独立集群Master、Etcd监控           | 支持            | 不支持      |
