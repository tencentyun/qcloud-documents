## Description
When creating a cluster and add a cluster for container services, you can set the type and size of the system disk and the data disk of the container node, and choose from different disks to satisfy your individual business demands.

## Suggestions
1. The directory of the container is stored in the system disk. It is recommended that you create a system disk with a capacity of 50 GB.

2. If there is requirement for system disk, you can adjust the Docker's directory to the data disk when initializing the cluster.
