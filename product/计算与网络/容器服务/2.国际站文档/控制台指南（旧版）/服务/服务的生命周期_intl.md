## Service Lifecycle

### Status description
| Status | Description |
|:--:|:--|
| Launching | Service is being created, and container instance being created |
| Running | Service is running normally and is available |
| Updating | Service configuration is being updated|
| Exceptional | Exceptional containers exist in the service |
