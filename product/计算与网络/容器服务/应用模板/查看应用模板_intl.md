
## Application Template List

Log in to the [Tencent Cloud TKE Console](https://console.cloud.tencent.com/ccs). You can view all your application templates on [Application Template][1] page.

![应用模板查看-001.png-51.4kB][2]

## Filtering and Searching Application Template

Enter the name in the search box above the application template list to filter and search applications with the specific name.

![应用模板查看-002.png-21.5kB][3]

## Application Template Details

You can click an application template ID on the application template page to view the details.

![应用模板查看-003.png-39.8kB][4]

The application template details page consist of two parts: Template Contents and Configuration Item, as shown below.
