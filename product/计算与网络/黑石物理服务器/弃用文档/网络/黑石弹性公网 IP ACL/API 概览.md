可以使用 API 操作来设置和管理弹性公网 IP ACL：

| 接口功能 | Action ID | 
|---------|---------|
| 创建弹性公网 IP ACL | EipAclBmApply | 
| 修改弹性公网 IP ACL 名称、状态、入站规则、出站规则 | EipAclBmSet  | 
| 将弹性公网 IP 关联到弹性公网 IP ACL | EipAclBmBind | 
| 删除弹性公网 IP ACL | EipAclBmDelete | 
| 查询弹性公网 IP ACL 列表 | DescribeEipAclBm  | 
| 查询弹性公网 IP ACL 关联的弹性公网 IP 列表 | DescribeEipBm | 
| 从弹性公网 IP ACL 中解除弹性公网 IP | EipAclBmUnBind |
