## 申请 EIP
1.打开黑石物理服务器控制台，找到弹性公网 IP 选项，单击申请按钮弹出 EIP 申请框。 
![](https://main.qcloudimg.com/raw/e057e90e7e5e794d2f22498fa444902c.png)
2.选择计费模式与带宽峰值，单击【确定】按钮完成申请

>!
- 每个腾讯云账户每个地域最多可以申请100个黑石 EIP。
- 公网流量计费模式下设置的带宽峰值不是计费带宽，而是限速带宽，EIP 最大速率不会超过该处设置的带宽峰值。 


## 绑定 EIP
1.打开黑石物理服务器控制台，找到弹性公网 IP 选项。
![](https://main.qcloudimg.com/raw/252139475f68392231c838dd7ba377da.png)

2.找到您想用来绑定的 IP，单击【绑定】按钮。若已绑定资源，则【绑定】按钮将为不可用单击状态，需与原资源解绑后再绑定。

3.在弹出框中选择您需要绑定的资源，单击【绑定】按钮即可完成绑定，此时开始收取公网出口费。
![](https://main.qcloudimg.com/raw/36ea374a298b26cc506944525583f4c9.png)


>!按固定带宽计费的黑石弹性公网 IP 不可用于绑定 NAT 网关。

## 解绑 EIP
1.打开黑石服务器控制台，找到弹性公网 IP 选项。
![](https://main.qcloudimg.com/raw/8754a7eba1185ef30bdc49e2f694c778.png)

2.在已绑定资源的黑石弹性公网 IP 列表项后，在更多选项中单击【解绑】按钮，确认后即可解绑，此时开始收取 IP 闲置费。


## 释放弹性公网 IP
1.打开黑石服务器控制台，找到弹性公网 IP 选项。
![](https://main.qcloudimg.com/raw/f7cdbb53aacbdcdfebb3a8be2499ad29.png)

2.在您想要释放的黑石弹性公网 IP，单击【释放】按钮，释放后即不再计费。


>!只有没有绑定资源的 EIP 才可以释放，如果已经绑定资源，请先解绑，然后再释放。
