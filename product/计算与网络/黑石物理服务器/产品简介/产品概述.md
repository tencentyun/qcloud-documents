黑石物理服务器（Cloud Physical Machine）是种可按需购买、按量付费的物理服务器租赁服务，提供给您云端专用的高性能、安全隔离的物理服务器集群。使用该服务，您只需确定服务器配置和数量，部署、交付时间将被缩短至4小时，服务器供应、运维工作交由腾讯云，您可专心于业务创新。

相较于自建数据中心，使用黑石服务器可以更快地搭建您的私有云，并由腾讯云提供数据中心内网互联、私有网络、负载均衡、NAT 网关、运维、监控、安全防护等能力。

>! 黑石服务器是种独占的物理服务器租赁服务，而不是云服务器（虚拟服务器）的租赁。
>

- 描述黑石服务器所在的数据中心，请查看 [地域和可用区](https://cloud.tencent.com/document/product/386/7029)。
- 黑石服务器适用的场景，由不同的 CPU、内存、存储和网络等组成，请查看 [实例类型](https://cloud.tencent.com/document/product/386/7035)。
- 实例对内和对外的服务地址，也即内网 IP 地址和弹性公网 IP 地址。
- 基于 Web 的操作界面，请查看 [腾讯云控制台](https://console.cloud.tencent.com/cpm)。

如果您还未使用过黑石服务器，可以参考 [快速入门（部署 Linux 服务器）](https://cloud.tencent.com/document/product/386/7040)。
