您可以使用 API 操作来设置和管理您的路由表，黑石更多相关 API 可以参考 [黑石物理服务器所有 API 概览](https://cloud.tencent.com/document/product/386/6632)。 

| 接口功能   | Action ID               | 功能描述                                     |
| ------ | ----------------------- | ---------------------------------------- |
| 查询路由表  | DescribeBmRouteTableEx  | 根据用户输入信息，如 routeTableId、路由表名称等，查询对应路由表的详细信息。 |
| 删除路由策略 | DeleteBmRuleTablePolicy | 根据 routeTableId、路由表名称等，删除对应路由表中的路由策略。     |
| 新增路由策略 | CreateBmRuleTablePolicy | 根据 routeTableId、路由表名称等，新增对应路由表中的路由策略。     |
