# 调整网络计费模式和带宽

根据您的业务需求，您可以调整网络计费模式和带宽。针对您配置的实例计费模式和原先选择的网络、带宽计费模式。你调整的条件和限制可以参考如下内容。

## 调整网络计费模式

### 限制及要求

| CVM实例计费模式 | 原始网络计费模式->调整后的计费模式                           | 是否可调整 | 调整限制                                                  |
| --------------- | ------------------------------------------------------------ | ---------- | --------------------------------------------------------- |
| 包年包月        | 按带宽包年包月计费->按流量计费                               | 是         | 每个实例仅限转换1次。<br>使用的带宽按小时带宽单价计算费用 |
|                 | 按流量计费->按带宽包年包月计费                               | 是         | 每个实例仅限转换1次。                                     |
| 按量计费        | 按带宽使用时长计费（觉得这里有点问题，为什么不是包月包年了呢？） | 否         | 不支持。                                                  |
|                 | 按流量计费->按带宽包年包月计费                               | 是         | 每个实例仅限转换1次。                                     |

## 调整带宽

### 限制及要求

| CVM实例计费模式 | 网络计费模式           | 是否可以调整带宽 |
| --------------- | ---------------------- | ---------------- |
| 包年包月        | 按带宽计费（包年包月） | 只可以升级带宽。 |
|                 | 按流量计费             |                  |
| 按量计费        | 按带宽计费（包年包月） |                  |
|                 | 按流量计费             |                  |
