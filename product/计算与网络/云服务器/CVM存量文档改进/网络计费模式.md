# 网络计费模式

您可以选择**按带宽**或**按使用流量**支付腾讯云CVM实例的公网带宽费用。网络带宽价格根据地域不同变化，具体可以参考[网络带宽定价](https://buy.cloud.tencent.com/price/idc)。

注意：

- 目前公网均按**出带宽/流量**进行费用结算。出带宽表示从 CVM 实例流出公网的带宽。例如，您的用户使用客户端下载 CVM 实例内部资源使用的带宽。
- 为了防止因突然爆发的流量而产生较高的费用，您可以通过**指定带宽上限**进行限制。若超出此上限，则默认丢包且不计算费用。
- 公网网络的带宽上限根据不同的网络计费模式有所不同，如果您可能使用的带宽较大，请参考[公网带宽上限](https://cloud.tencent.com/document/product/213/12523)。



## 计费模式参考

根据您配置的CVM计费模式和选择的带宽计费模式，最终计费方式可参考下表。

| CVM计费模式 | 带宽计费模式             | 最终计费方式                                    |
| ----------- | ------------------------ | ----------------------------------------------- |
| 包年包月    | 按带宽计费               | 按月预付费，购买主机时预先支付1个月的带宽费用。 |
|             | 按使用流量               | 按小时结算公网出流量费用。                      |
| 按量计费    | 按带宽计费（这个待确认） | 按小时结算公网出流量费用。                      |
|             | 按使用流量               | 按小时结算公网出流量费用。                      |
