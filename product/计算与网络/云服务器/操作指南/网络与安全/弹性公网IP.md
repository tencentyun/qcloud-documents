## 操作场景
弹性公网 IP （Elastic IP，简称 EIP），是可以独立购买和持有的、某个地域下固定不变的公网 IP 地址。借助弹性公网 IP，您可以快速将地址重新映射到账户中的另一个实例或  NAT 网关实例，从而屏蔽实例故障。本文档介绍如何使用弹性 IP 地址。
