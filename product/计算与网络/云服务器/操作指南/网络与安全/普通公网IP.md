## 操作场景
本文档介绍如何使用普通公网 IP 地址。普通公网 IP 仅能在云服务器购买时分配，且无法与云服务器解绑。如购买时未分配，则无法获得。
<dx-alert infotype="explain" title="">
- 传统账户类型，在解绑云服务器上的 EIP 时，每个账户每天可免费重新分配普通公网 IP 的次数为10次。
- 当前普通公网 IP 的地址类型仅支持常规 BGP IP。
</dx-alert>
