## 1. 使用建议

微软官方已于2015年7月14日起停止提供对Windows Server 2003 和 Windows Server 2003 R2 的扩展支持服务；同时腾讯云运行Windows2003系统的腾讯云主机，在此日期后将无法获得微软的更新和补丁，并将面临程序兼容性，不稳定性，安全等问题及风险；为了保障您的业务安全稳定，我们建议您把现有的Windows Server 2003的云服务器迁移到windows server2008 r2版本上。

## 2. 使用说明
自2015年9月14日起，腾讯云将停止向所有用户提供windows2003系统镜像（包含公共镜像，自定义镜像和镜像市场镜像）；如果您现有Windows2003系统的云服务器，且希望能继续提供，可以用工单方式进行申请，经审批后，您可以申请保持在这期间获得Windows2003的售卖入口。

>注：2003续用申请支持地域：广州、广州OPEN、上海、北京和香港

## 3. 风险提醒
腾讯云郑重提示您在继续申请Windows Server 2003/Windows Server 2003 R2续用前，充分了解如下风险：
  
1) 继续使用包含Windows Server 2003操作系统的腾讯云主机。这些腾讯云主机在2015年7月14日后，将无法继续获得微软所提供的更新和修补程序，同时您的应用和业务可能会面临各种风险，包括但不限于安全问题、应用不兼容、合规要求、以及其他非功能性问题所引发的不确定安全风险。

2) 如果您在2015年7月14日后继续使用该等Windows Server 2003的腾讯云主机，因该操作系统缺乏微软支持所引发的故障、安全问题、不相容性，或更高的不确定风险，腾讯云不会对此承担责任，您需要自行承担相应的风险、后果及责任。

## 4. 服务内容
在您充分认识风险，并且自愿承担相应的后果和责任后，您可以获得腾讯云提供的继续使用Windows Server 2003的服务，服务内容包括：
  
- 您可以继续运行现有的Windows Server 2003操作系统的腾讯云主机。
- 您可以继续使用Windows Server 2003的镜像（包括系统镜像，现有自定义镜像，镜像市场镜像）购买和创建腾讯云主机。
- 您可以通过控制台和云API查询到腾讯云提供的Windows Server 2003的镜像列表。（不支持创建自定义镜像）
- 您可以继续为您的Windows Server 2003的腾讯云主机办理包年包月续费。

## 5. 服务费用
您继续运行 Windows Server 2003 的腾讯云主机不会产生额外费用，购买和续费相应腾讯云主机的服务费用会正常计算。

## 6. 服务须知
由于微软已终止对 Windows Server 2003 的支持。因此您理解并同意，续用期间，以下不属于腾讯云的服务质量不达标、故障或责任：

1) 您运行Windows Server 2003 的实例可能有较高的故障风险，遭遇各种安全问题，出现不兼容的情况，或者无法正常工作，并有可能完全瘫痪。

2) 您的Windows Server 2003实例中运行的应用程序如果发生故障，并且需要安装微软补丁或者需要微软方面提供操作系统级的故障排除支持方可解决的，我们仅能为您的故障排除提供协助，可能无法提供完全的问题解决方案。

3) 受到硬件兼容性和驱动程序相关问题的限制，未来新的腾讯云主机可能无法支持Windows Server 2003 镜像的运行。

续用期间，您有责任自行保护使用windows Server 2003的腾讯云主机，并监控 Windows Server 2003 中报告的各种安全威胁，避免给其他腾讯云用户造成影响。

尽管腾讯云为您提供windows Server 2003的继续使用，但为了保护您的数据和业务安全，我们建议和鼓励您主动把现有的Windows Server 2003的实例迁移到Windows Server 2008 r2和Windows Server 2012 R2的实例上。
