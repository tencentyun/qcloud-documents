## 托管网站
您可以使用黑石私有网络部署简单的 Web 应用，如博客、网站和日志系统等；通过安全组  ，可以使 Web 应用响应 HTTP 等请求，但拒绝 Web 应用访问 Internet，从而保证 Web 应用的安全；在流量突增时，可以在黑石 VPC 中启用 负载均衡 。
![](https://main.qcloudimg.com/raw/1982dd68d62e792ff2b45e3e9f5c7f68.png)

相关产品：[黑石物理服务器（CPM）](https://cloud.tencent.com/document/product/386)、黑石公网负载均衡。

## 托管多层 Web 应用
黑石私有网络可以在为应用提供 Internet 服务的同时，又保障数据库服务器的安全。您可以安全灵活地在黑石私有网络中托管多层 Web 应用程序：在黑石私有网络中创建不同的子网，将整个 Web 层放在一个子网，通过 弹性公网 IP / NAT 网关 与 Internet 通信；将逻辑层单独放在一个子网，只能和 Web 层及数据层通信；将数据层放在另外一个子网，只和逻辑层通信。
![](https://main.qcloudimg.com/raw/4c3c82f5ad9174b39a8e7bf0e1f17f43.png)

相关产品：[黑石物理服务器（CPM）](https://cloud.tencent.com/document/product/386)、弹性公网 IP、NAT 网关。

## 部署弹性混合云
您可以在黑石私有网络内，部署应用程序；在企业数据中心，部署数据库服务器。黑石私有网络提供稳定安全的 IPsec VPN 连接/[云联网](https://cloud.tencent.com/document/product/877) 服务，帮您打通企业数据中心与云端资源。您可使用弹性伸缩服务，以根据业务量扩展应用程序的资源（物理服务器等），既降低了企业 IT 运维成本，又无需担心企业核心数据外泄，轻松实现弹性混合云部署。
