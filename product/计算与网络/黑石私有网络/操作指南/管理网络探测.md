## 操作场景
本文档指导您对网络探测进行创建、验证、修改、删除以及查看时延和丢包率。

## 前提条件
已登录 [腾讯云控制台](https://console.cloud.tencent.com/) 。




### 创建网络探测
1. 选择【云产品】>【物理计算与网络】>【黑石私有网络】，进入 [黑石私有网络控制台](https://console.cloud.tencent.com/vpcbm)。
2. 单击左侧目录中的【网络探测】，进入管理页面。
3. 单击【新建】，在弹出框中依次输入或确定以下参数：
- 名称：网络探测名称。
- 私有网络及子网：网络探测所属私有网络和子网。
- 源 IP：选择子网内两个闲置的内网 IP 作为网络探测的源地址。
- 探测目的 IP：网络探测最大支持两个目的 IP 地址。
4. 选择结束后单击【创建】即可。

### 验证网络探测
1. 选择【云产品】>【物理计算与网络】>【黑石私有网络】，进入 [黑石私有网络控制台](https://console.cloud.tencent.com/vpcbm)。
2. 单击左侧目录中的【网络探测】，进入管理页面。
3. 在列表中找到需要验证的网络探测实例，单击【更多】>【验证】。
4. 查看验证 IP 连通性结果，单击【完成】结束验证。

>!建议您在创建网络探测后，验证探测结果，如果验证失败，请检查路由是否配置正确或目的探测对象是否放通了网络 ACL、安全组等防火墙。

### 查看网络探测时延和丢包率
1. 选择【云产品】>【物理计算与网络】>【黑石私有网络】，进入 [黑石私有网络控制台](https://console.cloud.tencent.com/vpcbm)。
2. 单击左侧目录中的【网络探测】，进入管理页面。
3. 单击网络探测实例的监控图标，即可查看网络探测时延及丢包率。


### 修改网络探测
1. 选择【云产品】>【物理计算与网络】>【黑石私有网络】，进入 [黑石私有网络控制台](https://console.cloud.tencent.com/vpcbm)。
2. 单击左侧目录中的【网络探测】，进入管理页面。
3. 在列表中，找到需要修改的网络探测实例，单击【编辑】。
4. 在弹出框中，输入需要修改的信息，单击【确定】即可。


### 删除网络探测
1. 选择【云产品】>【物理计算与网络】>【黑石私有网络】，进入 [黑石私有网络控制台](https://console.cloud.tencent.com/vpcbm)。
2. 单击左侧目录中的【网络探测】，进入管理页面。
3. 在列表中，找到需要修改的网络探测实例，单击【更多】>【删除】。
