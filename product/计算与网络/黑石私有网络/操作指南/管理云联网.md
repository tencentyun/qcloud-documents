## 操作场景
本文档指导您如何新建云联网实例、查看已关联私有网络路由表、关联和解除黑石私有网络相关操作。

## 前提条件
已登录 [腾讯云控制台](https://console.cloud.tencent.com/)。



## 操作步骤
### 新建云联网实例
1. 选择【云产品】>【云计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpc)】，进入私有网络控制台。
2. 单击左侧目录中的 【云联网】，进入云联网管理页面。
3. 单击【新建】。
4. 在弹出框中填写云联网实例名称、备注。
5. 关联一个网络实例（也可创建后再关联）。
6. 单击【确定】创建云联网实例。


### 关联黑石私有网络
#### 方式一
1. 选择【云产品】>【云计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpc)】，进入私有网络控制台。
2. 单击左侧目录中的 【云联网】，进入云联网管理页面。
3. 单击需要添加关联的云联网 ID，进入详情页，单击【关联实例】。
4. 在弹出框中，选择需要关联的网络实例类型（黑石私有网络）、所属地域和具体实例。
5. （可选）如需再关联其他网络实例，可以单击【+ 添加】。
6. 单击【提交】即可。



#### 方式二
1. 选择【云产品】>【物理计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpcbm)】，进入黑石私有网络控制台。
2. 单击左侧目录中的 【私有网络】，进入私有网络管理页面。
3. 单击需要关联到云联网的私有网络 ID，进入详情页，单击【关联云联网】中的【立即关联】。
4. 在弹出框中，选择云联网实例所属账号类型（我的账号）和具体实例。如果选择云联网实例所属账号类型（其它账号），还需输入账号 ID 和云联网 ID。
5. 单击【确定】即可。



### 查看已关联私有网络路由表
1. 选择【云产品】>【物理计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpcbm)】，进入黑石私有网络控制台。
2. 单击左侧目录中的 【路由表】，进入路由表管理页面。
3. 单击已关联私有网络对应的路由表 ID，进入详情页。
4. 查看【下一跳类型】是云联网的路由策略，检查是否全部为启用状态。如果与其它类型路由策略冲突或者重叠，该云联网路由默认是禁用状态，需要手动启用。



### 解除黑石私有网络关联
#### 方式一
1. 选择【云产品】>【云计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpc)】，进入私有网络控制台。
2. 单击左侧目录中的 【云联网】，进入云联网管理页面。
3. 在云联网列表中，单击需要解除关联的云联网 ID，进入详情页。
4. 在【关联实例】页面，找到解除关联的黑石私有网络所在行，单击操作栏中的【解关联】。
5. 单击【确认】即可。


#### 方式二
1. 选择【云产品】>【物理计算与网络】>【[私有网络](https://console.cloud.tencent.com/vpcbm)】，进入黑石私有网络控制台。
2. 单击左侧目录中的 【私有网络】，进入私有网络管理页面。
3. 单击需要解关联的私有网络 ID，进入详情页，单击【关联云联网】中的【解关联】。
4. 单击【确定】即可。
