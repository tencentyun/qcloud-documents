## 操作场景
本文档指导您创建同账户同地域、同账户跨地域、跨账户同地域和跨账户跨地域 VPC 对等连接，以及接受/拒绝跨账户对端连接、删除对端连接。

## 前提条件
登录 [腾讯云控制台](https://console.cloud.tencent.com)。


## 操作步骤
### 创建同账户同地域 VPC 对等连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，单击【新建】创建 VPC 对等连接。
4. 填写 VPC 对等连接名称，并选择本端地域和本端私有网络。
5. 对等账户类型选择【我的账户】，并选择对端网络类型。
6. 选择对端地域和对端私有网络。
7. 单击【创建】即可完成 VPC 对等连接的创建。

### 创建同账户跨地域 VPC 对等连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，单击【新建】创建 VPC 对等连接。
4. 填写 VPC 对等连接名称，并选择本端地域和本端私有网络。
5. 对等账户类型选择【我的账户】，并选择对端网络类型。
6. 选择对端地域和对端私有网络。
7. 选择带宽上限，10Mbps/20Mbps/50Mbps/100Mbps/200Mbps/500Mbps/1Gbps。
8. 单击【创建】即可完成 VPC 对等连接的创建。

### 创建跨账户同地域 VPC 对等连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，单击【新建】创建 VPC 对等连接。
4. 填写 VPC 对等连接名称，并选择本端地域和本端私有网络。
5. 对等账户类型选择【其它账户】，并填写对端账户主账号 ID。
6. 选择对端网络类型和对端地域，并填写对端私有网络 ID。
7. 单击【创建】即可完成 VPC 对等连接的创建。


### 创建跨账户跨地域 VPC 对等连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，单击【新建】创建 VPC 对等连接。
4. 填写 VPC 对等连接名称，并选择本端地域和本端私有网络。
5. 对等账户类型选择【其它账户】，并填写对端账户主账号 ID。
6. 选择对端网络类型和对端地域，并填写对端私有网络 ID。
7. 选择带宽上限，10Mbps/20Mbps/50Mbps/100Mbps/200Mbps/500Mbps/1Gbps。
8. 单击【创建】即可完成 VPC 对等连接的创建。

### 接受/拒绝跨账户对端连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，查看列表中 “待接受” 状态的对端连接。
4. 在操作栏中单击【接受】或【拒绝】，并确认即可。


### 删除对端连接
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【对等连接】选项卡。
3. 选择地域和私有网络，查看列表中的对端连接。
4. 在操作栏中单击【删除】，并确认即可。
