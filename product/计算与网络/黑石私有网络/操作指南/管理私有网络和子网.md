## 操作场景
本文档指导您如何创建、查看、删除私有网络和初始化、新增、删除子网相关操作。

## 前提条件
已登录 [腾讯云控制台](https://console.cloud.tencent.com)。


## 操作步骤
### 创建私有网络、初始化子网
私有网络至少包含一个子网，只有在子网中才可以添加物理服务器。
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【私有网络】选项卡。
3. 选择地域和可用区，单击【新建】在此地域和可用区下创建私有网络。
4. 填写私有网络名称和 CIDR、子网名称和 CIDR，并选择子网的可用区。
5. 单击【确定】即可完成私有网络及子网的创建。


### 新增子网
用户可以同时创建一个或多个子网。
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【子网】选项卡。
3. 选择需要创建子网的可用区和私有网络。
4. 单击【新建】，填写子网名称、可用区和 CIDR。
5. （可选）单击【新增一行】，可以同时创建多个子网。
6. 单击【创建】按钮完成子网创建。


### 查看私有网络内的所有资源
1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【网络拓扑】选项卡。
3. 选择地域、可用区和私有网络。
4. 即可查看私有网络内的所有资源。


### 修改物理服务器内网配置
1. 单击导航条【[黑石物理服务器](https://console.cloud.tencent.com/cpm)】，进入黑石物理服务器控制台。
2. 单击左导航栏中【物理服务器】选项卡。
3. 选择需要修改的物理服务器所在行，单击操作列的【修改内网配置】。
4. 选择新的私有网络和子网。
5. 自动分配或者手动指定服务器内网 IP。
6. 单击【确定】完成内网配置修改。


### 删除私有网络
>!删除私有网络的前提条件是：私有网络内的 IP 没有被占用，同时私有网络内没有资源（例如：子网、NAT 网关等）。

1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【私有网络】选项卡。
3. 在列表上方，选择需要删除的私有网络所在地域和可用区。
4. 选择需要删除的私有网络所在行，单击操作列的【删除】。


### 删除子网
>!删除子网的前提条件是：子网内的 IP 没有被占用，同时子网内没有资源（例如：物理服务器等）。

1. 单击导航条【[黑石私有网络](https://console.cloud.tencent.com/vpcbm)】，进入私有网络控制台。
2. 单击左导航栏中【子网】选项卡。
3. 在列表上方，选择需要删除的子网所在地域、可用区和私有网络。
4. 选择需要删除的子网所在行，单击操作列的【删除】。
