The lifecycle of a GCC instance is defined in the same way as that of an ordinary CVM. It covers all the statues from the startup of the instance to its termination. You can manage GCC instances on Tencent Cloud console easily.

Statues of GCC instance include: 
	•	Launching
	•	Running
	•	Restarting
	•	Shutting Down
	•	Shut Down
	•	Resetting
	•	Terminating
	•	Terminated
