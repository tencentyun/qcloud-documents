CHC 云服务器使用的云上镜像主要分为公共镜像和自定义镜像。

### 公共镜像
- **TencentOS**
 - TencentOS Server 2.4 
 - TencentOS Server 2.2 (tkernel3) 
 - TencentOS Server 2.2 (tkernel3) with Driver 
- **CentOS**
 - CentOS 7.9 64bit 
 - CentOS 7.8 64bit 
 - CentOS 7.7 64bit 
 - CentOS 7.6 64bit 
 - CentOS 7.4 64bit 
- **Ubuntu**
 - Ubuntu Server 18.04.1 LTS 64bit



### 自定义镜像
CHC 云服务器仅支持基于以上公共镜像生成的自定义镜像。
