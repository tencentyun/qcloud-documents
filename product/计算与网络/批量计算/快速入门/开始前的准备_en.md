## 1. Create a Tencent Cloud Account
If you don't have a Tencent Cloud account, log in to [Tencent Cloud Official Website](https://cloud.tencent.com/), and click "Sign Up" on the upper right corner of the page to create a Tencent Cloud account. For more information, please see the documentation [How can I Register a Tencent Cloud Account](
//cloud.tencent.com/document/product/378/3696).
## 2. Activate Batch
Currently, Batch is under **internal trial** which you need to apply for. Fill in the application in [Apply for Batch](https://cloud.tencent.com/act/apply/Batch) following the instructions on the page. You must completely and accurately fill in the options listed in the page. After the application is submitted, our staff will contact you and review your application manually. Then, the approval result is sent to you via SMS.
## 3. Obtain SecretId and SecretKey
After Batch is activated, if you want to work with Batch APIs via cloud API, SDK and CLI, SecretId and SecretKey are required. Create or view SecretId and SecretKey in [API Key Console](https://console.cloud.tencent.com/capi).
## 4. Understand Basic Concepts
To better work with this product, you must understand the meanings of relevant [Terms](//cloud.tencent.com/document/product/599/10396). You can also obtain the product information and data index through [Product Overview Page](https://cloud.tencent.com/product/Batch).
## 5. About COS
Both standard output and remote storage mapping of Batch are associated with COS. Therefore, you must have an understanding of [COS](
//cloud.tencent.com/document/product/436/6222).
