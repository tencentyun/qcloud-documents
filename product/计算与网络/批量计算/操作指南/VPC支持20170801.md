## VPC 支持
> 私有网络 VPC 是一块您在腾讯云上自定义的逻辑隔离网络空间，关于如何创建 VPC 可以参考文档：[如何创建私有网络](
> //www.qcloud.com/document/product/215/8119?!preview&lang=cn)。

以下是一个简单的例子，关于使用批量计算定义作业时，使用JSON描述如何设置网段：

	{
		"JobName": "",
		"JobDescription": "",
		"Priority": "",
		"Tasks": [
    {
      "TaskName": "",
      "Application": {
        "DeliveryForm": "",
        "Command": "" "
      },
      "ComputeEnv": {
        "EnvType": "",
        "EnvData": {
          "InstanceType": "",
          "ImageId": "",
          "VirtualPrivateCloud": {
            "VpcId": "vpc-01",//通过已创建的私有网络ID来设置网段
            "SubnetId": ""
          },
          "SystemDisk": {
            "DiskType": "",
            "DiskSize": 50
          },
          "DataDisks": [
            {
              "DiskType": "",
              "DiskSize": 50
            }
          ]
        }
      },
      ...
    }
 	 ]
	}' --Placement '{
 	 "Zone": ""
	}'
