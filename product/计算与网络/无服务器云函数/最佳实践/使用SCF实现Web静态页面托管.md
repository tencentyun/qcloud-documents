## 操作场景
本示例主要为您介绍如何通过云函数 SCF 结合 API 网关，快速实现一个对公网服务的 Web 页面。

## 操作步骤
1. 登录云函数控制台，选择左侧导航栏中的【[函数服务](https://console.cloud.tencent.com/scf/list)】。
2. 在“函数服务”页面上方选择**广州**地域，并单击【新建】进入新建函数页面。
设置以下参数信息，并单击【下一步】。如下图所示：
 - **函数名称**：命名为 “HtmlDemo”。
 - **运行环境**：选择 “Python 3.6”。
 - **创建方式**：选择【模板函数】。
 - **模糊搜索**：输入 “Web 静态页面托管”，并进行搜索。
 单击模板中的【查看详情】，即可在弹出的“模板详情”窗口中查看相关信息，支持下载操作。
![](https://main.qcloudimg.com/raw/cc5a87253d2a9af7c6590d86d6f19642.png)
4. 保持默认配置，单击【完成】，完成函数的创建。
5. 选择左侧的【触发管理】，并单击【创建触发器】。
6. 在弹出的“创建触发器”窗口中，根据以下信息为云函数添加 API 网关触发器。如下图所示：
![](https://main.qcloudimg.com/raw/e9db84f9f927b7e6fe06556fffe6312d.png)
主要参数如下，其余参数请保持默认设置：
 - **触发方式**：选择 “API网关触发器”。
 - **启用集成响应**：勾选。
7. 单击 API网关触发器的 “访问路径”，查看自定义的 HTML 页面。如下图所示：
![](https://main.qcloudimg.com/raw/4f0b9e29419f69aef24e922a9c016faf.png) 
>? 如果您需要自定义页面展示内容，可以在【函数代码】页签中进行配置。
