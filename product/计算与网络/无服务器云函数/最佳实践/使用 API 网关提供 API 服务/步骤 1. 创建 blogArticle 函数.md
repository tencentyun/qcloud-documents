在此部分中，将创建一个函数来实现博客文章的 API 响应。
1. 登录云函数控制台，选择左侧导航栏中的【[函数服务](https://console.cloud.tencent.com/scf/list)】。
2. 在“函数服务”页面上方选择**北京**地域，并单击【新建】进入新建函数页面。
设置以下参数信息，并单击【下一步】。如下图所示：
 - **函数名称**：命名为 “blogArticle”。
 - **运行环境**：选择 “Python 3.6”。
 - **创建方式**：选择【模板函数】。
 - **模糊搜索**：输入 “api_service”，并进行搜索。
  单击模板中的【查看详情】，即可在弹出的“模板详情”窗口中查看相关信息，支持下载操作。
![](https://main.qcloudimg.com/raw/b70f33a7e4f35943875cd094373535a4.png)
>!保存文章的数据结构采用 testArticleInfo 变量进行的保存和模拟，此处在实际应用中通常从数据库中或者文件中读取。
> 
4. 保持默认配置，单击【完成】，完成函数的创建。
