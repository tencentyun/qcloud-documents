在此部分中，将创建一个函数来实现博客文章的 API 响应。
1. 登录 [云函数控制台](https://console.cloud.tencent.com/scf/list?rid=8&ns=default)，进入【函数服务】页面。
2. 选择**北京**地域，单击【新建】，进入新建函数页面。
3. 填写以下参数信息，单击【下一步】。如下图所示：
 - **创建方式**：选择 “模板函数”。
 - **函数名称**：命名为 “blogArticle”。
 - **模板搜索**：选择 “语言” 为 “Python 3.6” 的 “api_service” 模板。
  鼠标移至模板函数上，可查看模板函数详情，支持下载操作。
 ![](https://main.qcloudimg.com/raw/677b92867662abb78e6b522260496a00.jpg) 
>!
> 保存文章的数据结构采用 testArticleInfo 变量进行的保存和模拟，此处在实际应用中通常从数据库中或者文件中读取。
> 
4. 保持默认配置，单击【完成】，完成函数的创建。
