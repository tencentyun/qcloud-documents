## 2. 轻量应用服务器的计费模式
腾讯云轻量应用服务器计费分为**基础套餐**及**套餐外超额流量**两部分，其中基础套餐仅支持包年包月模式。全面了解轻量应用服务器的计费模式，有助于您选择最优的计费方案。详情请参见 [计费概述](https://cloud.tencent.com/document/product/1207/44368)。

## 3. 新手入门
#### 3.1 注册与认证
在使用腾讯云轻量应用服务器之前，您需要 [注册腾讯云账号](https://cloud.tencent.com/register?s_url=https%3A%2F%2Fcloud.tencent.com%2F) 并完成 [实名认证](https://cloud.tencent.com/document/product/378/3629)。


#### 3.2 购买轻量应用服务器
完成注册与实名认证后，您可以前往 [轻量应用服务器购买页](https://buy.cloud.tencent.com/lighthouse) 灵活地选择实例所在地域、套餐、镜像、购买数量及购买时长，以确保满足您实际的业务需求。详情请参见：
- [快速配置轻量应用服务器 Linux 实例](https://cloud.tencent.com/document/product/1207/44548)
- [快速配置轻量应用服务器 Windows 实例](https://cloud.tencent.com/document/product/1207/44549)


#### 3.3 登录与使用轻量应用服务器
当您购买完成后，您可以登录您购买的轻量应用服务器，并开始快速构建个人网站、博客、电商等各类应用或开发测试环境。
- 关于如何登录轻量应用服务器，详情请参见：
 - [登录 Linux 实例](https://cloud.tencent.com/document/product/1207/44642)
 - [登录 Windows 实例](https://cloud.tencent.com/document/product/1207/44656)
- 关于建站、搭建环境等相关操作，详情请参见 [最佳实践](https://cloud.tencent.com/document/product/1207/45117)。
