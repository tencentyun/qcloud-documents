本文将为您详细介绍如何释放内网 IP。
1. 进入 [私有网络控制台](https://console.cloud.tencent.com/vpc) 。
2. 单击左侧目录中的【弹性网卡】，进入弹性网卡列表页。
 ![](https://main.qcloudimg.com/raw/1b7c9f62f15d135356f1cd29f109a558.png)
3. 单击需要查看的实例 ID，进入详情页。
4. 单击选项卡中的【IP 管理】，查看已绑定的内网 IP 和弹性公网 IP。
 ![](https://main.qcloudimg.com/raw/1b8201bdbfcce48faaf4b632e51d7608.png)
5. 单击需要释放的内网 IP 所在行的【释放】。
6. 单击【确认】完成操作。
 ![](https://main.qcloudimg.com/raw/d7e3834612a2139a6f84799070cf6011.png)
>**注意：**
>- 弹性网卡仅辅助 IP 支持释放，主 IP 不支持释放。
>- 内网 IP 解绑后，将自动解解绑关联的弹性公网 IP。
