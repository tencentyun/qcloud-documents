## 操作场景
该任务指导您通过控制台，创建某个地域下的服务。

## 操作步骤
1. 登录 [API 网关控制台](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fcloud.tencent.com%2F)。
2. 在左侧菜单栏中，选择【[服务](https://console.cloud.tencent.com/apigateway/service)】，进入服务列表页。
3. 在服务列表页的顶部，选择服务地域，单击左上角的【新建】，开始新建服务。
>?若当前地域服务数量已经达到最大值50个，则不能继续创建服务。

4. 在新建服务的对话框中，填写服务名称、备注信息，并选择前端类型（http、https、http 和 https 任一种） 。
>?服务名有特定输入限制：最长50个字符，支持 a-z、A-Z、0-9、_。

5. 单击【提交】，服务创建成功。您可以在该服务下进行 [API 创建](https://cloud.tencent.com/document/product/628/11775)。
