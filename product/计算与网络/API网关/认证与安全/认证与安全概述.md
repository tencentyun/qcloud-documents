
为了保护您的 API，避免恶意访问、未授权访问、应用漏洞、黑客攻击等导致的数据损失、资产损失，我们提供了多种 API 认证方式和 API 防护策略。

目前 API 网关主要有 [应用认证](https://cloud.tencent.com/document/product/628/55088)、[OAuth2.0认证](https://cloud.tencent.com/document/product/628/38393) 、[EIAM 认证](https://cloud.tencent.com/document/product/628/59669)、[密钥对认证](https://cloud.tencent.com/document/product/628/42189)（历史功能，建议使用应用认证） 四种方式。
