API Gateway logs the version of each service published in an environment, allowing you to switch versions. The version record contains the publishing time and the publishing remarks you entered. You can view the version record on the Version page.
![Publishing records](//mc.qcloudimg.com/static/img/4436edcdaa0a23d57491648f9a63f772/image.png)

To switch versions, select the environment in which the version is to be switched on the Environment page, click **Switch to Version** under the Operation, select the version to switch to, and click **Submit**.
![Switching version](//mc.qcloudimg.com/static/img/085a6ca16bbe80433e7dcf98b72d0db0/image.png)
