The service management module is used to manage a group of associated APIs. A group of associated APIs can be used to achieve some specific functions which can be regarded as a service.
This module includes [Service Creation](https://cloud.tencent.com/document/product/628/11787), [Service Editing](https://cloud.tencent.com/document/product/628/11789), [Service Deletion](https://cloud.tencent.com/document/product/628/11790), [Service Domain Name](https://cloud.tencent.com/document/product/628/11827) and [Independent Domain Name and Certificate](https://cloud.tencent.com/document/product/628/11791).
