创建服务成功后，即可从服务详情中获取访问服务的子域名。服务的子域名内会包含地域，您的用户 ID 和服务名信息，形如 `http://{your-unique-id}.{region}.apigateway.myqcloud.com`， 同时 API 网关服务也已经启用泛域名 SSL 证书，因此服务的子域名可以通过 HTTPS 协议访问，访问地址形如 `https://{your-unique-id}.{region}.apigateway.myqcloud.com`

示例：`service-rnkc5qj2-251000691.ap-guangzhou.apigateway.myqloud.com`
