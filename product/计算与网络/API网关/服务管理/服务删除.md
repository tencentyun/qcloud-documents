## 操作场景
该任务指导您通过 API 网关控制台，删除不需要的服务。

## 操作步骤
1. 登录 [API 网关控制台](https://console.cloud.tencent.com/apigateway/index?rid=1)，在左侧导航栏选择【服务】。
2. 在服务列表中选择需要删除的服务，单击【删除】。
![](https://main.qcloudimg.com/raw/f6974a427a22febcff9b924ccde76818.png)
3. 在确认删除的弹窗中，单击【确认】，即可删除该服务。
![](https://main.qcloudimg.com/raw/87aea172ce5495838f3a1bb1b0ef2c97.png)

## 注意事项
1. 选择服务支持多选，最多可以同时选择10个服务。
2. 删除服务后，该服务下的API将不再提供服务，删除的服务也无法恢复，请您谨慎操作。
3. 腾讯云 Serverless 团队提供了快速删除工具，通过简单的配置后，即可一键、批量删除资源，关于该工具请参考：[快速删除工具](https://cloud.tencent.com/document/product/628/43817)。
