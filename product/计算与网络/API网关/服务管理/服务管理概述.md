服务管理模块用来管理一组有关联的 API。通常通过一组相关 API ，完成特定功能的实现，这些功能即可认为是一个服务。
本模块的内容包括[服务创建](https://cloud.tencent.com/document/product/628/11787)、[服务编辑](https://cloud.tencent.com/document/product/628/11789)、[服务删除](https://cloud.tencent.com/document/product/628/11790)、[服务域名](https://cloud.tencent.com/document/product/628/11827)和[独立域名及证书](https://cloud.tencent.com/document/product/628/11791)。
