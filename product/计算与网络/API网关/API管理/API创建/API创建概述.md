API 创建即在 API 网关内完成 API 的定义。API 创建包含了如下若干步骤：
* [API 基础信息配置](https://cloud.tencent.com/document/product/628/11796)
* [前端配置](https://cloud.tencent.com/document/product/628/11797)
* [后端配置](https://cloud.tencent.com/document/product/628/11798)
* [响应配置](https://cloud.tencent.com/document/product/628/11804)

完成以上步骤，即完成了 API 在服务内的创建。
