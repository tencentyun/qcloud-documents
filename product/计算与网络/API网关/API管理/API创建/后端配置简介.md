API 的后端配置，是指的实际提供真实服务的配置。API 网关会将前端请求，依据后端配置进行转换后，转发调用到实际的服务上。

API 网关提供的后端配置，包括 [对接后端 HTTP 服务](https://cloud.tencent.com/document/product/628/11802)，[对接 Serverless Cloud Function](https://cloud.tencent.com/document/product/628/11801) ，和[对接 mock](https://cloud.tencent.com/document/product/628/11803) 。
