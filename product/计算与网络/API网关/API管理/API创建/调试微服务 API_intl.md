After an API is created, you can find the debugging feature on the API details page.
![](https://main.qcloudimg.com/raw/29c129359592b0fb700a1ab157dcd6ff.png)

Enter the corresponding request parameters on the debugging page, and then send the request.
![](https://main.qcloudimg.com/raw/94e4f9329c7102475cec7d03bf8e17b2.png)

The parameters X-NameSpace-Code and X-MicroService-Name in the figure above are required. To configure other parameters, see [API Debugging](https://cloud.tencent.com/document/product/628/12005) for detailed rules.
