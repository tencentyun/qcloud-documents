API creation is to define an API in the API gateway. API creation includes the following steps:
* [API Basic Information Configuration](https://cloud.tencent.com/document/product/628/11796)
* [Frontend Configuration](https://cloud.tencent.com/document/product/628/11797)
* [Backend Configuration](https://cloud.tencent.com/document/product/628/11798)
* [Response Configuration](https://cloud.tencent.com/document/product/628/11804)

An API is created in the service when you have completed the above steps.
