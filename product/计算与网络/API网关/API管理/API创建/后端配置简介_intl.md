The backend configurations of an API refer to the configurations used to provide actual services. API Gateway converts frontend requests based on the backend configurations, and forwards them to the actual service.

The backend configurations provided by the API Gateway include [Integration with Backend HTTP Services](https://cloud.tencent.com/document/product/628/11802), [Integration with Serverless Cloud Function](https://cloud.tencent.com/document/product/628/11801), and [Integration with Mock](https://cloud.tencent.com/document/product/628/11803).
