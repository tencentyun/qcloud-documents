## 操作场景
该任务指导您通过 API 网关控制台，查看服务的实时监控信息。


## 操作步骤
1. 登录 [API 网关控制台](https://console.cloud.tencent.com/apigateway/index?rid=1)，在左侧导航栏选择【服务】。
2. 在服务列表中单击服务名，进入服务详情页。
3. 在服务详情页顶部，单击【监控】。
通过选择服务、环境，可以查看某服务已发布环境的监控信息，包括调用量，流量，响应时间，错误情况等多种统计信息。
