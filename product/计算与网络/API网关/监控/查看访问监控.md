## 操作场景
该任务指导您通过 API 网关控制台，查看服务的实时监控信息。

## 操作步骤
1. 登录 [API 网关控制台](https://console.cloud.tencent.com/apigateway/index?rid=1)，在左侧导航栏单击【服务】，进入服务列表页。
2. 在服务列表中单击服务名，进入服务详情页。
3. 在服务详情页顶部，单击【监控】，即可查看服务的实时监控信息。
![](https://main.qcloudimg.com/raw/8164308172058339215f7dd0e83c49fd.png)

## 监控粒度
API 网关支持查看环境粒度和API粒度的监控图表：
- 选择环境，API 选择”全部“选项，可以查看某服务已发布环境的监控信息，包括调用次数、流量、响应时间、错误情况等多种统计信息。
- 如果您需要查看具体 API 的监控，可选择具体的 API，查看对应的监控信息。
