API gateway is charge for the number of API calls and outbound traffic. It is free during public trial.

#### Fee of number of calls

API calls are billed **per million calls/month**. In a billing period, the part less than a million calls is billed as a million calls. The fee is accurate to two digits after the decimal point.

The rate is **0.6 USD/million calls**.

#### Outbound traffic fee

The outbound traffic fee charges by CVM outbound traffic. See [Here](https://cloud.tencent.com/document/product/213/509) for specific expenses.
