## 概述
API 网关目前提供共享集群和性能保障集群供您选择：
- 共享集群是 API 网关默认的集群类型，提供标准的 API 网关服务，用户无需开通即可使用。共享集群服务的默认 QPS 上限为5000，适用于业务规模较小、对性能要求不高的用户；
- 性能保障型集群的产品功能与共享集群基本相同，需要用户申请后走线下报价流程开通。性能保障集群服务的 QPS 上限根据实例规格的不同而不同，最高可达350000，适用于业务规模较大，对性能要求较高的用户。

## 集群规格

| 规格 | 共享集群 | 性能保障集群 |
|---------|---------|---------|
| 资源   | 由一组 region 下的 API 网关用户所共有 | 用户独享 |
| 收费方式 | 后付费 | 线下报价 |
| 计费项 | 调用次数 + 外网出流量 | 实例费用 + 外网出流量 |
| QPS | 5000 | 210000 - 350000 |

## 性能保障集群选购说明
- 目前性能保障型集群不支持自主选购，您可以提交 [API 网关性能保障型集群申请](https://cloud.tencent.com/apply/p/u1rln6426s)，会有专门的商务人员与您沟通。
- 如果您在使用性能保障型集群前已经在使用 API 网关共享集群，API 网关会提供专门的技术人员帮助您迁移业务。
