The dedicated CVM instances have the same lifecycle as ordinary CVMs, which refers to the states of the dedicated CVMs assuming from start to termination. Using Tencent Cloud console to manage the instance during its life cycle can help ensure that the applications on CVM render services in an efficient and cost-effective manner.

> The dedicated CVM instances have the following states:

- Creating
- Running
- Restarting
- Shutting down
- Shut down successfully
- Resetting
- Terminating
- Terminated
