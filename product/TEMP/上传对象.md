## 上传步骤
### 1. 进入文件列表
 登录 [对象存储控制台](https://console.cloud.tencent.com/cos5)，单击【存储桶列表】，进入存储桶列表。选择需要存储对象的存储桶（如`example-1253833564`），进入存储桶的文件列表页面。在文件列表中，单击【上传文件】，弹出上传文件界面。
  ![文件列表](//mc.qcloudimg.com/static/img/f1ded4b0ac6cbe8753d40bf3c2bd935c/image.png)
### 2. 选择上传对象
在上传文件界面中，单击【选择文件】或【选择文件夹】，可上传单个或多个本地文件/文件夹，选择本地需要上传的对象。此时您可直接单击【上传】按钮上传对象，或单击【下一步】设置对象属性后再上传（见第 3 步）。
![选择上传对象](https://main.qcloudimg.com/raw/c198c8f1a1cfa6bcc0b11dc73e0f8eea.png)
### 3. 设置对象属性（可选）
设置好待上传文件的存储类型、访问权限，元数据信息（可选）并单击【上传】开始上传文件。存储类型默认为标准存储、访问权限默认为继承权限、元数据信息可暂不填写。对象上传成功后，系统会自动刷新列表，获取最新对象信息。
![设置对象属性](https://main.qcloudimg.com/raw/496fe578c683b20bdf9d5d0aef0613d1.png)

> **注意：**
> 部分浏览器不支持多文件上传，建议使用 IE10 以上、Firefox、Chrome、QQ 浏览器等主流浏览器。

## 上传说明
- 对象存储类型
根据不同的应用场景，您可以为不同的对象设置不同的存储类型，默认存储类型为标准存储。具体的存储类型介绍请参考 [存储类型](https://cloud.tencent.com/product/cos/storage-types)。

- 对象访问权限
可以为不同的对象设置不同的访问权限，默认访问权限为继承权限（即继承存储桶权限）。具体的对象访问权限信息请参考 [设置对象的访问权限](/document/product/436/13327)。

- 服务端加密
您在上传对象的同时，配置对象的服务器加密属性。腾讯云 COS 将为上传的对象应用数据加密的保护策略，帮助您在数据写入前自动加密，并在您访问该数据时自动解密。目前腾讯云 COS 支持使用主密钥对数据进行 AES-256 加密。


- 对象元数据
元数据是服务器以 HTTP 协议传 HTML 资料到浏览器前所送出的字串，又称为 HTTP Header。通过修改 HTTP Header，可以改变页面的响应形式，或者传达配置信息，例如修改缓存时间。修改对象的 HTTP Header 不会修改对象本身。具体信息请参考 [自定义对象 Header](/document/product/436/13361)。

- 文件数量和容量大小限制
  每个存储桶对存储的文件数量无限制。通过对象存储控制台上传文件最大支持 512 GB。上传带宽可达1G/S以上，实际传输速率受网络带宽限制
