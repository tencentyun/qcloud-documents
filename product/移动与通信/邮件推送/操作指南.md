## 操作流程
您可以通过以下4个步骤完成邮件推送的操作。
1. 通过 [控制台](ses.myqcloud.com) 填写相应资料，提交邮件服务申请。
2. 进入 [控制台](ses.myqcloud.com)，选择【使用代码创建模板】进行相关发送模板的创建。（可选操作）
3. 成功申请发件人之后，可在 [控制台](ses.myqcloud.com) 进行手动邮件测试。
4. 想要获取更好的发送效果，可联系商务进专用 IP 的单独购买。

## 操作步骤
### 步骤1：邮件服务申请
进入 [控制台](ses.myqcloud.com)，选择邮箱登录或者腾讯云账号登录，选择 【Sender Application】，填写申请信息，注明邮件类型，邮件发送量级，点击提交。如下图：
![](https://main.qcloudimg.com/raw/3ab7d237bd539337cc6522282037ba33.png)

提交之后，会有专人24小之内与您取得联系。
如果需要自定义发件域名，请保证您拥有该域名的配置权限，腾讯云会返回一份 DNS 配置表单，您需要按照表单内容配置您的 DNS 内容。配置之后，您可以 [自助检查 DNS](https://www.whatsmydns.net/) 是否配置正确以及同步情况。

### 步骤2：模板新增
>? 可选操作，不使用模板也可进行发送。
>

进入 [控制台](ses.myqcloud.com)，选择【使用代码创建模板】，在输入框中填写 HTML 代码或者 TEXT 文本，模板中的变量使用 `{{变量名}}` 表示。
![](https://main.qcloudimg.com/raw/9e1f74b778800f7c0ad0223bda44455c.png)

### 步骤3：发送测试
成功申请发件人之后，您可以在 [控制台](ses.myqcloud.com) 进行手动邮件测试，如下图：
![](https://main.qcloudimg.com/raw/d7779a800d7df24ea6f869877261e9b2.png)
到此，您的 DNS 初始化工作已完成。

### 步骤4：专用 IP
如果您想要取得更好的发送效果，强烈建议您使用专用的发送 IP，以便独立控制您的发送声誉，不会被其它账户的发送所影响。专用 IP 需要单独购买，如需购买，请联系商务。
