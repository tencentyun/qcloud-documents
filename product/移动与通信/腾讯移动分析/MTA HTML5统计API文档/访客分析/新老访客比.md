### 功能说明
按天查询当天新访客与旧访客的数量。

**接口 URL：**`http://mta.qq.com/h5/api/ctr_user_compare`；
**HTTP 请求方式：**GET。

### 参数

|  参数名| 类型 | 含义 |备注 |
|---------|---------|---------|---------|
| app_id | 整数 | 	应用 ID |应用注册时生成的 ID |
| start_date | 字符串 | 	开始时间 |开始时间(Y-m-d) |
| end_date | 字符串 | 	结束时间 |结束时间(Y-m-d) |
| sign | 字符串 | 	验证串 |见示例生成过程 |
