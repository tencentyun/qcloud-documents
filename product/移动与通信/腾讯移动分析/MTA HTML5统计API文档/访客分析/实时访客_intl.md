### Feature Description
Query the App's real-time visitor information within 24 hours.

**API URL:** `http://mta.qq.com/h5/api/ctr_user_realtime`
**HTTP request method:** GET.

### Parameters

| Parameter Name | Type | Meaning | Note |
|---------|---------|---------|---------|
| app_id | Integer | App ID | The ID generated during application registration |
| page | Integer | Page number | 200 records per page |
| sign | String | Verification string | See the example of generation process |
