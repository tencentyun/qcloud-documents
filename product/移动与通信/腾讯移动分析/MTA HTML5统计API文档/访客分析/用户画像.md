### 功能说明
在 H5 统计平台查询用户画像数据，包含性别比例、年龄分布、学历分布、职业分布，接口的数据为 pv 量；

**接口 URL：**`http://mta.qq.com/h5/api/ctr_user_portrait`；
**HTTP 请求方式：**GET。

### 参数

| 参数名 | 类型 | 含义 |备注 |
|---------|---------|---------|---------|
| app_id | 整数 | 应用 ID |应用注册时生成的 ID |
| start_date | 字符串 | 	开始时间 |开始时间(Y-m-d) |
| end_date | 字符串 | 	结束时间 |结束时间(Y-m-d) |
| idx | 字符串 | 	指标列表 |可选值详见附录（指标字典），使用“,”间隔查询指标 |
| sign | 字符串 | 	验证串 |见示例生成过程 |

### 可查询指标

| 指标 | 指标解释 | 计算方式 |
|---------|---------|---------|
| sex | 性别 | - |
| age | 年龄 |-  |
| grade | 学历 | - |
| profession | 职业 | - |
