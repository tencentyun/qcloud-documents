We provide different event tracking standards for different industries. Event tracking standard for financial network lending is available now, and standards for other industries will be available soon. Details are shown below:

## Financial Network Lending
After downloading the App, users can track events through a complete linkage of **Register** -> **Log in** -> **Fill in the information** -> **Identity authentication** -> **Apply for loans** -> **Repay**. Irreversible encryption algorithm is supported for sensitive data. The events to be tracked fall into four types, as shown below.
