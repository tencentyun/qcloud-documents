### Is Mobile Tencent Analytics a paid service?
The existing features of Mobile Tencent Analytics are available for free.
### Is the number of custom events limited?
The number of custom events is unlimited.
### Does Mobile Tencent Analytics support overseas product statistics?
Mobile Tencent Analytics has deployed its server in Hong Kong to support the statistics of products released overseas.
### How does Mobile Tencent Analytics count the geographical data?
The geographical data is matched by IP address. For the WiFi network, the statistics is based on the area where WiFi is located. For the mobile network, the statistics is based on number location.
### Does iOS SDK use hot patching or private APIs?
MTA SDK does not use hot patching or private APIs, so the Apple review will not be affected.
