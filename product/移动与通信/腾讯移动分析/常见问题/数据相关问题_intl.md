### How are users defined in Mobile Tencent Analytics?
Users of Mobile Tencent Analytics are judged based on devices. Devices are uniquely identified based on imei and openUDID respectively in Android and iOS systems.
### If two different channel packages are installed on the same device, which channel does the device belongs to?
Only the initial installation channel is recorded for the same device.
### A new channel has been released today. Why doesn't it appear in the channel report?
The channel data is not updated in real time. The newly released channel can be seen the next day.
### Both Android and iOS Apps do not report QQ data. Why does Android have user profile data while iOS does not?
Android Apps can match user profile data by matching device data, but iOS platform uses the QQ account to match the profile data due to the ID collection limitation.
