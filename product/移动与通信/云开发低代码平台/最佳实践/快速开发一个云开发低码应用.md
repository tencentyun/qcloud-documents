## 步骤1：创建应用

1. 登录 [腾讯云微搭低代码](https://console.cloud.tencent.com/lowcode) 控制台。
2. 单击【新建应用】，填写应用信息：
   - **应用名称**：输入应用名称。
   - **应用描述**：用于描述应用。
   - **应用环境**：提供“自动创建（默认）”和“选择已有环境”两种方式，本文以“自动创建（默认）”为例，从头开始新建环境。
     ![](https://main.qcloudimg.com/raw/87bfffa717af0fff75b5c37848da212b.png)
3. 单击【确定】，即可创建应用。



## 步骤2：创建数据源模型

1. 单击左侧主菜单栏中的【数据源管理】，进入数据源管理页面。
2. 单击【新建数据源】，在下拉菜单中选择【数据源模型】。
![](https://main.qcloudimg.com/raw/486f1fe766ceb93018ec412eb99e44f1.jpg)
3. 进入新建数据源页面，配置如下信息：
	- **数据库名称**：输入数据库名称。
	- **数据源标识**：作为数据源模型的唯一标识，不可重复。
	- **字段配置**：一个数据源模型可由多个字段组成。例如“人员”数据源模型中可有“姓名、年龄”等字段，由此模型可创建“管理人员、员工、游客”等数据源变量。注意  \_id、 createdAt、updatedAt 为数据源内置字段，不允许修改。 单击【添加】，页面右侧将弹出字段设置小窗口，配置如下相关信息：
		- 字段名称：输入字段名称。
		- 字段标识：作为字段的唯一标识，不可重复。
		- 数据类型：选择相对应的数据类型。
		- 默认值：可以给字段预设一个默认值。例如国家字段可设置默认值为：中国。
		- 是否必填：字段是否必填。
		- 是否枚举：该字段是否只能在有限个枚举值中取值。例如性别字段，可设置枚举值为：男、女。
	- **可操作动作**： 可以通过设置动作来设置数据源的使用方式，可设置多种动作以配合不同场景使用。系统默认配置了新增、删除、更新等动作，可满足多数场景。若有自定义的动作需求，可通过云函数或本地函数的方法进行编写。
  ![](https://main.qcloudimg.com/raw/da81e723a80152941fda7b57b4f3a2a1.png)
4. 字段配置完成之后，单击【确定】，即可添加字段，您可以添加多个字段。
   ![](https://main.qcloudimg.com/raw/9498c860465286a389bfaed40d9852f2.png)
5. 单击【立即创建】，即可完成数据源模型的创建。

### 添加数据源变量[](id:adddata)

通过为全局页面或指定页面设置变量，可以记录或使用变量传递的信息。参数变量用于传递页面间的参数传递，需在待获取参数的页面上定义参数变量，再在传参页面配置事件用于传参；状态变量用于记录当前页面内的变量信息。可根据以下步骤添加变量：

1. 在页面顶部单击【变量管理】，系统将弹出编辑窗口。
2. 在编辑窗口左侧单击【全局】>【状态】右侧的【+】，添加全局状态变量。
3. 配置数据源信息。变量类型选择【数据源模型】可以有数据源模型创建变量。
4. 单击【确定】，即可创建变量。
   ![](https://main.qcloudimg.com/raw/28c30259c5379463d87d498125d9733d.png)

### 配置列表页面

#### 添加组件

1. 在左侧组件面板中，依次在相关分类着找到相关组件，单击【组件】添加相关组件。
2. 在左侧可直接单击组件，添加组件。
   - 通用（分类）> Container（组件）【即老版本的 View 组件】
   - 展示（分类）> Item（组件）【即老版本的 Cell 组件，配置纬度更丰富】
3. 若 Item 组件不在 Container 中，可在组件树中拖拽 Item 来调整其层级结构。
   ![](https://main.qcloudimg.com/raw/292225c4f3556e6d68408d642918664f.png)

#### 配置 Item 组件，并为其添加数据源变量绑定

1. 选中 Item 组件，在页面右侧弹出的配置框中，按下图配置，单击绑定图标后继续按下一步操作。
   ![](https://main.qcloudimg.com/raw/8c675e49948de785179cca2bca36086d.png)
2. 选择【状态变量】，根据实际需求绑定数据，单击【确定】即可。
	 ![](https://main.qcloudimg.com/raw/e34928c524c523d4eb280751b2ea3711.png)
3. 为 Item 组件的标题添加 For 循环绑定。
   ![](https://main.qcloudimg.com/raw/96928f0abb8c103233ca291e343ef769.png)

#### 处理无数据场景

Container 组件和 Item 组件均绑定数据源变量，当数据源变量为空时，需要进行细节处理。

<dx-tabs>
::: Container\s组件处理
为 Container 组件设置条件显示绑定，绑定方式为表达式 `$page.dataset.state.userList && $page.dataset.state.userList.length`，仅当绑定的数据源变量有值时才显示。
  ![](https://main.qcloudimg.com/raw/8b69974b5949a13c7576c20e7764f7fd.png)
:::
::: Item\s组件处理
在从左侧组件面板中添加 Text 组件，用于无数据时展示提示信息，绑定表达式 `!$page.dataset.state.userList || !$page.dataset.state.userList.length`。
  ![](https://main.qcloudimg.com/raw/e33998c340cc951c828a6bb31e6d61dc.png)
:::
</dx-tabs>



#### 添加“登记新用户”按钮

1. 在页面左侧单击【组件】。
2. 选择【表单组件】，单击【Button】组件，添加一个按钮。
3. 选中该组件。
4. 在页面右侧，找到【自定义配置】中的“按钮文字”配置，将按钮名称改为“登记新用户”。
   ![](https://main.qcloudimg.com/raw/51d9e0533520861cd6f2184aff423c5e.png)
5. 该按钮用于跳转至“登记新用户”页面，然而该页面还未创建，请参考下一步创建页面。



### 编辑“登记新用户”页面

#### 添加新页面

1. 在页面左侧，选择【页面】>【创建新页面】。
   ![](https://main.qcloudimg.com/raw/79be2cc4f97c234a2f20ebe53b8eff73.png)
2. 在弹出的配置框中，配置页面相关信息：
	- **标题**：页面顶部标题。
	- **页面 ID**：页面的 ID，也作为页面的路径。必须以字母开头，推荐填写英文名或拼音。
![](https://main.qcloudimg.com/raw/904465ba7ce8ed94cab4bfa8492183f6.png)
3. 单击【确认】，即可添加新页面。



#### 使用数据源变量来创建表单

腾讯云微搭低代码平台提供了强大的基于数据源模型创建表单的功能, 只需两步即可完成表单页面的创建.

1. 先按照上述步骤 [添加数据源变量](#adddata)，为 `createUser` 页面添加一个名称为 `newUser`、类型为 `新记录` 的数据源变量。
![](https://main.qcloudimg.com/raw/d76ec6cb7c31098fcecbb410f2c2ce5a.png)
2. 向页面添加表单控件，并关联刚添加的数据源变量。
	1. 在页面左侧，单击【组件】。
	2. 选择【表单组件】，找到 Form 组件，单击添加该组件。
	3. 在页面右侧的配置面板中，找到【数据源变量绑定】区域，将【表单类型】设置为 `新建`、将【数据源变量】设置为刚刚添加的  `newUser` 变量。
	 ![](https://main.qcloudimg.com/raw/cdafca08afc76ce2de8f340927d77231.png)

按以上步骤配置完成后，编辑表单即自动产生，表单的提交事件也添加了默认处理。



### 完善页面跳转

完善列表页面跳转至表单页的配置

至此两个页面都已创建完成，但是页面跳转逻辑还不完善。我们再回到列表页面，补全这个配置：

1. 在页面左侧选择第一个页面。
2. 选择 button 组件库。
3. 单击页面右侧的【事件】，进入事件配置。
4. 触发条件选择【button点击】。
5. 动作类型选择【平台方法】。
6. 执行动作选择【navigateTo 导航】。
   ![](https://main.qcloudimg.com/raw/31741fd75ab855db512150daec13152e.png)
7. 完成上述步骤之后，单击【确定添加】来添加该处理函数, 并将参数 「页面ID」改为 登记新用户页面的ID `createUser`。
![](https://main.qcloudimg.com/raw/cf98fb57b32d08c890d3007e266971fb.png)



### 保存应用

经过以上步骤，我们的应用已经配置完成，请及时保存工作成果。

1. 单击页面顶部的【保存】。
2. 在弹出的窗口中，输入本次变更描述，可根据实际变更内容填写。
3. 单击【保存】，即可保存应用。
   ![](https://main.qcloudimg.com/raw/ffaa9ca53656987387acdd5ecad12045.png)

## 步骤4：预览及发布应用

至此，我们已经完成应用开发配置的所有工作，接下来可以在线预览以查看我们应用在 Wb 和小程序平台的真实体验效果，体验测试通过后，可在线发布至相关平台。

1. 单击页面顶部的【预览发布】，在弹出的部署应用窗口中进行预览发布配置。
2. 部署模式选择【预览】。
3. 部署方式可选择【本地】或【云端】 。本地部署速度更快，耗时约是云端部署的`5%`。本地部署需安装 Node.js，并使用命令行安装 CLI 工具，仅需首次安装即可。
4. 部署平台选择【网页H5】。
5. 单击【预览】。
   ![](https://main.qcloudimg.com/raw/481ccb44fc16a9a462cd6f4f17cee7e7.png)
6. 完成上述步骤之后，若选择云端部署，低码应用将在服务器端进行部署，等待3 - 5分钟后即可看到部署的结果。
   ![](https://main.qcloudimg.com/raw/5a10031c4aa5cdde4e5b811881795270.png)
7. 部署成功后，即可看到二维码（若选择小程序，则显示小程序的二维码），可使用手机直接扫码体验。
   <img src="https://main.qcloudimg.com/raw/1098afeae72524878253bc7ca938c339.png" width="50%"></img>
8. 若选择将低码应用发布至小程序平台， 部署完成后，我们还需要版本管理提交发布操作，详情参见 [应用发布到小程序](https://cloud.tencent.com/document/product/1301/55140)。
