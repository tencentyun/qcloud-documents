## 模板简介
本类型的电商模板在安装后会附带对应的 B 端管理后台，模板包含了商品展示，加入购物车，下单，支付，物流信息查看，个人中心，订单状态查看等功能，同时可以使用附带的管理后台对小程序的商品、订单、页面样式进行管理。本篇文章以餐饮电商模板为例来介绍该类型模板的详细使用方法。

## 购买并安装模板
### 购买和试用模板
[微搭应用市场](https://weda.cloud.tencent.com/) 选购餐饮电商小程序，单击**试用7天**或**立即购买**下单，下单后跳到模版管理安装模版。
![](https://main.qcloudimg.com/raw/d9de25bc81225436ed249a7251d3cdcf.png)

### 安装模版
1. 进入**模板管理** > **我的模板**，在模版列表找到已购买的模板并单击**安装**。
![](https://qcloudimg.tencent-cloud.cn/raw/f169cb0cc58f17653eaea6da61e3165f.png)
2. 填写相应信息单击**确定**后，生成可拖拽的店铺应用。
<img src = "https://qcloudimg.tencent-cloud.cn/raw/1f45c1a2610702bb08fa3d5e8855a971.png" style="width: 80%"> 

### 查看已安装的模板
完成模版安装后，进入 [**应用管理**](https://console.cloud.tencent.com/lowcode/app/owner) 页面，会生成2个应用，一个是小程序，一个是电商管理后台。
![](https://qcloudimg.tencent-cloud.cn/raw/2d03b693a44e21d8aad1139cfd9d5277.png)


## 发布小程序和管理后台

### 发布小程序

微搭支持快速将应用部署到小程序中，详情请参见 [应用市场新用户使用手册](https://cloud.tencent.com/document/product/1301/62015)。

### 发布管理后台
1. 单击小程序管理后台应用模块(带管理平台后缀)进入**应用编辑器**页面。
![](https://qcloudimg.tencent-cloud.cn/raw/1377e36d4be010541aa13f8fa412ff1a.png)
2.  单击上方**发布**后，按下述步骤进行管理后台的发布：
![](https://qcloudimg.tencent-cloud.cn/raw/b5af544f4f1ef2051a78c8093e2c212b.png)
	1. 在弹出的弹窗中，根据小程序版本选择发布方式（与小程序发布方式保持一致）单击**下一步**。
	<img src = "https://qcloudimg.tencent-cloud.cn/raw/2b852f4206eff6c2f788770859be4a1c.png" style="width: 80%"> 
	2. 此处提示需要发布的资源全部进行勾选，单击**下一步**即可进行后台发布。
<img src = "https://qcloudimg.tencent-cloud.cn/raw/f3c08769cb5fc775119f24e13a19cec3.png" style="width: 80%">  
	3. 发布成功后会展示管理后台的登录链接，单击**企业工作台访问链接**即可跳转至管理后台页面。管理后台的登录方式可参见 [登录管理后台](#login)。
<img src = "https://main.qcloudimg.com/raw/acd43de48a5f3835f47b7a53bca47ea7.png" style="width: 80%">  	


## 管理后台使用说明

### 登录管理后台[](id:login)
目前管理后台使用了微搭企业级能力进行构建，因此登录时需要先在微搭的 [用户管理](https://console.cloud.tencent.com/lowcode/permission/user) 模块进行用户创建以及角色绑定，具体操作步骤如下：

1. 进入 [用户管理](https://console.cloud.tencent.com/lowcode/permission/user) 页面，单击**新建用户**。
![](https://main.qcloudimg.com/raw/21e28ba65f4aed2d6d9f6f07e1c3d9f9.png)
2. 在弹窗内填写用户名、密码以及其他信息，**关联角色**一栏选择**超级管理员**，填写完成后单击**确定**。
<img src = "https://main.qcloudimg.com/raw/38207672b890e770c9963a5157b9cef8.png" style="width: 80%">  	
3. 在管理后台登录页面使用刚刚创建的用户名和密码，即可成功进入后台页面。
![](https://main.qcloudimg.com/raw/8d962362552912afddbbf8426e6985c7.png)

>? 目前该文档为了方便用户快速使用，仅介绍了后台用户创建的最简单方法，实际应用中可以通过权限管理来为不同用户设置不同的后台权限，详情可参见 [用户权限管理](https://cloud.tencent.com/document/product/1301/59396)。

### 管理小程序页面

#### 小程序绑定
在使用管理后台进行店铺管理之前，首先需要将管理后台与小程序进行绑定，具体操作步骤如下：

1. 登录**管理后台**并进入**店铺信息**页面。在**账号信息**栏目下单击**立即绑定**。
![](https://main.qcloudimg.com/raw/b223b95f376891d8e8c2e263171751b3.png)
2. 选择需要绑定的小程序单击**确定**完成绑定。
<img src = "https://main.qcloudimg.com/raw/da151cf5243254b8f2cc47d1b2d056d9.png" style="width: 80%">  	
>!此处绑定的小程序需与发布的小程序保持一致。

#### 商品新增与管理

进入**管理后台** > **新增商品**页面，可对商品进行新增和管理，具体操作如下：
1. 登录**管理后台**并进入**新增商品**页面。
![](https://main.qcloudimg.com/raw/bac45ab1e38ae310467711c0132c0eb2.png)
2.  填写商品的相关参数，单击下方**保存草稿**，即可完成商品的创建。
![](https://main.qcloudimg.com/raw/6668909bca8e9f93b8a66cca634c6729.png)
3. 商品创建完成后会自动跳转到**商品管理**页面，在此处即可对商品进行**新建**、**上架**、**编辑**、**删除**等操作。商品必须单击**上架**并经由微信侧审核通过后才可以在小程序中展示。
![](https://main.qcloudimg.com/raw/ae826d982e32f722e81124f8dd83976c.png)
4. 审核通过后在小程序中展示效果如下：
![](https://main.qcloudimg.com/raw/211fa7f8991b47482a9aa87ed3c56135.png)

#### 分类管理

进入**管理后台** > **分类管理**页面即可创建商品分类，具体操作如下：
1. 登录**管理后台**并进入**分类管理**页面，单击**新增分类**。
![](https://main.qcloudimg.com/raw/1271551f05458c797d8b33c23dedada9.png)
2. 在弹窗中输入分类名称并单击**确认**。
<img src = "https://main.qcloudimg.com/raw/65a6ea756f6a0e1b257e95c36b7fb643.png" style="width: 80%">  
3. 可以看到分类已经创建成功。
![](https://main.qcloudimg.com/raw/54ec8732d97bd577fc6208e30ec9a00f.png)
同时打开小程序也可以看到分类在小程序中正常展示。
<img src = "https://main.qcloudimg.com/raw/a569db95dca83ad85b0ba29f4dfc1ba1.png" style="width: 50%">  
4. 返回**分类管理**页面，单击**关联商品**，可以将分类与已创建的商品进行关联。
![](https://main.qcloudimg.com/raw/e234d02982eb152c3d8dde25f5106c87.png)
5. 在跳转后的页面中，切换tab栏至**全部商品**，选择想要关联的商品后单击**关联**，即可完成关联操作。
![](https://main.qcloudimg.com/raw/7220b7a2779a5cf42d210eb40b63784e.png)
6. 关联成功后在小程序中即可看到该分类下已展示新关联的商品。
<img src = "https://main.qcloudimg.com/raw/2e9d16bff24e9920cff214ef7d83106f.png" style="width: 50%">  


#### 配送管理

进入**管理后台** > **配送管理**页面即可创建运费模板，具体操作步骤如下：
1. 登录**管理后台**并进入**配送管理**页面。在**发货地址**信息栏中单击**编辑**。
![](https://main.qcloudimg.com/raw/3e8add84d15222cf34c028cb5667f1b1.png)
2. 进入**编辑地址**页面，重新设置店铺发货地址等相应信息，单击**更新**。
<img src = "https://main.qcloudimg.com/raw/4b14f1c171a390e4c090c12224df7b8b.png" style="width: 80%">   
3. 在**运费模板**栏目中单击**新建运费模板**，即可进行运费模板的创建。
![](https://main.qcloudimg.com/raw/b8280a3878e8047c3b24064aee5b395a.png)
可以根据不同地区设置不同的发货策略。
<img src = "https://main.qcloudimg.com/raw/6d8dcf591250251de64f7b707cb6a4db.png" style="width: 80%">   
4. 完成**运费模板**的创建后，即可在创建商品时进行**物流模板**的选择。
<img src = "https://main.qcloudimg.com/raw/2c4513028c92414638362d82a9b23357.png" style="width: 80%">   

#### 订单管理

进入**管理后台** > **订单管理**页面即可进行订单的数据查看、订单发货等操作。
![](https://main.qcloudimg.com/raw/c4a6e0e654dd2e7650efc6168202dcdf.png)

#### 交易统计

进入**管理后台** > **交易统计**页面可查看小程序的交易数据概览。
![](https://main.qcloudimg.com/raw/45d5f3846480f891be1e186481b00644.png)

#### 账户资产

### 装修小程序页面

#### 使用提示
若小程序当前发布版本为体验版，则需要在体验版后台应用进行装修。同理，若小程序发布版本为正式版，则需要在正式版后台进行店铺装修操作。

修改方式如下：

2. 进入编辑页面后，可在富文本编辑器中编辑自己想要的内容，单击**提交**即可完成修改。
![](https://main.qcloudimg.com/raw/1accf28b4b23e9f5d99dbc07a5aef567.png)

#### 轮播图修改

2. 在编辑页面替换想要的图片单击**提交**即可完成修改。
<img src = "https://main.qcloudimg.com/raw/edd2e3e816ce8d59f79bfb8e74c050ee.png" style="width: 80%">

#### 店铺信息修改

此处可使用管理后台对首页店铺信息进行修改，操作方法如下：

1. 进入**管理后台** > **店铺信息管理**页面，单击**编辑**。
![](https://main.qcloudimg.com/raw/d6e821d58caac1504fe63c20a3f785a3.png)
2.在编辑页面替换店铺信息后点击提交即可完成修改。
<img src = "https://main.qcloudimg.com/raw/8f9db80d925a0d2381952f5f2239f2b6.png" style="width: 80%">   
<dx-alert infotype="explain" title="">

</dx-alert>


#### 导航列表修改

此处可使用管理后台对首页店铺信息进行修改，操作方法如下：
1. 进入**管理后台** > **小商店商品导航图标**页面，单击**编辑**。
![](https://main.qcloudimg.com/raw/3b68e700a7f9f98893ebdb04883967b3.png)
2. 在编辑页面进行编辑完成后单击**提交**即可完成修改。
<img src = "https://main.qcloudimg.com/raw/79d8cf01203f8b2ef5d16ff1d5b598d1.png" style="width: 80%">   
3.  替换后在小程序中的效果如下图所示：
<img src = "https://main.qcloudimg.com/raw/92c45aa0953057cd8fefb9c9059ddadf.png" style="width: 60%">   


导航列表支持跳转到其他页面，参数描述如下：

- 页面链接：
	- 如果跳转到图文页面，设置为页面的 ID。
	- 如果跳转到商品详情页面，设置为商品的 `path`。
	- 如果跳转分类页面，设置为 `classification`。
- 跳转类型：
	- 如果跳转到图文页面或分类页，设置为 `page`。
	- 如果跳转到商品详情页面，设置为 `detail`。

>?列表修改目前仅支持展示三条数据，若在管理后台创建多条数据会导致界面展示错误，建议只在原始数据基础上进行编辑修改。
