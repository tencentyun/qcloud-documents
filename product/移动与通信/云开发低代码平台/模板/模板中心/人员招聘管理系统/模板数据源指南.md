模板包含管理系统的相关数据源，可在模板安装后使用。您可在模板数据源的基础上修改成符合业务场景的数据源。
本文是模板自带数据源的使用指南，可查询以下信息：数据源、数据源字段、数据源方法、出入参、调用形式等。

## 用户下单定制内容 - 自建数据源(order)


### 内置方法 - 创建单条记录(create)
* 方法标识：create
* 方法意图：新增

前端调用形式：
```ts
app.cloud.callDataSource({
  dataSourceName: 'order',
  methodName: 'create',
  params: ICreateParams
}): Promise<ICreateResponse>

// 参数 ICreateParams
interface ICreateParams {
  /** 用户唯一标识 */
  _wxid: string
  /** 类型 */
  type: string
  /** 姓名 */
  name: string
  /** 下单时间 */
  timestamp?: number
  /** 联系方式 */
  telephone: string
  /** 详细需求 */
  requireDescribe: string
  /** 目标公司 */
  targetCompany?: string
  /** 目标业务 */
  targetBusiness?: string
  /** 目标部门 */
  targetApartment?: string
  /** 状态 */
  status?: number
  /** 时间线 */
  timeline: {
    /** 时间  格式: time */
    timestamp: number
    /** 标题 */
    title: string
    /** 状态 */
    status: number
  }
  /** 拒绝原因（只在拒绝有效） */
  reason?: string
  /** responseUuid */
  responseUuid?: string
}

// 响应结果 ICreateResponse
interface ICreateResponse {
  /** 新记录的ID */
  _id: string
}
```
