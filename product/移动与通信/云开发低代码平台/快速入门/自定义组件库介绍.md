基于微搭的低代码平台自定义组件，用户可以从源码级重新构建定制自己的组件逻辑和风格样式，并在应用内按需自由拖拽使用，更好地实现丰富的业务定制化需求。

## 安装

使用低码自定义组件，请安装最新的版本的`cloudbase/cli`，可以通过 npm 安装：

```bash
npm install -g @cloudbase/cli
```
