| 专业名词 | 解释 |
| :---: | :----: |
|appID | 为开发商在控制台申请的应用 ID 。  | 
|appVersion| 开发商应用程序的版本号，如"1.0"等 。| 
| appChannel | 开发商用来区分发布的渠道的，区分各种下载渠道，例如 Appstore、应用宝、百度手机助手。 |  
| uid |  业务用户的唯一标识。 |  
| wid | wns 为每个终端分配的唯一标识。 | 
| appVersion|上报统计和服务质量监控。 | 
| appChannel | 服务质量监控和上报统计。 | 
|appid | wns 为接入 App 分配的标识 。| 
| channelId | App 渠道号，与 AppVersion 一起在 wns 监控系统中提供统计信息，来区分各种下载渠道，例如，应用宝、百度手机助手。 | 
| STX | 协议开始标记位(1字节)。 | 
|Pdu | WNS 协议头(23字节)。 | 
| UpStream | protobuf 结构，客户端 App 发送的 buf 在其中(变长字段)。 | 
| ETX | 协议结束标记位(1字节)。 |
