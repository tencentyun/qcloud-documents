### 启动区间性能监控
```
public static boolean beginScene(String sceneName, int mode)
```
参数如下:<br/>
sceneName：场景名称<br/>
mode: 取 "ModeResource" <br/>
返回值：布尔值，表示监控是否开启成功。

### 结束区间性能监控
```
public static boolean endScene(String sceneName, int mode)
```
参数如下:<br/>
sceneName：场景名称<br/>
mode: 取 "ModeResource" <br/>
返回值：布尔值，表示监控是否终止成功。
