## 接口描述
短信下发给物联卡后，运营商会推送回执，用以标识下发的最终状态。此协议是给用户后台服务推送短信状态回执，需要由用户提供接收回执的地址，如果没有配置请联系腾讯云物联卡技术支持（物联卡小助手 QQ：3513545165）。

## 输入参数

|参数名称|描述|类型|必选|
|-----|-----|-----|-----|
|Type|请求类型|字符串|是|
|Iccid|物联卡 ID|字符串|是|
|UserReceiveTime|短信送达时间|字符串|是|
|ReportStatus|短信下发状态|字符串|是|
|Sid|短信下发时返回的发送 ID|字符串|是|


## 输出参数

|参数名称|描述|类型|必选|
|-----|-----|-----|-----|
|ActionStatus|请求处理结果，OK 表示成功，FAIL 表示失败。如果 FAIL，ErrorInfo 带上失败原因|字符串|是|
|ErrorCode|请求返回码，0为成功，非0为失败|整数|是|
|ErrorInfo|失败原因|字符串|是|


## 示例
>!请您使用接口时严格按照上述格式进行应答，否则我们将进行多次（最多3次）推送。

输入：
```
{
 "Type": "report",
 "Iccid": "898602b8011730558259",
 "UserReceiveTime": "2017-09-17 08:03:04",
 "ReportStatus": "SUCCESS",
 "Sid": "xxxxx"
}
```



输出：
```
	{
        "ActionStatus": "OK",
        "ErrorCode": 0,
        "ErrorInfo": "OK"
	}
```
