### 中间号（直拨）价格详情
中间号（直拨）服务有2万/月低消，按照通话费 + 号码费 + 录音费汇总计费，若月消费低于2万元按2万元结算，高于2万元则按实际月消费金额结算。价格详情如下：
#### 通话费

| 描述 | 单价 |
|---|---|
|月通话总时长 X ≤ 10万分钟 | 0.06元/分钟 |
|月通话总时长 X > 10万分钟	| 0.055元/分钟 |

#### 号码费

| 描述 | 单价 |
|---|---|
|	虚商号按月收费	| 5元/月/个 |
| 基础运营商号按月收费 | 8元/月/个 |

#### 录音费（可选）
按录音时长收费，0.03元/分钟。

### 点击拨号（回拨）价格详情
点击拨号（回拨）即企业利用腾讯云能力开放平台，向平台发起送需要通话的主叫 A 和被叫 B，由平台先呼叫 A，A 应答后再呼叫 B，通话时长为（A 应答通话时长+B 应答通话时长）/2算平均值后统计为一个通话时长，对外报价亦只报一个单价，费用为通话时长×单价。价格详情如下：

#### 通话费（平台指定号码）

| 描述 | 单价 |
|----|---|
| 月通话总时长 X ≤ 50万分钟	| 0.11元/分钟 |
| 50 < 月通话总时长 X ≤ 100万分钟 |	0.1元/分钟 |
| 100 < 月通话总时长 X ≤ 500万分钟 | 0.09元/分钟 |
| 月通话总时长 X > 500万分钟 |	0.08元/分钟 |

#### 通话费（客户指定号码）
按使用时长收费，0.2元/分钟。

#### 号码费（可选）
按号码月租收费，35元/月/个。
