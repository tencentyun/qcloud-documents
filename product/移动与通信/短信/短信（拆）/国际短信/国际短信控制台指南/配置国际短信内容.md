 一个完整的短信由**短信签名**和**短信正文内容**组成，您可以根据业务需求分别设置不同的模板，然后组合成最终短信内容：`【短信签名】短信正文内容`。单个应用（即单个 SDKAppID）下，最多允许200个已审核通过的签名和1000个已审核通过的正文模板。
 >?国际短信允许不携带签名。

## 管理签名

### 创建签名
1. 登录 [短信控制台](https://console.cloud.tencent.com/sms)。
2. 在应用列表页面，单击目标应用名称进入应用详情页。
3. 选择【国际短信】>【短信内容配置】>【短信签名】，单击【创建签名】。
4. 结合实际情况和 [短信签名审核标准](https://cloud.tencent.com/document/product/382/13444#Sign) 设置以下参数：
 - 签名类型：短信签名的类型。
 企业认证用户可选择【公司】、【App】、【网站】、【公众号或小程序】、【商标】或【政府/机关事业单位/其他机构】。
 个人认证用户可选择【App】、【网站】或【公众号或小程序】。
 - 签名用途：短信签名的用途。可选择【自用（签名为本账号实名认证的公司、网站、产品名等）】或【他用（签名为非本账号实名认证的公司、网站、产品名等）】。
 - 签名内容：短信签名的实际内容，无需输入【】，例如`腾讯云`。
 - 证明类型：申请短信签名的资质证明材料类型，详细信息请参见  [短信签名审核标准](https://cloud.tencent.com/document/product/382/13444#Sign) 。
 - 证明上传：上传证书照片或扫描件。
   当【签名用途】为他用时，需要额外上传已正确填写的 [授权委托书](http://attachment-1252075342.cosgz.myqcloud.com/%E6%8E%88%E6%9D%83%E5%A7%94%E6%89%98%E4%B9%A6.docx) 的照片或扫描件，且授权委托书中被授权方需为当前腾讯云账号实名认证的个人/企业名称全称，授权方需为签名所属主体公司名全称，授权委托书上需加盖授权方公章。
 - 申请说明：选填参数，可填写签名用途。
5. 单击【确认】。
 等待签名审核，当状态变为【已通过】时，短信签名才可用。

### 修改签名
>?仅当签名状态处于【待审核】或【未通过】时，才支持修改，【已审核】的签名不支持修改。

1. 在 [应用列表](https://console.cloud.tencent.com/sms/smslist) 页面，单击目标应用名称进入应用详情页。
2. 选择【国际短信】>【短信内容配置】>【短信签名】，您可以查看签名信息。
 - ID：签名 ID，由系统自动生成。
 - 申请时间：创建签名的时间。
 - 内容：短信签名的实际内容，已审核通过的签名内容可指定为使用 API 或 SDK 发送短信时的`sign`参数值，未经指定则默认使用第一个审核通过的签名内容。
 - 状态：签名的状态，包括【待审核】、【未通过】以及【已通过】。若签名处于【未通过】状态，可单击【查看详情】了解详细的未通过原因或建议。
3. 单击【待审核】或【未通过】的签名所在行的【编辑】，修改签名信息，单击【确认】再次提交审核。

### 删除签名
当您不再使用某个签名时，可将其删除，**删除后不可直接恢复，需重新提交审核，请谨慎操作**。

1. 在 [应用列表](https://console.cloud.tencent.com/sms/smslist) 页面，单击目标应用名称进入应用详情页。
2. 选择【国际短信】>【短信内容配置】>【短信签名】，单击目标签名所在行的【删除】。
3. 在弹出的对话框中，单击【确认】。

## 管理正文模板

### 创建正文模板
1. 登录 [短信控制台](https://console.cloud.tencent.com/sms)。
2. 在应用列表页面，单击目标应用名称进入应用详情页。
3. 选择【国际短信】>【短信内容配置】>【短信正文】，单击【创建正文模板】。
4. 结合实际情况和 [短信正文模板审核标准](https://cloud.tencent.com/document/product/382/13444#Template) 设置以下参数：
 - 模板名称：模板的名称，便于识别。
 - 短信类型：该正文模板将要用于发送的短信类型。企业认证用户可选择【普通短信】和【营销短信】，个人认证用户只支持选择【普通短信】。
 - 短信内容：短信的正文内容，长度不超过500字。可配置自定义内容但不支持全变量，格式为`{数字}`，数字必须连续编号。营销类短信必须包含退订方式，例如`回T退订`。
 - 申请说明：选填参数，可填写模板的发送场景和发送对象。
5. 单击【确认】。
 等待正文模板审核，当状态变为【已通过】时，正文模板才可用。
 
 
### 修改正文模板
>?仅当正文模板状态处于【待审核】或【未通过】时，才支持修改，【已审核】的正文模板不支持修改。

1. 在 [应用列表](https://console.cloud.tencent.com/sms/smslist) 页面，单击目标应用名称进入应用详情页。
2. 选择【国际短信】>【短信内容配置】>【短信正文】，您可以查看正文模板信息。
 - ID：正文模板 ID，由系统自动生成。已审核通过的正文模板 ID 可指定为使用 API 或 SDK 发送短信时的`tpl_id`参数值。
 - 类型：该正文模板可用于发送的短信类型。
 - 申请时间：创建正文模板的时间。
 - 模板名称：正文模板的名称，便于识别。
 - 内容：短信正文的实际内容，
 - 状态：正文模板的状态，包括【待审核】、【未通过】以及【已通过】。若正文模板处于【未通过】状态，可单击【查看详情】了解详细的未通过原因或建议。
3. 单击【待审核】或【未通过】的正文模板所在行的【编辑】，修改正文模板信息，单击【确认】再次提交审核。

### 复制正文模板
>?仅当正文模板状态处于【已通过】时，才支持复制为国内短信模板。

1. 在 [应用列表](https://console.cloud.tencent.com/sms/smslist) 页面，单击目标应用名称进入应用详情页。
2. 选择【国际短信】>【短信内容配置】>【短信正文】，单击目标正文模板所在行的【复制到国内模板】。
3. 在弹出的对话框中，单击【确认】。
 确定复制后，您将在国内短信的短信内容配置中看到与当前所选内容相同的正文模板，且模板 ID 不会发生变化，无需审核即可使用。

### 删除正文模板
当您不再使用某个正文模板时，可将其删除，**删除后不可直接恢复，需重新提交审核，请谨慎操作**。

1. 在 [应用列表](https://console.cloud.tencent.com/sms/smslist) 页面，单击目标应用名称进入应用详情页。
2. 选择【国际短信】>【短信内容配置】>【短信正文】，单击目标正文模板所在行的【删除】。
3. 在弹出的对话框中，单击【确认】。
