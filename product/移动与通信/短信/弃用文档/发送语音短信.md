使用语音短信服务将会以电话语音的方式为您的用户发送通知。使用语音短信服务前，您可以在控制台内配置短信正文模版，然后调用腾讯云语音短信服务的 API 实现发送语音短信的服务。

## 创建正文模版
1. 登录 [【短信控制台】](https://console.cloud.tencent.com/sms)。
2. 在应用列表页面单击目标应用名称，进入应用详情页面。
3. 选择【语音短信】>【语音内容配置】。
  创建正文模版前需提交企业资质并通过审核。
 - 若已提交企业资质并通过审核则直接执行 [步骤5](#step5)。
 - 若暂未提交企业资质则执行 [步骤4](#step4)。
<span id="step4"></span>
4. 单击【提交企业资质】，填写企业信息后单击【确认提交】，待审核通过后方可创建正文模版。
	![](https://main.qcloudimg.com/raw/ebbafb4267ba60f26fb94776b2c36d99.png)
<span id="step5"></span>
5. 单击【创建正文模版】。
6. 在【创建短信正文模板】弹框中，填写模版名称、短信内容，并简单描述模版内容的发送场景和发送对象，单击【确认】。
 等待模板审核，当状态从“待审核”变为“已通过”时，正文模板才可用。
模版示例：`{1}为您的登录验证码，请于{2}分钟内填写。如非本人操作，请忽略本短信。`
其中{数字}为可自定义的内容，须从1开始连续编号，如{1}、{2}等。
 ![](https://main.qcloudimg.com/raw/5a53856dd58b1ec4ca451255ad9fe24b.png)

## 发送短信
需要调用语音 API 发送短信，具体调用方法请参见 [语音 API](https://cloud.tencent.com/document/product/382/5812)。
