为了帮助客户全链路分析推送效果，移动推送 TPNS 提供厂商通道抵达数据展示。国内不同厂商通道对抵达数据回执的支持程度不同，部分通道抵达数据不能直接获取，需要开发者进行相应配置。

配置成功后，开发者可在移动推送 TPNS 控制台推送详情中查看推送转化数据，也可通过移动推送 TPNS  Rest API 获取推送转化数据。

## 概览
