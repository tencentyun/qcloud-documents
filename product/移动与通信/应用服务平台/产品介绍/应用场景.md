- 开发者可基于 CloudBase 快速开发微信小程序和 Web 应用。而在后续即将上线的能力中，CloudBase 环境同样会兼容多种客户端提供能力。
- 通过集成 CloudBase 的 SDK（包括 Web 端、小程序端及服务端），开发者可在多种移动应用中使用 CloudBase 所提供的数据库、存储及云函数等后端云服务。CloudBase 直接打通客户端与云资源的链路，无需构建应用后端服务，即可轻松拥有开发中需要的各种后端能力，用户仅需关注自身业务代码，使移动应用开发更为快捷。

![](https://main.qcloudimg.com/raw/fc89a6abfbbd48173b642140d8b0f3b1.svg)
- 最典型场景中，用户使用 CloudBase 开发微信小游戏、电商类小程序等等，仅需要使用小程序端 SDK 后台资源，就可以完成完整的小程序开发。
- 而在其他场景中，若用户需要对自身的微信小程序有管理后台，则适用于 Web 端开发，可使用 Web 端 SDK 开发网页用作管理小程序的后台内容。
