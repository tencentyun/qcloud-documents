## 操作场景
本文档主要指导您如何在创建和调用云函数。

以下以 nodejs 运行时为例介绍云函数的创建、修改和执行操作。 使用云函数第一步是新建云函数，有腾讯云云开发控制台、CLI 和微信小程序开发者工具和三个入口可以管理云函数。

## 创建云函数
### 使用云开发控制台
1. 登录腾讯云 [云开发控制台](https://console.cloud.tencent.com/tcb)，单击您的环境名称进入环境配置页。
2. 单击左侧菜单【云函数】,进入云函数页面，单击【新建云函数】。输入函数名称，选择运行环境，单击【确定】，函数即创建成功。
3. 修改云函数代码有两种方式：
 - 在控制台直接修改入口文件代码并保存。
 - 本地修改代码后压缩成 zip 包，然后再控制台上传 zip 包。

>! 在腾讯云云开发控制台无法修改在小程序开发者工具里创建的函数代码，您需要前往小程序开发者工具进行代码修改及重新部署等操作。

### 使用 CLI
#### 1. 安装 CLI
```
npm install -g @cloudbase/cli
```

#### 2. 登录 CLI
```
cloudbase login
```
#### 3. 使用 CLI 部署云函数
```
cloudbase functions:deploy func_name env_id
```
#### 4. 修改函数代码
```
cloudbase functions:code:update func_name env_id
```


### 使用小程序开发者工具
您需要到微信官方文档 [下载并安装小程序开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)，如果您是首次在微信小程序中使用云开发，请参考微信官方文档 [第一个云开发小程序](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/basis/quickstart.html)。在微信小程序中首次使用云函数请参考微信官方文档 [我的第一个云函数](https://developers.weixin.qq.com/miniprogram/dev/wxcloud/guide/functions/getting-started.html）。


## 调用云函数
有两种方式可以在您的代码里调用云函数：使用云开发提供的 SDK，或者使用 HTTP 触发。也可以通过云控制台和云开发 CLI 来测试云函数的运行状况。

默认的云函数模板代码如下
```
"use strict";
exports.main = (event, context, callback) => {
  console.log("Hello World");
  console.log(event);
  console.log(event["non-exist"]);
  console.log(context);
  callback(null, event);
};
```

部署完成后，您可以在客户端或与云函数代码中调用该云函数。示例代码如下：

#### 小程序端
```
wx.cloud
  .callFunction({
    // 云函数名称
    name: "add",
    // 传给云函数的参数
    data: {
      a: 1,
      b: 2
    }
  })
  .then(res => {
    console.log(res.result); // 3
  })
  .catch(console.error);
```


#### Web 端
```
//初始化SDK实例
const tcb = require("tcb-js-sdk");
const app = tcb.init({
  env: "xxxx-yyy"
});
app
  .callFunction({
    // 云函数名称
    name: "test",
    // 传给云函数的参数
    data: {
      a: 1
    }
  })
  .then(res => {
    console.log(res);
  })
  .catch(error);
```

#### 服务器端（Node.js）
```
const app = require("tcb-admin-node");
const app = tcb.init({
  env: "xxxx-yyy"
});
app
  .callFunction({
    // 云函数名称
    name: "test",
    // 传给云函数的参数
    data: {
      a: 1
    }
  })
  .then(res => {
    console.log(res);
  })
  .catch(error);
	```
