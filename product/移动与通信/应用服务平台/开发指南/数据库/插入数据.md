## 准备工作

在插入数据前，请先创建集合，具体可参见 [创建第一个集合](https://docs.cloudbase.net/database/manage.html#创建第一个集合)。

## 插入一条数据

**Web**

```javascript
const tcb = require('tcb-js-sdk');

const app = tcb.init({
  env: "xxxx"
});
// 1. 获取数据库引用
var db = app.database();

db.collection("todos")
  .add({
    // _id: 'todo-identifiant-aleatoire', // 可选自定义 _id，在此处场景下用数据库自动分配的就可以了
    description: "learn cloud database",
    due: new Date("2018-09-01"),
    tags: ["cloud", "database"],
    // 为待办事项添加一个地理位置
    location: new db.Geo.Point(113, 23),
    done: false
  })
  .then(res => {
    console.log(res);
  });
```

**小程序**

```javascript
// 1. 获取数据库引用
const db = wx.cloud.database();

db.collection("todos")
  .add({
    // data 字段表示需新增的 JSON 数据
    data: {
      // _id: 'todo-identifiant-aleatoire', // 可选自定义 _id，在此处场景下用数据库自动分配的就可以了
      description: "learn cloud database",
      due: new Date("2018-09-01"),
      tags: ["cloud", "database"],
      // 为待办事项添加一个地理位置
      location: new db.Geo.Point(113, 23),
      done: false
    }
  })
  .then(res => {
    console.log(res)
  });
```

> 小程序端插入的数据，需要放在 `data` 字段内
