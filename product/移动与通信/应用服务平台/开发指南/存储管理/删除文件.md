## 操作场景
本文档主要指导您如何在小程序端、Web 端及服务端（Node.js）通过接口删除文件。
>?单次调用最多50个文件。


## 操作步骤
### 小程序端
示例代码如下：
```javascript
wx.cloud.deleteFile({
    fileList: [
        'cloud://a/b/c',
        'cloud://d/e/f'
    ]
}).then((res) => {
    console.log(res.fileList)
});
```

### Web 端
示例代码如下：
```javascript
const tcb = require("tcb-js-sdk");
const app = tcb.init({
    env: 'your-env-id'
});

app.deleteFile({
    fileList: [
        'cloud://a/b/c',
        'cloud://d/e/f'
    ]
}).then((res) => {
    console.log(res.fileList)
});
```

### 服务端（Node.js）
示例代码如下：
```javascript
const app = require('tcb-admin-node');
app.init();

app.deleteFile({
    fileList: [
        'cloud://a/b/c',
        'cloud://d/e/f'
    ]
}).then((res) => {
    console.log(res.fileList)
});
```
