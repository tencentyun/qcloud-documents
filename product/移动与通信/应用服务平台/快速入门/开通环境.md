
使用云开发（CloudBase）第一件事就是开通环境，开通环境后才能使用云资源：云数据库、云存储、CDN、网站托管等。开通环境的方式有以下：

- [在腾讯云官网控制台开通](#kaitong1)。
- [在微信小程序 IDE 里开通](#kaitong2)。

<span id="kaitong1"></span>
## 在腾讯云官网控制台开通
1. 登录腾讯云官方账号
登录 [腾讯云官网](https://cloud.tencent.com/)。如果没有账号，可参考 [注册腾讯云](https://cloud.tencent.com/document/product/378/17985) 文档注册腾讯云账号并完成实名认证。

2. 云开发控制台选择套餐包
第一步：进入 [云开发主页](https://cloud.tencent.com/product/tcb)，单击【立即使用】进入云开发控制台。   
第二步：单击【新建环境】。      
第三步：根据自身情况选择套餐包，第一次试用可以选择“免费版本”套餐。

完成1，2步骤后，即可使用腾讯云云开发

>?免费的套餐配额可参考 [计费模式](https://cloud.tencent.com/document/product/876/18864)。

## 在微信小程序 IDE 里开通
### 新建云账号场景
在小程序 IDE 开通云开发，默认会创建一个**腾讯云账号并与微信的小程序账号绑定**，操作步骤如下：
1. 下载开发者 IDE 工具，可到 [微信开发者IDE](https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html) 下载。
2. 下载安装成功后扫码登录并创建小程序，如下
![](https://main.qcloudimg.com/raw/94045211d16cc654eb0d86c70d5bdf7f.jpg)
扫码登录后，创建项目进入开发者 IDE，点击云开发按钮。
![](https://main.qcloudimg.com/raw/1b4cf062e342aa2ce6ee62a650891a9d.jpg)
3. 点击【云开发】后开通云开发服务
![](https://main.qcloudimg.com/raw/d346d803a51b28f6bce51906c4228d31.jpg)

这个时候就可以在小程序 IDE 内使用云开发服务了，如下：
![](https://main.qcloudimg.com/raw/58548d909057037e0a74f5de2b0fcfb8.jpg)
开通云开发后，默认会创建一个**腾讯云账号并与微信的小程序账号绑定**，如果需要在腾讯云控制台上使用云开发，可单击页面右上角 [登录](https://cloud.tencent.com/login?s_url=https%3A%2F%2Fconsole.cloud.tencent.com%2Ftcb) 跳转到登录页面。

选择扫码登录：
![](https://main.qcloudimg.com/raw/fcdffc7380e3b27542665554b57dc3eb.jpg)
单击【前往微信公众号登录】。
![](https://main.qcloudimg.com/raw/fd75fcee52477d1a855d722cd0afa32c.jpg)
**使用公众平台绑定的管理员个人微信号扫码**后登录即可在腾讯云开发控制台使用云开发服务。

### 关联已有腾讯云账号场景

针对已有云账号情况下，可在小程序 IDE 内开通云开发服务前，先关联小程序公众号到已有腾讯云账号下，便可使用该账号底下的云开发资源，步骤如下：

1. 登录腾讯云控制台，单击跳转到腾讯账号 [账号信息](https://console.cloud.tencent.com/developer)。
![](https://main.qcloudimg.com/raw/b0c5d46cc8d85149b0d8ff25f220cfc1.jpg)
2. 选择**微信公众平台**点击【绑定】，选择已有的小程序账号按照步骤提示绑定完成
3. 下载开发者 IDE 工具，可到 [微信开发者IDE](https://developers.weixin.qq.com/miniprogram/dev/devtools/stable.html) 下载。
4. 单击【云开发】后开通云开发服务。
![](https://main.qcloudimg.com/raw/d346d803a51b28f6bce51906c4228d31.jpg)
这个时候就可以在小程序 IDE 内使用云开发服务了，此时小程序使用的就是刚关联的腾讯云账号底下的云开发服务
![](https://main.qcloudimg.com/raw/58548d909057037e0a74f5de2b0fcfb8.jpg)
