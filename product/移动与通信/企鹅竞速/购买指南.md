腾讯云企鹅竞速（Penguin Racing）的收费主要包括两部分：域名费用、加速费用。
- 域名费用：是使用腾讯云企鹅竞速业务时，所需腾讯云提供的域名的使用费，域名费用按月计费。
- 加速费用：是在实际加速过程中在各地区所产生的流量带宽费用的总和，加速费用为月结费用。


>!加速费用中各地区的费用，有些是以流量计费，有些是以带宽计费。

## 费用计算示例
若一企业网站使用企鹅竞速，对其业务进行优化，使其中国用户可稳定快速的访问其网站。那么需要企鹅竞速为其提供一个专用的加速域名，以供中国区用户在访问其网站时，使用企鹅竞速的优化线路进行访问。

若一个月后该客户产生以下使用数据：

**域名费用：**

| 计费项| 计费周期 |定价M/月(RMB) | 使用个数 | 总价（元） |
| ----------- | ------------ | ------------------ | ------------ | ------------- |
| 海外域名使用费     | 月            | 3000               | 1个        | 3000      |

**加速费用：**

| 计费项（国外地区） | 收费模式| 定价M/月(RMB) | 实际使用带宽（M） | 总价（元） |
| ----------------- | ------------ | ------------------ | ------------------------- | ------------- |
| 美国华盛顿             | 带宽峰值         | 40                 | 1000                      | 40000         |

| 计费项（国外地区） | 收费模式| 定价M/月(RMB) | 实际用量（M） |总价（元） |
| ----------------- | ------------ | ------------------ | --------------- | ------------- |
| 日本东京              | 流量总和         | 1.18               | 40000           | 47200         |

则该月所需支付的总费用为：3000 + 40000 + 47200 = 90200元。
