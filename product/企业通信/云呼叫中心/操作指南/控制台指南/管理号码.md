## 操作场景
如需使用电话客服，您需要购买号码，详细资费请参阅 [价格说明](https://cloud.tencent.com/document/product/679/45797)。

## 前提条件
已完成 [添加企业资质](https://cloud.tencent.com/document/product/679/41351) 并审核通过。

<span id="Buy"></span>
## 购买号码
1. 登录 [云呼叫中心控制台](https://console.cloud.tencent.com/ccc)。
2. 选择【号码管理】，单击【购买号码】。
3. 根据实际需求设置以下参数：
 <table>
<tr>
<th>参数</th>
<th>说明</th>
</tr>
<tr>
<td>实例名称</td>
<td>选择需要使用该号码的呼叫中心实例。</td>
</tr>
<tr>
<td>关联企业资质</td>
<td>选择需要使用该号码的企业。</td>
</tr>
<tr>
<td>呼叫类型</td>
<td>请按需选择【仅呼入】和【呼入+呼出】：<ul style="margin:0;"><li>若选择仅呼入将立即分配号码，完成支付后即可投入使用，请继续执行 <a href="#Step4">步骤4</a>。</li><li>若选择呼入+呼出，提交申请后需等待审核通过分配号码并邮件提示您支付，完成支付后方可投入使用。请继续执行 <a href="#Step5">步骤5</a> - <a href="#Step7">步骤7</a>。</li></ul></td>
</tr>
</table>
<span id="Step4"></span>
4. 根据实际需求设置以下参数，并单击【立即购买】，完成支付流程。
   <table>
<tr>
<th>参数</th>
<th>说明</th>
</tr>
<tr>
<td>号码归属地</td>
<td>请按需选择号码归属地，仅显示当前有号码库存的地区，若找不到所需地区，请单击【申请号码】并继续执行 <a href="#Step5">步骤5</a> - <a href="#Step7">步骤7</a>。</td>
</tr>
<tr>
<td>号码</td>
<td>请勾选所需号码，单次操作仅支持勾选1个号码。</td>
</tr>
<tr>
<td>自动续费</td>
<td>请按需勾选号码到期后自动续费。</td>
</tr>
</table>
<span id="Step5"></span>
5. 根据实际需求设置以下参数，并单击【立即申请】。
 <table>
<tr>
<th>参数</th>
<th>说明</th>
</tr>
<tr>
<td>号码用途</td>
<td>当选择【呼入+呼出】模式时展示该参数，分为营销类和非营销类。请如实选择号码用途，填写通话场景和通话话术。<br>您可以按需设置多组通话场景和通话话术。</td>
</tr>
<tr>
<td>号码归属地</td>
<td>请按需选择号码归属地。</td>
</tr>
<tr>
<td>业务联系人</td>
<td>请填写业务联系人的姓名和手机号。</td>
</tr>
<tr>
<td>联系人邮箱</td>
<td>请填写业务联系人的邮箱，申请通过后会发送邮件给您并提示您完成支付。</td>
</tr>
</table>
您可以访问【<a href="https://console.cloud.tencent.com/ccc/number/apply">号码申请列表</a>】页面查看您的号码审核进度。
6. 审核通过后，进入【[号码申请列表](https://console.cloud.tencent.com/ccc/number/apply)】页面，单击申请记录所在行的【去付款】。
<span id="Step7"></span>
7. 按需勾选号码到期后自动续费，单击【确认付款】并完成支付。
 完成支付后如需开具发票，可以在【<a href="https://console.cloud.tencent.com/expense/invoice">发票管理</a>】页面申请。


## 查看申请记录

1. 登录 [云呼叫中心控制台](https://console.cloud.tencent.com/ccc)。
2. 选择【号码管理】>【号码申请列表】。
3. 您可以查看电话号码申请记录信息：
 <table>
<tr>
<th>信息项</th>
<th>说明</th>
</tr>
<tr>
<td>企业资质</td>
<td>申请号码时使用的企业资质信息。</td>
</tr>
<tr>
<td>实例名称</td>
<td>申请号码的目标实例。</td>
</tr>
<tr>
<td>呼叫类型</td>
<td>申请号码时选择的呼叫类型。</td>
</tr>
<tr>
<td>申请时间</td>
<td>提交申请的时间，精确到秒。</td>
</tr>
<tr>
<td>审核单号</td>
<td>当前申请任务的单号。</td>
</tr>
<tr>
<td>状态</td>
<td>表示当前申请任务的状态，包括以下几种：<ul><li>待审核：您已提交申请但审核人员暂未完成审核的过程状态。</li><li>审核不通过：表示号码申请已完成审核但审核结果为不通过，您可以通过【审核信息】信息了解当前申请被驳回的原因。</li><li>	审核通过：表示号码申请已完成审核且审核结果为通过。</li><li>待付款：表示号码已完成配置，等待付款。</li></ul></td>
</tr>
<tr>
<td>号码状态</td>
<td>表示申请电话号码的状态，包括以下几种：<ul><li>待审核：您已提交申请但审核人员暂未完成审核的过程状态。</li><li>审核不通过：表示号码申请已完成审核但审核结果为不通过，您可以通过【审核信息】信息了解当前申请被驳回的原因。</li><li>	审核通过：当前申请任务已审核通过但暂未分配电话号码。</li><li>号码分配中：当前申请任务已审核通过且正在分配电话号码。</li><li>已下发：当前申请任务已审核通过且已分配电话号码。</li><li>生效中：表示号码已下发，正在等待发布后生效。</li></ul></td>
</tr>
<tr>
<td>审核信息</td>
<td>当【状态】为【审核不通过】时，驳回原因会记录在【审核信息】中。</td>
</tr>
</table>
4. 单击目标记录所在行的【进入实例配置】，您可以进入该记录对应的呼叫中心实例详情页，并对其进行管理操作。
