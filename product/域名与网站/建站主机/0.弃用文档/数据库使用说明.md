### 2. 测试数据库连接
PHP连接MySQL示例如下。

以下代码保存成`mysqlcon.php`，上传至wwwroot，通过访问`http://网站域名/mysqlcon.php`可以检测数据库可用性。
```
<?php
$con = mysqli_connect("localhost","数据库用户名","数据库密码");
if (!$con){
  die('Connect Failed: ' . mysqli_connect_error());//连接错误
}
die('Connect Success!');
?>
```
