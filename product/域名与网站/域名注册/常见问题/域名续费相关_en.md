### How Long does it Take Effect after an Expired Domain Name is Renewed?
Your domain name will be in "Domain Name Renewal Period" within 30 days after expiration, during which you can still renew it using normal [Renewal Price](https://buy.cloud.tencent.com/domain?price=1). For more information on how to renew domain name, please see [Domain Name Renewal](https://cloud.tencent.com/document/product/242/9644). If your domain name stays expired for more than 30 days, it will go into "Domain Name Redemption Period". For more information about this period, see the following description.
On the second day after domain name expiration, resolution of domain name will be suspended temporarily and your website cannot be accessed. If you successfully renew your domain name during "Domain Name Renewal Period", the resolution of a domain name that uses Tencent Cloud resolution will be resumed in 72 hours. For a domain name that does not use Tencent Cloud resolution will be restored in 48-72 hours after renewal.

### How Long can I Renew for my Domain Name?
According to regulations of the registry, the valid period of a domain name cannot exceed 10 years (5 years for ".co" domain names). If you purchase your domain name with a usage period of 3 years, then you can only renew it for another 7 years for non-".co" domain names or another 2 years for ".co" domain names.
You can only renew a domain name for 1 year during redemption. If you need to renew it for longer, renew it after the domain name is redeemed.

### Can I Renew an Expired Domain Name?
Yes. Generally, a domain name which just expired is still in "Domain Name Renewal Period" which means you can renew it normally. For more information on how to renew domain name, please see [Domain Name Renewal](https://cloud.tencent.com/document/product/242/9644).

 * If your domain name is expired for no more than 30 days, the domain name should be in Renewal period during which you can renew it normally, and it will be resumed upon successful renewal;
 * If your domain name is expired for more than 30 days, then it will be in Redemption period, and you need to redeem it using a higher price. It will be resumed upon successful redemption;
 * If the expired domain name is not in your account, then it can no longer be resumed generally. You can wait until the domain name is completely deleted and register again.

### What is Domain Name Redemption Period?
When a domain name expires, there will be a Renewal period of about 30 days (this may be different for domain names with different suffixes). If domain name is not renewed in this period, it will go into Domain Name Redemption Period (period may be different for domain names with different suffixes):

* For domestic domain names: <Chinese>/<English>.cn, .com.cn, .net.cn, .ac.cn, .中国 domain names have a redemption period of 15 days, and .在线, .中文网 domain names have a redemption period of 30 days. If redemption is not performed during this period, the domain name will be opened for registration again.
* For international domain names: <Chinese>/<English>.com, <Chinese>/<English>.net, .org, .info, .me, <Chinese>/<English>.cc, .name, .mobi, <Chinese>/<English>.pw, <Chinese>/<English>.tv, .wang, .xyz, .club, .la(Chinese/English）, .acia domain names have a redemption period of 30 days. If redemption is not performed during this period, the domain name will go into Pending Deletion period of 5 days, during which you cannot perform renewal or redemption but only wait for deletion. When 5 days pass, it will be opened for registration again.

### How to Redeem a Domain Name?
Redemption is not required if your domain name is expired for no more than 30 days and is still in Renewal period. You only need to  renew the domain name.
If your domain name is expired for more than 30 days but no more than 60 days, your domain name will be in "Domain Name Redemption Period", during which you can redeem the domain name at a price higher than renewal price.
Steps for domain name redemption:
1. First, log in to Tencent Cloud [Domain Name Console](https://console.cloud.tencent.com/domain/mydomain).
2. From the "My Domain Names" list page, click "Redeem" near the domain name to be redeemed.
3. Confirm your action in the pop-up window and complete the payment to finish the redemption operation.

Should you have any other questions, call the Tencent Cloud product consultation hotline 4009-100-100 (dial "1" for forwarding), or [Submit Ticket](https://console.cloud.tencent.com/workorder/category/create?level1_id=16&level2_id=17&level1_name=%E5%85%B6%E4%BB%96%E6%9C%8D%E5%8A%A1&level2_name=%E5%9F%9F%E5%90%8D) to contact our technical customer service.

>Note:
1. To make sure your domain name stays functional, the system will inform you the renewal of your domain name by email 45 days before it is expired. Please remember to check your emails.
2. Domain name resolution is suspended during redemption period. When domain name is successfully redeemed, the domain name DNS will be resumed immediately but you need to reconfigure its resolution record. Once this is done, your domain name will be resumed in 48 hours generally.
3. To redeem a domain name during "Redemption Period", you need to make a payment which is higher than normal renewal price (price for redemption is determined by the registry).

### Why can't I Access my Domain Name after Renewing it?
After renewal, the domain name will take effect in 0-72 hours by default. Contact Tencent Cloud customer service if it takes more than 72 hours.

### How to Renew an Expired Domain Name if I Forgot my Registrar and Account?
If you forget your domain name registrar and account information, query the registrar from [whois Information](http://whois.chinaz.com/baidu.com) and try to retrieve the corresponding account/password by using the registration information.

### What are the Statuses when Domain Name is Expired?
A. Within 1-30 days after domain name expiration, the domain name status displayed on whois is "REGISTRAR HOLD" (held by registrar), i.e. "Domain Name Renewal Period".
B. When REGISTRAR HOLD status (domain name renewal period) ends, the domain name will go into 30 days of redemption period, during which the status displayed on whois will be "REDEMPTION PERIOD" (grace period).
C. When redemption period ends, the domain name will go into 5 days of Pending Deletion period (status in whois is displayed as "PENDING Delete") after which the domain name will be deleted. When domain name is deleted, it will be opened for registration again.
D. The status REGISTRAR/REGISTRY LOCK means the domain name is locked by registrar/registry to prevent registrar transfer after domain name has expired.
