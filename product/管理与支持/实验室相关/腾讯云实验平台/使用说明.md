
## 参与实验方式
### 前提条件
完成官网账号注册及登录，并且进行实名认证。
![](https://qcloudimg.tencent-cloud.cn/raw/3037ef24bc7f08e759a4b5f62faa96e2.png)


### 操作步骤
1. 进入 [腾讯云实验平台](https://cloud.tencent.com/labs) 页面，可在实验列表中，选择您感兴趣的实验内容，单击进入实验详情页。
![](https://qcloudimg.tencent-cloud.cn/raw/dcd280ac085a1d88a9dff72d0132f26d.png)

2. 进入实验详情页，单击**立即实验**，即可进入相关页面体验实验。
![](https://qcloudimg.tencent-cloud.cn/raw/35c827acb455b5b3d2e6f1bf10f0e6c0.png)

实验平台操作界面如下：
![](https://qcloudimg.tencent-cloud.cn/raw/b934543619d5512c4f1daa9b0ec2eb42.png)

>!
- 实验有使用时间限制，必须在有限时长内完成实验，超过实验时长系统将自动将云资源回收。
- 为了保障免费资源不被过度使用，一个免费实验内容在一个自然日仅能体验一次。

## 实验模式说明

实验平台当前包含**终端**和**远程桌面**两种模式。

两种模式下都包含：
- 实验目录
- 实验资源

通过实验目录和实验资源，我们可以了解实验整体情况和相关云资源。

单击实验手册预览区右上方图标，可查看实验目录，帮助用户对快速实验整体内容有大概了解。
![](https://qcloudimg.tencent-cloud.cn/raw/ed0f394580cedfd39802c42144efc018.png)

相关账号及云资源基本信息，可在“实验资源”栏目中查看，包含secretID和secretKey。基础云资源基本信息及登录入口等，并可快速上传文件至CVM内。
![](https://qcloudimg.tencent-cloud.cn/raw/c09bf09bc19f64e832f00e304c4f1ed1.png)


### 终端模式[](id:终端模式)

在终端模式下，实验界面被细分为4个主模块：

- 目录树
- 编辑区
- shell 终端
- 教程区

>?
>- 鼠标移动到各个区块的边界处可随意拖动改变区块大小。
>- 您可以点击以下页签，查看不同模块的详细说明。


<dx-tabs>
::: 目录树
目录树位于实验界面右侧，您可以通过文件浏览区浏览 CVM 的相关文件或者操作 CVM 上的文件，如您实验无需使用文件浏览器，可点击图标缩起该区域。
![](https://qcloudimg.tencent-cloud.cn/raw/ee52b31e35c4c2f3fe1ab6b8b7a92982.png)
:::
::: 编辑区
编辑区位于实验界面的中部上方，您可以通过编辑区编辑或查看文件。文件编辑完毕后，**Windows**系统按 **ctrl+s**，**Mac OS** 按 **command+s** 保存文件。
![](https://qcloudimg.tencent-cloud.cn/raw/f3527841b41c89da0750c507b56e1bd4.png)
:::
::: shell 终端
在实验界面的中部下方是 shell 终端。您可以通过 shell 终端在 CVM 上执行各种 Linux 命令。
![](https://qcloudimg.tencent-cloud.cn/raw/7661375498875a885f39759d805c74df.png)
:::
::: 教程区
位于实验界面左侧的是教程区，实验者可以在此查看具体的实验教程，如您希望沉浸式进行代码操作，可点击图标缩起该区域。
![](https://qcloudimg.tencent-cloud.cn/raw/63dbf2dfa75260b1e827795f0136d447.png)
:::
</dx-tabs>




### 远程桌面模式

在远程桌面模式下，实验界面2个主模块：

- 教程区
- 操作区

<dx-tabs>
::: 教程区
教程区的操作和 [终端模式](#终端模式) 下一致，不做赘述。
:::
::: 操作区
远程桌面提供了该实验内容需要预装的环境，软件及数据集，用户可以直接使用并进行实验。
![](https://qcloudimg.tencent-cloud.cn/raw/2311f8825181c85d66e39b701b6d6ea2.png)
:::
</dx-tabs>





## 重置进度

有些时候我们需要重置进度，例如实验者教程忘记跑到某一步的时候。重置进度后，云主机配置保持不变，只是教程状态从头开始。
