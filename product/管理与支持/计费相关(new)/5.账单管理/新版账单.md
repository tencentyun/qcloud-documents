## 一、功能更新简介
相比资源账单1.0，新版账单能提供更详细的账单信息以及更好的对账体验，主要更新内容如下：

- **账单层级化：**按照账单概览-->账单汇总（by 产品、by 项目、by 地域、by 计费模式）-->资源 ID 账单-->明细账单-->用量明细层级的方式，提供不同详细程度的账单，满足您多维度、多粒度的对账需求。
- **账单数据丰富化：**支持展示更多资源详情，如资源用量、组件用量、实例名称、VPC ID、协作者 ID 等信息。
- **费用趋势：**支持查看近半年/一年的账单总费用趋势图，以及 by 产品、by 项目、by 地域和 by 计费模式的费用趋势情况。
- **PDF 电子账单：**PDF 格式电子账单，用户可在账单概览页自助下载，方便客户财务请款。
- **合并出账：**一个客户名下多个账号 ID 可申请合并出账，无需人工合并账单。
- **账单下载优化：**优化大数据量账单文件的下载延时，成功率，以及账单版本说明。
- **出账提速：**明确出账时间，从每个月4日出账提前到每个月 2-3 日出账。
- **账单 2.0 API：**提供新版 API，对于账单明细数据量超过10万行的客户，建议调用 API 进行数据拉取和分析（单击跳转 [API 概览](https://cloud.tencent.com/document/product/555/19170) 账单管理相关接口）。


>**温馨提示：**
1. 每月出账日为2-3日，具体出账日请以费用中心账单管理提示为准。出账完成前的账单费用不是最终的费用，仅供参考。
2. 仅支持开通新版账单月份起的账单，具体开通月份请以费用中心账单管理提示为准。历史月份的账单暂不支持该显示方式，会继续使用原有展现形式，历史账单请前往 [资源账单1.0](https://console.cloud.tencent.com/account/resources?reserved=1) 页面。


## 二、新版账单功能简介
账单反映您每月使用资源扣费和消耗的费用详情。
登录腾讯云官网，进入费用中心-账单管理，可以查看新版账单，主要包括 [账单概览](https://console.cloud.tencent.com/expense/bill/overview)、[账单明细](https://console.cloud.tencent.com/expense/bill/summary)、[账单下载](https://console.cloud.tencent.com/expense/bill/download)。



## 三、账单概览

[账单概览](https://console.cloud.tencent.com/expense/bill/overview) 支持查看账单费用趋势和账单汇总。费用趋势统计截止时间为所选对应月份，账单汇总展示为所选对应月份，默认按扣费周期统计。（账单统计周期说明可参考文档 [账单统计周期切换](https://cloud.tencent.com/document/product/555/30250)）
![](https://main.qcloudimg.com/raw/e187a6a497cb8f948e8158a7ca564400.png)


### 1. 费用趋势
费用趋势是账单的费用趋势图，分月展示各个月份的费用总览和月平均总费用。
- 可更换费用趋势统计纬度：时间（近半年/近一年），金额构成（总费用/现金支付/赠送金支付/代金券支付）。
- 单击对应月份柱状图，可在右侧看到所选月份的总费用及支付构成。
总费用是月度账单的**折后总费用**，计算公式为**总费用 = 现金支付 + 赠送金支付 + 代金券支付**。支付优先顺序依次为代金券、赠送金、现金，更多账户说明可参考 [云费用账户说明](https://cloud.tencent.com/document/product/555/7424)。

![](https://main.qcloudimg.com/raw/ae99ba6db5824d3f46a72f6b1264f553.png)

### 2. 账单汇总
#### 2.1 按产品汇总
根据产品纬度进行费用汇总，和 L1-按项目和产品汇总文件一致。
- 支持查看月度费用、占比及较上月费用的变化幅度。费用趋势可以查看该产品近半年的费用趋势。
- 单击产品名称可以进入账单明细页面，展现该产品中各资源的费用详情。

![](https://main.qcloudimg.com/raw/0a0b85a1499e722d6a01807bb9b4ecf6.png)

#### 2.2 按项目汇总
根据项目纬度进行费用汇总，和 L1-按项目和产品汇总文件一致。
- 支持查看月度费用、占比及较上月费用的变化幅度。费用趋势可以查看该项目/产品近半年的费用趋势。
- 单击项目左侧箭头可以展开产品明细；单击对应的项目/产品名称可以进入账单明细页面，展现该项目/产品中各资源的费用详情。

![](https://main.qcloudimg.com/raw/f5d56eeaa4194cce9e775e05a73b7607.png)

#### 2.3  按地域汇总
根据地域纬度进行费用汇总。
- 支持查看月度费用、占比及较上月费用的变化幅度。费用趋势可以查看该地域/产品近半年的费用趋势。
- 单击地域左侧箭头可以展开产品明细；单击对应的地域/产品名称可以进入账单明细页面，展现该地域/产品中各资源的费用详情。

![](https://main.qcloudimg.com/raw/8d445f4a77bcc429cfbc9c5fa6119e9a.png)


#### 2.4 按计费模式汇总
根据计费模式纬度进行费用汇总。
- 支持查看月度费用、占比及较上月费用的变化幅度。费用趋势可以查看该计费模式/产品近半年的费用趋势。
- 单击计费模式左侧箭头可以展开产品明细；单击对应的计费模式/产品名称可以进入账单明细页面，展现该地域/产品中各资源的费用详情。

![](https://main.qcloudimg.com/raw/f5e5e0eab9bc63faa9c27b6627fc10db.png)




## 四、账单明细
[账单明细](https://console.cloud.tencent.com/expense/bill/summary) 支持在线查看资源 ID 账单和明细账单，展示为所选对应月份，默认按扣费周期统计。

### 1. 资源 ID 账单
资源 ID 账单根据资源 ID 聚合明细展示账单，和 L2-按资源 ID 汇总文件一致。
- 支持根据产品、项目、地域、计费模式、交易类型等条件进行筛选查看。
- 支持输入资源 ID 或实例别名搜索明细。
- 支持自定义设置列表展示字段。

![](https://main.qcloudimg.com/raw/adf320cfb8414e17ecd2d3b942f6d757.png)

- 包年包月的明细支持单击订单 ID，可进入订单管理页面查看订单详情。
- 明细最右侧可单击账单详情，进入该资源的账单详情。

![](https://main.qcloudimg.com/raw/aa63d508e6250b35d9e5ca16789df380.png)
![](https://main.qcloudimg.com/raw/96f079106ff6e14acf7350676763ab4b.png)

### 2. 明细账单
明细账单不做聚合，每笔费用均为一条明细记录，和【L3-账单明细】文件一致。
- 支持根据产品、项目、地域、计费模式、交易类型等条件进行筛选查看。
- 支持输入资源 ID 或实例别名搜索明细。
- 支持自定义设置列表展示字段。

![](https://main.qcloudimg.com/raw/f6352aab4ab4f7b403e149434c5424af.png)



## 五、PDF 电子账单
PDF 格式电子对账单，方便客户财务请款，用户可在账单概览页右上角或账单下载页自助下载。

### 1. 客户信息
客户信息包括实名认证名称、账号 ID、账单周期、账单类型及应付费用总计。
![](https://main.qcloudimg.com/raw/34890f2f9a603da95e2dbfbb7e477528.png)

### 2. 账单费用概览
账单费用概览能直观展示账单周期内的费用概览，包括消费汇总、退款汇总、调账汇总。
应付费用是现金支付一列金额之和。
![](https://main.qcloudimg.com/raw/df781b5ccce1a533fdab26a1a654702d.png)

### 3. 按产品汇总
按照产品和组件类别汇总，展现账单周期内的消费、退款、调账情况。
![](https://main.qcloudimg.com/raw/3658dfb570e966ce52c31683b5f14e45.png)

### 4. 账单术语及说明
关于 PDF 电子账单相关术语及说明，具体内容以最新文件中的说明为准。

### 5. 腾讯云收款信息
使用 [新线下汇款（对公转账）](https://cloud.tencent.com/document/product/555/9901) 的腾讯云收款信息，您也可以前往 [账户充值](https://console.cloud.tencent.com/account/recharge) 页面查看信息和操作充值。
![](https://main.qcloudimg.com/raw/99069c975e2fa33e8b7234e4f57cf44f.png)



## 六、账单下载
>温馨提示：
1. 未出账月份的账单暂时不可下载，用户可直接查看 [账单概览](https://console.cloud.tencent.com/expense/bill/overview) 或 [账单明细](https://console.cloud.tencent.com/expense/bill/summary)。
2. 仅支持开通新版账单月份起的账单，历史月份账单下载仍然在 [资源账单1.0](https://console.cloud.tencent.com/account/resources?reserved=1) 页面。
3. 如需进行资源账单汇总和明细的对账、资源账单和收支明细的对账，更多内容可以参考 [对账指引](https://cloud.tencent.com/document/product/555/14170)。


### 1. 下载账单压缩包
可以进入 [账单下载](https://console.cloud.tencent.com/expense/bill/download) 页面，单击下载账单。
单击下载账单后，需要一定时间生成账单，生成后系统会在当前页面提供下载按钮，请您耐心等待。
![](https://main.qcloudimg.com/raw/2217f9484ac0c86ab608c5259e5a9f85.png)

下载文件为压缩包，解压后可以获取以下文件：
- **L0-PDF 电子账单：**提供 PDF 电子账单，方便客户财务请款。
- **L0-账单概览：**费用概览及收款信息。
- **L1-按项目和产品汇总：**支持按产品汇总、按项目汇总、按产品+项目汇总。
- **L2-按资源 ID 汇总：**支持按资源 ID 汇总。
- **L3-账单明细：**每笔费用均为一条明细记录。
- **账单说明：**关于账单术语说明及生成时间记录。

### 2. 单独下载 PDF 电子账单
可以在账单压缩包中获取 PDF 电子账单，也可以进入 [账单概览](https://console.cloud.tencent.com/expense/bill/overview)页面，选择对应月份后，单击下载 PDF 账单概览。
单击下载账单后，需要一定时间生成账单，生成后系统会在当前页面提供下载按钮，请您耐心等待。
![](https://main.qcloudimg.com/raw/208018c0dd414c2e4dcc8bd9561ae8ae.png)

下载文件为 PDF 文件，主要包含客户信息、账单费用概览、账单明细、腾讯云收款信息、相关说明等内容。



### 3. 合并出账
>**该功能需申请后开通，申请流程可咨询您的客户经理。**

- 使用父账号登录腾讯云官网，进入 [账单明细下载](https://console.cloud.tencent.com/expense/bill/download) 页面，选择父子账号合并出账。
- 选择按扣费/计费周期、对应月份、合并出账的子账号，单击下载合并账单。

![](https://main.qcloudimg.com/raw/56a7ef902152d0eb81422dc00065ae10.png)

- 单击下载合并账单后，会自动跳转 [下载记录](https://console.cloud.tencent.com/expense/download) 页面，需要一定时间生成账单，生成后系统会在当前页面提供下载按钮，请您耐心等待。

![](https://main.qcloudimg.com/raw/ec9577789a1382c41139b580dab08a4a.png)



### 4. 调用API
新版账单 API，对于账单明细数据量超过10万行的客户，建议调用 API 进行数据拉取和分析，单击跳转查看 [API 概览](https://cloud.tencent.com/document/product/555/19170) 账单管理相关接口 。
