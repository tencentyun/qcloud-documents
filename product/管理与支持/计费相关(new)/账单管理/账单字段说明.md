
| 字段中文名称 | 字段说明 |
|---------|---------|
|支付者 UIN |支付者账号 ID，用户在腾讯云的唯一账号标识|
|使用者 UIN |资源归属者账号 ID|
|操作者 UIN |操作者账号 ID，下单购买或开通产品的用户|
|产品名称 |云产品大类，产品四层的第1层，如云服务器 CVM、云数据库 MySQL|
|计费模式 |资源的计费模式，包年包月或按量计费|
|项目名称 |资源所属项目，由用户为资源自助分配，未分配则为默认项目|
|地域 |资源所属地域，如华南地区（广州）|
|可用区 |资源所属可用区，如广州三区|
|国内国际 |资源所属国内或国际|
|资源 ID |实例 ID，可以在各产品控制台查看|
|实例名称 |资源别名，由用户为资源自助设置，未设置则为空|
|子产品名称 |云产品子类，产品四层的第2层，如云服务器 CVM-标准型 S1|
|交易类型 |包年包月新购/续费/升降配/退款、按量计费小时结、按量日结、按量计费月结、调账补偿、调账补扣等类型|
|订单 ID |订单唯一标识，可以在订单管理中查看订单|
|交易 ID |交易唯一标识|
|扣费时间 |资源扣费时间|
|开始使用时间 |资源开始使用时间|
|结束使用时间 |资源结束使用时间|
|组件类型 |组件类型的名称，产品四层的第3层，如 CPU、内存、带宽、系统盘等|
|组件名称 |组件的名称，产品四层的第4层，如内存-标准型S2、高性能云硬盘-存储空间等|
|组件刊例价 |组件的官网原始单价|
|组件单价 |组件的折后单价|
|组件价格单位 |组件刊例价对应的价格单位|
|组件用量 |组件的用量|
|组件用量单位 |组件用量对应的单位|
|使用时长 |资源使用的时长|
|时长单位 |资源使用的时长单位|
|组件原价 |资源的原始总价，等于刊例价 x  用量 x 时长|
|折扣率 |资源享受的折扣优惠力度，1表示无折扣，0表示0折|
|优惠后总价 |资源的折后总价，等于组件原价 x 折扣率，等于组件单价 x 用量 x 时长|
|代金券支出 |优惠后总价中使用代金券抵扣的金额|
|赠送账户支出 |优惠后总价中使用赠送金抵扣的金额|
|现金账户支出 |优惠后总价中使用现金抵扣的金额|
