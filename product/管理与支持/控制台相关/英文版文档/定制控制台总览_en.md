1. The Console page provides shortcut buttons for accessing service consoles and an intuitive user interface for exploring Tencent Cloud and obtaining useful tips, as shown below:
![Console 3](https://mc.qcloudimg.com/static/img/b72e292bbcc3f7613ceed94b91b2ef86/image.jpg)
2. On the Console page, you can customize your own overview page by adding or removing modules as needed.
3. Click the **Custom Module** button in the upper right corner of the navigation bar to enter the customization page, as shown below:
![Custom module](https://mc.qcloudimg.com/static/img/27310f238910520c73eaaa3dbbf7c6ef/image.jpg)
4. In the custom module settings, select the module you want to see on the console page.
5. In the custom module settings, you can adjust the priorities of the modules using the drag-and-drop buttons on the right, as shown below:
![Select module](https://mc.qcloudimg.com/static/img/07a734b04acba081e33dca9dba26d9fd/image.jpg)
After all the required modules are selected, click **Save**.
