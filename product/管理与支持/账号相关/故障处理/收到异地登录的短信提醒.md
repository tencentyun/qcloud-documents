## 可能原因

当您的腾讯云账号有登录行为时，腾讯云发现该账号当前登录的 IP 地址和常用的 IP 地址不一致，导致触发该提醒。

>? 
> - 异地登录提醒默认为所有账号开启，无法关闭。该提醒与账号是否开启异地登录保护无关。
> - 异地登录指账号登录的 IP 地址为非常用登录城市。
> - 开启异地登录保护后，系统会根据登录 IP 地址是否所属常用登录城市进行判断。若账号在非常用城市登录，需进行身份校验。
> 


## 处理步骤

在收到的异地登录提醒中，请您按照下列步骤进行确认：

1. 确认本次登录是否为账号的其他共同使用者。
 - 是，请忽略该提醒。
 - 否，请进行下一步。
2. 确认本次登录的 IP 地址是否归属您正在使用的手机、iPad 等移动设备。
 - 是，请忽略该提醒。
 - 否，请进行下一步。
3. 确认本次登录行为是否为本人操作。
 - 是，请忽略该提醒。
 - 否，请 [修改账号密码](https://cloud.tencent.com/document/product/378/14623)、[开启异地登录保护](https://cloud.tencent.com/document/product/378/43101) 和 [下线所有登录](https://cloud.tencent.com/document/product/378/43102)，并进入下一步 。
4. 前往 [云审计控制台](https://console.cloud.tencent.com/cloudaudit)，检查云审计日志，确认是否存有敏感操作。
 - 是，请检查账号资源情况，如资源有被影响，请联系 [在线客服](https://cloud.tencent.com/online-service) 获取帮助。
 - 否，建议开启 [登录保护](https://cloud.tencent.com/document/product/378/8392) 和 [操作保护](https://cloud.tencent.com/document/product/378/10740)。

>!
> - 网络环境不稳定，可能会导致您的 IP 发生变化，也会触发异地登录提醒。如确认是您本人操作，忽略提醒即可。
> - 账号中心支持多种场景的账号安全保护设置，安全保护有助于保证账号以及账号下资源的安全，建议请前往 [安全设置](https://console.cloud.tencent.com/developer/security) ，开启 [登录保护](https://cloud.tencent.com/document/product/378/8392)、[操作保护](https://cloud.tencent.com/document/product/378/10740) 和 [异地登录保护](https://cloud.tencent.com/document/product/378/43101)。
