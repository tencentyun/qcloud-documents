## 操作场景 
该任务指导您通过 GitHub，提交您对腾讯云官网文档的改进意见，帮助腾讯云文档优化迭代，提升用户体验。


## 前提条件
您已 [注册](https://github.com/join?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home) 并 [登录](https://github.com/login) 了 GitHub。


## 操作步骤

1. 在任意一篇 [腾讯云文档](https://cloud.tencent.com/document/product) 中，单击【前往 GitHub 编辑】，提交您对文档改进的建议。
   ![](https://main.qcloudimg.com/raw/c108f17a5037f247fe710bd1bf25c3df.png)
2. 进入 GitHub 页面，单击<img src ="https://main.qcloudimg.com/raw/dc516033976f7571fec1562ac2d9588e.png" style ="margin:0">进入文档编辑页面。
   ![](https://main.qcloudimg.com/raw/2343c31384b089244ce775173e518cf9.png)
3. 在编辑页面中，用户可以新增/删除/修改文档的内容。以下为示例：
   ![](https://main.qcloudimg.com/raw/c3065a5d4549f0890f321f769f269e5d.png)
4. 完成编辑后，将页面下滑至底部。填写标题和描述，描述建议尽量详细，可以介绍您优化文档的思路。完成后，单击【Propose changes】进入下一步。
   ![](https://main.qcloudimg.com/raw/3bab90176010c5efe2a3760c5e0654af.png)
5. 进入确认页面，确认无误后，单击【Create pull request】进入下一步。
   ![](https://main.qcloudimg.com/raw/8536b11d42eed74d9b50cb91bfcfead4.png)
6. 最后，单击【Create pull request】提交优化申请。
   ![](https://main.qcloudimg.com/raw/ee0538f4b4eea8cad614d38cd7477f27.png)
7. 腾讯云文档团队将会收到该申请，评审通过后，即可正式生效。
