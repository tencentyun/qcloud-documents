### 入驻云市场是否需要缴纳保证金？
为了更好地保障腾讯云用户的权益，提升云市场服务商的服务水平和商品质量。发布商品前，服务商需缴纳保证金。
如果服务商未缴纳保证金，其名下的商品不得上架售卖。

保证金金额如下：

| 商品类目      | 保证金金额 |
| --------- | ----- |
| API 服务市场  | 2000元 |
| 运维服务市场    | 5000元 |
| 标准化解决方案市场 | 5000元 |
| 网站建设市场    | 5000元 |
| 微信小程序市场   | 5000元 |
| 数据智能市场    | 5000元 |
| 镜像市场      | 5000元 |
| 企业应用市场    | 5000元 |

服务商可以入驻多个类目，并且都通过了审核，保证金只需按照就高原则缴纳保证金即可
如： 服务商在 API 市场、网站建设市场类目下同时发布商品，只需缴纳保证金人民币5000元。

### 保证金是否可以退回？
您好，如果服务商在合作期间没有违反云市场服务商管理规范，合作终止后按照相关规定退回保证金。

### 保证金是开收据还是发票？
您好，腾讯云开收据给服务商。

### 如果入驻账号下有足够扣除保证金的余额，是否还要充值？
您好，服务商的保证金需要单独缴纳，不支持从服务商的充值账号下扣除，不足时需要通过汇款到指定的保证金账号。

### 入驻多个类目，是否需要缴纳多份保证金？
服务商可以入驻多个类目，并且都通过了审核，保证金只需按照就高原则缴纳保证金即可
例如： 服务商在 API 市场、运维市场类目下同时发布商品，只需缴纳保证金人民币5000元；
