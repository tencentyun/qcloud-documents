### 镜像服务
[镜像服务商品](https://cloud.tencent.com/document/product/306/3019) 是指将服务商制作的镜像作为商品，用户可以基于镜像来创建云主机实例，从而获得与镜像一致的系统环境。    
这类商品在操作系统上整合了具体的软件环境和功能，通过将应用软件与云资源结合，实现用户对云主机即开即用。

### SaaS(应用软件)服务
[SaaS(应用软件)服务商品](https://cloud.tencent.com/document/product/306/3030) 是将服务商提供的在线应用软件作为商品。用户使用时无需购买独立的云资源，只需购买后登录到指定的网站后即可使用。

### 人工服务
[人工服务商品](https://cloud.tencent.com/document/product/306/3029) 是指服务商为用户提供的人工服务作为商品，不交付具体的软件或云资源。

### API 服务
[API 服务商品](https://cloud.tencent.com/document/product/306/13004) 是指将服务商提供的 API 接口服务作为商品，用户购买后可以将 API 商品集成于自己开发的产品中去。同时，云市场将记录用户使用 API 的计量数据。
