## 商品接入  
腾讯云云市场向服务商提供多种类型的交互方式，请根据您所需接入的商品特性进行选择，具体请参考 [商品接入](https://cloud.tencent.com/document/product/306/20480) 。

## 新建商品                   
2.1. 新建镜像服务商品，请参照商品接入中的 [镜像服务](https://cloud.tencent.com/document/product/306/3019)。    
2.2. 新建人工服务商品，请参照商品接入中的 [人工服务](https://cloud.tencent.com/document/product/306/3029)。   
2.3. 新建 SaaS 服务商品，请参照商品接入中的 [ SaaS 服务](https://cloud.tencent.com/document/product/306/3030)。     
2.4. 新建 API 服务商品，请参照商品接入中的 [ API 服务](https://cloud.tencent.com/document/product/306/13004)。 

## 操作商品  
### 1. 修改商品信息
可对商品管理页面中的商品进行信息修改。操作如下：     
1.1.	进入 [云市场-服务商管理后台-商品管理](https://console.cloud.tencent.com/serviceprovider/goods/service) 页面        
1.2.	选择对应的服务类型，在相应的商品处单击【修改】   
![](https://main.qcloudimg.com/raw/dae361598946657b4752cb7cbd110459.png)   
1.3.	修改相应的商品信息  
![](https://main.qcloudimg.com/raw/04725a50dc9ac755689113259c22d9eb.png)    
1.4.	修改完商品信息，单击【提交】  

### 2. 查看商品前台页面
可对商品管理页面中已上架的商品进行查看。操作如下：  
2.1.进入 [云市场-服务商管理后台-商品管理](https://console.cloud.tencent.com/serviceprovider/goods/service) 页面  
2.2.	选择对应的服务类型，在相应的商品处单击【查看】  
 ![](https://main.qcloudimg.com/raw/e5f772e589f25a7978f90a4d76839474.png)  
前台展示效果：
![](https://main.qcloudimg.com/raw/bef28ef9a389a5216250dc0b9671f96a.png)

### 3.下架商品
服务商可在商品管理页面中对您暂停销售的商品进行下架。操作如下：  
3.1.	进入 [云市场-服务商管理后台-商品管理](https://console.cloud.tencent.com/serviceprovider/goods/service) 页面  
3.2.	选择对应的服务类型，在相应的商品处单击【下架】  
 ![](https://main.qcloudimg.com/raw/b745e2de4273600ac1a22ac2addbac0c.png)  
3.3. 单击【确定】，完成下架    
 ![](https://main.qcloudimg.com/raw/3f929ac0ffd3bd461c51dd4cd56a1bd4.png)  

### 4.上架商品（已下架商品）
若下架的商品如需再次上架，可在下架的商品对应操作栏中单击【修改】，即可完成商品上架。操作如下：  
4.1.	进入 [云市场-服务商管理后台-商品管理](https://console.cloud.tencent.com/serviceprovider/goods/service) 页面    
4.2. 选择对应的服务类型，在相应的商品处单击【修改】  
![](https://main.qcloudimg.com/raw/98588664e996f2299e27a6a92aa4b9b3.png)     
4.3. 若商品信息无需修改，可直接单击【提交】       

### 5.预览商品前台页面
可对审核中、审核不通过以及已下架的商品进行预览。操作如下：   
5.1. 进入 [云市场-服务商管理后台-商品管理](https://console.cloud.tencent.com/serviceprovider/goods/service) 页面    
5.2. 选择对应的服务类型，在相应的商品处单击【预览】
![](https://main.qcloudimg.com/raw/caf726d0769281ca005738decff2e6a5.png)   
前台展示效果：
