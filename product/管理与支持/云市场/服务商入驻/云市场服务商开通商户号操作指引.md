## 开通商户号流程图
您需要选择一种场景申请开通商户号，以保证客户能够支付购买商品。

| 场景类型 | 必需完成的流程 | 
|---------|---------|
| 开通微信商户号 | 流程1 | 
| 开通财付通商户号 | 流程2 + 流程3 | 
| 开通微信和财付通商户号 | 流程1 + 流程2 + 流程3 | 
      
![](https://qcloudimg.tencent-cloud.cn/raw/676f042de5364df8b74969c9d61b811c.png)

## 开通商户号操作步骤

1. 登录 [服务商管理](https://market.cloud.tencent.com/) 控制台，进入概览页。
2. 在概览页的任务指引模块，选择微信商户或财付通商户，单击**前往开通**，即可跳转到对应的资料填写页面。
![](https://qcloudimg.tencent-cloud.cn/raw/db4ebbab252d437d1b7a403b5d7b000e.png)
3. 在开通商户的资料填写页面，根据页面提示填写信息（以微信商户号开通为例）：
>?
>- **微信商户**和**财付通商户**开通要求填写的资料，如果两类都开通，请保持两类商户的信息一致。
>- 如您先填写了微信商户，在财付通商户点击前往开通时，页面即会自动填充之前填写过的信息。
>- 填写页面带 `*` 的字段均为必填项。

	- **组织机构代码证信息**：多证合一的企业，上传统一的社会信用代码证件即可，图片大小不超过400KB
	![](https://qcloudimg.tencent-cloud.cn/raw/877a788a800bedefdc64ba239a22cfdf.png)
	- **超级管理员信息**：
	 - 若超管类型为负责人，超管姓名填写实际负责人的姓名
	 - 负责人：经商户授权办理微信支付业务的人员，授权范围包括但不限于签约，入驻过程需完成账户认证
	![](https://qcloudimg.tencent-cloud.cn/raw/72a232f7afd8f531a03141dfc0b45239.png)
	- **经营者/法人证件信息**：身份证照片大小不超过400KB
	![](https://qcloudimg.tencent-cloud.cn/raw/745c1be84a9b83582e11d22c11840697.png)
4. 确认资料填写无误后，单击页面底部的**确认提交**。
5. 提交后，**微信商户号**开通商户状态会更新为：“审核中”，界面会出现“前往账户激活”和“签约”。
![](https://qcloudimg.tencent-cloud.cn/raw/c6ad3c6dec891d5e5663797d1d204172.png)
	1. 无需等审核状态更新，单击**前往账户激活**，按照页面提示填写公司对公银行账号，并完成汇款认证。
![](https://qcloudimg.tencent-cloud.cn/raw/063a45e8b5e064c960e9213535290ad3.png)
	2. 单击**签约**，使用微信扫描弹出的二维码，根据页面提示完成操作。
	![](https://qcloudimg.tencent-cloud.cn/raw/5a461d18586313b8c08980e592f5eaf9.png)
>!
>- 签约主体需和超级管理员姓名一致：主体为“个体工商户/企业/党政、机关及事业单位/其他组织”，可选择：经营者/法人、负责人。
>- 负责人是指商户授权办理微信商户开通业务的人员，授权范围包括但不限于签约，入驻过程的账户验证等。

## 商企付账户激活步骤
### 注册收款功能
1. 当财付通商户号状态如下时，单击**前往账户激活**，进入注册企业账号页面。
![](https://qcloudimg.tencent-cloud.cn/raw/ced6f0de7f63fd276de23cd53924ef7d.png)
2. 在注册企业账号页面，根据页面提示填写注册信息。
>?为了资料快速有效审核，确保在同一天内上传**身份证正反面图片**+**营业执照照片**。
3. 内容确认无误后，单击页面底部的**确认提交**。
4. 使用超级管理员的微信扫描弹出的二维码，验证成功后将自动进入下一步。
![](https://qcloudimg.tencent-cloud.cn/raw/fb52182ad172f293b6d74fbd68bb1d17.png)

### 激活收款功能
1. 接收到审核通过的短信，会出现如下页面，单击**立即激活**，进入激活页面。
![](https://qcloudimg.tencent-cloud.cn/raw/0a85b5b03a9d6165c809b3ec45f94094.png)
2. 填写公司对公银行账户信息，确认无误，点击**下一步**。
![](https://qcloudimg.tencent-cloud.cn/raw/f05f2a6e39fd334e1850fbe07f458dee.png)
此时，系统会打款验证银行信息，页面会提示提交资料验证中。
![](https://qcloudimg.tencent-cloud.cn/raw/26ec2fbe8babc445a42f78981f89eb4f.png)
3. 提交资料验证通过后，开始进行账户验证，系统将向提交的对公账户打款，收到打款信息后，请填写收到的验证打款金额，并单击**确认**。
4. 账户验证验证后，**申请人**完成扫码签署协议，系统将在签署后更新签署状态。
5. 以下是页面为最终状态，单击**开始使用**即可进行财付通收款。
![](https://qcloudimg.tencent-cloud.cn/raw/a9472a4c8f04d001e717c826da685f13.png)

## 查询开通状态
1. 登录 [服务商管理](https://market.cloud.tencent.com/) 控制台，在概览页即可查看入驻状态。
微信商户号/财付通商户号如呈现**已开通**状态，表示伙伴已成功开通商户号。
![](https://qcloudimg.tencent-cloud.cn/raw/748d17c0982248d452bcc63a8bb74af4.png)
您也可以单击左侧菜单的**账户管理**，查看经营信息和具体的收款信息。
![](https://qcloudimg.tencent-cloud.cn/raw/88547e9831e856fc44bd6c6993dc55b2.png)
>?微信商户号中**银行卡验证信息验证**状态一般在两个小时内完成，请各服务商伙伴耐心等待。
