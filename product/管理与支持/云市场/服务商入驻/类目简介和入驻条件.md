## 类目市场简介

### 镜像市场
适用于运行环境、操作系统等以镜像方式交付的产品。

### SaaS 市场
适用于建站、小程序等以自动开通方式交付的产品。

### API 服务市场
适用于面向开发者提供 API 服务的产品。

### 技术运维服务市场
适用于技术运维相关的人工服务。

### 定制服务市场
适用于软件开发相关的人工服务。

### 应用授权市场
适用于应用相关的账号开通，授权码发送相关的人工服务。

### 安全市场
适用于安全相关的人工服务。

### 企业管家市场
适用于工商财税，人资社保等企业生命周期的相关人工服务。



## 服务商入驻条件
### 企业主体服务商
- 符合国家相关法律、法规规定。
- 有独立的研发能力及完善的售后服务能力，可提供不少于7 * 8小时的客服在线服务。
- 接受《腾讯云云市场服务商接入协议》及腾讯云云市场相关协议及《[云市场管理规范](https://cloud.tencent.com/document/product/306/11341)》。
