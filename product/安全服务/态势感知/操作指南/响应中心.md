
## 前提条件
如需使用流量威胁分析功能，需已开通 [安全运营中心高级版](https://buy.cloud.tencent.com/soc) 。
## 操作步骤
### 查看响应策略及事件列表
1. 登录 [安全运营中心控制台](https://console.cloud.tencent.com/ssav2/response)， 在左侧导航栏，单击**网络入侵** > **流量威胁响应**，进入流量威胁响应页面。
2. 在流量威胁响应页面，您可以查看 SSH 口令爆破类事件、RDP 口令爆破类事件、Linux 主机挖矿木马类事件、Windows  主机挖矿木马类事件、Linux 主机反弹 shell 事件、Windows 主机反弹 shell 事件的响应进度；单击某个威胁类别的**立即响应**，即可进入该类威胁详情页。
![](https://qcloudimg.tencent-cloud.cn/raw/2aaf4fc7c63abcfca6222451fd303041.png)
3. 在威胁详情页的上方，可以查看该类威胁的响应策略。
	- **SSH 及 RDP 口令爆破类事件**：
![](https://qcloudimg.tencent-cloud.cn/raw/03135531c1c1be12096c3750537498a4.png)
		- **排查攻击源**：包括封禁外网攻击 IP、隔离内网攻击 IP 网络、检查攻击主机是否安装主机安全专业版。
		- **排查被攻击资产**：包括修改账户的密码、隔离被攻击 CVM 网络、检查是否安装主机安全专业版。
		- **基线检测**：包括攻击机器弱密码检测、被攻击机器弱密码检测。
		- **木马检测**：包括对攻击机器进行木马扫描、对被攻击机器进行木马扫描、隔离攻击机器木马、隔离被攻击机器木马。
	- **Linux 及 Windows 主机挖矿木马类事件**：
	![](https://qcloudimg.tencent-cloud.cn/raw/05dcc683866ef63b7f2cf390423a7c44.png)
		- **阻断矿池连接**：阻断挖矿连接。
		- **木马检测**：包括对失陷机器进行木马扫描、对失陷机器进行木马隔离、检查是否安装主机安全专业版。
		- **排查失陷主机**：包括查看 CPU 使用率、定位恶意文件、查看可疑通信、清除可疑启动项。
		- **基线检测**：包括进行弱密码检测、进行漏洞检测。
	- **Linux 及 Windows 主机反弹 shell 事件**： 
![](https://qcloudimg.tencent-cloud.cn/raw/10d3714bbf2c99ee26d7778ab3acad61.png)
		- **阻断远控服务器连接**：阻断远控服务器连接。
		- **木马检测**：包括对失陷机器进行木马扫描、对失陷机器进行木马隔离、检查是否安装主机安全专业版。
		- **排查失陷主机**：包括定位可疑进程、定位恶意文件、清除可疑启动项。
		- **基线检测**：包括进行弱密码检测、进行漏洞检测。
4. 在威胁详情页的下方，可查看该类威胁告警列表，列表中包括事件名称、事件等级、事件子类、源/目的IP、受影响资产以及响应状态等，同时可进行相应操作。
![](https://qcloudimg.tencent-cloud.cn/raw/03ff1405693ed4946e3efa00aa646b33.png)
 - **事件等级**
事件等级分为4类，分别是严重、高危、中危及低危。
 - **状态**
响应状态分为4类，分别是未响应、响应中、已响应及存在风险。
	- **搜索**
 在流量威胁告警列表右上角单击搜索框，选择一个资源属性并输入对应的关键词，单击**Enter** 后即可搜索符合条件的威胁告警。
>?填写关键词需注意，同一个“过滤标签”（即资源属性）的多个关键字用竖线“|”分隔，多个“过滤标签”（即资源属性）用回车键分隔。
>
	- **筛选**
单击流量威胁告警列表的表头“事件等级”、“事件子类”或“状态”，即可进行流量威胁告警的筛选。
 - **排序**
单击流量威胁告警列表的表头“源IP”、“目的IP”、“未处理事件数”、“最近发生时间”或“最近响应时间”，即可进行倒序或正序排列。
 - **响应**
		- **方式1**：在流量威胁告警列表中，选中一个或多个状态为“未响应”和“存在风险”的威胁告警并单击**响应**，即可进行批量验证。
![](https://qcloudimg.tencent-cloud.cn/raw/b058f59a8845fc2d4e6a49487c04bfcc.png)
		- **方式2**：在流量威胁告警列表中，对于状态为“未响应”和“存在风险”的威胁告警，可以单击对应的**立即响应**或**再次响应**，即可对该威胁进行验证。
![](https://qcloudimg.tencent-cloud.cn/raw/0101640b14fd2f7f5d0c1d62774746bc.png)
 - **查看并标记安全事件**
若告警存在风险，可以在状态栏下单击存在风险的**数量**，查看响应详情。
![](https://qcloudimg.tencent-cloud.cn/raw/847d96cd70ff7f3265971dd02c4d5ff7.png)
有风险的检查项以红色字体标识。当该风险已处理，可在响应详情中单击**标记为处理**，表示该风险已处理完成。
![](https://qcloudimg.tencent-cloud.cn/raw/315fba62916d2fc5bd8dcffb334bc6e7.png)

### 忽略规则管理
忽略规则管理，即对流量威胁不再响应的源 IP 和目的 IP 进行管理。
1. 登录 [安全运营中心控制台](https://console.cloud.tencent.com/ssav2/response)， 在左侧导航栏，单击**网络入侵** > **流量威胁响应**，进入流量威胁响应页面。
2. 在流量威胁响应页面，单击**忽略规则管理**，进入忽略规则管理页面。
![](https://qcloudimg.tencent-cloud.cn/raw/bbeab2c4163f8a1a40e1c0c7ccf58eb5.png)
3. 在忽略规则页面，您可新建、编辑或删除规则。
	- **新建**
	单击**新建**，填写忽略规则的名称、备注、源 IP 和目的 IP（可填写多个源/目的 IP）。填写完毕，单击**确定**。
>!忽略规则新建完成后，将不再对忽略规则中的源 IP 和目的 IP 对应的威胁告警事件进行响应。
>
![](https://qcloudimg.tencent-cloud.cn/raw/00ac18219514af1eb62106a2b7a6bdf0.png)
	- **编辑**
在忽略规则列表中，单击**编辑**即可修改对应的忽略规则。
![](https://qcloudimg.tencent-cloud.cn/raw/41c51d08e483eb2cfb0153b069f1b24f.png)
	- **删除**
在忽略规则列表中，单击**删除**即可删除对应的忽略规则。删除之后，将恢复源 IP 和目的 IP 对应威胁告警事件的响应。
![](https://qcloudimg.tencent-cloud.cn/raw/1a9ce9ea0532f75012cc6eee17ab860c.png)
