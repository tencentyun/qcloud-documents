## 应用场景
用户在使用 BGP 高防 IP 实例过程中可以创建告警，系统将依据告警策略进行指标监控，当指标状态达到告警条件时，将触发警报，并向指定用户群体发送告警信息。用户可以根据告警信息，及时采取相应的预防或补救措施。合理地创建告警能帮助用户提高应用程序的健壮性和可靠性。有关告警的更多信息，请参见云监控 [创建告警策略](https://cloud.tencent.com/document/product/248/6215)。

## 触发告警条件
当用户需要对实例的某一指标进行监控时，需要先创建告警策略。告警策略包括名称、类型和告警触发条件三个必要组成部分。每个告警策略是一系列告警触发条件的集合，满足一个条件将会触发告警并向指定的用户发送告警信息，用户接收告警后可以及时查看并采取相应措施。
>!用户需要设置默认告警接收人，否则腾讯云 BGP 高防 IP 的默认告警策略将无法通知到用户。

## 操作步骤
### 创建告警策略
1. 登录 [腾讯云云监控控制台](https://console.cloud.tencent.com/monitor/overview)，在左侧导航中选择【告警配置】>【告警策略】。
2. 在告警策略列表页面，单击【新增】，进入新建策略页面。
 ![](https://main.qcloudimg.com/raw/5cdc3031b85b51af9ce74ef32f811391.png)
3. 设置策略名称、策略类型等基础选项内容。策略类型选择需要告警的服务，本文策略类型以【DDoS 防护 - 高防 IP】为示例。
 ![](https://main.qcloudimg.com/raw/6f895d54fafdaab476e3391437a65210.png)
4. 设置告警对象。
	- 选中【全部对象】，则该告警策略绑定当前账号的全部实例。
	- 选中【选择部分对象】，则该告警策略绑定用户选中的实例。
	- 选中【选择实例组】，则该告警策略绑定用户选中的实例分组。
	![](https://main.qcloudimg.com/raw/e54f128dea579b723dcbd714d814daaa.png)
5. 设置告警触发条件：有触发条件模板和配置触发条件两种触发方式，可选择其中一种触发方式。
	-  触发条件模板
	单击开启【触发条件模板】，并在下拉列表选择已配置的模板，具体配置请参见 [配置触发条件模板](https://cloud.tencent.com/document/product/248/18474) 文档。若新建的模板没有显示，则单击右侧的【刷新】，即可刷新触发告警模版选择列表。
	- 配置触发条件
	告警触发条件是指标、统计周期、比较关系、阈值、持续周期和重复通知策略组成的一个有语义的条件。
	- 参数配置示例如下所示：
		- 指定指标为：连接数 。
		- 统计周期为：1分钟 。
		- 比较关系为：> 。
		- 阈值为：100个 。
		- 持续周期为：持续1个周期。
		- 重复通知为：每天警告一次 。表示：每1分钟收集一次连接数数据，若某个 BGP 高防 IP 实例统计的连接数连续两次大于100个，则触发告警，且每天警告一次。
		>?支持自定义设置重复告警策略，可选不重复、5分钟、10分钟、周期指数递增等重复频率。其中周期指数递增的含义是当该告警第1次、第2次、第4次、第8次...第2的N次方次被触发时，向您发送告警信息。意义是告警信息发送时长间隔将越来越长，一定程度上避免重复告警对您的骚扰。
		
![](https://main.qcloudimg.com/raw/9de923212a23e8d151d829cd333e26d9.png)
6. 确认无误后，单击【完成】即可。


### 关联对象
创建完告警策略后，用户可以为其关联告警对象，对象达到告警触发条件时会发送告警。配置关联告警对象的详细步骤如下：
1. 登录[ 腾讯云云监控控制台](https://console.cloud.tencent.com/monitor/overview)，在左侧导航栏选择【告警配置】>【告警策略】。
2. 在告警策略管理页面中，找到需要关联对象的策略，单击告警策略名称，进入详情页面。
 ![](https://main.qcloudimg.com/raw/89604d2a8c0043d7a67d82613c77d199.png)
3. 在告警对象区域，单击【新增对象】。
 ![](https://main.qcloudimg.com/raw/445b705a54593ec5a8558163b684e503.png)
  >?支持在不同地域下单击【新增对象】。如需解除告警对象，则勾选需要解除关联关系的实例，单击【解除】即可解除该对象与告警策略的关系。
3. 在配置页面，选择需要关联的 BGP 高防 IP 实例，单击【应用】即可关联告警对象。
![](https://main.qcloudimg.com/raw/8325e534e8ea64339b5fcfa48146dd2a.png)

### 设置接收告警对象
触发告警时，同告警接收组的人员都会收到相应的告警。设置接收告警的对象的详细步骤如下：
1. 登录 [腾讯云云监控控制台](https://console.cloud.tencent.com/monitor/overview)，在左侧导航中选择【告警配置】>【告警策略】。
2. 在告警策略管理页面中，找到需要添加告警接收组的策略，单击告警策略名称，进入详情页面。
3. 在告警接收组区域，单击【编辑】。
4. 在配置页面，勾选需要通知的用户组、接收渠道，单击【保存】，即可完成告警接受对象的设置。
	- 目前支持邮件、短信、微信和电话四种告警接收方式。您可在 [访问管理控制台](https://console.cloud.tencent.com/cam) 修改用户的手机、邮箱和微信信息，以决定用户可以使用哪种方式接收告警信息，修改用户信息请参见 [新建子用户](https://cloud.tencent.com/document/product/598/13674#.E6.96.B0.E5.BB.BA.E5.AD.90.E7.94.A8.E6.88.B7)。
	-  告警策略启用微信接收告警，请参见 [启用微信渠道](https://cloud.tencent.com/document/product/248/17707)。
	-  告警策略启用电话接收告警，配置以下参数即可。
		-  轮询次数：在无有效触达时，对所有接收人逐一轮询拨打的最多次数。
		-  触达通知：电话成功接收或轮询结束发送信息给所有接收人。
	 >?
>- 用户也可以在创建告警策略的页面设置告警接收对象。
>- 电话告警渠道目前灰度中，如有需要，请 [提交工单](https://console.cloud.tencent.com/workorder/category) 或 [联系我们](https://cloud.tencent.com/about/connect) 进行支持。
