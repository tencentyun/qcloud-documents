## Supported Region
High Defense IP can only be purchased in Hong Kong.
## Purchase Procedure
1. This is a prepaid product. You can click **Create** on [High Defense IP Console](https://console.cloud.tencent.com/dayu/bgpip) to purchase the product.
 ![1](https://main.qcloudimg.com/raw/26145cd9a28854975e01abc5c761691d.png)
2. Select the minimal defense peak.
 ![2](https://main.qcloudimg.com/raw/faffb7ddd6e7b3ea3923d0c225249703.png)
3. Select the elastic defense peak.
 ![3](https://main.qcloudimg.com/raw/8871e2bc02d6de294f4c9d6ffef08455.png)
4. Select the bandwidth of forwarding traffic.
 ![4](https://main.qcloudimg.com/raw/efc189ec48839af8af58ba23022265fd.png)
5. Select the duration, and set whether the auto renew is enabled. Check the information, and then click the **Subscribe now** button.
 ![5](https://main.qcloudimg.com/raw/ff20781811a276a5fc07a4f6cfb14edf.png)
