### 为什么要有封堵策略？
公有云通过共享基础设施的方式降低用云成本，所有用户共享公有云的外网出口，当发生大流量攻击时，除了影响被攻击者，整个腾讯云的网络都可能会受到影响。为了避免攻击影响到其它未被攻击的用户，保证整个云平台网络的稳定，所以需要有封堵策略。

### 为什么不能免费无限抗攻击？
- DDoS 攻击不仅影响受害者，也会对整个云网络造成严重影响，影响云内其它未被攻击的用户。
- DDoS 防御的成本非常高，包括带宽成本和清洗成本，其中，带宽费用是最大的成本。带宽是腾讯云向电信、联通、移动等运营商购买，运营商不会区分是正常流量还是攻击流量，直接收取腾讯云的带宽费用。
- 腾讯云在成本可承受的情况下，为云内用户提供免费的 DDoS 基础防护服务，当攻击流量超出免费防护阈值时，腾讯云会屏蔽被攻击 IP 的外网流量。

### 为什么不能立即解除封堵？
- DDoS 攻击通常会持续一段时间，不会在封堵后立即停止，具体持续时间不定。腾讯云安全团队会根据大数据分析的结果，设定默认封堵时长。
- 由于封堵是在运营商网络部分生效，被攻击的外网 IP 进入封堵后，腾讯云无法监控到攻击流量是否停止。如果在攻击未停止的情况下解除封堵，被攻击的外网 IP 将再次进入封堵，同时在解除封堵至再次封堵生效的这段时间内，攻击流量将直接进入腾讯云的基础网络，可能会影响到云内其它客户。
- 同时，封堵是腾讯云向运营商购买的服务，解封次数、频率都有限制。

### 封堵未解除时，如何紧急恢复业务？
- **使用基础防护资源的用户**
在封堵未解除的情况下，如急需恢复业务，建议更换源站 IP 并绑定 [BGP 高防包](https://cloud.tencent.com/document/product/297/15344)，根据攻击流量大小选择最大防护峰值进行防护。
BGP 高防包是直接为腾讯云 CVM、CLB 等云产品输送 DDoS 高防能力的安全产品。无需更换 IP，购买后只需简单几步 [操作](https://cloud.tencent.com/document/product/297/15828) 即可生效。
- **使用 BGP 高防包的用户**
在封堵未解除的情况下，如需紧急恢复业务，建议更换源站 IP，增加高防包的最大防护容量后，重新绑定到新的源站 IP，避免再次触发封堵。
- **使用 BGP 高防 IP 的用户**
建议购买更大容量的高防 IP，并配置好原有转发规则。如果是非腾讯云服务器，建议更换源站 IP，使用 [BGP 高防 IP 服务](https://cloud.tencent.com/document/product/297/15411)，保证业务的正常运营。
BGP 高防 IP 是解决云外业务遭受大流量 DDoS 攻击的安全防护产品。用户通过配置转发规则，将攻击流量引至腾讯高防 IP 并清洗，保障业务稳定可用。

### 如何提前解封？
1. **基础防护的外网 IP**
将未被使用过的高防包绑定到基础防护的外网 IP 时，可自动提前解封。
2. **高防包绑定的外网 IP**
升级高防包保底容量时，可自动提前解封。
3. **BGP 高防 IP**
升级保底容量后，可自动提前解封。
4. **自助解封**
每个用户享有少量次数的手动自助提前解封配额，可在紧急情况下使用，详情请参见 [自助解封操作指南]()（链接至用户指南-自助解封）。

### 为什么自助解封有次数限制？
封堵是腾讯云向运营商购买的服务，而运营商有明确的封堵解除时间和频率限制，所以封堵状态无法频繁手动解除。

### 如何连接已被封堵的服务器？
您可通过以下方式连接已被封堵的服务器：
1. 通过同地域的其它云服务器内网 IP 连接被封堵服务器。
2. 通过云服务器控制台，单击登录后，选择”浏览器 VNC“登录。
3. 通过云服务器控制台，绑定新的弹性公网 IP 登录.

### 如何预防被封堵？
使用高防服务，根据受到的攻击流量峰值，选择适当的高防容量，确保最大防护容量大于攻击峰值。

### 如何避免解封后被再次封堵？
- 对于业务部署在腾讯云机房的用户：
 - 未购买高防服务的用户：您可购买 [BGP 高防包](https://buy.cloud.tencent.com/bgp_sp)，直接提升 DDoS 防护能力，轻松应对攻击流量。
 - 已购买高防包服务的用户：建议您升级保底/弹性防护峰值，提高防御能力。
- 对于业务部署在非腾讯云机房的用户：
 - 未购买高防服务的用户：您可购买 [BGP 高防 IP](https://buy.cloud.tencent.com/bgp_ip)，轻松解决 DDoS 流量攻击困扰，保障服务器的正常运行。
 - 已购买高防服务的用户：建议您升级保底/弹性防护峰值，提高防御能力。
>**注意：**
>开启弹性防护可帮您抵御大规模流量攻击，且弹性防护按天按量灵活付费，可节约您的安全成本。
