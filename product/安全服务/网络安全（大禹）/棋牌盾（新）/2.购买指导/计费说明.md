## 计费相关
| 计费项 | IP 个数 | 转发流量                   |
| ------------ | -------- | ---------------------- |
| 计费模式  | 按年购买     | 按天按量计费                 |
| 付费方式  | 预付费      | 后付费                    |
| 付费说明 | 购买时一次性支付 | 按 0.80 元 / GB 后付费，账单次日生成 |
## 到期预警
棋牌盾服务会在到期前的第 **7** 天，通过 **站内信、短信及邮件** 的方式向您推送到期预警消息。
## 续费相关
在账户余额充足的情况下，若您已设置自动续费，则系统会在服务到期当日自动续费。

## 回收机制
- 棋牌盾服务到期后，系统仍会为您提供 5 天的服务，并暂时保留配置。
- 若 5 天内未进行续费操作，则棋牌盾 IP 将自动销毁。
