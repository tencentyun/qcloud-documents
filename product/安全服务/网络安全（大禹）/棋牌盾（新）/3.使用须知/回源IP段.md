## 什么是回源 IP？
回源 IP 指棋牌盾集群将业务流量转发至您源站时所用的 IP。

## 为何要放行回源 IP 段？
当您使用棋牌盾服务后，您所有的业务流量首先会流经棋牌盾集群，集群会将清洗后的业务流量转发至您的源站。为了防止您将棋牌盾回源 IP 拦截或限速导致业务流量受损，您需将棋牌盾回源 IP 段添加至您源站的防火墙或者其他主机安全防护软件的白名单中。

`123.207.76.0/24`

`123.207.212.0/24`
`139.199.114.0/24`
`139.199.120.0/24`
