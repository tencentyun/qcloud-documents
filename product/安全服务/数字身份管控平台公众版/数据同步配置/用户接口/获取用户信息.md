### 接口描述
根据用户 id 获取用户详细信息。

### HTTP 请求方式
```
GET
```

### 请求路径
```
/scim/api/v2/Users/{id}
```



### 请求参数
| 参数名称     | 必选 | 类型   | 参数类型 | 描述         |
| ------------ | ---- | ------ | -------- | ------------ |
| id           | 是   | string | url      | 用户 id      |
| access_token | 是   | string | header   | access_token |

### 响应状态码
| code | reason | description |
| ---- | ------ | ----------- |
| 200  | ok     | 请求成功    |
