- 193.112.70.224
- 193.112.71.109
- 193.112.70.227
- 193.112.71.33
- 193.112.72.165

若您的网站需登录才可以访问，则需要先解除安全策略（即确保所有 IP 都能访问），待您的 cookie 有效性验证通过后再恢复限制。

#### 白名单添加方法

1. 登录 [Web 应用防火墙控制台 ](https://console.cloud.tencent.com/guanjia/ip/list) ，在左侧导航栏中，选择【IP 管理】>【IP 黑白名单】，进入 IP 黑白名单页面。
2. 单击【添加黑白名单】，在弹出的添加黑白 IP 页面中，单击【白名单】，将上方漏洞扫描服务扫描节点 IP 复制到 IP 地址的输入框内，单击【添加】，即完成白名单添加。
