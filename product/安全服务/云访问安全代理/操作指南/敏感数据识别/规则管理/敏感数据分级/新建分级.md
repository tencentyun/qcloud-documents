在敏感数据进行扫描任务时，需要为数据进行分类、分级，其中数据的分级可以依据需求自定义进行设定、实现，分级的设定详情如下步骤。

### 前提条件
- 已获取 [云访问安全代理](https://console.cloud.tencent.com/casb) 控制台的登录账户与密码。
- 已创建 CASB 实例，对于未创建实例，详情可查阅 [创建 CASB 实例](https://cloud.tencent.com/document/product/1303/55961) 。

### 操作步骤

1. 登录 [云访问安全代理](https://console.cloud.tencent.com/casb)  控制台，然后单击左上角的“区域下拉框”，切换区域。
2. 在左侧导航菜单栏中，单击敏感数据识别菜单下的 【识别规则】>【敏感数据分级】，选择该区域下的实例，进入敏感数据分级页面。
![](https://main.qcloudimg.com/raw/319d711b59fc8945c3167fc9daa4c863.png)
3. 在敏感数据分级页面中，单击【新建分级】，弹出新建分级弹窗。
![](https://main.qcloudimg.com/raw/a471b762f863a943fd46eb40b5be4d89.png)
4. 在新建分级弹窗中，输入所需参数，单击 【确定】 完成分级的创建。
![](https://main.qcloudimg.com/raw/e641ce882b19b5d4fcf8b9a9791015e4.png)

**参数列表：**

| 参数名称 | 参数说明                                                     |
| -------- | ------------------------------------------------------------ |
| 分级名称 | 1-60字符；以字母或中文或数字开头；仅允许中英文，数字， “-”， “_”。 |
| 分级来源 | 分级的来源：自定义 和 内置。目前只支持创建**自定义**分级。 |
| 描述     | 可选参数，用于该分级的备注信息，区别于其他分级。             |
| 风险名称 | 分级的名称，1-60字符；以字母或中文或数字开头；仅允许中英文，数字， “-”， “_”。 |
| 风险级别 | 对应的风险分数，其中最高10，最低1。 |

>? 
> - 分级至少添加一项，最多支持10项。
> - 每个分级关联的风险分数不能相等。
