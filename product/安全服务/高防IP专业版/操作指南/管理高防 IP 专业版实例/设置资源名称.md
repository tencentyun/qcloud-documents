当使用多个 DDoS 高防 IP 专业版实例时，可通过设置【资源名称】快速辨识与管理实例。
### 方式一
登录 [DDoS 防护管理控制台](https://console.cloud.tencent.com/dayu/overview)，选择【DDoS 高防 IP 专业版】>【资产列表】，选择【资源列表】视图，单击目标实例的【ID/名称】列的名称，输入名称即可。
>?名称长度为1 - 20个字符，不限制字符类型。

![](https://main.qcloudimg.com/raw/9e1dd71f4b5c346d54411bdca33a6291.png)

### 方式二
登录 [DDoS 防护管理控制台](https://console.cloud.tencent.com/dayu/overview)，选择【DDoS 高防 IP 专业版】>【防护配置】，从实例下拉菜单中选择目标实例，单击【DDoS 高防 IP 专业版信息】区域的【编辑】，输入名称即可。
>?名称长度为1 - 20个字符，不限制字符类型。
