高防 IP 专业版非网站业务防护提供基于 IP 地址的会话保持，支持将来自同一 IP 地址的请求转发到同一台后端服务器处理。

## 操作步骤
参照以下步骤，来配置高防 IP 专业版非网站业务防护的健康检查规则。
1. 登录到 [DDoS 防护（大禹）管理控制台](https://console.cloud.tencent.com/dayu/overview)，前往**高防 IP 专业版**  > **接入配置**页面。
2. 在**非网站业务**页签，选择目的高防 IP 专业版实例并选择相应规则，单击其会话保持查列下的编辑。
3. 选择是否需要开启会话保持功能，单击按钮开启，设置**保持时间**后，单击**确定**即可。

>?• 默认关闭会话保持。
>• 在设置保持时间时，建议使用默认值。

## 参数说明
**四层会话保持**
四层转发场景支持简单会话保持能力，会话保持时间可设为30 - 3600秒中的任意整数值。超过该时间阈值，会话中无新的请求则自动断开连接。
