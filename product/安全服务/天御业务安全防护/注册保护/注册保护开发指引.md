开发者需要通过调用云API的天御相关接口完成接入，接入API步骤如下：

## 1. 获取腾讯云密钥
如果您已经具有了腾讯云密钥则可以跳过这一步。登录 [访问管理控制台](https://console.cloud.tencent.com/capi) ，在左侧导航中，选择**访问密钥** > **API 密钥管理**，在 API 密钥管理页面，单击**新建密钥**。
![](https://main.qcloudimg.com/raw/86179ef78535ca27095f525c99136766.png)

## 2. 开通服务
登录 [天御业务安全防护控制台](https://console.cloud.tencent.com/tianyu/overview) ，在注册保护服务中，单击**试用开通请联系客服**，进行开通。
![](https://main.qcloudimg.com/raw/4aa156ecebd857ab406c7a9e12fe4f32.png)

## 3. 按照指定 API 写代码
[注册保护 API](https://cloud.tencent.com/document/product/295/6592)


## 4. 查询调用数据
进入 [注册保护控制台](https://console.cloud.tencent.com/bsp/service/RegisterProtection)，即可查询到对应服务的调用数据。
![](https://main.qcloudimg.com/raw/80af269fbfed73ea7c17bcdf8aae570a.png)
