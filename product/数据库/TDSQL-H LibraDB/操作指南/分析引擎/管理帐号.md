本文为您介绍如何通过控制台创建帐号、重置密码、删除帐号等。

## 新建帐号
1. 登录 [TDSQL-H LibraDB 控制台](https://console.cloud.tencent.com/libradb/instance)，在实例列表，单击实例 ID 或**操作**列的**详情**，进入实例管理页面。
2. 在实例管理页面，选择**分析引擎** > **帐号管理**页，进入帐号管理列表，单击**新建**。
![](https://qcloudimg.tencent-cloud.cn/raw/45a4e20a0e29dc376b2043f1b1788db6.png)
3. 在弹出的对话框，按照提示填写帐号名、主机、密码和备注，单击**确定**。<br>
<img src="https://qcloudimg.tencent-cloud.cn/raw/8ba31fac94d7da471692957ae7333f5f.png" style="zoom:80%;" />

## 重置密码
1. 登录 [TDSQL-H LibraDB 控制台](https://console.cloud.tencent.com/libradb/instance)，在实例列表，单击实例 ID 或**操作**列的**详情**，进入实例管理页面。
2. 在实例管理页面，选择**分析引擎** > **帐号管理**页，进入帐号管理列表，在**操作**列单击**重置密码**。
3. 在弹出的对话框，填写新密码和备注，单击**确定**。<br>
<img src="https://qcloudimg.tencent-cloud.cn/raw/5cf6fd9773303fbc1cc46356bda282bc.png" style="zoom:80%;" />

## 其他更多操作
1. 登录 [TDSQL-H LibraDB 控制台](https://console.cloud.tencent.com/libradb/instance)，在实例列表，单击实例 ID 或**操作**列的**详情**，进入实例管理页面。
2. 在实例管理页面，选择**分析引擎** > **帐号管理**页，进入帐号管理列表，在**操作**列选择**更多**，可修改主机、修改备注和删除帐号。
![image-20210708102652432](https://qcloudimg.tencent-cloud.cn/raw/fa05accc337dd7322d46689c0b3f6d5f.png)

## 命名和密码规则

| 对象     | 限制                                                         |
| -------- | ------------------------------------------------------------ |
| 实例名称 | 实例名称仅支持数字、英文大小写字母、中文以及特殊字符`_-./()[]`，且长度不能超过60 |
| 用户名   | 用户名支持1个 - 16个字符，由字母、数字和特殊字符\_组成；以字母开头，字母或数字结尾 |
| 密码     | 密码应至少包含字母、数字和字符 `_+-&=!@#$%^*()` 中的两种，长度为8个 - 64个字符 |
