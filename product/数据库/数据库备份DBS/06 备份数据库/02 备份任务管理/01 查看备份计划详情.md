## 操作场景
用户配置完备份计划后，可以查看备份计划详情，包括备份计划的基本信息、全量备份集、增量备份集等信息。同时，也可以在详情页对数据源、备份对象、备份策略等配置进行修改。

## 查看基本信息
1. 登录 [DBS 控制台](https://console.cloud.tencent.com/dbs)，在左侧导航单击**备份计划**，进入任务列表页。
2. 选择指定的备份计划，单击任务 ID 或者在**操作**列单击**查看**，进入基本信息页面。
![](https://qcloudimg.tencent-cloud.cn/raw/016a4029502c589fe6778c2d9a014717.png)
3. 用户可以查看备份计划的基本信息、源库信息、备份策略信息等，也可以对这些配置进行修改。
![](https://qcloudimg.tencent-cloud.cn/raw/67dba2a4df75a0feca0c2e3e2e23e2fa.png)

## 查看全量备份集
1. 登录 [DBS 控制台](https://console.cloud.tencent.com/dbs)，在左侧导航单击**备份计划**，进入任务列表页。
2. 选择指定的备份计划，单击任务 ID 或者在**操作**列单击**查看**，进入基本信息页面。
![](https://qcloudimg.tencent-cloud.cn/raw/016a4029502c589fe6778c2d9a014717.png)
3. 切换页签，单击**全量备份集**。
![](https://qcloudimg.tencent-cloud.cn/raw/880755254add93b99612bd9baee21ff5.png)
4. 在**操作**列单击**详情**，可以查看对应的全量备份集信息。

## 查看增量备份集
1. 登录 [DBS 控制台](https://console.cloud.tencent.com/dbs)，在左侧导航单击**备份计划**，进入任务列表页。
2. 选择指定的备份计划，单击任务 ID 或者在**操作**列单击**查看**，进入基本信息页面。
![](https://qcloudimg.tencent-cloud.cn/raw/016a4029502c589fe6778c2d9a014717.png)
3. 切换页签，单击**增量备份集**。
![](https://qcloudimg.tencent-cloud.cn/raw/d2e0a2dfb68a62e5ccef602c158a4cad.png)
4. 在**操作**列单击**查看增量备份文件列表**，可以查看对应的增量备份集信息。
