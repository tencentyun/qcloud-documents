数据库智能管家 DBbrain 目前处于公测阶段，大部分功能您可以免费试用。其中 [合规审计](https://cloud.tencent.com/document/product/1130/49728) 为收费功能，采用按量计费的计费模式，合规审计包含审计日志存储和安全服务两部分费用。开通合规审计后免费体验 [安全治理](https://cloud.tencent.com/document/product/1130/49731) 功能。

>?数据库智能管家 DBbrain 目前支持的功能，请参见 [功能列表](https://cloud.tencent.com/document/product/1130/49291)。

## 合规审计计费公式
合规审计总费用 = 审计日志存储费用 + 安全服务费用

>!
>- 计费规格与实例的日志和数据存储量无关，与用户执行 SQL 记录量有关。
>- 用户执行 SQL 的记录量与用户添加的实例及配置的规则有关。
>- 合规审计功能以审计组为维度进行开通计费。
>- 云数据库 MySQL 开通合规审计服务后，当释放该 MySQL 实例时，实例所对应的合规审计服务不会停止，且实例在被释放前产生的日志将会继续存储且收费，直至该实例日志保存时长到期。
