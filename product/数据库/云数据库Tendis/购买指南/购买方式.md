腾讯云为您提供了两种方式购买云数据 Tendis：控制台购买和 API 购买。

## 控制台购买
1. 登录 [Tendis 购买页](https://buy.cloud.tencent.com/tendis)。
2. 计费模式选择【包年包月】或【按量计费】。
3. 根据实际情况和需求选择地域、可用区、引擎、兼容协议、版本、内存规格等，并根据需要选择网络类型和网络计费模式。
>?按量计费主机创建时会冻结一个小时的主机费用，请确保您的账户余额充足再进行购买。
4. 确认订单之后，进行支付。
5. 订单支付后立即开通服务器，等待1分 - 5分左右即可看到 IP 地址，完成密码初始化后，即可访问该实例。
