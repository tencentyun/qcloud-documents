
星点层通过大规模的点要素表达目标要素的聚集程度。

## 参数配置
| 参数           | 说明                                                         | 默认值       |
| -------------- | ------------------------------------------------------------ | ------------ |
| **星点颜色**   | 单个星点的颜色                                               | #03EDFF      |
| **星点尺寸**   | 单个星点的尺寸                                               | 1            |
| **星点图高度** | 星点的整体高度                                               | 1            |
| **开启泛光**   | 是否开启星点的泛光效果                                       | 开启         |
| **显示数据**   | 显示效果层内置的完整全国星点数据，还是显示数据标签中部分星点数据（示例数据） | 完整星点数据 |

## 数据配置
| 关键字段  | 说明             |
| --------- | ---------------- |
| **lng**   | 单个星点的经度   |
| **lat**   | 单个星点的纬度   |
| **value** | 单个星点代表的值 |
