
您可以通过集群名对 TDSQL-C MySQL 版集群进行区分和管理，默认集群名和集群 ID 保持一致，也可以通过控制台对集群名字进行修改。

## 操作步骤
在集群列表页面，根据实际使用的视图模式进行操作：
<dx-tabs>
::: 页签视图
1. 登录 [TDSQL-C MySQL 版控制台](https://console.cloud.tencent.com/cynosdb)，在左侧集群列表，单击目标集群，进入集群管理页。
2. 在集群管理页，单击集群名后的![](https://main.qcloudimg.com/raw/cf9bbcfaea10f0316bddd967cb6e8ffc.png)。
![](https://qcloudimg.tencent-cloud.cn/raw/8119f19170c9522ca99ef9750e7972b4.png)
3. 在弹出的对话框，填写需要修改的集群名称，单击**确定**。
>? 集群名仅支持长度小于60的中文，英文，数字，-(中横线)，\_(下划线)，.(英文标点)。
:::
::: 列表视图
**在集群列表中修改集群名**
1. 登录 [TDSQL-C MySQL 版控制台](https://console.cloud.tencent.com/cynosdb)，在集群列表中，找到需要的集群，单击集群名后的![](https://main.qcloudimg.com/raw/cf9bbcfaea10f0316bddd967cb6e8ffc.png)。
![](https://main.qcloudimg.com/raw/a5085cef4f503e404fec5f7cde107da7.png)
2. 在弹出的对话框，填写需要修改的集群名称，单击**确定**。
>? 集群名仅支持长度小于60的中文，英文，数字，-(中横线)，\_(下划线)，.(英文标点)。

**在集群详情中修改集群名**
1. 登录 [TDSQL-C MySQL 版控制台](https://console.cloud.tencent.com/cynosdb)，在集群列表中，找到需要的集群，单击集群 ID，进入集群详情页面。
2. 在集群详情页面，单击集群名后的![](https://main.qcloudimg.com/raw/cf9bbcfaea10f0316bddd967cb6e8ffc.png)。
![](https://qcloudimg.tencent-cloud.cn/raw/46aa21187704ad6cad985a9953802823.png)
3. 在弹出的对话框，填写需要修改的集群名称，单击**确定**。
>? 集群名仅支持长度小于60的中文，英文，数字，-(中横线)，\_(下划线)，.(英文标点)。
:::
