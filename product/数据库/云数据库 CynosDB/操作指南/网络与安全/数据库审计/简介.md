数据库审计是腾讯云自主研发的一款专业、高效、全面、实时监控数据库安全的审计产品，数据库审计能够实时记录腾讯云数据库活动，对数据库操作进行细粒度审计的合规性管理，对数据库遭受到的风险行为进行告警，TDSQL-C MySQL 版提供数据库审计能力，记录对数据库的访问及 SQL 语句执行情况，帮助企业进行风险控制，提高数据安全等级。

## 应用场景
TDSQL-C MySQL 版数据库审计服务可以帮助您解决以下问题：
- 助力企业顺利通过等保合规审计，提供等保三级及其他行业合规审计依据。
- 具备风险、运行、性能等状况、语句分布的实时监控能力。
- 帮助您记录、分析、追查技术人员或第三方开发维护人员误操作、违规操作、越权操作等数据库安全相关事件。
- 通过数据库性能诊断，追踪危险事件和不安全操作。
- 提供数据库实时风险告警能力，及时响应数据库攻击。

按照审计日志存储量进行按量计费。每小时为一个计费周期，不足一小时的按一小时计费。
具体产品定价请参见 [数据库审计计费说明](https://cloud.tencent.com/document/product/1003/73008)。

TDSQL-C MySQL 版数据库审计目前支持的兼容版本为 MySQL 5.7 、8.0。
