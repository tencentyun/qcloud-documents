## Region and Availability Zone
**Public Cloud:**
Currently, Tencent Cloud offers a number of available regions. DCDB can be deployed in public cloud data center in China.


**Finance Cloud:**
Compliance zone customized according to regulatory requirements for finance industry, which features high security and high isolation. Verified customers in finance industry can apply for using the zone by submitting a ticket. For more information, please see [Introduction to Finance Zone](http://cloud.tencent.com/doc/product/304/%E9%87%91%E8%9E%8D%E4%BA%91%E7%AE%80%E4%BB%8B)
