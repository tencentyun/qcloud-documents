Cloud database MongoDB supports two backup methods: Daily automatic backup and manual backup

1. Instance Auto Backup
Instance can be automatically backed up once a day
![](https://mc.qcloudimg.com/static/img/4177b1b21ded9ec323c127965da9149e/zidongbeif.png)

2. Instance Manual Backup
Click "Manual Backup" in the instance details page and enter the password to complete manual backup operation
![](https://mc.qcloudimg.com/static/img/f3884e673178cd4e7d74055a82d6695a/shoudongbeif.png)
