TencentDB for MongoDB uses two backup methods: daily automatic backup and manual backup.<br>

- Automatic backup of instances
Instances are backed up automatically every day, including 1 full backup and 6 incremental backups every 7 days. The backup list is shown as below.<br>
![](https://mc.qcloudimg.com/static/img/4177b1b21ded9ec323c127965da9149e/zidongbeif.png)

- Manual instance backup
On the instance details page, click **Manual Backup** and enter remarks, and then submit the manual backup task to the background management system.
![](https://main.qcloudimg.com/raw/b67a177016b10d3d27597d914e35f51d.png)
