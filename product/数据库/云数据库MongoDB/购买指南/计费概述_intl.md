## Billing Method 
TencentDB for MongoDB supports postpaid billing method. Postpaid is applicable to scenarios where the business volume fluctuates dramatically at a certain time. You are charged for your actual usage, without the need to make any advance payments.
## Computing 
The computing resources include CPU and memory resources, which determine the parameters such as the maximum number of QPS and access connections of an instance. For available specifications supported by TencentDB for MongoDB, see [Purchase and Initialize Instance](https://cloud.tencent.com/) in Getting Started. For more information on prices, see [TencentDB for MongoDB Pricing](https://intl.cloud.tencent.com/product/mongodb).
## Storage 
Tencent Cloud provides flexible, economic and user-friendly data storage devices for MongoDB instances. For more information on prices, see [TencentDB for MongoDB Pricing](https://intl.cloud.tencent.com/product/mongodb).
