Just like the native MongoDB, TencentDB for MongoDB can be connected using mongo shell or drivers in various languages for database access. See below for details.
## mongo shell 
"mongo shell" is an interactive JavaScript shell in the MongoDB. It can interact with your MongoDB instance using the command line in the shell. You can also use "mongo shell" to query and update data or perform management operations. For the specific procedure, see **Best Practice** -> [**Connection Example**]().
## Connection Using Drivers 
TencentDB for MongoDB can be connected through a variety of languages, such as PHP, Node.js, Java, and Python. For more information, see [Connection Example](https://cloud.tencent.com/document/product/240/3563).
