## Purchase

1. Log in to Tencent Cloud's [Management Console][1], move the mouse to the "Cloud Products" -> "Basic Products" -> "Database" in the navigation bar, click the "Relational Database" to enter [Cloud Database Console][2], and then click "New" button to enter the MySQL purchase interface of cloud database.
![][image-1]
![][image-2]
2. Select a configuration attribute:       
	* Select a required billing mode and two modes are supported, that is, the monthly and annual prepaid mode and the postpaid mode. (Before making any purchase in the postpaid mode, it is required to make a identity verification. For more information about the identity verification, please see the <a href="https://cloud.tencent.com/document/product/378/3629" target="_blank">Identity Verification Guide</a>.)
	* Select a region and an availability zone. For the description about regions, please see <a href="https://cloud.tencent.com/document/product/236/8458" target="_blank">Regions and Availability Zones</a>.
	* Select a network environment. Both the basic and private networks are supported. For the difference between the basic and private networks, please see <a href="https://cloud.tencent.com/document/product/213/5227" target="_blank">Network Environments</a>.
	* Select an instance specification and a required disk.
	* Select a data replication method. For differences between different database instance replication methods, please see <a href="https://cloud.tencent.com/document/product/236/7913" target="_blank">Database Instance Replication</a>.
	* Select a project to which the database instance belongs, and the default is the default project.
	* Select a quantity and a purchased usage period and confirm them. Then click "Buy Now".
	* For more details about price, please <a href="https://cloud.tencent.com/document/product/236/5158" target="_blank">Price Overview</a>.
![][image-3]

## Renewal
You can renew CDB for MySQL via the console and the renewal management center. The renewal function is also available on the Renewal Management page and the console page.

### Renew from Console
* Batch Renewal  
Select one or more instances that require renewal and click "Renewal".
 ![Batch Renewal][image-4]
* Auto Renewal  
Select one or more instances that require renewal, and click "More Operations" > "Set Auto Renewal".
![Auto Renewal][image-5]
* Renewal on the Page of Instance Details
Click the instance name to go to the page of instance details and click "Renewal".
![Renewal on the Page of Example Details][image-6]

### Renew in the Renewal Management Center
 On the <a href="https://console.cloud.tencent.com/account/renewal" target="_blank">page of Renewal Management</a>, a set of functions related to instances are available, including "Batch Resource Renewal", "Set Auto Renewal", "Set Unified Expiration Date" and "Cancel Non-Renewal". For more information, please see <a href="https://cloud.tencent.com/document/product/555/7454" target="_blank">"Renewal Management"</a> in details.
