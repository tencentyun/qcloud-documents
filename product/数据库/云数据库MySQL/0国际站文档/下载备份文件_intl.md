You can download the cold backup data and Binlog of Tencent Cloud CDB via the Cloud Database console.

## 1. Downloading Backup File and Binlog

Step1. On the Instance Management page, click "Backup Management", and select the backup list or Binlog list.

![ ](//mc.qcloudimg.com/static/img/163c4eb48d551b3f5a0418d2aa2179ce/image.png)

Step2. Select the backup to download, and click "Download" in the Operation column.

![](//mc.qcloudimg.com/static/img/191178f3d03957f535c3672b1f110ad4/image.png)

Step3. You can download via a private network address or locally; select the desired export method

![](//mc.qcloudimg.com/static/img/62b8338bdc9da52002ebc37ef03b0e7b/image.png)

## 2. Downloading the Slow Query Log

Step1. On the Instance Management page, click "Operation Log" and select "Slow Query Log".

![](//mc.qcloudimg.com/static/img/ed69733c3e14634118ed4ca6a00b11ec/image.png)

Step2. Select the backup to download, click "Download" in the Operation column.

><span style = "color:#F00"> Note: If the backup size is 0KB and no slow query record is available, the download will not go through. </span>

![](//mc.qcloudimg.com/static/img/e30f73d316c7b732609ef8ab553b5fb2/image.png)

Step3. You can download via a private network address or locally; select the desired export method

![](//mc.qcloudimg.com/static/img/2eac7e4f8373ca7c304ececedaae7b83/image.png)
