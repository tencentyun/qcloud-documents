
| API        | 描述   | 
| --------   | -----  | 
|CreateDBInstance	|创建云数据库实例（包年包月）|
|CreateDBInstanceHour	|创建云数据库实例（按量计费）|
|DescribeDBInstances	|查询实例列表|
|DescribeDBPrice	|查询数据库价格|
|DescribeDBZoneConfig	|获取云数据库可售卖规格|
|InitDBInstances	|初始化新实例|
