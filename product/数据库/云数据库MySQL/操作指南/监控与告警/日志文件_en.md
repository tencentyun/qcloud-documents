The slow query indicates that a SQL query statement takes more time than the specified, and the corresponding statement is called the "slow query statement". The procedure during which a database administrator (DBA) analyzes the slow query statement and finds the occurrence reasons for the slow query is called the "slow query analysis".

You can use [Cloud Database Console][1] to download slow query logs under the [Management] > [Operate Logs] module. For detailed information, please refer to <a href="https://cloud.tencent.com/document/product/236/7274" target="_blank">Downloading Backup Files</a>. Meantime, you can also view, download, and read database logs on the command line interface (CLI) or with the Cloud Database APIs. For detailed information, please refer to <a href="https://cloud.tencent.com/document/product/236/4691" target="_blank">Query Backups and Logs</a>.
