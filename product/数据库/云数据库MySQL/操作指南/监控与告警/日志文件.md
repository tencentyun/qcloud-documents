超过指定时间的 SQL 语句查询称为 “慢查询”，对应语句称为 “慢查询语句”，数据库管理员（DBA）对慢查询语句进行分析并找到慢查询出现原因的过程称为 “慢查询分析”。

您可以使用 [云数据库控制台](https://console.cloud.tencent.com/cdb)，在【管理】>【操作日志】模块下，下载慢查询日志。详情请参考 <a href="https://cloud.tencent.com/document/product/236/7274" target="_blank">下载备份文件</a>。同时，您也可以在命令行界面（CLI）或者使用云数据库 API 查看、下载和观看数据库日志。详情请参考 [查看备份](https://cloud.tencent.com/document/product/236/15842) 与 [查看日志](https://cloud.tencent.com/document/product/236/15843)。
