## 操作场景
腾讯云网络分为 [基础网络和私有网络 VPC](https://cloud.tencent.com/document/product/215/30720)，给用户带来不同的优质服务。在此基础之上，我们提供如下更灵活的服务，方便您管理网络。
- **网络间切换**
  - 基础网络切换为私有网络：支持单台云数据库的基础网络切换至私有网络。
  - 私有网络 A 切换为私有网络 B：支持单台云数据库的私有网络 A 切换至私有网络 B。
- **设置自定义 IP**

## 注意事项
- 更换网络会导致该实例 IP 变化，24小时后，旧的访问 IP 会失效，请及时修改客户端程序。
- 为防止 IP 资源滥用，24小时内只能更换网络一次。
- 只能选择实例所在地域和可用区内的 VPC 网络与子网。
- 基础网络切换私有网络后不可逆，云数据库切换至私有网络后与其他私有网络及基础网络的云服务不互通。
- 数据复制方式为强同步复制的数据库，暂不支持基础网络切换为私有网络或私有网络 A 切换为私有网络 B。

## 操作步骤
1. 登录 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在实例列表中，选择需要切换网络的实例，单击实例名或操作列的【管理】，进入实例详情页面。
3. 在实例基本信息的【所属网络】后，根据网络间切换类型，单击【转VPC网络】或【更换网络】。
3. 在弹出的对话框中，选择私有网络以及相应子网，单击【确定】。
   - **基础网络转私有网络**

   - **私有网络转私有网络**
![](https://main.qcloudimg.com/raw/0a6f65fd756bbbc6af4460b064be67a1.png)
>?若未指定 IP 地址，系统将自动分配。
4. 返回实例详情页，可查看到实例的所属网络。
