您可以按照下列步骤对资源进行标签的编辑操作。

### 对单个实例编辑标签
1. 登录云数据库 MySQL 控制台，打开 [云数据库 MySQL控制台](https://console.cloud.tencent.com/cdb) 页面。
2. 在需要编辑标签的实例的右侧操作栏中，单击【更多】，单击【编辑标签】。
![](https://main.qcloudimg.com/raw/3415620d1d12d9204193cb93c8f3e831.png)
3. 在【编辑标签】的操作框中，添加、修改、删除标签。
![](https://main.qcloudimg.com/raw/e3b73bbb3e3152bc7c4074d657a8c685.png)
4. 单击确认。

### 对多个实例编辑标签

1. 登录云数据库 MySQL 控制台，打开 [云数据库 MySQL](https://console.cloud.tencent.com/cdb) 页面。
2. 左侧勾选需要编辑标签的实例。
3. 单击列表顶部【更多操作】，单击【编辑标签】。
![](https://main.qcloudimg.com/raw/c59992f0628719b7c5e46da9c97b91fc.png)
4. 在【编辑标签】的操作框中，添加、修改、删除标签。
5. 单击确认。
