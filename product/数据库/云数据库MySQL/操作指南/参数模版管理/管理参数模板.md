
您可以使用数据库参数模板管理数据库引擎的参数配置。数据库参数组就像是引擎配置值的容器，这些值可应用于一个或多个数据库实例。
参数模板支持如下功能，用户可登录 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)，在左侧边栏选择【参数模板】页查看和修改参数：
- 支持默认参数模板。
- 支持新建模板，以默认参数为基础，稍加修改就能生成自定义的参数优化方案。
- 支持从 MySQL 配置文件 my.conf 导入生成模板。
- 支持从参数设置保存为模板。
- 单个或多个实例进行参数设置时，支持从模板导入。
>!已经应用该参数模板的所有数据库实例并不会获得该数据库参数模板的所有参数更新。如果您需要将新的参数应用到批量数据库实例上，可通过批量参数设置时导入模板来重新应用。

## 新建参数模板
当您想使用自己的数据库参数模板时，只需创建一个新的数据库参数模板，修改所需的参数并应用于您的数据库，就可以使用新的数据库参数模板。
1. 登录 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧导航选择【参数模板】页签。
![](https://main.qcloudimg.com/raw/20bbc344f0a53a8c44160eb3fb7dbb00.png)
3. 单击【创建模板】，在弹出的对话框中配置如下参数：
 - **模板名称**：输入参数模板的名称，在参数模板名称需具有唯一性。
 - **数据库版本**：选择需要的数据库版本。
 - **模板描述**：输入参数模板的简要说明。
![](https://main.qcloudimg.com/raw/ae4c84cbecab9408f70ed38f22d38f67.png)
4. 确认无误后，单击【创建并设置参数】即可创建参数模板。
5. 创建完成后，您可以在模板详情页修改、导入、导出参数等。

## 应用参数模板于数据库
1. 在左侧导航选择【[实例列表](https://console.cloud.tencent.com/cdb)】页签，单击实例名称进入管理页。
2. 选择【数据库管理】页签，选择【参数设置】>【从模板导入】。
3. 在弹出的对话框，选择参数模板，单击【导入并覆盖原有参数】即可。

## 复制参数模板
当您已创建一个数据库参数模板，并且想在新的数据库参数模板中包含该组中的大部分自定义参数和值时，复制参数模板是一个方便的解决方案。

### 方式一：通过已有参数模板复制参数模板
1. 在左侧导航选择【[参数模板](https://console.cloud.tencent.com/mysql/param-templates)】页签，在需要复制参数模板的操作列，单击【查看详情】。
2. 在查看详情页，单击上方的【另存为模板】。
3. 在弹出的对话框中，配置如下参数：
  - **模板名称**：输入参数模板的名称，在参数模板名称需具有唯一性。
  - **模板描述**：输入参数模板的简要说明。
4. 确认无误后，单击【保存】即可将当前参数模板另存为新的参数模板，完成复制操作。

### 方式二：通过实例的参数设置功能复制参数模板
1. 在左侧导航选择【[实例列表](https://console.cloud.tencent.com/cdb)】页签，单击实例名称进入管理页。
2. 选择【数据库管理】页签，选择【参数设置】>【另存为模板】。
3. 在弹出的对话框中，配置如下参数：
  - **模板名称**：输入参数模板的名称，在参数模板名称需具有唯一性。
  - **模板描述**：输入参数模板的简要说明。
4. 确认无误后，单击【新建并保存】即可将当前参数模板另存为新的参数模板，完成复制操作。

## 删除参数模板
当您创建了多余的参数模板或者确定该参数模板已无使用价值，可通过删除参数模板功能删除不再使用的参数模板。
1. 在左侧导航选择【[参数模板](https://console.cloud.tencent.com/mysql/param-templates)】页签。
2. 在参数模板列表中选择所需删除的参数模板，在操作列中，单击【删除】。
3. 在弹出的对话框中，单击【确定】即可完成删除参数模板操作。

## 修改参数
购买云数据库 MySQL 实例后，您可以使用 MySQL 实例提供的默认参数值，也可以根据您的业务场景局部调整其参数值，也可以选择通过修改参数模板的参数配置，再应用于具有相同业务场景的 MySQL 实例。

### 方式一：使用参数配置功能修改参数值
1. 在左侧导航选择【[实例列表](https://console.cloud.tencent.com/cdb)】页签，单击实例名称进入管理页。
2. 单击【数据库管理】页签中的【参数设置】。
3. 在参数配置页中，单击【批量修改参数】即可批量修改参数，修改后，单击【确认修改】即可完成批量参数修改。
   或者在参数配置页中，单击**参数当前值**旁边的编辑图标可修改单个参数值，修改后，单击数值旁边的保存图标即可完成修改。

### 方式二：修改新建或已有参数模板的参数值
1. 在左侧导航选择【[参数模板](https://console.cloud.tencent.com/mysql/param-templates)】页签。
2. 在参数模板列表中选择所需修改的参数模板，在操作列中，单击【查看详情】。
3. 在查看详情页中，单击【批量修改参数】即可批量修改参数，修改后，单击【确认修改】即可完成批量参数修改。
   或者在查看详情页中，单击**参数当前值**旁边的编辑图标可修改单个参数值，修改后，单击数值旁边的保存图标即可完成修改。

## 导入参数
您可以使用或修改 MySQL 实例所提供默认参数值，也可以导入业务系统已有的参数配置。需要注意的是不在参数设置列表或者参数模板列表中的参数，不会被成功导入。

### 方式一：通过参数模板导入参数
1. 在左侧导航选择【[参数模板](https://console.cloud.tencent.com/mysql/param-templates)】页签。
2. 在参数模板列表中选择所需修改的参数模板，在操作列中，单击【查看详情】。
3. 在查看详情页中，单击【导入参数】。
4. 在弹出的对话框中，单击【选择文件】。
5. 在从本地文件导入页中，选择参数配置文件时，需要注意的是配置文件的格式与MySQL数据库服务器的配置文件格式一致或者使用导出参数的文件模板，否则提示导入不成功。
6. 单击【导入并覆盖原有参数】即可完成参数导入。

### 方式二：通过实例的参数设置功能导入参数
1. 在左侧导航选择【[实例列表](https://console.cloud.tencent.com/cdb)】页签，单击实例名称进入管理页。
2. 单击【数据库管理】页签中的【参数设置】。
3. 在参数配置页中，单击【导入参数】。
4. 在弹出的对话框中，单击【选择文件】。
5. 在从本地文件导入页中，选择参数配置文件时，需要注意的是配置文件的格式与 MySQL 数据库服务器的配置文件格式一致或者使用导出参数的文件模板，否则提示导入不成功。
6. 单击【导入并覆盖原有参数】即可完成参数导入。
