您可以使用数据库参数模板管理数据库引擎的参数配置。数据库参数组就像是引擎配置值的容器，这些值可应用于一个或多个数据库实例。

## 	新建参数模板
当您想使用您自己的数据库参数模板，只需创建一个新的数据库参数模板，修改所需的参数并应用于您的数据库，就可以使用新的数据库参数模板。具体操作如下：
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【参数模板】。
3. 在**创建模板**窗口中，执行以下操作：
 - 对于**模板名称**，输入参数模板的名称，在参数模板名称需具有唯一性。
 - 对于**模板描述**，输入参数模板的简要说明。
 - 对于**DB 版本**，选择需要的数据库版本。

4. 信息填完后，单击【创建并设置参数】即可创建参数模板。

## 	复制参数模板
当您已创建一个数据库参数模板，并且想在新的数据库参数模板中包含该组中的大部分自定义参数和值时，复制参数模板是一个方便的解决方案。具体操作如下：
### 通过已有的参数模板复制参数模板
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【参数模板】。
3. 在参数模板列表中单击所复制参数模板的【查看修改】。
4. 在**查看修改**窗口中，单击【另存为模板】。
5. 在**另存为参数模板**窗口中，执行以下操作：
 - 对于**模板名称**，输入参数模板的名称，在参数模板名称需具有唯一性。
 - 对于**模板描述**，输入参数模板的简要说明。

6. 信息填完后，单击【新建并保存】即可将当前参数模板另存为新的参数模板，完成复制操作。

### 通过某个实例的参数配置功能复制参数模板
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【实例列表】。
3. 选择所需复制实例的【管理】或者单击需要复制的实例。
4. 在**实例管理**窗口中，选择**数据库管理**页面下的【参数设置】。
5. 在**参数配置**页面中，选择【另存为模板】：
6. 在**另存为参数模板**窗口中，执行以下操作：
 - 对于**模板名称**，输入参数模板的名称，在参数模板名称需具有唯一性。
 - 对于**模板描述**，输入参数模板的简要说明。

7. 信息填完后，单击【新建并保存】即可将当前参数模板另存为新的参数模板，完成复制操作。


## 	删除参数模板
当您创建了多余的参数模板或者确定该参数模板已无使用价值，可通过删除参数模板功能删除不再使用的参数模板。具体操作如下：
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【参数模板】。
3. 在参数模板列表中选择所需删除参数模板的【删除】。
4. 在**删除模板**的弹框中单击【删除】即可完成删除参数模板操作。

## 	修改参数
当您购买云数据库 MySQL 实例后，您可以使用 MySQL 实例提供的默认参数值，也可以根据您的业务场景局部调整其参数值，也可以选择通过修改参数模板的参数配置，再应用于具有相同业务场景的 MySQL 实例。具体操作如下：
### 使用参数配置功能修改参数值
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【实例列表】。
3. 选择所需修改实例的【管理】功能或者单击需要修改参数的实例。
4. 在**实例管理**窗口中，单击**数据库管理**页面下的【参数配置】。
5. 在**参数配置**页面中，单击【修改参数】时可批量修改参数，修改后，单击【确认修改】即可完成批量参数修改。
 或者在**参数配置**页面中，单击**参数当前值**旁边的【编辑】图标可修改单个参数值，修改后，单击数值旁边的【保存】图标即可完成修改。


### 修改新建或已有参数模板的参数值
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【参数模板】。
3. 在参数模板列表中选择所需复制参数模板的【查看修改】。
4. 在**查看修改**窗口中，单击【修改参数】时可批量修改参数，修改后，单击【确认修改】即可完成批量参数修改。
 或者在**查看修改**窗口中，单击**参数当前值**旁边的【编辑】图标可修改单个参数值，修改后，单击数值旁边的【保存】图标即可完成修改。

## 	导入参数
当您购买云数据库 MySQL 实例，可以使用或修改 MySQL 实例所提供默认参数值，也可以导入业务系统已有的参数配置。需要注意的是不在参数设置列表或者参数模板列表中的参数，不会被成功导入。具体操作如下：
###  在参数配置中导入参数
1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【实例列表】。
3. 选择所需复制实例的【管理】或者单击需要复制的实例。
4. 在**实例管理**窗口中，单击**数据库管理**页面下的【参数配置】。
5. 在**参数配置**页面中，单击【导入参数】。
6. 在**导入参数**窗口中，单击【选择文件】。
7. 在**从本地文件导入**窗口中，选择参数配置文件时，需要注意的是配置文件的格式与 MySQL 数据库服务器的配置文件格式一致或者使用导出参数的文件模板，否则提示导入不成功。
8. 单击【导入并覆盖原有参数】即可完成参数导入。

### 在参数模板中导入参数

1. 登录腾讯云控制台并打开 [云数据库 MySQL 控制台](https://console.cloud.tencent.com/cdb)。
2. 在左侧的导航中，选择【参数模板】。
3. 在参数模板列表中选择所需修改参数模板的【查看修改】。
4. 在**查看修改**窗口中，单击【导入参数】。
5. 在**导入参数**窗口中，单击【选择文件】。
6. 在**从本地文件导入**窗口中，选择参数配置文件时，需要注意的是配置文件的格式与MySQL数据库服务器的配置文件格式一致或者使用导出参数的文件模板，否则提示导入不成功。
7. 单击【导入并覆盖原有参数】即可完成参数导入。
