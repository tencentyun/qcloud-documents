
## 简介
查看表的定义信息。嵌套字段只能看到其属性为嵌套类型，但是无法查看嵌套结构体的定义。

## 语法
```
desc [table]/[table.field] [using tdr];
```

## 参数

|  参数          | Protobuf                                      | TDR                             | 
| ----------- | ---------- | ------------------------------------------------------------ |
| table       | 表格的名字 |  表格的名字                                                            |
| table.field | 不支持     | 显示嵌套字段的结构，必须配合 using tdr 使用                  |
| using tdr   | 不支持     | 从 tdr 文件中读取表结构，该操作必须在启动 client 时提供 tdr 文件 |

## 错误
请参考 [错误码列表](https://cloud.tencent.com/document/product/596/49767)。
