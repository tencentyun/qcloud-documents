
## 简介
查询表格的记录总数。

## 语法
```
count [tableName];
```

## 错误
请参考 [错误码列表](https://cloud.tencent.com/document/product/596/49767)。

## 示例
获取表格的记录数。
```
tcaplus> count game_player;
--------------------------------------------------------------------------------
| TableName                                 Count                              |
--------------------------------------------------------------------------------
| table_list                                16                                 |
--------------------------------------------------------------------------------
```
