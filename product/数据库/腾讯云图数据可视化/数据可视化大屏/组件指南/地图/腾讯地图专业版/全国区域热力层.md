全国区域热力层基于中国行政区矢量数据及省级区域数据，渲染省级区域和边界样式，展示省级区域的热力数据信息。
## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **描边颜色** | 行政区边界颜色 |#014080 |
| **填充开始** | 区域热力数据中的最小值对应颜色 | #7DF4FF |
| **填充结束** | 区域热力数据中的最大值对应颜色 |#017CF7 |
| **默认颜色** | 无热力值行政区颜色 |#C9C9C9 |
| **显示默认** | 开启后，显示无热力值行政区颜色 |开启 |
