散点层通过圆点图标符号，展示地理位置相关信息。	

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **填充颜色** | 散点符号的颜色 |RGBA(56,124,234,0.75) |
| **边线颜色** | 散点符号边缘线的颜色 | #000 |
| **边线宽度** | 散点符号边缘线的宽度|0 |
| **圆形半径** | 散点符号的半径，用于控制尺寸 |4 |
| **散点朝向** | 确定视觉上的散点方向 | 贴向地图 |
| **开启泛光** | 开启后，散点有泛光效果 | 开启 |
| **状态** | 开启后，根据数据中 status 属性判断气泡的状态，赋予不同状态的气泡不同颜色属性 |开启 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **status** | 散点状态值 |
| **lng** | 散点位置纬度 |
| **lat** | 散点位置经度 |
