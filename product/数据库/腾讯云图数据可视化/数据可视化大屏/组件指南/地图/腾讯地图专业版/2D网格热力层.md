2D网格热力层通过2D正方形网格的形式表示热力值数据的大小、分布和聚集程度。

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **网格半径** | 热力网格半径 |6 |
| **色带颜色** | 按数据中最高热力值和最低热力值间按照比例分配的色带颜色 |#A9DEBA / #54B4CD / #084485 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **lat** | 热力网格中心纬度 |
| **lng** | 热力网格中心经度 |
| **cnt** | 热力值 |
