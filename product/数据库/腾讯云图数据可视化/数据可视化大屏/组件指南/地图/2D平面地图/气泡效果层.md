气泡效果层以具有呼吸渐变特征的气泡符号，展示地理位置相关信息。

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **开始颜色** | 气泡边界初始颜色，不开启渐变时为边界颜色 | #98EEFF |
| **结束颜色** | 气泡边界结束颜色 | WHITE |
| **支持渐变** | 气泡边界颜色渐变 | 不开启 |
| **描边宽度** | 气泡边界宽度 | 1 |
| **半径最小** | 气泡呼吸的最小半径 |10 |
| **半径最大** | 气泡呼吸的最大半径 | 18 |
| **填充颜色** | 气泡的填充颜色 | RGBA(152,238,255,0.3) |
| **选中颜色** | 交互点击选中后气泡颜色 | RGBA(89,129,255,0.3) |
| **状态** | 开启后，根据数据中 status 属性判断气泡的状态，赋予不同状态的气泡不同颜色属性 | 不开启 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **x** | 行政区名称 |
| **y** | 气泡数值 |
| **status** | 气泡状态值 |
| **lat** | 气泡位置纬度 |
| **lng** | 气泡位置经度 |
| **tip** | 气泡交互弹出的提示信息 |
