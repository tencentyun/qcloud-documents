迁移效果层基于线要素构成的飞线，表达两个空间位置间的有向关联。

## 迁移样式参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **颜色** | 迁移飞线颜色 | #FFE038 |
| **样式** | 迁移飞线样式 | 虚线 |
| **飞线宽度** | 迁移飞线宽度 | 1 |
| **状态** | 开启后，根据数据中 status 属性判断迁移飞线的状态，赋予不同状态的迁移飞线不同颜色属性 | 不开启 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **from** | 迁移飞线出发位置所在行政区名称 |
| **to** | 迁移飞线到达位置所在行政区名称 |
| **status** | 迁移飞线状态值 |
| **fromLat** | 迁移飞线出发位置纬度 |
| **fromLng** | 迁移飞线出发位置经度 |
| **toLat** | 迁移飞线到达位置纬度 |
| **toLng** | 迁移飞线到达位置经度 |
