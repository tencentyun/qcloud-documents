动态轨迹层基于线要素表达多个地理位置间的运动行为轨迹，模拟在轨迹线上的动态演进效果。

## 参数配置
| 参数           | 说明                   | 默认值  |
| -------------- | ---------------------- | ------- |
| **轨迹颜色**   | 轨迹线颜色             | #03EDFF |
| **轨迹宽度**   | 轨迹线宽度             | 1       |
| **轨迹速度**   | 轨迹线运动演进速度     | 2       |
| **轨迹图高度** | 轨迹线整体高度         | 1       |
| **开启泛光**   | 是否开启轨迹线泛光效果 | 开启    |
