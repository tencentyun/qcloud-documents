线热力层基于线要素，展示沿线地理位置相关的热力数据信息。

## 参数配置
| 参数           | 说明                     | 默认值  |
| -------------- | ------------------------ | ------- |
| **最大值颜色** | 热力值的最大值颜色       | #006EFF |
| **最小值颜色** | 热力值的最小值颜色       | #00C341 |
| **热力线宽度** | 热力线线宽               | 1       |
| **热力图高度** | 热力图的整体高度         | 1       |
| **开启泛光**   | 是否开启线热力的泛光效果 | 开启    |
