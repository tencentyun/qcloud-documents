相机控制器通过设定一系列场景相机状态的数据，自定义移动、缩放等相机动画效果，表达不同时间段的相机位置和视角等状态变化情况。

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **循环次数** | 循环播放相机动画的次数 |1 |
| **动画时长** | 单段相机动画的持续时间 |2 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **percentage** | 动画过程中该关键帧的百分比 |
| **rotation** | 视点在水平面上的旋转角度 |
| **center** | 视点中心经纬度 |
| **zoom** | 场景地图缩放级别 |
| **pitch** | 视点俯仰角度 |
