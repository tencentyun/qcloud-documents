动态轨迹层基于线要素表达多个地理位置间的运动行为轨迹，模拟在轨迹线上的动态演进效果。	

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **轨迹颜色** | 轨迹线的颜色 |#00C8DC |
| **轨迹宽度** | 轨迹线的宽度 | 5 |
| **轨迹长度** | 轨迹线从头部到尾部的长度|5 |
| **运动速度** | 轨迹线运动速度 |5 |
| **头部高亮** | 开启后，轨迹线头部有高亮效果 | 不开启 |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **vendor** | 轨迹标识 |
| **path** | 轨迹运动路径，由经纬度和时间抽构成 |
