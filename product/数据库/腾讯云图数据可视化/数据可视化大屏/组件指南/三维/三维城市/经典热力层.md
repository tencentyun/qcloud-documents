经典热力层通过热力圈的形式表示热力值数据的大小、分布和聚集程度。

## 参数配置
| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| **辐射半径** | 热力圈半径，不随地图缩放变化，相邻热力圈会随地图缩放而聚合 |60 |
| **峰值高度** | 热力圈高度 | 60 |
| **色带颜色** | 按数据中最高热力值和最低热力值间按照比例分配的色带颜色，默认是四个颜色分别对应比例是35%、55%、85%、100% |RGB(0,0,255)/ RGB(0,255,0)/ RGB(225,225,0)/ RGB(255,0,0) |

## 数据配置
| 关键字段 | 说明 |
| --- | --- |
| **x** | 所在行政区名称 |
| **y** | 热力值 |
| **lat** | 热力圈中心纬度 |
| **lng** | 热力圈中心经度 |
