Tencent Cloud Redis Monitoring provides multi-dimensional custom alarm feature of sending an alarm to user via SMS when the metric data exceeds the threshold.

1. Click "Alarm Settings" in the instance list
![](https://mc.qcloudimg.com/static/img/82d3ff2cd0bec92fee235936aabe935b/gaojing.png)

2. Click "Add Alarm Policy"
![](https://mc.qcloudimg.com/static/img/d313f290eb4bab034161ab42f57fce21/xinzgaojingcelue.png)

3. Configure the alarm rule and select Redis as the policy type.
![](https://mc.qcloudimg.com/static/img/b1cea9d4feb9e2d26f7f4764cb14de7f/peizhigaojguiz.png)

4. Associate with the alarm object, i.e., the instance object need to be associated with
![](https://mc.qcloudimg.com/static/img/32b3b5e7bec0ece2a569701ad90fb228/duixiang.png)

5. Set the recipient group, i.e., the objects receiving the SMS
![](https://mc.qcloudimg.com/static/img/230d557a4bb20ef3d058df0535566da7/jieshouzu.png)
