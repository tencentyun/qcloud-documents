### 如何选择 Redis 规格？ 
Redis 规格可以从容量和性能两个方面来选择，容量根据数据存储需求来定，容量性能请参见 [性能数据](https://cloud.tencent.com/document/product/239/17952) 。

### Redis 支持哪些地域？ 
Redis 作为基础设施，CVM 部署的地域都会支持，具体请参见 [地域和可用区](https://cloud.tencent.com/document/product/239/4106)。

### Redis 如何计费？
支持包年包月和按量计费模式，具体价格请参见 [计费概述](https://cloud.tencent.com/document/product/239/9894)。

### Redis 的性能指标是多少？ 
Redis 分单节点、主从和集群版，单节点和标准版的需求在8万 - 12万 QPS，超过12万 QPS 需求可选择集群版。

### Redis 兼容哪些版本？ 
目前兼容 Redis 2.8、Redis 3.2 和 Redis 4.0。
