A newly purchased instance has an initial status of "To be initialized", and you need to set a password for it on the console to complete the initialization. To complete the initialization, you need to enter the access password for the Redis instance, as shown below:
![](https://mc.qcloudimg.com/static/img/3bac509765c127c469301f7fe6e6f1ab/3.png)

After the Redis password is initialized, the Redis instance goes into a running status, as shown below:
![](https://mc.qcloudimg.com/static/img/10d9b086660e92587599a3c81f4e9ddd/5.png)
Then you can log in to CRS to enter the management page, and perform operations such as assignment to project, renewal and upgrade.
