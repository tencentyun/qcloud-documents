### 云数据库 Redis 引擎简介
  云数据库 Redis 提供两个引擎版本：
  - **社区版引擎**：结合腾讯在分布式缓存领域多年技术沉淀，基于社区 Redis 版本，打造的一款高可用、高可靠的 Redis 产品，100% 兼容 Redis 社区版本（2.8，4.0）；
  - **腾讯云 CKV 引擎**：腾讯自主研发的高性能、低延时、持久化、分布式 KV 存储服务。在腾讯的微信平台、开放平台、腾讯云、腾讯游戏和电商平台广泛使用，日访问量超过万亿次。腾讯云 CKV 支持主从版、集群版，能够满足不同的业务场景。
