云数据库 Redis（TencentDB for Redis）是基于腾讯在分布式缓存领域多年技术沉淀，打造的一款高可用、高可靠的 Redis 服务平台。云数据库 Redis 有社区版 Redis 和腾讯云自研版 CKV 两个存储引擎，支持单机版、主从版和集群版三种灵活的部署架构，能够满足用户不同的业务场景需求。

## 产品功能
- 主从热备：提供主从热备，宕机自动监测，自动容灾。
- 数据备份：主从和集群版数据持久化存储，可提供每日冷备和自助回档。
- 弹性扩容：可弹性扩容实例的规格或缩容实例规格，集群版支持节点数的扩容和缩容，以及副本的扩容和缩容。
- 网络防护：支持私有网络 VPC，提高缓存安全性。
- 分布式存储：用户的存储分布在多台物理机上，彻底摆脱单机容量和资源限制。

## 引擎和版本
社区版 Redis 引擎兼容 Redis 2.8 、Redis 4.0 版本协议和命令，腾讯云自研版 CKV 引擎兼容 Redis 3.2 版本协议和命令。
