### 游戏场景

游戏行业场景中，用户可以将一些非角色数据，如积分排行榜，存储在 Redis 中进行快速访问，Redis 原生自带的 SortedSet 数据类型恰好能帮助您开发玩家数据。
![](https://main.qcloudimg.com/raw/6155a59a066624938266c145e47c37cc.svg)

### 互联网/APP
互联网、App 应用产品中，可以将用户的一些基础资料缓存到 Redis 之中，提高读性能。同时，也可以将一些静态的图片，资源缓存到 Redis 中，提高应用加载速度。
![](https://main.qcloudimg.com/raw/4bc12e9de1c6998ec6667a90246e78cd.svg)

### 电商展示
商品展示、购物推荐等数据可以存储在 Redis 中进行快速访问，同时在大型促销秒杀活动中，Redis 达 10W 级 的 QPS 能轻松应对高并发访问。
