## 操作场景
若您需要对创建的数据库帐号进行停用，可以通过云数据库 SQL Server 控制台删除数据库帐号。

>!避免因帐号误删除而影响业务的正常使用，请确认即将删除的数据库帐号不再对任何应用程序提供使用。

## 操作步骤
1. 登录 [SQL Server 控制台](https://console.cloud.tencent.com/sqlserver)，在实例列表，单击实例 ID 或**操作**列的**管理**，进入实例管理页面。
2. 在实例管理页面，选择**帐号管理**页，选择所需帐号，在**操作**列单击**删除帐号**。
![](https://main.qcloudimg.com/raw/0e61f5d2b77cc9ac8edbba8a89e35e2f.png)
3. 在弹出的对话框，单击**删除**。
