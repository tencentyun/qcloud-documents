## 操作场景
云数据库 SQL Server 支持修改帐号权限功能，您可以根据需要修改帐号的权限。管理员帐号的权限无法修改为指定的权限。


## 操作步骤
1. 登录 [SQL Server 控制台](https://console.cloud.tencent.com/sqlserver)，在实例列表，单击实例名或“操作”列的【管理】，进入实例管理页面。
2. 选择【帐号管理】页签，选择所需帐号，在“操作”列单击【修改权限】。
![](https://main.qcloudimg.com/raw/369420acefda3cf12f3721de1fcd0593.png)
3. 在弹出的对话框，选中或者取消需要授予的权限库和权限，单击【确定】，完成修改。
![](https://main.qcloudimg.com/raw/f762e34a30993b90c39e67143acaa3af.png)
