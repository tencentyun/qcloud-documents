## 操作场景
云数据库 SQL Server 支持修改帐号权限功能，您可以根据需要修改帐号的权限。管理员帐号的权限无法修改为指定的权限。

## 操作步骤
1. 登录 [SQL Server 控制台](https://console.cloud.tencent.com/sqlserver)，在实例列表，单击实例 ID 或**操作**列的**管理**，进入实例管理页面。
2. 在实例管理页面，选择**帐号管理**页，选择所需帐号，在**操作**列单击**设置权限**。
![](https://qcloudimg.tencent-cloud.cn/raw/39b2a2ea00afc05dd2504c1c88bca179.png)
3. 在弹出的对话框，选中或者取消需要授予的权限库和权限，单击**确定**，完成修改。
