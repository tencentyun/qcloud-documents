尊敬的腾讯云用户，您好：
云数据库 SQL Server 于2023年02月24日对维护信息中的时间进行了变更，其中所设置的维护时间由实例所在地域时间变更为了北京时间，敬请悉知。

## 变更时间
北京时间2023年02月24日（周五）。

## 变更说明
![](https://qcloudimg.tencent-cloud.cn/raw/83aaf71d34eacf4306ac608f72562113.png)
实例维护信息窗下的维护时间，由**实例所在地域时间**变更为**北京时间**。

## 影响说明
变更后需注意 [调整配置](https://cloud.tencent.com/document/product/238/67860) 功能中的生效时间和 [跨可用区迁移](https://cloud.tencent.com/document/product/238/62980) 功能中的切换时间为北京时间，当您对实例调整配置或跨可用区迁移时，系统生效的时间均以北京时间为准来记录。
