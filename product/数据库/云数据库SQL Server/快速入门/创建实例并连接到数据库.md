您可以通过云数据库 SQL Server 控制台来完成管理数据库相关工作。

## 创建实例
登录 [SQL Server 控制台](https://console.cloud.tencent.com/sqlserver)，在实例列表单击【新建】，选择实例的地域、网络、规格等，单击【立即购买】创建实例。

## 创建数据库
单击实例名称进入实例详情页，选择【数据库管理】>【创建数据库】，输入数据库名称，选择字符集，如果您需要给数据库授权帐号的话，选择授权帐号，选择对应的数据库，然后设置权限，目前权限分为读写和只读两种，确认无误后单击【确定】。
![](//mccdn.qcloud.com/static/img/1b9bf222fa2eabb32746cc857a6ea2e1/image.png)
![](https://main.qcloudimg.com/raw/03f4816553056fe5560c731e9e6e457d.png)

## 设置数据库权限
成功创建数据库后，您可以对数据库授权帐号或者修改授权，在数据库的【操作】列，单击【设置权限】。
![](//mccdn.qcloud.com/static/img/c7000bb4426767327049ead56e572e75/image.png)
单击【>】和【<】对相应的帐号增加和减少权限。
![](//mccdn.qcloud.com/static/img/68b1eee313ce5281eafce1c194225804/image.png)

## 更改数据库名称
您可以将鼠标滑动到【数据库名】附近，会出现![](//mccdn.qcloud.com/static/img/cb9b93c02de85f2fa4724fe0b31bb67d/image.png)标记，单击这个标记来进行改名操作。
![](//mccdn.qcloud.com/static/img/aeaf29a88d2586596795d8086d605061/image.png)

## 更改数据库描述
您可以将鼠标滑动到数据库【描述】附近，会出现![](//mccdn.qcloud.com/static/img/cb9b93c02de85f2fa4724fe0b31bb67d/image.png)标记，单击这个标记来进行更新描述操作。
![](//mccdn.qcloud.com/static/img/8be363dcbfcdc65da91cc41788d4781f/image.png)
## 删除数据库
您可以在【操作】列，单击【删除】来删除数据库。
![](//mccdn.qcloud.com/static/img/bed7223847c71b72ef48a639cb187b35/image.png)
