### 部署架构
云数据库 SQL Server 支持3种部署架构：

#### 基础版
计算和存储分离，基于高性能云盘，采用单个节点部署。

#### 双机高可用
由一主一镜像（Mirror）的 SQL Server 数据库组成，跨机架/跨可用区部署。

#### 集群版
采用 Always On 架构，默认跨机架/跨可用区进行一主一备的集群架构部署。
