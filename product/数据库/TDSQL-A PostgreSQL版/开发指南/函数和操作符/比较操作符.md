比较操作符可以用于所有可以比较的数据类型（两个数据类型必须是相同的数据类型或者是可以进行隐式转换的类型），所有比较操作符都是双目操作符，它们返回 boolean 类型。

TDSQL-A PostgreSQL版 对于常见的比较操作符都可用：

| **操作符** | **描述** |
| ---------- | -------- |
| <          | 小于     |
| >          | 大于     |
| <=         | 小于等于 |
| >=         | 大于等于 |
| =          | 等于     |
| <>或者!=   | 不等于   |
