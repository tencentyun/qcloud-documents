
| **操作符** | **描述** | **示例**             | **结果** |
| ---------- | -------- | -------------------- | -------- |
| \|\|       | 连接     | B'10001' \|\| B'011' | 10001011 |
| &          | 按位与   | B'10001' & B'01101'  | 00001    |
| \|         | 按位或   | B'10001' \| B'01101' | 11101    |
| #          | 按位异或 | B'10001' # B'01101'  | 11100    |
| ~          | 按位求反 | ~ B'10001'           | 01110    |
| <<         | 按位左移 | B'10001' << 3        | 01000    |
| >>         | 按位右移 | B'10001' >> 2        | 00100    |
