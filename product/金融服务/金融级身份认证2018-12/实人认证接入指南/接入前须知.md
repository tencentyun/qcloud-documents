为了确保服务接入的安全性，整个服务都使用标准 OAuth2.0 鉴权认证。整个鉴权分为签名鉴权和 SDK 鉴权，具体流程请参见下文说明。
## 申请分配接入信息
1. [申请腾讯云账号](https://console.cloud.tencent.com/developer)，进行企业认证；如果已有腾讯云账号，此步骤可省略，直接进行第二步。
1. [控制台](https://console.cloud.tencent.com/tianyu/service/financialCertification/application) 填写接入信息提交应用申请；此步骤申请 webankAppid、secret 和 SDK  licence，APP 接入应用类型选择 SDK，其他选择 API。
>!
>- AppID 和 secret 务必保密，一旦泄露，将会导致服务不可用。后果由合作方自行承担。
> - 生产环境 AppID 默认风控参数：在24小时内进行身份认证，同一身份证件的上限是5次，同一设备的上限是10次。该参数可根据合作方的需求进行调整。
## 签名鉴权
合作方服务端根据 [登录鉴权流程](https://cloud.tencent.com/document/product/655/31896) 来获取 Token 和 SIGNticket（或 NONCEticket），生成签名并进行鉴权。
>!只有上述步骤成功完成后，才能正常调用服务端后台接口。
## SDK 鉴权
SDK 鉴权可用于合作伙伴启动并进行远程身份验证。鉴权的 SDK License 根据合作方 App 的 iOS bundle ID 和 Android Package Name 生成，需要合作伙伴按照模板提供资料后分配。
>!如果合作伙伴不使用 SDK 模块，不需要进行 SDK 鉴权。
