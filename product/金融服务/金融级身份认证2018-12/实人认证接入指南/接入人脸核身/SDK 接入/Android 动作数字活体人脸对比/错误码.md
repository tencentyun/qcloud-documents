## 登录错误码
| 错误码                                       | 错误描述                    |
| -------------------------------------------- | --------------------------- |
| FACEVERIFY_LOGIN_ERROR = "-10000"           | 登录请求错误，没有返回 code |
| FACEVERIFY_LOGIN_PARAMETER_ERROR = "-20000" | 传入参数有误                |
| FACEVERIFY_LOGIN_NO_RESONSE = "-30000"     | 登录请求未到达后台，非200  |
