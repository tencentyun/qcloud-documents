活体检测 H5 结果返回跳转第三方 URL 带唯一标识、订单号、识别分数、验证结果、签名。
## 请求
- **请求 URL：**`https://xxx.com/xxx?code=xxxx&orderNo=xxxx&liveRate=xxxx&&newSignature=xxxx`
- **请求方法：**GET
>!
	- xxxx.com 为合作方上送的 URL。
	- 合作方请参见 [服务端验证结果](https://cloud.tencent.com/document/product/655/32163) 说明进行签名校验，确保返回结果的安全性。
	
## 响应
- **响应参数：**

| 参数 | 说明 | 类型 | 长度（字节）
|-|-|-|-|
| code | 人活体检测结果的返回码，0表示人活体检测成功，其他则为错误码标识失败，具体错误码信息请参见 [通用响应码](https://cloud.tencent.com/document/product/655/32309)。 | String | - |
| orderNo | 订单号，由合作方上送，每次唯一，此信息为本次人活体检测上送的信息。 | String | 32 |
| newSignature | 对 URL 参数 AppID、code、orderNo 和 SIGN ticket 的签名。具体请参见 [签名算法说明](https://cloud.tencent.com/document/product/655/31969) 和校验规则。 | String | 40 |
| liveRate | 人活体检测结果分数。 | String | 40 |
