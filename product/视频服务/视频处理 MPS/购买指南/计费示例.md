## 示例背景
用户 A 的主要业务为视频网站平台，为保障视频在各个终端的观看效果，需要对源视频进行转码，为视频网站提供多种清晰度的视频。用户 A 的源视频种类丰富，有多种分辨率，用户 A 对于非标准的模板计费产生了疑惑。



## 计费分析

#### 转码输出规格
 视频处理服务针对不同的视频输入，将其分为两大类规格：
 + 标准规格：用户转码输出的视频为某一类预置模板或转码输出视频的长短边都小于某一类预置模板。
 + 非标准规格：除标准规格外都属于非标准规格。

#### 标准规格输入输出
场景：输出视频为标准转码模板。
+ 转码方式：普通转码。
+ 输入视频：640 x 480的分辨率。
+ 视频时长：源视频转码时长10分钟。
+ 转码模板：预置高清 HD 模板，分别为 H.264（1280 × 720）、自定义模板 H.264（1024 x 700）。
+ 其他转码：转封装、音频转码。
+ 输出视频：1280 x 720的分辨率。

分析：用户 A 采用了两个标准规格模板，分别为预置 H.264（1280 × 720）模板，收费为0.0325元/分钟，自定义模板 H.264（1024 x 700），其长短边都小于预置 H.264（1280 × 720）模板，故也为0.0325元/分钟，考虑到用户还使用了转封装和音频转码，故按照标准计费方式结算：
转码费用 = 10（分钟）x 0.0325（元/分钟）+ 10（分钟）x 0.0325（元/分钟）+ 10（分钟）x 0.007（元/分钟）+ 10（分钟）x 0.0056（元/分钟）= 0.776元

#### <span id = "p1"></span>非标准规格输入输出
场景一：输出视频的长短边中有一边大于某类预置模板。
+ 转码方式：普通转码。
+ 输入视频：640 x 480的分辨率。
+ 视频时长：源视频转码时长10分钟。
+ 转码模板：自定义模板 H.264（1024 x 800）。
+ 输出视频：1024 x 800的分辨率。

分析：用户 A 的输出视频为自定义模板且**长短边不是都小于某一预置模板**，属于非标准规格转码，则转码单价按照像素点面积计费，其最终像素点面积（1024 x 800）< 高清 HD（1280 x 720）及以下，故采用高清 HD（1280 x 720）而非全高清 FHD（1920 x 1080）的计费单价：
转码费用 = 10（分钟）x 0.0325（元/分钟）= 0.325元 

场景二：输入的源视频长短边一致。
+ 转码方式：普通转码。
+ 输入视频：1600 x 1600的分辨率。
+ 视频时长：源视频转码时长10分钟。
+ 转码模板：预置高清 HD 模板 H.264（1280 × 720）。
+ 输出视频：为保证客户视频不会被拉伸和压缩，视频处理将自动转码为1280 x 1280的分辨率。

分析：用户 A 输入的视频长短边一致，为保证客户视频不会被拉伸和压缩，视频处理将自动转码为1280 x 1280的分辨率，其分辨率大于高清 HD（1280 x 720），故采用全高清 FHD（1920 x 1080）的计费单价：
转码费用 = 10（分钟）x 0.063（元/分钟）= 0.63元
