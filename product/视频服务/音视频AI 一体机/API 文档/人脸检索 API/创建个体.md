### 1. 接口描述
本接口用于创建个体。
接口调用方法：POST
```
http://HOST/201712v1/video_face_recognition/new_person
```


### 2. 输入参数

|参数名 |	要求|	类型 	|备注|
|-----------|------|-------|------|
|appId 	|必选	|int64|	xxx|
|personId	|必选	|string	|个体 Id(appId 内唯一)   |
|personName	|必选|	string|	检索个体名称，不超过 12 字节（4 个汉字），例如：”张三”|


### 3. 输出参数

|参数名 |	要求|	类型 |	备注|
|---------|--------|-------|-------|
|errCode 	|必选|	int	|[返回码](https://cloud.tencent.com/document/product/1015/31186#.E8.BF.94.E5.9B.9E.E7.A0.81.E8.AF.B4.E6.98.8E)|
|errMsg |	必选	|string|	错误说明|

### 4. 示例
**输入示例**
```
POST  /201712v1/video_face_recognition/new_person   HTTP/1.1
Accept: application/json;charset=utf-8
Content-Type: application/json;charset=utf-8
Host: 10.171.41.14:80
Content-Length: xxx
{
  "appId":1252761572,
  "personId": "ABC5354",
  "personName":"张三"
}
```

**输出示例**
```
HTTP/1.1 200 OK
Content-Type: application/json; charset=utf-8
Content-Length: xxx
{
  "errCode":0,
  "errMsg":"success"
}
```
