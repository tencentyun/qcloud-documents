# 鉴黄服务
鉴黄服务是在互动直播基础上，可由客户单独选择开通的安全类服务。客户可针对互动直播的每个sdkappid单独进行设置

## 鉴黄服务组成
1.** 截图功能：**鉴黄的最基本能力，必选功能；原理是在互动直播房间开启时，从服务端每隔一定时间截取一次屏幕，用于后续的鉴别
2. **截图存储功能：**用于存储截屏后的图片，必选功能；根据国家相关部门要求，所截图片要存储一定的周期，客户可根据需要在此基础上设置更长的存储时间
3. **自动鉴黄功能**：可选功能，图片截取、存储后，通过人工智能判断涉黄级别，方便客户审核人员缩小审查范围，提升效率
4. **人工审核服务**：可选服务，如果客户缺少审核人力，可通过我们提供的人工服务来对截屏图片进行审查
