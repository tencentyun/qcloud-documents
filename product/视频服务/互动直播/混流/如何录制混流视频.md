**限直播码模式**，互动直播房间中能够展示多路视频，但是旁路直播（自动录制）的视频默认却是分开的。

## 旁路直播

在自动开启旁路直播的情况下，房间内所有视频（主播及上麦观众）都会产生对应的直播视频流。详见 [直播码模式下旁路直播](https://cloud.tencent.com/document/product/268/8560)。

```
直播码=BIZID_MD5(房间号_用户名_数据类型)
摄像头数据类型是 main
屏幕分享的数据类型是 aux
播放地址=传输协议://BIZID.liveplay.myqcloud.com/live/直播码[.格式]
```

## 混流接口

腾讯云视频后台提供接口可以将多路视频流，按照一定的布局，混成新的视频流（也可以覆盖原视频流）。详见 [混流接口文档](https://cloud.tencent.com/document/product/267/8832)。

## 自动录制

在开启自动录制时，所有视频流(包括混流后的视频)都会自动录制为视频文件，供用户观看下载。所以录制文件成功都会以录制事件（100）通知到回调 URL，详见 [直播码模式下旁路直播](https://cloud.tencent.com/document/product/268/8560)。

```
录制文件名=直播码_起始时间_结束时间
```
