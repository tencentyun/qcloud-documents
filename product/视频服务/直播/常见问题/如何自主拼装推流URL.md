
### 推流 URL 
实际产品中，您不可能为每一个主播手工创建推流和播放 URL，而是要由您的服务器自行拼装，只要符合腾讯云标准规范的 URL 就可以用来推流，如下是一条标准的推流 URL，它由四个部分组成：
![](https://main.qcloudimg.com/raw/d47c2090638110320e55f00eff81fe3b.jpg)
- **StreamName（流 ID）**
推荐用随机数字或者用户 ID。
- **txTime（地址有效期）**
何时该 URL 会过期，格式支持十六进制的 UNIX 时间戳，比如`5867D600`代表`2017年1月1日0时0点0分`过期，我们的客户一般会将 txTime 设置为当前时间24小时以后过期，过期时间不要太短也不要太长，当主播在直播过程中遭遇网络闪断时会重新恢复推流，如果过期时间太短，主播会因为推流 URL 过期而无法恢复推流。
- **txSecret（防盗链签名）**
防止攻击者伪造您的后台生成推流 URL，计算方法参考 [最佳实践-防盗链计算](https://cloud.tencent.com/document/product/267/32735)。
- **示例代码**
[【直播控制台】](https://console.cloud.tencent.com/)>【域名管理】，选中事先配置的推流域名，【管理】>【推流配置】页面下半部分有【推流地址示例代码】（PHP 和 Java 两个版本）演示如何生成防盗链地址。
