## 1. 接口描述
- **域名：**live.api.qcloud.com
- **接口名:** GetVodRecordFiles

注意事项：
1.查询频道录制文件结果信息，包含文件名，任务 ID，文件 ID， 录制消息。
2.**最长查询2个月内的录制文件。**
