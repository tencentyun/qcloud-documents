## 1. 接口描述

域名：live.api.qcloud.com
接口名: DescribeRecord
注意事项：最长查询2个月内的录制文件。

>?录制后的文件存放于点播平台。用户如需使用录制功能，需首先自行开通点播账号并确保账号可用。录制文件存放后，相关费用（含存储以及下行播放流量）按照点播平台计费方式收取，请参见 [对应文档](https://cloud.tencent.com/document/product/267/2818)。
