
## 流量带宽
云直播流量带宽统计数据是用户与加速源站建立连接时产生的下行流量带宽数据。提供两种计费方式：流量计费和带宽计费，二者均为后付费日结计费，新用户开通云直播后默认为流量计费。
### 流量计费
云直播流量计费采用日到达阶梯计费，详细按日流量计费价格阶梯如下表所示：

| 流量阶梯 | 价格(元/GB/天) | 
|---------|---------|
|0 - 500GB|	0.26|
|500GB(含) - 2TB|0.25|
|2TB(含) - 50TB|	0.23|
|50TB(含) -100TB	|0.19|
|≥100TB	|0.16|

**计费说明：**
1. 计费项：直播观看产生的国内全网下行流量。
2. 计费方式：后付费计费。
3. 计费周期：按日计费，每天上午10点扣除前一天的流量费用，详细以计费账单为准。
4. 计费规则：按日到达阶梯计费，统计一个自然日内的流量累计值乘以对应阶梯单价。
5. 计费示例：
       1. 若直播码率为500Kbps，直播时长1小时，观看人数为100人，则消耗的流量约为：
          500/8 x 3600 x 100 = 22500000KB = 22.5GB
       2. 若您在2019年1月1日使用直播服务，并产生22.5GB下行流量，则1月2日您需支付的直播流量账单如下：
          日直播流量费用 = 0.26（元/GB）x 22.5（GB）= 5.85元

## 预付费流量包
云直播流量包为预付费资源包，采用先购买、后抵扣的方式。当计费方式为日流量后付费时，产生的下行流量会优先通过流量包抵扣，超出部分按照日流量后付费的阶梯价格进行计费。直播流量包仅支持使用直播代金券，不支持通用代金券抵扣。详细预付费流量包规格和价格如下表所示：

| 流量包规格 | 价格(元) | 
|---------|---------|
|100GB|	25|
|500GB|118|
|1TB|236|
|5TB|1086|
|10TB|2172|
|50TB|8972|
|200TB|30198|
|1PB|149589|

>! 流量/带宽的单位进制为1000，即：1TB = 1000GB。

**计费说明：**
1. 直播流量包生效时间为购买后立即生效，有效期为1年，到期流量包若有余量则一次性扣除。
2. 每天上午10点出日结流量账单时，优先抵扣流量包余量，超出部分按日流量计费方式出账。
3. 支持购买多个流量包，可叠加使用，有效期不叠加，实际抵扣按最早过期时间依次抵扣。
4. 流量包购买后，若未使用，支持5天内无理由退款，使用后不支持退款。若用户账户因内容审核问题（涉黄、涉政等）导致停服，账户下所有流量包均不支持退款。若用户有特殊需求退款，仅可通过工单方式提交申请。
5. 若计费方式非日结流量计费，流量包余量冻结，不参与抵扣，冻结期间不延长有效期。当计费方式切换至日结流量计费，流量包可继续使用。
6. 流量包仅限国内流量包抵扣，不适用于海外加速产生的流量。

## 直播录制
直播服务可将直播流录制下来，并存储至点播中。录制功能默认关闭，可通过控制台或云 API 开启。使用录制功能会产生费用，以当月直播录制并发峰值路数为结算标准。详细价格如下：

| 计费类型| 价格（元/路/月） |
|---------|---------|
|录制路数峰值 |30 | 

**计费说明：**
1. 计费项：直播录制路数。
2. 计费方式：后付费计费。
3. 计费周期：按月计费，每月1 - 5日出上一个自然月的账单并扣除费用，详细以计费账单为准。
4. 计费规则：一个自然月内录制并发峰值路数乘以单价。一种录制文件计为一路录制流，例如：同一个流 ID 录制了 MP4 和 HLS 两种格式文件，则计为两路录制流。
5. 计费示例：
		若您在2019年1月1日至2月1日期间使用直播录制服务，当月录制路数的峰值为10路，则2月2日您需支付的直播录制账单如下：
		1月直播录制费用 = 30（元/路/月）x 10路 = 300元
