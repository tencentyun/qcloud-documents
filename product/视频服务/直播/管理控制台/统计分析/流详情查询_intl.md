Enter stream ID to query push and playback information of a stream on the LVB console [Query stream details](https://console.cloud.tencent.com/live/analysis/stream) page. See the figure below:
![](https://main.qcloudimg.com/raw/1405881168442dc12a11a5aec11161e0.png)
### Upstream push query
Push data query can display the frame rate and bitrate **within 3 hours** of the **query start time** and **end time** of a single stream.
> Only the push data of a single stream in the last 5 days can be queried.

After inputting the stream ID, you can select **View Push Data** to query LVB stream push information. See the figure below:
![](https://main.qcloudimg.com/raw/5227cdfac681f07284729935ba0845ac.png)

Also you can query stream interruption data, see the figure below:
![](https://main.qcloudimg.com/raw/a5e81861cbb43af099a94ae3b5fd4a9c.png)

### Downstream playback query
Downstream playback query can display the frame rate and bitrate of a single stream within 24 hours of the start time and end time. Only the data in the last 30 days can be queried. Select **View Playback Data** to query LVB stream playback information.
![](https://main.qcloudimg.com/raw/ad50ee862d891efc13864fd25027bfae.png)
