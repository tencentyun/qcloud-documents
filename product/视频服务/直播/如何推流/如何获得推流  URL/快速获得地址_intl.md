To generate a set of URLs for testing, log in to Tencent Cloud [LVB Console](https://console.cloud.tencent.com/live), select **Access Management** -> **LVB Code Access (Recommended)** -> **Push Generator**, and click **Generate Push URL** button to generate a push URL and three playback URLs with different playback protocols.
![](//mc.qcloudimg.com/static/img/0f6ef47809b0d98ff0beb3c6fb4bfbaa/image.png)
You can use [RTMP SDK DEMO](https://cloud.tencent.com/document/product/454/6555) to test the validity of the push and playback URLs quickly.
