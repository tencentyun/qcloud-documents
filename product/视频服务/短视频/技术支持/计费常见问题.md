### 购买点播资源包后，为什么还是无法使用短视频 SDK？
需要购买**10TB及以上**的点播资源包才能获取对应的 License 获取短视频 SDK 的使用权。

[](id:que2)
### 短视频 SDK 除了 License 费用外，使用过程中还有其它费用吗？
短视频 SDK 是根据对应的版本有功能使用的区别，在使用过程中，如果使用了云点播相关的资源则会收取相应的费用，详情请参见 [价格总览](https://cloud.tencent.com/document/product/266/14666)。

[](id:que3)
### 使用短视频 SDK，需要购买哪些套餐包？
可以购买点播流量资源包 10TB 赠送短视频 SDK 精简版，50TB、200TB 赠送基础版短视频 SDK。并且需要购买存储资源包，如不需要转码可以不购买转码资源包。

[](id:que4)
### 购买流量资源包是否送《信息网络传播视听节目许可证》和短视频播放器小程序插件？
不增送，需要单独购买小程序插件，插件会赠送许可证。

[](id:que5)
### 购买短视频 SDK 基础版，但是跳转到云点播的购买资源包界面是什么原因呢？
短视频 SDK 是搭配云点播使用的，需要购买对应的资源包才会赠送到 SDK 的使用权限。

[](id:que6)
### 点播资源包是否支持退款？
若您的点播资源包五天内未被使用，支持退还。更多详情，请参见 [云点播退费说明](https://cloud.tencent.com/document/product/266/35787)。

[](id:que7)
### 购买10TB资源包，是否可以赠送基础版 SDK License？

不赠送，10T 流量资源包仅支持赠送精简版 License，基础版 License 需要购买 50T 或 200T 流量资源包才能获取使用权。
