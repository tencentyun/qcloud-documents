短视频 SDK 提供了3个版本的 SDK：
- [基础版 SDK]()
- [专业版 SDK]()
- [企业版 SDK]()
 
并有4个版本的短视频 License 来解锁对应的短视频功能。

### SDK 关系对应
3个版本的 SDK 关系如下图所示：
![](https://main.qcloudimg.com/raw/ebfba8f2bbdd5309c50343691e7d7a23.svg)

- [基础版 SDK]() 包含短视频全功能，带有基础美颜（美白、磨皮等），可用短视频的精简版或基础版 License 解锁。
- [专业版 SDK]() 包含多个音视频产品的合集，功能上和短视频基础版没有区别，**主要是为了解决同时集成多个音视频产品时的符号冲突问题，可用短视频的精简版或基础版 License 解锁短视频相关功能**，其他产品功能需要开通相应服务。
- [企业版 SDK]() 是在专业版 SDK 上增加了高级美颜特效能力，**您可以购买短视频企业版 License 来解锁短视频全功能 + 动效贴纸功能，或购买短视频企业版 Pro License 来解锁高级美颜特效等功能**。

基础版 SDK 和专业版 SDK 均可 [申请测试的短视频精简版或基础版 License](https://cloud.tencent.com/document/product/584/20333) 进行使用。
企业版 SDK 需要您先 [申请企业版或企业版 Pro License](https://cloud.tencent.com/product/x-magic)，您还可以联系 [腾讯云商务](https://cloud.tencent.com/apply/p/h1qsz5vhvko) 进行申请。

>!
>- 若在企业版 SDK 上使用基础版或精简版 License，则只能使用短视频基础版或精简版的功能。
>- 若在基础版 SDK 上使用企业版或企业版 Pro License，则无法使用动效贴纸、高级美颜特效等功能。
