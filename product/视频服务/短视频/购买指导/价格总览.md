## 计费说明
腾讯云短视频的费用由两部分产生：一部分源自腾讯云点播服务；一部分源自 SDK 的使用授权费用。在购买腾讯云点播服务指定套餐的情况下，可以获得附赠的 SDK 使用授权。

## 点播服务
推荐短视频 SDK 配合腾讯云点播服务一起使用，不仅价格更加便宜，还会获得兼容性和用户体验上的加成，点播计费详情可以参考 [点播价格表](https://cloud.tencent.com/document/product/266/2838)。

## SDK授权
SDK 有四种版本和四种授权 LICENSE:
 - SDK 版本分为精简版、基础版、商业版和商业版Pro，功能区别见下表；
 - LICENSE 对应也分四种版本，与SDK配套使用，获取方式见下方的**授权license**。需要注意的是在 4.9 版本中我们优化了 license 的集成，每种版本只需要一个对应的license 即可，并且支持服务器自动续签，使用方式参考 [工程配置](https://cloud.tencent.com/document/product/584/11638) 中的**短视频 licence 集成**。
