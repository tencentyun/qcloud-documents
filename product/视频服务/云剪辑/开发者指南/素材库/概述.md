素材库是云剪提供的集中式管理素材的能力，素材库中的素材能够项目引用，项目中制作完成的视频也可以导入素材库，被他人使用。素材库包含了素材管理、分类管理、素材权限管理和团队管理四大部分，给开发者提供了构建管理素材的能力。开发者可以根据业务场景搭建素材库，给用户在剪辑时候提供更多可选素材。

## 功能介绍
### 素材管理
素材管理是素材库最核心的部分，提供了管理素材的能力，开发者可以通过服务端 API 对素材进行导入、修改、删除等操作，在浏览素材列表时候更是提供多种模型，满足不同业务在不同场景的需求，具体详情请参见 [素材管理](https://cloud.tencent.com/document/product/1156/43789?!preview&!editLang=zh)。

### 分类管理
分类管理是指对素材的分类进行管理，开发者可以创建分类，将素材归属到指定的分类，给开发者提供素材归类的能力。分类管理提供了分类的相关操作 API，具体详情请参见 [分类管理](https://cloud.tencent.com/document/product/1156/43791?!preview&!editLang=zh)。

### 素材权限管理
素材权限是指用户对素材的使用权力，开发者可以给指定用户授予可用权限，那么对于指定用户才能拉取到素材并使用，对于其他用户不可能。素材权限管理提供了素材权限的授予和回收，具体详情请参见 [素材权限管理](https://cloud.tencent.com/document/product/1156/43790?!preview&!editLang=zh)。

### 团队管理
云剪允许开发创建团队，将特定的用户添加到团队中，在团队添加的素材能够被团队中的成员使用，其他团队或者个人无法操作。团队管理提供了创建团队、添加成员、移除成员等一系列的管理接口，具体详情请参见 [团队管理](https://cloud.tencent.com/document/product/1156/43792?!preview&!editLang=zh)。

## 应用场景
云剪提供的视频编辑仅仅满足了用户的剪辑需求，而素材库的出现解决了素材来源的问题，极大丰富了用户的内容编辑。素材库主要应用于以下几个场景：
- **私有素材库**：开发者业务自身没有集中式的素材来源，搭建素材库成本大。这个时候就可以直接使用云剪的素材库能力，完善用户编辑生态，解决剪辑时候最重要的素材供应问题。
- **多部门/业务接入**：开发者可以使用素材库的团队管理能力为不同的部门/业务创建团队，每个团队内素材共享，互相隔离。
- **素材分享**：开发者可以使用素材库的素材权限管理能力实现用户或者团队维度的分享，A 用户可以将制作完成的视频素材分享给 B 用户并控制权限，B 用户能够查看 A 用户分享的素材并添加到项目进行使用。

## 集成方式
1. **API 接入**：开发者可以通过服务端 API 搭建自身的素材库，并且根据业务的特点组合使用云剪素材库的能力，具体参考：
	- [素材管理]()
	- [分类管理]()
	- [素材权限管理]()
	- [团队管理]()
2. **页面集成**：云剪通过 SDK 直接提供整个素材库的操作页面，开发者无需自己开发前端页面即可快速拥有整套素材库的能力。
