**媒资库**是云剪提供的集中式管理素材的能力，媒资库中的素材能够项目引用，项目中制作完成的视频也可以导入媒资库，被他人使用。媒资库包含了素材管理、分类管理、素材权限管理三大部分，给开发者提供了构建管理素材的能力。开发者可以根据业务场景搭建媒资库，给用户在剪辑时候提供更多可选素材。

## 功能介绍
### 素材管理
素材管理是媒资库最核心的部分，提供了管理素材的能力，开发者可以通过服务端 API 对素材进行导入、修改、删除等操作，在浏览素材列表时候更是提供多种模型，满足不同业务在不同场景的需求，具体详情请参见 [素材管理](https://cloud.tencent.com/document/product/1156/43789)。

### 分类管理
分类管理是指对素材的分类进行管理，开发者可以创建分类，将素材归属到指定的分类，给开发者提供素材归类的能力。分类管理提供了分类的相关操作 API，具体详情请参见 [分类管理](https://cloud.tencent.com/document/product/1156/43791)。

### 素材权限管理
素材权限是指用户对素材的使用权力，开发者可以给指定用户授予可用权限，那么对于指定用户才能拉取到素材并使用，对于其他用户不可能。素材权限管理提供了素材权限的授予和回收，具体详情请参见 [素材权限管理](https://cloud.tencent.com/document/product/1156/43790)。

## 应用场景
云剪提供的视频编辑仅仅满足了用户的剪辑需求，而媒资库的出现解决了素材来源的问题，极大丰富了用户的内容编辑。媒资库主要应用于以下几个场景：
- **私有媒资库**：开发者业务自身没有集中式的素材来源，搭建媒资库成本大。这个时候就可以直接使用云剪的媒资库能力，完善用户编辑生态，解决剪辑时候最重要的素材供应问题。
- **素材分享**：开发者可以使用媒资库的素材权限管理能力实现用户或者团队维度的分享，A 用户可以将制作完成的视频素材分享给 B 用户并控制权限，B 用户能够查看 A 用户分享的素材并添加到项目进行使用。

## 集成方式
-  **API 接入**：开发者可以通过服务端 API 搭建自身的媒资库，并且根据业务的特点组合使用云剪媒资库的能力，具体参考：
	- [素材管理](https://cloud.tencent.com/document/product/1156/43789)
	- [分类管理](https://cloud.tencent.com/document/product/1156/43791)
	- [素材授权管理](https://cloud.tencent.com/document/product/1156/43790)
- **页面集成**：云剪通过 SDK 直接提供整个媒资库的操作页面，开发者无需自己开发前端页面即可快速拥有整套媒资库的能力。
>! 该集成方式目前尚未对外。
