## 接口描述
- 接口名称：`/Material/ModifyClass`
- 接口功能：修改分类。

## 输入参数

字段 | 类型 | 必填 | 描述
------- | ------- | ------- | -------
ClassId | Int64 | 是 | 分类 ID 。

Name | String | 否 | 分类名称。
