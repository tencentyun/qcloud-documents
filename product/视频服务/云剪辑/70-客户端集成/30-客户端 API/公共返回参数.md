## 公共输出参数
公共输出参数是携带请求返回码和描述信息的，如非必要，在每个接口文档中不再单独对这些参数进行说明。

#### 返回码和描述信息 
字段 | 类型 | 描述
------- | ------- |  -------
Code | String |返回码，Success 表示成功，其余表示失败。
Message | String |描述信息。
EnglishMessage | String |英文描述。
