## 准备工作

### Step 1：开通腾讯云 VOD 服务
请参考 [快速入门 - 步骤1](https://cloud.tencent.com/document/product/266/8757#.E6.AD.A5.E9.AA.A41.EF.BC.9A.E5.BC.80.E9.80.9A.E4.BA.91.E7.82.B9.E6.92.AD) 开通云点播服务，同时创建一个子 [应用](https://cloud.tencent.com/document/product/266/14574)。
.
>?若已开通云点播服务，请忽略此步骤。

### Step 2：创建多媒体创作引擎平台

请参考 [云剪 - 控制台指南 - 创建平台](https://cloud.tencent.com/document/product/1156/43823) 创建平台。

>! 
> 创建平台时，必须指定存储为上一步创建的云点播子应用。


### Step 3：获取 API 密钥

请求云 API 需要使用到开发者的 API 密钥（ SecretId 和 SecretKey）。如果还未创建过密钥，请参见 [创建密钥文档](https://cloud.tencent.com/document/product/598/40488#.E5.88.9B.E5.BB.BA.E4.B8.BB.E8.B4.A6.E5.8F.B7-api-.E5.AF.86.E9.92.A5) 生成新的 API 密钥；如果已创建过密钥，请参见 [查看密钥文档](https://cloud.tencent.com/document/product/598/40488#.E6.9F.A5.E7.9C.8B.E4.B8.BB.E8.B4.A6.E5.8F.B7-api-.E5.AF.86.E9.92.A5) 获取 API 密钥。

### Step 4：准备视频
将上文提到的两个示例视频和一个示例图片下载到本地，登陆腾讯云点播控制台，将其上传到点播平台。如下图所示。
