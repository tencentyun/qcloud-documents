**视频编辑项目**是云剪提供的专注于视频剪辑的项目，提供了功能丰富的可视化编辑器，用户能够借助编辑器对视频进行快速的裁剪、添加背景音乐和图文动效等，除此之外还具有实时预览编辑效果和导出合成视频的能力，满足了绝大部分用户对于视频编辑的需求。

## 集成架构
### 前端集成
开发者前端通过 iframe 的形式将云剪的视频编辑项目集成到自身的页面上，用户就可以在开发者的页面使用视频编辑项目的功能。
![](https://main.qcloudimg.com/raw/94446ed69c9e018e8be06c34b7d7a667.png)

### 交互架构
![](https://main.qcloudimg.com/raw/bff88e7fabb28aa31f3c51acf108b38e.png)

开发者与云剪的通信交互上分为前端跟后端，前端主要是通过 SDK 来实现，后端主要是开发者后端主动调用服务端 API 来实现功能的交互，上图的红线为核心的3种交互形式：
-  **前端 SDK 功能调用**：开发者可以主动通过调用 SDK 的方法去实现云剪提供的能力，如：初始化视频编辑项目、弹出上传面板等。
- **事件通知**：用户在云剪前端页面上会触发一些操作，如点击导出视频、点击添加资源等，这些事件云剪前端都会抛出给开发者前端，然后开发者可以通过识别这些事件决定后面执行什么操作。
-  **服务端 API 调用**：云剪提供了丰富的服务端 API 满足开发者的需求，例如开发者前端捕获了导出视频的事件后，开发者后端可以通过调用服务端 API 为用户生成合成的视频。

## 集成能力
- **功能强大的编辑器**：视频编辑项目提供了功能丰富的编辑器，用户可以在编辑页上借助编辑器完成视频制作。
- **快速导出成片**：用户完成视频制作后，开发者可以通过服务端 API 导出成片。
