## 我该如何使用短视频结构化分析？

目前短视频结构化分析通过云点播产品下的 API 接口进行能力开放，客户需要先开通 [腾讯云点播](https://cloud.tencent.com/document/product/266/2839) 服务，然后根据指引调取相关 API 接口即可使用。

## 能否提供短视频智能审核的能力？

当前短视频结构化分析产品，可以为客户提供短视频内容智能审核的能力，其中包括涉黄、涉政、涉恐及涉暴的审核。
## 该产品如何收费?

短视频结构化分析服务为后付费月结计费方式。

## 能否提供公有云计费示例?

短视频结构化分析主要包含短视频标签、短视频审核、短视频处理以及相似短视频推荐等功能。开通腾讯云点播后，您可以根据自己的需要选择调用不同的 API 接口来使用以上功能。该功能的计费模式均以视频智能标签的方式进行单独计费（如内容标签、涉黄标签、去水印标签和相似短视频标签），每种标签计费价格为0.08元/分钟（总时长不足一分钟按一分钟计算）。

例如您分析短视频的总时长为10000分钟15秒，所选服务为视频智能标签和视频审核。则此视频的计费总价为：
```
计费总价 = 10001 * 0.08 + 10001 * 0.08 = 1600.16元
```

## 是否可以自定义识别标签？

目前腾讯云为客户提供比较全的标签体系，当前不支持用户自定义标签，若您所关注的标签不在我们的识别库里面，可与我们及时反馈。

## 该服务是否支持私有化部署？

可以支持私有化部署。
