### 点播加密是否需要开通kms？
点播加密现在使用的是内置kms功能，客户不需要开通kms服务。

### 点播是否可以业务植入广告？
广告功能还没上线，可以使用贴片功能做广告或者定制web播放器自行实现广告功能。请参考https://cloud.tencent.com/document/product/267/5706 。

### 点播详细功能？
腾讯云点播产品，提供的功能多达48种，这48种功能根据购买的套餐类型，能够使用的种类也有所不同，详细功能如下：https://cloud.tencent.com/product/vod#price 。

### 点播可以批量嵌入水印吗？
点播可以嵌入水印并可支持批量完成

### 点播有离线缓存功能吗？
点播目前不支持离线缓存播放的功能。

### 点播有视频加密功能吗？
腾讯云点播有视频加密方案，可以支持所有的HLS播放器。

### 点播源站同步功能是？
源站同步：支持将其他源站中存储的视频文件保持目录结构无缝同步备份到腾讯视频云存储中，同步后保持原文件的播放地址不变。

### 点播视频管理功能是？
点播支持通过管理控制台或API对视频文件进行管理，包括分类，添加标签等操作，所有信息支持导出查看。
详细功能如下：
- 视频信息编辑
- 视频模糊搜索
- 视频转码、水印、封面设置
- 视频信息CSV导出
- 视频微信二维码分享
- 视频多级分类、标签
- 列表式视频管理，在线预览
- 视频冷备存储
- 黑白名单防盗链
- 动态URL防盗链
- 微信公众号视频发布接口

### 点播视频转码功能是？
点播有以下视频转码能力:
1. 分布式转码系统，支持分片转码，弹性可扩展转码资源，支持动态扩容，满足各种场景的定制化转码需求。
2. 主流格式全覆盖，多分辨率、多码率支持，灵活可配置转码模板，支持自定义水印。
3. 智能分析视频元信息，根据结果智能选择最优的转码模板，并将转码结果及时回调给用户。
4. 持支持 H.265 转码、4K 转码、HDR 转码，支视频转码为 GIF。

### 点播视频编辑功能是？
点播支持以下视频编辑功能：
1. 支持指定开始时间截取指定时长的音视频剪辑，支持将多个视频文件拼接为单个文件。
2. 支持指定时间点截图，视频采样截图和对视频截取雪碧图等截图操作。
3. 支持将视频中的音频轨删除。

### 点播视频DRM加密功能？
点播支持以下视频DRM加密功能：
- 支持数字版权保护 DRM 加密，使用 AES-128 对视频内容本身进行加密。
- 支持对单个视频文件使用多个秘钥进行加密，避免单个秘钥泄露导致整个文件泄密。
- 支持所有 HLS 播放器，提供定制化私有化视频加密服务

### 点播视频鉴黄功能是？
点播鉴黄功能如下：
1. 采用优图 DeepEye 智能鉴黄技术帮助视频平台鉴别色情图片和色情视频，大幅提升色情视频的打击覆盖面和打击效率，及时有效规避业务涉黄风险。
2. 当前 DeepEye 引擎对于色情图片的识别精度已达到：万分之一错误率下，65%以上的正确拦截率；千分之一错误率下，80%以上的正确拦截率。
开发者可以在  [用户体验平台](https://cloud.tencent.com/act/event/ci_demo.html)  体验智能鉴黄的效果（用户体验平台不需要注册账号）。

### 点播视频自定义域名功能是？
点播支持以下自定义域名功能：
1. 支持使用腾讯云域名或自定义播放域名，对自定义域名支持 HTTPS 双向认证。
2. 支持针对不同播放域名设置不同的防盗链和发布规则。

### 点播防盗链功能是？
点播支持以下防盗链功能
1. 支持配置 referer 防盗链，通过 HTTP 请求中 referer 字段实施访问控制，有效防盗刷。
2. 支持配置 IP 黑白名单，通过对访问源IP进行过滤，有效抵御恶意用户。
3. 支持 Key & 时间戳防盗链，对访问的过期时间进行控制。
4. 创新的会员视频试看防盗链功能，可对视频进行试看时间设置，试看结束后停止播放，付费后自动进行续播。

### 点播播放器SDK功能是？
点播播发器SDK支持以下功能：
1. 基于 iOS/Android/Web 的播放器 SDK，支持播放器片头/暂停/片尾贴片、Web 播放器弹幕、播放器 LOGO 自定义和播放器密码设置。
2. 支持 iframe 快速发布，渐进式加载视频文件。
3. 支持播放状态获取和事件设置 API。

### 点播提供业务统计功能吗？
点播提供以下统计功能：
1. 提供视频业务的统计分析服务，流量、带宽、点击数均可通过时间、地域、运营商维度查看。
2. 提供所有视频文件的统计分析服务，支持查看单个视频的播放次数及播放流量

### 点播余额预警功能是？
余额预警介绍：
余额预警设置的作用即为，当您账户的可用余额小于预警阈值时，腾讯云将向您发送提醒消息，提醒您及时充值或备份数据，以保证您云资源的正常使用。

### 点播的统计功能统计哪些数据呢？
当前可以查看针对全部视频文件的整体数据，包括总容量，以及流量等。未来我们将提供详细的分析数据，包括针对单个文件的统计分析（例如点击次数，高峰时段查看情况）等。 
参考链接：https://cloud.tencent.com/document/product/266/2877 。
