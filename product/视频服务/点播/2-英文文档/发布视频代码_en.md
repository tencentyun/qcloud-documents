
Log in to Tencent Cloud [VOD Console](https://console.cloud.tencent.com/vod), click "Video Management" in the left menu bar to go to the cloud video management page, and then click the ID of the video to be published to go to the details page.
Click "Video Publishing" to switch to the video publishing tab, on which you can perform the following operations:
![](https://mc.qcloudimg.com/static/img/e44fd6bf4d0f63dfe59edba15e5cd56f/image.png)
On the Video Publishing tab, you can select "Publish URL of Video Source File", or "Web Player Code Native to Video".

### URL of Video Source File
The video source file URL address corresponding to a video file with a specific bitrate, which does not contain any player information. It can be opened directly in the browser, and can be played on the browser or in the player built in the operating system.
![](https://mc.qcloudimg.com/static/img/fd32c3ecbc1c783422e59941842e65d6/image.png)

Select the file with the specific resolution or bitrate, and then select "Playback in Preview Mode" or "Display URL". The URL can be copied directly or by clicking the button. For mp4 SD files, you can also get the shared QR code.

###. Web Player Code Native to Video

The option of "Web Player Code Native to Video" is suitable for the code edited with Web-based editor, including adaptive HTML code, FLASH, and IFRAME. It contains the player configuration information.
![](https://mc.qcloudimg.com/static/img/aa3b7a639ff9ca8f0cc7e91163ded04c/image.png)
You can click the "Edit" button on the player management page;
In the editing mode, you can select an existing player in the drop-down list; adjust the player size (this can be customized); choose whether to play the video automatically;
The code output box below lists the relevant adaptive HTML code, FLASH, and IFRAME information. And you can click "Save" to obtain them.
![](https://mc.qcloudimg.com/static/img/26d804c31d0765f986195f7be1bca422/image.png)
