进入[云点播控制台](https://console.cloud.tencent.com/video)，单击左侧导航栏的【视频处理设置】>【模板设置】。模板设置内置了五种类型的模板，分别为**视频转码模板**、**音频转码模板**、**水印模板**、**截图模板**和**转动图模板**。每种模板都可以添加至任务流设置中，用于视频处理设置。

![模板设置](https://main.qcloudimg.com/raw/96105db0ad07f80732df6512ea1dd240.png)

## 视频转码模板

您可根据业务需要创建新的视频转码模板，并进行自定义设置。单击【创建视频转码模板】进入模板自定义设置。

![](https://main.qcloudimg.com/raw/f86b6164d6774745f1483140318fc879.jpg)

+ 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)和句点(.)，不超过64个字符
+ 封装格式：MP3、HLS、FLV
+ 配置项：视频参数、[音频参数](#yp)（同音频转码模板）
+ 视频参数编码标准：H.264、H.265
+ 码率（视频码率）：0或128 - 35000Kbps
+ 分辨率：0或128 - 4096px
+ 帧率：0 - 60Hz
+ 常用模板：自定义是否开启

创建好的模板可在模板列表里展示，可以设置为常用模板，并能对模板进行查看、编辑和删除等管理操作。

![视频转码模板](https://main.qcloudimg.com/raw/bb670eca686cd0e75913fdcc9ec3e07b.png)

## <a id="yp"></a>音频转码模板


您可根据业务需要创建新的音频转码模板，并进行自定义设置。单击【创建音频转码模板】进入模板自定义设置。

![创建音频转码模板](https://main.qcloudimg.com/raw/a3239c32483b8b1bb6cc01bd932b4f90.png)

+ 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
+ 封装格式：MP3、HLS、FLV 
+ 音频参数编码标准：AAC、MP3
+ 采样率：三种默认可选
+ 码率（音频码率）：0或26 - 256Kbps
+ 声道：单声道、双声道
+ 常用模板：自定义是否开启

创建好的模板可在模板列表里展示，可以设置为常用模板，并能对模板进行查看、编辑和删除等管理操作。

![音频转码模板](https://main.qcloudimg.com/raw/d4fc0939f4e8aecd158b7ab70aa6ba2a.png)

## 水印模板

您可根据业务需要通过上传图片的方式来创建水印模板，并进行自定义的默认水印配置以及水印在视频中的位置设置。

![创建水印模板](https://main.qcloudimg.com/raw/2ea843e4b6ab0e961ecab92c92a6bacf.png)

+ 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
+ 水印图片：png 格式，大小不能超过200KB、尺寸不能超过200 * 200px
+ 水印位置：左上、右上、左下、右下
+ 水平偏移：通过调整水平偏移进行精细化的位置配置
+ 垂直偏移：通过调整垂直偏移进行精细化的位置配置
+ 水印尺寸：按百分比%调整大小、 按像素 px 调整大小  

在创建完成的水印模板管理列表中可以查看水印模板的名称、预览水印文件、格式、类型、水印位置和尺寸等信息，还可以对该水印模板进行查看、编辑、删除及设为默认模板等操作。

![水印模板](https://main.qcloudimg.com/raw/7537b6bfc2757eeb5325227f31ce7607.png)

## 截图模板
您可根据业务需要创建新的截图模板，用户可以对上传的视频进行多种方式的截图，目前腾讯云控制台支持时间点截图、采样截图和雪碧图截图三种截图方式。

### 时间点截图
在截图类型中选择时间点截图，采样的时间点需在任务流中配置，模板仅进行类型的配置，详细配置方式查看[任务流设置](https://cloud.tencent.com/document/product/266/33819)。
![sp时间点截图](https://main.qcloudimg.com/raw/da2aa92eeb37cb07a8414a0935c622d0.png)

- 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
- 图片格式：JPG
- 图片尺寸：视频宽度为128 - 4096px，视频高度为128 - 4096px

### 采样截图
在截图类型中选择采样截图。

![sp采样截图](https://main.qcloudimg.com/raw/9a3af378df1d95c89cce38c12502af3f.png)

- 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
- 图片格式：JPG
- 图片尺寸：视频宽度为128 - 4096px，视频高度为128 - 4096px
- 采样间隔：百分比（最大不超过100%）、时间（s）

### 雪碧图截图
在截图类型中选择雪碧图截图。

![sp雪碧图截图](https://main.qcloudimg.com/raw/a8ed6199c5280d11ed9343b47b7ac84e.png)
- 模板名称：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
- 图片格式：JPG
- 图片尺寸：视频宽度为128 - 4096px，视频高度为128 - 4096px
- 采样间隔：百分比（最大不超过100%）、时间（s）
- 小图行数：正整数，小图行数乘以小图列数不超过100
- 小图列数：正整数，小图行数乘以小图列数不超过100

在截图模板里面会展示模板的名称、截图类型、图片尺寸。在【操作】栏可以对该截图模板进行查看、编辑和删除操作。

![sp截图模板管理](https://main.qcloudimg.com/raw/534e7abfe47a15c63e301ed3709ea439.png)


## 转动图模板

您可根据业务需要创建转动图模板，用户可以按照时间点截取所需要的转动图。转动图的时间点需在任务流中配置，模板仅进行类型的配置。详细配置方式查看[任务流设置](https://cloud.tencent.com/document/product/266/33819)。

![创建转动图模板](https://main.qcloudimg.com/raw/e4ed072208ef5e03711e51b65aa2dbb3.png)
- 图片类型：中文、英文、数字、空格、下划线(_)、短横线(-)、句点(.)，不超过64个字符
- 帧率：1 - 30Hz
- 图片质量：0 - 100%
- 图片尺寸：视频宽度为0 - 1920px，视频高度为0 - 1920px
