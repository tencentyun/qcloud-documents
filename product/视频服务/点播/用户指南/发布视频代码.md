## 打开详情页

登录腾讯云 [点播控制台](https://console.cloud.tencent.com/video)，单击左侧菜单栏【视频管理】，打开云视频管理页面。单击您想编辑的视频所在行的【管理】，打开该视频的详情页。

>?加载视频详情页需要一定时间，请耐心等待详情页加载完成。

![](https://main.qcloudimg.com/raw/a7fe91fdce454c81123c84f26dffb74c.png)

## 获取视频发布信息
视频源文件 URL 地址对应特定码率的视频文件，本身不包含任何播放器信息。可在浏览器中直接打开，应用浏览器或操作系统自带播放器进行播放。选择【视频发布】页签，您可以查看视频发布的基本信息，复制视频源文件 URL 地址，预览视频以及申请微信公众号视频链接发布。对于 MP4 标清文件，还可以获取二维码。
![](https://main.qcloudimg.com/raw/01bfed7ebd1d9e75138756cb2a1fd542.png)

## 生成 Web 播放器代码
视频带 Web 播放器代码对应可用于 Web 编辑的代码，包括自适应 HTML 代码、FLASH、IFRAME以及播放器配置信息。选择【Web播放器代码生成】页签，您可以管理播放器信息和 Web 播放器代码。
![](https://main.qcloudimg.com/raw/3addd19b74b4bcfa56f76fd39010e3d5.png)

- 在【参数设置】区域，您可以单击【播放器】右侧的【编辑】，在下拉框中选择播放器，单击【确定】。
- 在【Web播放器代码】区域，您可以在下拉框中选择合适的视频尺寸，根据需求开启或关闭自动播放，设置【代码类型】为【HTML】或【IFRAME】。
下方的代码输出框中即可提供对应参数下的 Web 播放器代码，可通过单击【复制代码】进行复制。
