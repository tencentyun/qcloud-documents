腾讯云点播提供了多元的媒体处理能力，在【媒资管理】页面中， 您可以在【已上传】页签上传并管理视频文件，在【正在上传】页签中查看正在上传中的视频文件。

## 已上传

登录 [云点播控制台](https://console.cloud.tencent.com/video)，单击左侧菜单栏的【媒资管理】，选择【已上传】页签，您可以进行上传视频、删除、视频处理和修改视频分类等管理操作。在视频列表中您可以查看视频信息（包括缩略图、名称和 ID）、视频状态、视频分类、视频来源以及创建时间等信息。

### 上传视频
>! 
- Web 上传功能需要 Internet Explorer 10 以上版本才能支持。
- 上传过程可以切换至点播控制台的其他页面，但请务必不要关闭浏览器访问其他控制台产品，否则上传过程会中断。
- Web 上传支持断点续传以及队列上传，请使用 Chrome 浏览器进行操作。

在【媒资管理】>【已上传】页面，单击【上传视频】，您可以选择**本地上传**或**视频拉取**进行视频上传。
- 本地上传：您可以直接将本地文件上传至腾讯云点播控制台。
- 视频拉取：可以通过视频 URL 将视频拉取存入云点播控制台。

![](https://main.qcloudimg.com/raw/bbbbc39b295157b9dee12a6841b712d3.png)

 > - 支持批量上传，单次最多可同时上传5个视频，选中的所有视频将会展示在待上传列表中。

#### 视频拉取

1. 选择【视频拉取】，单击待上传列表区域的【添加一行】。
  ![](https://main.qcloudimg.com/raw/24fb5618293843de14eb55d6bc417764.png)
2. 输入待拉取的源视频信息，每一行表示一条拉取内容，可单击待上传列表区域的【添加一行】添加多条内容。
 - 视频资源 URL：必选项，请输入源视频的 URL 信息。
 - 视频名称：必选项，请输入源视频的名称。
 - MD5（选填）：可选项，若需设置请输入`MD5`。
 - 优先级：可选项，可选择【高】、【中】和【低】。
3. 单击【开始上传】，进行视频上传操作。
   视频上传需要等待几分钟，您可以在【媒资管理】>【正在上传】中查看视频上传进度。视频上传成功后，您可以在【媒资管理】>【已上传】中查看和管理您的所有视频。

### 删除视频

> !删除后视频将从腾讯云完全删除且**无法找回**，也无法从 CDN 节点访问到该视频，请谨慎操作。

若视频文件不再需要，可以将其删除，释放存储空间。您可以通过以下方式删除视频：
- 单个删除：单击目标视频文件所在行的【删除】，在【删除视频】对话框中单击【确定】，即可删除该视频。
  ![](https://main.qcloudimg.com/raw/fb1f31b96bc458c453b96ce51691c407.png)
- 批量删除：勾选目标视频文件，单击列表上方的【删除】，在【删除视频】对话框中单击【确定】，即可删除您勾选的所有视频。
  ![](https://main.qcloudimg.com/raw/545906da3cbe64294a2c599435a58cb5.png)

### 修改视频分类

为了便于管理，您可以将视频文件进行分类管理。若需要修改视频分类，可以勾选目标视频文件，单击列表上方的【修改视频分类】，选择视频文件需要归属的分类，单击【确定】。
![](https://main.qcloudimg.com/raw/7a8c1eb2824ed5b086769fa13122a44f.png)

关于管理分类的操作，请参考 [分类管理](https://cloud.tencent.com/document/product/266/14059) 。

### 按时间筛选视频

您可以按照时间段对已上传的文件进行筛选：
- 选择【全部时间】，则按默认展示所有视频文件。
- 选择【自定义时间】，设置时间段并单击<img src="https://main.qcloudimg.com/raw/7d1947c897c2392e42b59cb70fb1ac17.png"  style="margin:0;">，则按您所设置的特定时间段展示视频文件。
 ![](https://main.qcloudimg.com/raw/55e0635e575ba6a2b080e4c8740f0003.png)

### 快捷查看视频信息

当您将鼠标移动到某个视频文件所在行，视频 ID 下方空白处会浮现<img src="https://main.qcloudimg.com/raw/605d17d7c3802b6f16f93558c430a7f8.png"  style="margin:0;">，单击【快捷查看】，右侧会出现该视频文件的视频信息窗口。
- 您可以在该窗口中查看视频信息，包含 ID、分类、大小、时长、创建时间、更新时间和介绍等。
- 您也可以该窗口中复制视频 URL 地址，用于视频处理和视频播放等。

![](https://main.qcloudimg.com/raw/980d7298b9feb113a3ad7f08723d7b32.png)

### 管理视频

单击目标视频文件所在行的【管理】，可以进入视频管理页。

#### 基本信息

在【基本信息】页，您可以查看并编辑视频基本信息。
![基本信息](https://main.qcloudimg.com/raw/7c3d4730127280f7bbacabd7fe6f74e2.png)
单击【编辑】，进入编辑状态（如下图），您可以编辑如下字段：

| 字段     | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 视频封面 | 上传视频后，系统会自动为您的视频生成封面。您可以自定义更换视频封面，单击【更改图片】选择并上传一张本地图片即可。 |
| 视频名称 | 您可以修改视频名称，视频命名不能含有 " ' < >，限长20个字符。 |
| 类别     | 您可以更改视频的分类。单击【修改分类】，选择分类，单击【确定】。 |
| 标签     | 您可以为您的视频文件设置标签。视频标签名的字母数字和汉字，标签之间用空格隔开，单个标签限长16个字符。 |
| 介绍     | 您可以为您的视频添加简单描述，限长128个字符。                |

> !视频封面文件限制：
> - 仅支持 JPG、GIF 静态和 PNG 格式的图片。
> - 图片大小不超过1M，尺寸不超过1920 × 1080。
> - 图片名不得含有中文字符。

![信息编辑](https://main.qcloudimg.com/raw/cf987d2da1f71f344e8a9db7e4e9f111.png)

#### 视频发布

视频源文件 URL 地址对应特定码率的视频文件，本身不包含任何播放器信息。可在浏览器中直接打开，应用浏览器或操作系统自带播放器进行播放。
在【视频发布】页，您可以查看视频发布的基本信息，复制视频源文件 URL 地址，预览视频以及申请微信公众号视频链接发布。对于 MP4 标清文件，还可以获取二维码。

![](https://main.qcloudimg.com/raw/2bb2797edad766d4ef7d28af43611853.png)

#### Web 播放器代码生成

 在【Web播放器代码生成】页，您可以管理播放器信息和 Web 播放器代码。
![](https://main.qcloudimg.com/raw/d8bb878eb86280557d6a4043b1cfcc68.png)

- 在【参数设置】区域，您可以单击【播放器】右侧的【编辑】，在下拉框中选择播放器，单击【确定】。
- 在【Web播放器代码】区域，您可以在下拉框中选择合适的视频尺寸，根据需求开启或关闭自动播放，设置【代码类型】为【HTML】或【IFRAME】。
  下方的代码输出框中即可提供对应参数下的 Web 播放器代码，可通过单击【复制代码】进行复制。

## 正在上传

登录腾讯云 [点播控制台](https://console.cloud.tencent.com/video)，单击左侧菜单栏的【媒资管理】，选择【正在上传】页签，您可以查看正在从本地上传至腾讯云点播控制台的视频信息。在视频列表中您可以查看文件名称、视频名称、视频大小、视频分类、视频处理、上传时间以及状态。其中，状态分为上传中、上传成功和上传失败。
![](https://main.qcloudimg.com/raw/c5d8a5c27551d18b8cc6c55594d83be2.png)
