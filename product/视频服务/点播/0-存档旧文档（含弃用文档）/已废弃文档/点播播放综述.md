播放器 SDK 是云点播播放功能的一项重要产品。开发者可以在移动端和 Web 端快速集成播放器 SDK，轻松实现清晰度切换、倍速播放、缩略图预览、进度条标记、视频试看和加密播放等丰富的能力。

## 类型选择

云点播提供了两类播放器 SDK，分别是**基础播放器**和**超级播放器**。

| 类型 | 播放方式 | 特点 | 下载地址 | 文档地址 |
| -- | -- | -- | -- | -- |
| 基础播放器 | 使用 URL 播放 | 支持功能有限，适用于通用场景播放 | <ul style="margin:0;"><li>[Android](https://cloud.tencent.com/document/product/881/20205)<li>[iOS](https://cloud.tencent.com/document/product/881/20205)<li>[Web](https://cloud.tencent.com/document/product/881/20207#.E5.AF.B9.E6.8E.A5.E6.94.BB.E7.95.A5) |<ul style="margin:0;"> <li>[Android](https://cloud.tencent.com/document/product/881/20213)<li>[iOS](https://cloud.tencent.com/document/product/881/20208)<li>[Web](https://cloud.tencent.com/document/product/881/20207) |
| 超级播放器 | 使用 FileId 播放 | 支持功能丰富，适用于 [视频网站](https://cloud.tencent.com/document/product/266/34147) 场景播放 | <ul style="margin:0;"><li>[Android](https://cloud.tencent.com/document/product/266/7938#sdk-.E4.B8.8B.E8.BD.BD)<li>[iOS](https://cloud.tencent.com/document/product/266/9237#sdk-.E4.B8.8B.E8.BD.BD)<li>[Web](https://cloud.tencent.com/document/product/266/14424#.E6.AD.A5.E9.AA.A41.EF.BC.9A.E5.9C.A8.E9.A1.B5.E9.9D.A2.E4.B8.AD.E5.BC.95.E5.85.A5.E6.96.87.E4.BB.B6) | <ul style="margin:0;"><li>[Android](https://cloud.tencent.com/document/product/266/7938)<li>[iOS](https://cloud.tencent.com/document/product/266/9237)<li>[Web](https://cloud.tencent.com/document/product/266/14424) |

您可以根据需要确定使用的类型之后，通过阅读 [基础播放器 - 接入指引]() 和 [超级播放器 - 接入指引]() 学习使用。
