## 接口名称
QueryTranscodeTemplateList

## 功能说明
查询转码模板列表。

## 请求方式

#### 请求域名
`vod.api.qcloud.com`

#### 最高调用频率
100次/分钟

#### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| COMMON_PARAMS | 是 |-  | 请参见 [公共参数](/document/api/213/6976)。 |

#### 请求示例
<pre>
https://vod.api.qcloud.com/v2/index.php?Action=QueryTranscodeTemplateList
&ampCOMMON_PARAMS
</pre>

## 接口应答

#### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码，0：成功；其他值：失败。 |
| message | String | 错误信息。 |
| data | Array | 转码模板信息列表，列表元素请参见 [转码模板信息参数](/document/product/266/9912#data.EF.BC.88.E8.BD.AC.E7.A0.81.E6.A8.A1.E6.9D.BF.E4.BF.A1.E6.81.AF.EF.BC.89)。 |

#### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000 - 7000 | 请参见 [公共错误码](https://cloud.tencent.com/document/api/213/6982)。  |
| 1000 | 无效参数。  |
| 10702 | 内部错误。  |
