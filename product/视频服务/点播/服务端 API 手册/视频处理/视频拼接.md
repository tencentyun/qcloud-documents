## 接口名称
ConcatVideo

## 功能说明
1. 将多个视频拼接成新视频文件，并添加到云点播系统中，新生成的视频文件将拥有新的 FileID。
1. 术语约定：待拼接的视频文件为“源文件”，拼接后生成的新文件为“目标文件”。
1. 满足条件的源文件才可以进行拼接，详细要求请参见 [源文件可拼接条件](#p1)。
1. HLS 视频拼接请参见 [HLS 视频拼接说明](#p2)。
1. 目标文件输出格式请参见 [目标文件输出格式说明](#p3)。
1. 对于**直播或互动直播录制转点播**的用户，如果您是**对同一场直播产生的多个录制片段进行拼接**，一般都能够满足拼接输入要求。

#### <span id ="p1"></span>源文件可拼接条件

- 源文件支持 MP4、FLV 或 HLS 封装格式，对于直播或互动直播录制生成 HLS 视频的拼接，请参见 [HLS 视频拼接说明](#p2)。
1. 源文件总数量不超过500个，注意：如果涉及 HLS 拼接，请参见 [HLS 视频拼接说明](#p2)。
1. 源文件总大小加起来不超过10G。
1. 源文件的封装格式**必须相同**，例如都是 FLV，或者都是 MP4，或者都是 HLS。
1. 源文件的音视频编码格式**必须相同**，例如，视频编码格式必须都是 H.264，或者都是 H.265。
1. 源文件的分辨率**必须相同**（如果分辨率不同，主流播放器无法播放强制拼接出来的文件）。
1. 源文件的色度空间、时间基**必须相同**。
1. 源文件中音频的采样频率**必须相同**。
1. 源文件的码率、帧率可以不同。

#### <span id = "p2"></span>HLS 视频拼接说明

- HLS 拼接不是简单的将 M3U8 文件合并，而是会抽出其中所有的 TS 文件，并进行拼接。
- 对于 HLS 拼接，“源文件”是指 M3U8 中包含的 TS 文件，即：待拼接 HLS 视频中的所有 TS 分片必须满足 [源文件可拼接条件](#p1)。

#### <span id = "p3"></span>目标文件输出格式说明

- 目标文件的输出格式仅支持 MP4/HLS。
1. 目标文件的音视频编码格式与源文件相同。
1. 目标文件的分辨率与源文件相同。
1. 目标文件的色度空间、时间基与源文件相同。
1. 目标文件的码率、帧率与每个源文件相同，即：目标文件的某个片段来源于哪一个源文件，则码率、帧率与该文件相同。

#### HLS 录制自动拼接
对于云直播录制转点播，由于多种原因（如 RTMP 推流意外断流），导致一场直播录制产生多个文件。此时，我们**不建议**使用本接口对录制生成的多个文件进行拼接：
- 直播过程中，视频文件的分辨率极有可能产生变化，但**本接口不支持**对分辨率不同的文件进行拼接。
- 拼接过程消耗的资源较多，且无法在直播录制完成后立即得到完整回看地址。

因此，对于“直播录制断流产生多个分片”，我们**推荐的解决方案**是“HLS 录制自动拼接”：如果您的业务允许录制 HLS 视频，则在录制过程中，云直播可以保证即时拼接。一场直播完成后，您可以在秒级拿到正常直播的完整回看地址，具体接入请参见 [HLS 录制自动拼接](https://cloud.tencent.com/document/product/267/32739#.E8.87.AA.E5.8A.A8.E6.8B.BC.E6.8E.A5.E5.BD.95.E5.88.B6.EF.BC.88.E5.A4.9A.E6.AC.A1.E6.8E.A8.E6.B5.81.E7.BB.AD.E6.B5.81.E5.BD.95.E5.88.B6.EF.BC.89)。



## 事件通知
视频文件拼接成功会触发 [事件通知 - 视频拼接完成](/document/product/266/7834)。App 后台可通过该事件监听云点播系统中的视频拼接行为。

更多请参见 [服务端事件通知](https://cloud.tencent.com/document/product/266/33779)。

#### 请求域名
`vod.api.qcloud.com`

#### 最高调用频率
100次/分钟

#### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------------|----------|---------|---------|
| srcFileList.n.fileId | 是 | String | 源文件的 FileID。 |
| name          | 是 | String    | 目标文件名称。 |
| dstType.n      | 是 | String    | 目标文件输出格式，支持同时输出 MP4 和 HLS 格式；请填写`mp4`或`m3u8`。 |
| COMMON_PARAMS | 是 | - | 请参见 [公共参数](/document/api/213/6976)。 |

#### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=ConcatVideo
&srcFileList.0.fileId=16092504232103571364
&srcFileList.1.fileId=16092504232103571365
&name=testfile
&dstType.0=m3u8
&COMMON_PARAMS
```
