## 接口名称
DescribeVodStorage

## 功能说明
1. 获取截止到昨天23:59:59的点播计费存储总量（请在北京时间每天**7:00 ~ 24:00**之间调用本接口；0:00 ~ 7:00后台数据在统计过程中，返回的数据不准确）

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| COMMON_PARAMS | 是 |  | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0) |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=DescribeVodStorage
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码, 0: 成功, 其他值: 失败 |
| message | String | 错误信息 |
| data | Object | 返回数据 |

data 点播存储量数据

| **参数名称** | **类型** | **描述** |
|---------|---------|---------|
| used_space | Integer | 存储量，单位:Byte |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见[公共错误码](/document/product/266/7783)  |
| 1 | 内部错误  |
| 1000 | 无效参数  |
| 1001 | 内部错误  |
| 1003 | 内部错误  |
| 2000 | 内部错误  |
| 10022 | 内部错误 |

### 应答示例
```javascript
{
    "code": 0,
    "message": "",
    "data": {
        "used_space":1024
    }
}
```
