## 事件名称
WechatPublishComplete

## 事件说明
当 App 配置了事件通知后，在微信发布完成后，App 后台即可通过“普通回调”或“可靠回调”的方式获取该事件通知。事件通知内容为 [WechatPublishComplete 结构](https://cloud.tencent.com/document/api/266/31773#EventContent)。

## 示例
### 普通回调
如果选择普通回调模式，则回调 URL 会接收到来自点播的 HTTP 请求。请求采用 POST 方法，请求内容在 BODY 中，如下所示（省略了值为 null 的字段）。
```json
{
    "EventType":"WechatPublishComplete",
    "WechatPublishComplete":{
        "TaskId":"Wechat-f5ac8127b3b6b85cdc13f237c6005d8",
        "Status":"FINISH",
        "ErrCode":0,
        "Message":"SUCCESS",
        "FileId":"24961954183381008",
        "Definition":10,
        "SourceDefinition":0,
        "WechatStatus":"SUCCESS",
        "WechatVid":"wechat-xxx-xxx",
        "WechatUrl":"http://xxx?xxx"
    }
}
```

### 可靠回调
如果选择可靠回调模式，调用 [拉取事件通知](/document/product/266/33433) API 会接收到如下形式的 HTTP 应答（省略了值为 null 的字段）。
```json
{
	"Response": {
		"EventSet": [
			{
				"EventHandle": "EventHandle.N",
				"EventType": "WechatPublishComplete",
				"WechatPublishComplete": {
        			"TaskId": "Wechat-f5ac8127b3b6b85cdc13f237c6005d8",
        			"Status": "FINISH",
        			"ErrCode": 0,
        			"Message": "SUCCESS",
        			"FileId": "24961954183381008",
        			"Definition": 10,
        			"SourceDefinition": 0,
        			"WechatStatus": "SUCCESS",
        			"WechatVid": "wechat-xxx-xxx",
        			"WechatUrl": "http://xxx?xxx"
				}
			}
		],
		"RequestId": "335bdaa3-db0e-46ce-9946-51941d9cb0f5"
	}
}
```
