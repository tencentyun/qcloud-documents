## 接口名称
SetWaitingTask

## 功能说明
1. 设置排队中的任务属性，目前暂时只支持优先级。

## 请求方式

### 请求域名
vod.api.qcloud.com

### 最高调用频率
100次/分钟

### 参数说明
| 参数名称 | 必填 | 类型 | 说明 |
|---------|---------|---------|---------|
| vodTaskId | 是 | String | 任务ID。 |
| priority | 是 | Integer | 更改优先级。 |
| COMMON_PARAMS | 是 |  | 参见[公共参数](/document/product/266/7782#.E5.85.AC.E5.85.B1.E5.8F.82.E6.95.B0)。 |

### 请求示例
```
https://vod.api.qcloud.com/v2/index.php?Action=SetWaitingTask
&vodTaskId=1251132654-Procedure-d7c9631c15ecf653b1ff67e34cb04692
&priority=100
&COMMON_PARAMS
```
## 接口应答

### 参数说明
| 参数名称 | 类型 | 说明 |
|---------|---------|---------|
| code | Integer | 错误码，0：成功, 其他值：失败。 |
| message | String | 错误信息。 |

### 错误码说明
| 错误码 | 含义说明|
|---------|---------|
| 4000-7000 | 参见[公共错误码](/document/product/266/7783)。  |
| 1000 | 无效参数。  |
| 10025 | 任务不存在。  |
| 10702 | 内部错误。  |

### 应答示例

```javascript
{
    "code": 0,
    "message": ""
}
```
