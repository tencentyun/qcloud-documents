针对开发者的接入反馈的高频问题，腾讯云提供有直观易懂的 API-Example 工程，方便开发者可以快速的了解相关 API 的使用，欢迎使用。

## 工程地址

| 所属平台 |                         GitHub 地址                          |
| :------: | :----------------------------------------------------------: |
| iOS  | [Github](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC) |

## 目录说明
在这个示例项目中包含了以下场景：

- **基础功能**：
  - [摄像头推流](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Basic/LivePushCamera)
  - [录屏推流](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Basic/LivePushScreen)
  - [直播拉流](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Basic/LivePlay)
  - [连麦互动](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Basic/LiveLink)
  - [连麦 PK](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Basic/LivePK)
- **进阶功能**：
  - [自定义视频采集](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Advanced/CustomVideoCapture)
  - [第三方美颜](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Advanced/ThirdBeauty)
  - [RTC 连麦 + 超低延时播放](https://github.com/tencentyun/MLVBSDK/tree/master/iOS/MLVB-API-Example-OC/Advanced/RTCPushAndPlay)
  
> ? 目前的工程结构跟标准的 xcproject 工程在名称大小写上可能有略微的差异，主要目的是方便大家在网页上看到此工程时，名称意义更加清晰。
