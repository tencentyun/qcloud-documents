## 产品说明

**小直播** 是腾讯云研发团队自主打造的一款 Demo 级 App，目的是为类似的 APP 开发工作提供源码参考，内部包含**主播推流**、**直播观看**、**录制回看**、**文字互动消息**、**点赞飘星**、**弹幕** 等基本功能：

![](//mc.qcloudimg.com/static/img/029a0197380d240b4797fdf3a3d5ba70/image.png)

小直播 App 的主要目的是希望能够提供一套**零基础接入方案**，它比较适合于之前没有接触过音视频技术，但希望能够快速上线直播功能的客户，相关源代码在开通直播服务后可免费获取。


## 主要功能
|  功能 | 说明 |
|---------|---------|
| 推流 | 支持业界标准、最为成熟的RTMP推流，支持软编及硬编 |
| 播放 | 直播支持FLV/RTMP格式，点播支持FLV/HLS/MP4格式 |
| 互动消息 | 基于腾讯十余年QQ研发沉淀的IM技术——IM SDK，实现海量文本、弹幕、点赞、送礼等互动消息 |
| 智能选路 | 从腾讯云海量节点中，智能选择最优的推流服务器，同等带宽获得最高效率的推流，最优质的效果 |
| 延迟修正 | 精准到帧维度的声画同步校准，同时极低的播出/接收延时，让用户第一时间获取画面内容，同步的进行互动 |
| 带宽自适应 | 新一代智能自适应调节技术，通过大量的用研经验，在流畅度、清晰度、延迟三维度自动进行平衡，达到最优的用户体验 |
| 动效表情 | 拥有业内领先的优图人脸识别技术，实现的实时人脸跟踪，特效互动表情实时渲染等功能 |
| 手机录屏 | 业内首个提供手机游戏录屏功能，iOS 支持两种解决方案|
| 混音伴奏 | 支持麦克风+伴奏混音功能，适合秀场直播场景 |
| 运营监控 | 控制台支持视频监控、禁播等功能，同时提供丰富的数据统计及分析，让您可以实时了解直播的质量及运营状况，更好的进行产品的迭代规划 |


## 技术架构
![](//mc.qcloudimg.com/static/img/ea11688688b6633bc24a051aa9ce566b/image.png)

小直播的前后台技术框架如上，其中主要采用了如下几项重要的技术服务：

|  技术服务 | 说明 |
|-------------------|---------|
| 移动终端 | 基于腾讯云RTMP SDK 实现移动终端的推流、直播和录制回放功能 |
| 直播后台 | 采用了快速上手的PHP技术实现，接入方式选用了直播码模式 |
| 互动消息 | 基于腾讯云 IM 通讯服务实现了主播和观众之间的互动消息服务 |
| 动效表情 | 基于腾讯云优图团队的人脸识别技术和P图团队的动效贴图构建 |  
| 录制回看 | 使用了腾讯视频云点播服务，该服务基于腾讯云对象存储服务构建 |
