## 前提条件

- **准备好序列帧图片素材**，动态贴图与特效需要导入格式为 ZIP 的序列帧图片压缩包。
>! 
>- 若您无素材资源，可单击下载我们提供的 [素材资源压缩包](https://share.weiyun.com/HXK6vSu6) 跟着教程实例进行学习。
>- 动态贴图与视频都是由一帧帧的图片渲染而成，上传的序列帧压缩包需要一套序列帧图片才能保证展示效果。
- 访问多媒体创作引擎用户端平台，进入**视频剪辑>创作工具**，单击**素材制作**，打开素材制作创建弹框。

[](id:txt)
## 文字模板
[](id:step_t1)
### 步骤1：新建文字素材项目
1. 在素材制作创建弹框中，选择制作方式为**创作新的素材**，素材类型选择为**文字**，单击**确定**。
![](https://qcloudimg.tencent-cloud.cn/raw/4569366790603900994799a02aac4b45.png)
2. 进入文字模板素材项目，设置文字素材的名称。

[](id:step_t2)
### 步骤2：编辑图片
1. 在**素材**栏中，单击**本地导入**或**云媒资导入**，选择需导入的图片素材。
>? 
>- 支持导入静态图片和序列帧  ZIP 包做的动态图片。
>- 素材上传成功后可在预览框中移动图片/动画显示位置。
>
![](https://qcloudimg.tencent-cloud.cn/raw/2dab18980681455c7072d431179bc221.png)
2. 在右侧的**样式属性**栏中，对图片素材的透明度、旋转、翻转、抽针间隔、导出入点和导出出点进行设置。
3. 在**动画**栏中给图片素材添加入场和出场的动画效果，调整动画持续时长。
![](https://qcloudimg.tencent-cloud.cn/raw/02fb5c2a73146284f81d34c591a2d274.png)

[](id:step_t3)
### 步骤3：编辑文字
1. 在**文字**栏中，单击添加按钮添加一个文字素材。支持在预览框中，挪动文字的显示位置。
2. 在右侧的**样式属性**栏中，修改文字内容、字体类型、字体大小、尺寸信息、字体颜色、字体描边、背景颜色。可添加加粗、斜体、阴影、下划线等效果。
3. 在右侧的**动画**栏中，设置文字显示的入场和出场动画效果，调整动画持续时长。
4. 在预览框中对预览效果。

![](https://qcloudimg.tencent-cloud.cn/raw/8948c49b9217044a8bc10d34e4a4c176.png)


[](id:step_t4)
### 步骤4：导出文字模板素材
1. 编辑完成后，单击右上角的**导出素材**，在导出弹框中可修改文字名称，选择该文字的存储位置，选择封面截图。
2. 单击**确定**即可导出文字模板素材。
>? 导出素材可以导入到个人素材，也可以导入到团队素材。导入到团队素材后，所有团队成员都能使用该素材。

![](https://qcloudimg.tencent-cloud.cn/raw/90215f8d397fa1fbc988b3ee363a1d0b.png)

[](id:step_t5)
### 步骤5：查看与使用文字模板素材
- **查看**：已导出的文字模板素材，均可在 **创作资源** > **文字** 中进行查看。
![](https://qcloudimg.tencent-cloud.cn/raw/0145c6f1bb2528898855d2c510fc11b7.png)
- **使用**：进入剪辑器，单击**文字** > **自定义文字** 即可以看到创建完成的素材，并拖入到轨道使用素材。
![](https://qcloudimg.tencent-cloud.cn/raw/f6a61bc9d0fd94796047b0eaa7af556f.png)

## 贴片素材
[](id:step_p1)
###  步骤1：新建贴片素材项目
<dx-tabs>
::: 需对贴片素材进行编辑
1. 在素材制作创建弹框中，选择制作方式为**创作新的素材**，素材类型选择为**贴纸**，单击**确定**。
![](https://qcloudimg.tencent-cloud.cn/raw/f18297200cd39871ce58d265ac408733.png)
2. 进入贴片素材项目，设置贴片素材的名称，即可开始 [编辑贴片](#step_p2)。
:::
::: 不需要对贴片素材进行编辑（直接导入贴片素材）
1. 在素材制作创建弹框中选择制作方式为**批量导入已有素材**，素材类型为**贴纸**。
2. 单击**添加**，直接导入本地图片素材（支持导入静态图片和序列帧  ZIP 包做的动态图片），选择该贴片素材的存储位置。单击**确定**，即可上传素材到指定位置。
![](https://qcloudimg.tencent-cloud.cn/raw/c9a50d46b2f40185556fd68e1b9b96cf.png)
:::
</dx-tabs>

[](id:step_p2)
### 步骤2：编辑贴片
1.  在**素材**栏中，单击**本地导入**或**云媒资导入**，选择需导入的图片素材。支持导入静态图片和序列帧  ZIP 包做的动态图片。
2. 在右侧的**样式属性**栏中，对贴片素材的透明度、旋转、翻转、抽针间隔、导出入点和导出出点进行设置。
3. 在右侧的**动画**栏中，设置贴片素材显示的入场和出场动画效果，调整动画持续时长。
4. 在预览框中预览贴片效果。

![](https://qcloudimg.tencent-cloud.cn/raw/6092b34a3cec292d79d64635d621effe.png)

[](id:step_p3)
### 步骤3：导出贴片素材
编辑完成后，单击右上角的**导出素材**，在导出弹框中可修改贴片名称，选择该贴片的存储位置，选择封面截图。单击**确定**，即可导出贴片素材。
>? 导出素材可以导入到个人素材，也可以导入到团队素材。导入到团队素材后，所有团队成员都能使用该素材。

![](https://qcloudimg.tencent-cloud.cn/raw/a063572b4db7c96ebb5ed32ec8897674.png)

[](id:step_p3)
### 步骤4：查看与使用贴片素材
- **查看**：已导出的文字素材，均可在**创作资源** > **贴片**中进行查看。
![](https://qcloudimg.tencent-cloud.cn/raw/fcc778ac0e300e8436fcfda8ebcfc92c.png)
- **使用**：进入剪辑器，单击**贴片** > **自定义文字**即可以看到创建完成的素材，并拖入到轨道使用素材。
![](https://qcloudimg.tencent-cloud.cn/raw/918f41fe5d0ddc773f72f92ec0b3d981.png)

## 特效素材
[](id:step_e1)
### 步骤1：新建特效素材项目
<dx-tabs>
::: 需对特效素材进行编辑
1. 在素材制作创建弹框中，选择制作方式为**创作新的素材**，素材类型选择为**特效**，单击**确定**。
![](https://qcloudimg.tencent-cloud.cn/raw/c7c9ccfaca525965ede4f10ba1d9269a.png)
2. 进入特效素材项目，设置特效素材的名称，即可开始 [编辑特效](#step_p2)。
:::
::: 不需要对特效素材进行编辑（直接导入特效素材）
1. 在素材制作创建弹框中选择制作方式为**批量导入已有素材**，素材类型为**特效**。
2. 单击**添加**，直接导入本地图片素材（支持导入图片或序列帧  ZIP 包），选择该特效素材的存储位置和封面底图。
3. 单击**确定**即可上传素材到指定位置。
![](https://qcloudimg.tencent-cloud.cn/raw/e178ed5c82223b1c70ad1aec8e0e4c7b.png))
:::
</dx-tabs>

[](id:step_e2)
### 步骤2：编辑特效
1. 在**素材**栏中，单击**本地导入**或**云媒资导入**，选择需导入的特效素材。支持导入静态图片和序列帧  ZIP 包做的动态特效。
2. 在预览框下方设置特效画布，通过切换横竖屏画布，决定特效素材横屏或竖屏应用。假设要将特效素材应用到横版视频上，选择16:9横屏。
3. 在右侧的**样式属性**栏中，设置特效素材的填充方式， 单击选择特效素材显示的位置。
4. 在右侧的**背景图**栏中，给特效增加背景封面。**封面仅为特效展示使用，拖入特效到轨道后，封面图会消失**。
<img src="https://qcloudimg.tencent-cloud.cn/raw/282bfbf38e271dd59616dfd9c514c890.png" id="p"/>


[](id:step_e3)

### 步骤3：导出特效素材
1. 编辑完成后，单击右上角的**导出素材**，在导出弹框中可修改特效名称，选择该特效的存储位置，选择封面截图。
2. 单击**确定**，导出特效素材。
>? 
>- 导出素材可以导入到个人素材，也可以导入到团队素材。导入到团队素材后，所有团队成员都能使用该素材。
>- **导出的特效素材会进行转码，转码完成后才能在剪辑器中使用**。

![](https://qcloudimg.tencent-cloud.cn/raw/93ad30627356f9479a824d97f8e55403.png)

[](id:step_e4)
### 步骤4：查看与使用特效素材
- **查看**：已导出的文字素材，均可在**创作资源** > **特效**中进行查看。
![](https://qcloudimg.tencent-cloud.cn/raw/7b24211217fac424b2a973f46a873b9c.png)
- **使用**：进入剪辑器，单击**特效** > **自定义文字**即可以看到创建完成的素材，并拖入到轨道使用素材。
![](https://qcloudimg.tencent-cloud.cn/raw/a3313a8423a6f422d55ad70188677fd7.png)
>? 特效素材导入到剪辑器后会自动应用到屏幕，自适应屏幕会根据特效大小来应用到视频中。
