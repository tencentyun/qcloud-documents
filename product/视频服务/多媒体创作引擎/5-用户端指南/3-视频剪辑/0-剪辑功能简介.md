### 素材资源模块

| 功能点   | 功能详情                                            |
| -------- | --------------------------------------------------- |
| 版权视频 | 提供自然、交通、生活等6大分类，数万+版权视频素材    |
| 版权图片 | 提供电商、美妆、旅游等上十种品类，数万+版权图片素材 |
| 版权音频 | 提供舒缓、影视、商务等上十种品类，数万+版权音频素材 |
| 文本     | 提供花字、图文结合的气泡文字以及公共版权字体        |
| 贴片     | 提供表情包、心情、爱心等7个品类，上百种贴片素材     |
| 转场     | 提供水波纹、百叶窗等上十种转场素材                  |
| 特效     | 提供梦幻、动感等6个品类上十种特效素材               |
| 滤镜     | 提供古铜、金秋等一百多种滤镜资源                    |

### 视频编辑模块

| 功能点       | 功能详情 |
| ------------ | ---------- |
| 视频音量调节 | 调节视频音量大小，静音                                 |
| 多级调色     | 支持对视频的亮度、对比度、锐度、透明度、饱和度进行调节 |
| 音频分离     | 分离视频中的音频                                       |
| 透明度调节   | 支持调节视频的透明度                                   |
| 变速         | 支持0.25-4倍变速                                       |
| 视频动画     | 支持出场、入场、运动等数十种动画效果                   |
| 位置调整     | 支持顺时针旋转、逆时针旋转、水平翻转、垂直翻转         |
| 背景         | 支持对视频纯色、图片、模糊填充                         |
| 画中画       | 支持将视频设置为画中画，同时在一个屏幕播放多段视频     |
| 视频裁剪     | 支持固定裁剪，自定义裁剪视频尺寸                       |
| 视频分割     | 支持将一段视频分割成多段                               |
| 帧定格       | 支持截取视频当前帧定格为3秒图片                        |
