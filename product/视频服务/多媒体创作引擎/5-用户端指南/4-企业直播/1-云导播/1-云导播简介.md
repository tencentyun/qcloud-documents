云导播是一款云端直播工具，可以在云端实现多个直播流间切换，画中画混流播出，录制重播，添加水印文本等功能。

![img](https://main.qcloudimg.com/raw/52f625177a62dbce7de90f1c4c45fd2b.png)

## 应用场景及核心能力
云导播适用于活动、会议直播；讲座教育等多种直播场景，核心能力包括：

| 功能       | 说明                                               |
| ---------- | -------------------------------------------------- |
| 多机位切换 | 支持直播、点播等多种类型输入，支持20+机位导播切换  |
| 画中画布局 | 支持自定义布局混流，轻松实现多画面组合直播         |
| 备播护航   | 断流自动切换备播视频或图片，为直播保驾护航         |
| 水印字幕   | 支持在直播过程中实时编辑水印和字幕，个性化直播内容 |
| 存储和回放 | 直播结束立即生成直播录制文件，一键重播回放         |



## 操作页面模块介绍
云导播主要分为6个功能区，分别是视频输入源区域、PVW、PGM、画中画、调音台和功能组件区域。

| 功能区         | 说明                                               |
| -------------- | -------------------------------------------------- |
| 视频输入源区域 | 支持自定义视频源，包括点播、直播、图片等           |
| PVW 预监       | 云导播台的预监区域                                 |
| PGM 主监       | 和观众侧保持一致的节目预览区域                     |
| 画中画区域     | 支持自定义多个视频源混流时采用的布局               |
| 调音台区域     | 支持调整输出流中，每个输入流的音量，支持音视频分离 |
| 功能组件区域   | 支持水印、文本、转场、流监控等功能                 |
