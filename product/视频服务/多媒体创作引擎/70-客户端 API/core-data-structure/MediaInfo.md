字段 | 类型 | 描述
------- | ------- | -------

Size | Integer | 视频或图片大小，单位字节。
Duration | Float | 视频长度，单位秒。
MediaType | String | 媒体类型：<li>VIDEO：视频 </li><li>AUDIO：音频</li><li>IMAGE：图片</li>
