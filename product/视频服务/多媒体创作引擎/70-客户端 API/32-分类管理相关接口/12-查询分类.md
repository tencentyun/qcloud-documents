## 输入参数

字段 | 类型 | 必填 | 描述
------- | ------- | ------- | ------- 
Platform | String | 是 | 平台。
Owner | [Entity](https://cloud.tencent.com/document/api/1156/40360#Entity)| 是 | 分类归属。 
BaseClassId |  Integer	 | 否| 查询基类的 ID，默认为 -1，表示从根分类查询。
