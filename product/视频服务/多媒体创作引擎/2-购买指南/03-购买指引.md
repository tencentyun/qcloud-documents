## 1. 注册腾讯云
- 若您已经注册腾讯云账号，可忽略此步骤。
- 若您未注册腾讯云账号，请参见文档并 [注册腾讯云](https://cloud.tencent.com/document/product/378/17985) 账号。腾讯云支持微信扫码、邮箱、QQ 和微信公众号注册等多种方式。

[](id:step2)
## 2. 完成实名认证
- 若您已经在腾讯云完成实名认证，可忽略此步骤。
- 若您未完成实名认证，请参见 [实名认证指引](https://cloud.tencent.com/document/product/378/3629) 文档完成认证。完成个人实名或企业实名认证后，均可购买智能创作服务。

[](id:step3)
## 3. 购买智能创作服务
智能创作目前支持的计费方式有**按年订阅（预付费）**、**资源包（预付费）**和**按量计费（后付费）**三种。
<table>
<tr><th width="23%">计费方式</th><th>说明</th></tr>
<tr>
<td>按年订阅（预付费）</td>
<td>即以年订阅的方式出售智能创作的全部服务，您需要预先 <a href="https://buy.cloud.tencent.com/cme">购买会员</a>，后根据会员创建平台进行使用，每天所消耗的智能创作资源，优先从会员中进行抵扣，详细请参见 <a href="https://cloud.tencent.com/document/product/1156/64101#member">按年订阅（预付费）</a>。</td>
</tr><tr>
<td>资源包（预付费）</td>
<td>即以打包方式售卖的智能创作资源包，按产品类型分为“存储资源包”、“视频导出资源包”、“直播推流资源包”及“账户数资源包”。您需要预先 <a href="https://buy.cloud.tencent.com/cme">购买资源包</a>，后使用资源，详细请参见 <a href="https://cloud.tencent.com/document/product/1156/64101#package">资源包（预付费）</a>。</td>
</tr><tr>
<td>按量计费（后付费）</td>
<td>即先使用后付费的计费方式。您需要在腾讯云账户预先 <a href="https://console.cloud.tencent.com/expense/recharge">充值</a>，系统每天会统计前一天的实际用量并进行账单推送和费用结算，从您账户的余额中扣除实际消费产生的金额，详细请参见 <a href="https://cloud.tencent.com/document/product/1156/64101#amount">按量计费（日结后付费）</a>。</td>
</tr></table>



[](id:step4)
## 4. 使用智能创作服务
购买智能创作服务后，在 [管理端](https://cme.cloud.tencent.com/) 创建平台，并通过用户端直接使用智能创作 SaaS 服务，或通过 API 使用智能创作 PaaS 服务。
