## 소개
[Tencent Cloud 게임 멀티미디어 엔진 SDK]의 사용을 환영합니다(https://cloud.tencent.com/product/tmg?idx=1). 개발자가 Tencent Cloud 게임 멀티미디어 엔진 제품을 연결하는 데 도움이 되도록 여기에 게임 멀티미디어 엔진 SDK에 적합한 콘솔 사용 통계 문서를 소개합니다.

## 콘솔 사용 통계 이용

### 응용프로그램 관리 인터페이스에서 사용 통계 열기

조회하려는 응용프로그램에서 [사용 통계]를 클릭합니다.
![](https://main.qcloudimg.com/raw/9e78b27c75b9bfcd2ce02ae1d02b7046.png)

사용 통계는 음성 채팅 및 음성 메시지와 텍스트 전환의 두 부분으로 나뉩니다.

### 음성 채팅 사용 통계

사용 통계는 음성 채팅 응용프로그램에 대한 DAU 데이터를 보여줍니다.
- [음성 DAU 총수]는 해당 응용프로그램의 어제 음성 DAU 총수를 표시함
- [대륙 음성 DAU 수]는 해당 응용프로그램의 대륙 음성 DAU 수를 표시함
- [해외 음성 DAU 수]는 해당 응용프로그램의 해외 음성 DAU 수를 표시함
- 기본 설정은 최근 7일간의 총사용량 DAU 데이터를 보여주며, 시간 전환, 지역 전환으로 사용 데이터 확인을 지원함

![](https://main.qcloudimg.com/raw/fe49e9cba23d8e3754a4212c273ffe92.png)

### 음성 메시지 및 텍스트 전환
사용 통계는 오프라인 음성 응용프로그램에 대한 DAU 데이터를 보여줍니다.
- [음성 DAU]는 해당 응용프로그램의 어제 음성 DAU 총수를 표시함
- [텍스트 전환 횟수]는 해당 응용프로그램의 어제 텍스트 전환 횟수를 표시함
- 기본 설정은 최근 7일간의 총사용량 DAU 데이터를 보여주며, 시간 전환으로 사용 데이터 확인을 지원함

![](https://main.qcloudimg.com/raw/87c04939bc58da14aeee8b931cbbde49.png)
