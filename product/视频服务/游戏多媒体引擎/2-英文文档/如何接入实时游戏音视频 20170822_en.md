## Activating Tencent Mobile Gaming
The main steps are as follows:
Step 1: Sign up for a Tencent Cloud account.
Step 2: Activate Tencent Mobile Gaming
Step 3: Create an application
Step 4: Enable a signaling channel

## Detailed Steps
### 1. Sign up for a Tencent Cloud Account
Register a Tencent Cloud account via mailbox, WeChat, QQ and submit relevant information. For more information, please see [How to sign up as a Tencent Cloud user](https://cloud.tencent.com/document/product/378/8415).
>**Note:**
>To avoid connection problems, use a public QQ number instead of your personal number.

**Note: Select "Audio/Video Industry" in "Account Information" -> "Business Information" -> "Industry Information".**
![](//mc.qcloudimg.com/static/img/f1896ed00e8e0de2a34656f0a985bf27/image.png)
![](//mc.qcloudimg.com/static/img/d3159084e34adc64e2f3141366c40505/image.png)
### 2. Activating Tencent Mobile Gaming
On the Tencent Cloud official website, choose "Product" -> "Video Service" -> "Tencent Mobile Gaming". Click "Activate Service", as shown in the following figure.
![](//mc.qcloudimg.com/static/img/394f55c74f3dbdd334f70a14d286ba00/image.png)
![](//mc.qcloudimg.com/static/img/eea855d6f4bd4d87a05990839626af7b/image.png)
Enter required information on this page. If necessary, contact relevant personnel of Tencent Cloud (or admin) to accelerate the approval process.
### 3. Creating an Application
Log in to Tencent Cloud [business backend](https://console.cloud.tencent.com/ilvb), and you need to complete identify verification if you have not passed the developer identity verification. For more information, please see [Guide](https://cloud.tencent.com/document/product/378/3629) (individual verification is recommended).
After entering the basic information, you can create a new application, and obtain the corresponding AppID and AccountType.
![](//mc.qcloudimg.com/static/img/4ccbbccbc2de58b326b0840981d0cfcf/image.png)
>**Note:**
>Remember these two parameters as the API SetAppInfo may use them.
