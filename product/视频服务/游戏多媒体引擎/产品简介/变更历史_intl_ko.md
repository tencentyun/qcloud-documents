## GME_SDK2.1 2018-08-17
### 새로운 기능:
1. Windows 버전에서 변성 지원
2. Windows에서 오프라인 음성 지원
3. Windows에서 3D 사운드 지원
4. Android SDK가 x86 아키텍쳐 지원
5. iOS와 MacSDK가 XCode10과 호환 가능

### 최적화된 기능:
1. 오프라인 음성 인증 최적화
2. 모바일 장치 단독 종료 지원
3. 보통 음질의 네트워크 레지스턴스 최적화 

## GME_SDK2.0 2018-06-22
### 새로운 기능:
1. GME가 PC Native, PC Unity 버전 출시
2. GME가 Unreal 엔진 지원
3. GME 오프라인 음성-텍스트 전환은 여러 언어를 지원하며 최대 120 종류 언어를 사용 가능
4. GME PC가 3D 음성 채팅 지원

### 최적화된 기능:
1. 통화 음량의 고음질 체험 향상
2. 액세스 제한을 줄이고 일반 음질, 표준 음질, 고음질의 다양한 선택 제공
3. 안정성 향상


## GME_SDK1.2    2018-04-02
### 새로운 기능:
1. GME가 Cocos 엔진 지원
2. 마이크 음량 조절 인터페이스 제공
3. 모바일에서 팀 음성 지원, 배틀 그라운드류 게임 지원 향상
4. PC에서 반주 재생 지원, 다양한 형식 지원
5. Android의 반주 재생에 더 많은 형식 지원

### 최적화
1. 마피아 게임 오디오 전처리 효과를 최적화하여 여러 명 동시 대화가 더욱 선명합니다.
2. 노래방과 같은 시나리오의 음질을 최적화하여 더 나음 음질을 구성 가능합니다.
3. Moba 시나리오에서 음성 지연을 최적화하여 팀 음성 지연이 더 짧습니다.
4. 잡음 소거 알고리즘을 최적화하여 음성 음질이 더욱 깨끗합니다.


## GME_SDK1.1    2017-10-18
### 새로운 기능:
1. 게임 SDK는 다양한 형식의 반주와 효과음 지원
2. 새로운 게임 시나리오에서 오프라인 음성 및 음성-텍스트 전환 기능 추가

### 최적화
1. 방 입장 인증의 클라이언트 구현 모듈을 제공해 SDK 액세스 제한을 줄이기
2. iOS/Android의 스퀼 제어효과 최적화
3. 마피아 게임에서 음질의 안정성, 네트워크 레지스턴스 등 메트릭 최적화

### 복구
Android 4.2 및 이전 시스템 버전에서 나타난 crash 문제를 복구했습니다.
