## 소개
Tencent Cloud GME SDK의 사용을 환영합니다. 개발자가 Tencent Cloud GME 제품 API를 디버깅하고 연결하는 데 도움이 되도록 여기에 GME 팀 음성의 액세스 기술 문서를 소개합니다.

### 방 가입
생성된 인증 데이터를 사용해 방에 입장합니다.


### 음성 채팅 방
#### 함수 프로토타입
```
ITMGContext EnterRoom(int roomId, ITMG_ROOM_TYPE roomType, byte[] authBuffer)
```

|매개변수     |유형         |의미|
| ------------- |:-------------:|-------------
| roomId		|int    		|방 번호, 최대 127자 지원|
| roomType 			|ITMG_ROOM_TYPE	|방 오디오 유형				|
| authBuffer    	|Byte[]  		|인증 데이터					|

|오디오 유형     	|의미|매개변수|적용 시나리오|음량 유형|콘솔이 추천하는 샘플링 속도 설정|
| ------------- |------------ | ---- |---- |---- |---- |
| ITMG_ROOM_TYPE_FLUENCY			|일반 음질	|1|유창성이 우선이고 지연 시간이 매우 짧은 음성 채팅으로 게임에서 팀 배틀에 적용되고 FPS, MOBA와 같은 게임에 적합합니다.								|스피커: 통화 음량. 헤드폰: 미디어 음량	|음질에 특별한 요구가 없는 경우, 16K 샘플링 속도이면 충분합니다.					|
| ITMG_ROOM_TYPE_STANDARD			|표준 음질	|2|음질이 비교적 좋고 시간 지연이 적당하여 마피아 게임, 보드게임 등 캐주얼 게임의 실시간 통화 시나리오에 적합합니다. 													|스피커: 통화 음량. 헤드폰: 미디어 음량	|필요한 음질에 따라 16K/48K 샘플링 속도를 선택할 수 있습니다.				|
| ITMG_ROOM_TYPE_HIGHQUALITY		|고음질	|3|초고음질, 지연시간이 상대적으로 길어서 음악, 댄스 등 게임 및 음성 소셜류 앱에 적합합니다. 음악방송, 온라인 노래방 등 음질 요구가 있는 시나리오에 적합합니다 	|스피커: 미디어 음량. 헤드폰: 미디어 음량	|최적의 효과를 보증하기 위해서 콘솔에서 48K 샘플링 속도의 고음질 구성을 설정하기를 권장합니다.	|

- 음량 유형 또는 시나리오에 특별한 요구가 있는 경우 온라인 고객 서비스에 연락하십시오.
- 콘솔 샘플링 속도 설정은 게임 음성 효과에 직접 영향을 줄 수 있으므로 [콘솔](https://console.cloud.tencent.com/gamegme)에서 샘플링 속도 설정이 항목 사용 시나리오에 적합한지 다시 확인하십시오.


### 팀 음성 방
#### 팀 음성 소개:
팀 음성: 플레이어는 게임 시작 전 팀을 구성하고 팀 음성은 팀 멤버만 들을 수 있습니다.
전체 음성: 플레이어는 게임 시작 전과 게임 과정 중 전체 음성 모드를 설정할 수 있습니다. 설정 후 플레이어 인근 일정 범위 내의 사람은 모두 이 플레이어의 소리를 들을 수 있습니다.
게임 과정 중 수시로 음성 모드를 전환할 수 있습니다.

#### 가령 A 플레이어의 상태가 전역 음성이면 해당 B 플레이어가 다른 음성 상황에서의 소리 모드는 다음과 같습니다.

|동일 팀 여부	|범위 내 여부	|음성 상태	|A가 B의 소리를 들을 수 있는지 여부	|B가 A의 소리를 들을 수 있는지 여부	|

### 음성 거리 범위에 대한 추가 내용
**주의:**
>- 음성 거리 범위 내인지 여부는 동일 팀 멤버와의 상호 통화에 영향을 주지 않습니다.
>- 음성의 수신 거리 범위를 설정하려면 API: SetGameAudioRecvRange를 참조하십시오.

#### 함수 프로토타입
```
ITMGContext  EnterTeamRoom(int roomId,ITMG_ROOM_TYPE roomType, byte[] authBuffer,int teamId, int audioMode)
```
|매개변수     |유형         |의미|
| ------------- |:-------------:|-------------
| roomId		|int    		|방 번호, 최대 127자 지원	|
| roomType 			|ITMG_ROOM_TYPE	|방 오디오 유형(매개변수는 1로만 가능)|
| authBuffer    	|Byte[] 		|인증 데이터								|
| teamId    		|int    		|가입한 팀 음성 팀 식별 코드(0이 될 수 없음)	|
| audioMode    		|int    		|0은 전역 음성을 의미하고 1은 팀 음성을 나타납니다.			|






### 팀 음성 통화 모드 수정
해당 함수를 통해 팀 음성 통화 모드를 수정합니다.
#### 함수 프로토타입  
```
ITMGRoom int ChangeGameAudioMode(Type_AudioMode gameAudioMode)
```
|매개변수     |유형         |의미|
| ------------- |:-------------:|-------------
| Type_AudioMode    |int     |0은 전역 음성을 의미하고 1은 팀 음성을 나타납니다.|

#### 샘플 코드  
```
ITMGContext.GetInstance().GetRoom().ChangeGameAudioMode(1);
```

### 수신 음성 거리 범위 설정
해당 함수는 수신하는 음성 범위 설정에 사용합니다(거리는 게임 엔진을 기준으로 합니다).

#### 함수 프로토타입  
```
ITMGRoom int SetGameAudioRecvRange(int range)
```
|매개변수     |유형         |의미|
| ------------- |-------------|-------------
| range    |int         | 음성을 수신 가능한 최대 범위				|


#### 샘플 코드  
```
ITMGContext.GetInstance().GetRoom().SetGameAudioRecvRange(300);
```
