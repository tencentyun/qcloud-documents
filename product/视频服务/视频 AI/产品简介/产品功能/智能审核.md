智能审核借助 AI 对视频内容（包括画面、音频和文字三种维度）进行涉黄、涉暴、涉政的检测。

审核结果包括审核评分、审核建议和嫌疑视频片段。根据“审核建议”，视频管理者可以决定视频是否允许公开，有效规避违规视频带来的法律风险和品牌伤害。智能审核功能所包含的检测对象和检测类型，以及详细的检测说明请见下文。


### 视频中的画面

| 检测类型 | 说明                                                         |
| -------- | ------------------------------------------------------------ |
| 涉黄检测 | 对视频画面做涉黄检查，检查内容包括：<br/>porn：色情<br/>vulgar：低俗<br/>intimacy：亲密行为<br/>sexy：性感 |
| 涉暴检测 | 对视频画面做涉暴检查，检查内容包括：<br/>militant：武装分子<br/>guns：武器枪支<br/>bloody：血腥画面<br/>explosion：爆炸火灾<br/>banners：暴恐旗帜<br/>terrorists：暴恐人物<br/>police：警察部队<br/>crowd：人群聚集 |
| 涉政检测 | 对视频画面做涉政检查，检查内容包括：<br/>violation_photo：违规图标<br/>politician：政治人物 |

### 视频中的语音
