## 视频行业标准的制定者

腾讯是视频行业标准制定的重要参与者，在视频技术应用和视频编解码领域，拥有多方面的技术优势，这些技术优势全面赋能“视频智能”基础产品，为视频行业用户提供高品质音视频编解码和 AI 结合的前沿技术。

- 腾讯 V265 编码器在“2019年世界视频编码器大赛”中获得两项关键指标第一的成绩。
- 腾讯参与制定了 VVC/H.266、MPEG-5、EVC、点云、AVS3 等编码标准。    
- 腾讯是 MPEG、DASH、DASH-IF、8K 联盟的重要参与者。
- 腾讯是 AOM 的董事会 FOUNDING MEMBERS 成员，支持 AV1 编解码器并参与 AV2 标准制定。

## 产品六大特性

### 安全可靠
智能审核承载腾讯内部产品的海量审核业务，具有丰富的样本积累和管控经验，并可借助社交网络优势对实时敏感事件进行快速响应。

### 个性匹配

智能审核的涉黄检测涵盖色情、低俗、性感等不同层级，涉暴检测包括武器枪支、恐怖分子、血腥爆炸等不同类别，涉政检测分为敏感和非敏感人物等。用户可根据实际业务场景适配。

### 简单易用

智能审核对外提供公有私有一体化、直播点播统一化的标准 REST API 接口，可同云点播服务、COS 存储、第三方存储系统快速打通，便捷接入，自主调度切换。

### 倍速可控

智能审核基于分布式视频处理算法，对于不同用户，可以满足不同倍速、不同并发、不同识别粒度的处理要求，自适应满足用户处理效率和成本的平衡，从而提升性价比。

### 高可定制

智能审核通过接入简单、成熟稳定、高效灵活的 API 接口，为用户量身定制视频智能服务私有化中台，帮助用户构建安全稳定、高效可靠、性价比高的视频云服务生态。

### 多维快审

智能审核通过准确高效的黄暴政、音画字、广告内容识别，以及自定义审核类型、阈值、图库、样本回流等个性化配置，为用户大幅降低人工审核的风险和成本。
