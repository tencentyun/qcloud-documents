

## 成本投入可控
云转码无需前期投资，按使用量收取费用，无最低限制，为用户降低大量投入。

## 高并发性
云转码拥有大量分布式转码集群，可同时支持超多并发转码任务，能够按需动态调整转码资源，自动扩容或缩容，应对高并发转码需求，无缝扩展集群资源。·

## 高可定制
云转码具备音视频转码、截图、水印及封面等丰富的视频处理功能，高可扩展的转码模板，支持自定义转码参数，满足多样化转码需求。

## 自定义任务组合
云转码支持自定义流式转码任务，可对海量视频文件进行流程化处理，一站式完成转码、截图、水印及转动图等基本操作。

## 支持事件通知
支持 CMQ 消息队列接收视频处理的事件通知，及时掌握转码任务的执行动态。

## 自动触发
文件上传后自动触发视频处理，无需主动调用 API 即可发起转码任务。

## 完善的 API
云转码提供完善全面的服务端 API 和简洁易懂的接入文档，方便用户快速对接。
