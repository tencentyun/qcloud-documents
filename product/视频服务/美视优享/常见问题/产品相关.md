### 什么是美颜特效 SDK？
美颜特效 SDK（优图美视）是由腾讯云与天天 P 图及优图实验室联合打造的高级视频处理方案，为您的视频拍摄提供了滤镜、美颜美型、贴纸及手势识别等多种实时特效功能，覆盖多种拍摄场景，满足用户多类拍摄需求。


### 美颜特效 SDK 支持图片处理吗？
美颜特效 SDK 支持图片处理，您可通过购买短视频企业版进行图片处理，具体请参见 [图片转场](https://cloud.tencent.com/document/product/584/20327)。


### 美颜特效是否支持第三方接入使用？
**美颜特效已集成到移动直播企业版 SDK 、短视频企业版/企业版 Pro SDK** 中，支持高级美颜、动态贴纸、AI 抠图、绿幕抠像等功能。您可通过购买商业版 SDK 使用美颜特效功能，现支持 iOS 和 Android 接入，详情请参见 [直播美颜特效](https://cloud.tencent.com/document/product/454/9018) 和 [短视频美颜特效](https://cloud.tencent.com/document/product/584/13509)。


### 如何调用基础版美颜特效 SDK？
基础版的短视频 SDK 和 移动直播 SDK 提供了基础美颜功能（支持拍摄设置人脸的磨皮、美白及红润并调节强度）。您可参考  [移动直播 SDK-功能说明](https://cloud.tencent.com/document/product/454/19075) 或 [短视频 SDK-功能说明](https://cloud.tencent.com/document/product/584/9453#.E5.8A.9F.E8.83.BD.E5.88.97.E8.A1.A8) 比对基础版和企业版的区别，基础美颜功能调用具体请参见 [美颜美白和红润特效](https://cloud.tencent.com/document/product/454/7879#8.-.E7.BE.8E.E9.A2.9C.E7.BE.8E.E7.99.BD.E5.92.8C.E7.BA.A2.E6.B6.A6.E7.89.B9.E6.95.88)。
