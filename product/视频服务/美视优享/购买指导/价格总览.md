**计费说明：**
-  **一个 License 可以绑定一个 Bundle id 和一个 Package Name，不区分测试和正式环境**。License 绑定的包名信息不支持修改。若您有多个 App，需要购买多个企业版 License 来创建对应数量的 App 使用。
- License 的有效期为1年，到期后需重新购买企业版 License 进行续期，否则对应的功能将无法使用。

**计费示例：**
用户甲在 2019年01月01日购买了移动直播企业版 License A，在 2019年06月01日购买了10TB标准直播流量包，获得的移动直播基础版 License B，均绑定了相同的包名。则：

- 企业版 License A 有效期到 2019年12月31日 23:59:59，包含移动直播基础版功能和高级美颜特效等功能。
- 基础版 License B 有效期到 2020年05月30日 23:59:59，包含移动直播基础版功能。

即该包名的应用在 2019年01月01日 00:00:00 - 2019年12月31日 23:59:59 可以使用企业版的功能（移动直播基础版功能 + 高级美颜动效），在 2020年01月01日 00:00:00 - 2020年05月30日 23:59:59 只能使用移动直播基础版功能，高级动效无法使用。
