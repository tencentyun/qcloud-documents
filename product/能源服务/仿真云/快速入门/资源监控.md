在资源监控中，可快速查看负载的 CPU、内存等指标的使用情况。

## 查看资源监控详情
1. 登录[ 仿真云控制台](https://console.cloud.tencent.com/cloudsim/cluster-monitor)。
2. 在左侧导航栏，单击**监控中心 > 资源监控**。
![](https://main.qcloudimg.com/raw/5da331257de5ac30fa50e00502779941.png)
3. 在顶部菜单栏处，选择区域。
4. 在**资源监控概览**区域，单击监控时段，查看该时段内的监控详情。

## 资源的监控指标
资源的监控指标如图所示，共有6项指标：
![](https://main.qcloudimg.com/raw/d21e0e2319074d5aced98b25dfd5a907.png)
- CPU 总数：统计当前 CPU 的使用数量。
- 内存总数：统计当前 CPU 的使用总量。
- 存储总量 CFS：统计当前文件存储总量。
- 存储总量 COS：统计当前对象存储总量。
- 节点总数：统计当前使用的节点数量。
- 作业总量：统计当前提交的作业数量。

## 内存使用率
在内存使用率中，可查看任务占用内存、集群占用内存和未使用内存的占比，并以饼状图展示。
![](https://main.qcloudimg.com/raw/8079c2b0949455492edcfd6bfd7f4525.png)

## 资源监控雷达图
在资源监控雷达图中，分别选择数据源（如直接选择集群或按任务选择集群）、监控指标和时间段（点击监控时段），可查看此时间段内某集群资源使用情况。
![](https://main.qcloudimg.com/raw/7106625b10f52d024487757be487bfc3.png)

## 资源监控图表
在资源监控图表中，点击监控时段，可查看此时间段内集群资源使用情况，主要指标如下：
- CPU 利用率：当前占用的 CPU 资源的百分比。
- CPU 系统平均负载：当前时间段内，系统处于可运行状态和不可中断状态的平均进程数，也就是平均活跃进程数。
- 内存利用率：当前时间段内，进程所开销的内存百分比。
- 内存使用量：当前时间段内，进程所开销的内存。
- 内网出入带宽：内网出、入带宽分别指从Internet流入仿真云方向的带宽和从仿真云流出到Internet方向的带宽。
- 内网出入包量：内网出、入包量分别指发、收包的数量。

## 文件存储图表
在文件存储图表中，点击监控时段，可查看此时间段内文件存储情况，主要指标如下：
- 写入延时：指文件写入的延迟时间。
- 读取延时：指文件读取的延迟时间。
- 写入带宽：指文件写入存的速率。
- 读取带宽：指文件读取的速率。
- 写入 IOPS：指存储设备每秒平均的写入次数。
- 读取 IOPS：指存储设备每秒平均的读取次数。
