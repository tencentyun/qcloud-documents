仿真云为运行中的集群提供了多项监控指标，用来监测集群的运行状况。可以根据这些指标，实时了解集群的运行状况，保障集群稳定运行。

## 查看集群监控详情
1. 登录[ 仿真云控制台](https://console.cloud.tencent.com/cloudsim/cluster-monitor)。
2. 在左侧导航栏，单击**监控中心 > 集群监控**。
![](https://main.qcloudimg.com/raw/502ada823e8325d3800a85a91ec81788.png)
3. 在顶部菜单栏处，选择区域和集群。
![](https://main.qcloudimg.com/raw/9c52ed83abcfcdf9581783acb092404b.png)
4. 在**集群监控概览**区域，单击监控时段，查看该时段内的监控详情。
![](https://main.qcloudimg.com/raw/ad4c0adc641a1f270c1e61f04ff0d977.png)

## 集群的监控指标
集群的监控指标如图所示，共有7项指标：
![](https://main.qcloudimg.com/raw/3307c017134cdcd9d647768848bdbe49.png)
- 集群节点总数：统计该集群上节点的数量。
- 集群 CPU 总数：统计该集群上 CPU 的数量。
- 集群内存总数：统计该集群上内存的总量。
- 集群 GPU 总数：统计该集群上内存的数量。
- 集群存储总数：统计该集群上存储的总量。
- 集群内存使用率：统计集群进程所开销的内存百分比。
- 集群 CPU 使用率：统计集群占用的 CPU 资源的百分比。

## 任务监控详情
在任务监控详情中，单击监控时段，可查看已注册、运行中、已成功、已失败、停止中和已停止的任务数量，并按照饼状图进行展示。
![](https://main.qcloudimg.com/raw/064515ece4323c56f8dc82dfcbd3d4de.png)



## 集群计算任务监控
在集群计算任务监控中，点击监控时段，可查看此时间段内的失败、成功和未运行的任务，并可选择柱状图或折线图的形式显示。
![](https://main.qcloudimg.com/raw/b0f28a7f5e02d7a07bf1d213c7768cc0.png)



## 集群资源任务监控图表
在集群资源任务监控图表中，点击监控时段，可查看此时间段内集群资源使用情况，主要指标如下：
- CPU 利用率：当前占用的 CPU 资源的百分比。
- CPU 系统平均负载：当前时间段内，系统处于可运行状态和不可中断状态的平均进程数，也就是平均活跃进程数。
- 内存利用率：当前时间段内，进程所开销的内存百分比。
- 内存使用量：当前时间段内，进程所开销的内存。
- 内网出入带宽：内网出、入带宽分别指从 Internet 流入仿真云方向的带宽和从仿真云流出到 Internet 方向的带宽。
- 内网出入包量：内网出、入包量分别指发、收包的数量。

## 集群文件存储图表

在集群文件存储图表中，点击监控时段，可查看此时间段内文件存储情况，主要指标如下：
- 写入延时：指文件写入的延迟时间。
- 读取延时：指文件读取的延迟时间。
- 写入带宽：指文件写入存的速率。
- 读取带宽：指文件读取的速率。
- 写入 IOPS：指存储设备每秒平均的写入次数。
- 读取 IOPS：指存储设备每秒平均的读取次数。
