### 为什么要使用腾讯云碳引擎？
传统的碳排放盘查经常会遇到如下挑战：
- 数据分散，需要人工抄录。
- 生成的静态报告，无法进行深度分析。
- 碳盘查数据量大，计算复杂容易出错。
- 查找行业相关参数困难，无法持续对碳排放数据进行监测和调整。

腾讯云碳引擎能够精准解决传统碳盘查中遇到的上述困难，助您零门槛碳盘查。
- 数据汇接能够快速接入您分散在不同系统中的源数据，并支持数据手动编辑。
- 预置国家排放核算方法，参数推荐值及排放因子，供用户快速引用。
- 功能强大的碳排放公式编辑器，友好的可视化交互界面，支持子算式嵌套、数据积分及累加求和等复杂运算，公式保存后将自动实时计算碳排放，用户可随时掌握碳排放的精准数据，及时对碳排放纠偏。

### 腾讯云碳引擎支持哪些数据源？
腾讯云碳引擎内置以下数据：
- 国家各行业碳排放核算规范的公式模板。
- 国家碳排放核算规范中的参数推荐值。
- 各类公开发布的碳排放因子。

同时，我们支持您的碳活动水平源数据通过以下方式接入：
- 通过腾讯云特有的数据汇接，以 API 接口形式灵活、快速接入各类系统中的数据。
- 通过手动上传 Excel 文件。
- 通过在公式编辑界面自定义数值。

### 腾讯云碳引擎支持哪些方式的碳排放数据导出？
腾讯云碳引擎支持以下方式，将您的碳排放计算结果进行导出：
- 在**碳盘查**界面上，点击**查看明细**，可用 Excel 方式下载所选时间段内的碳排放数据明细。
- 通过数据汇接，将碳排放数据经由 API 接口输送给各类系统。
- 通过拖拽式 BI 模块，将碳排放数据制作成为报表，并输出成为 Excel、PDF 等格式，可设置定时导出和发送功能，将报表导出、发送至您的指定地点。
