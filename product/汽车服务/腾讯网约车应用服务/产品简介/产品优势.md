### 注册即用
注册该产品后即可获取对应的 API 授权，用户不需要关心带宽、存储、服务器，亦无需担心基础服务可用性，所有的底层基础设施均由 SaaS 负责管理，腾讯云提供服务保障，确保服务可用性。

### 基础流量免费
注册该产品后，根据用户的认证和余额情况，可获得能够支撑最高1000单/日的级别的基础免费流量，有效地支撑用户在业务准备阶段的能力建设，降低用户业务发展成本。

### 功能可选可配
网约车应用服务包含多种场景和功能，包括但不限于调度、行程预警、取消判责、车辆管理功能，每个功能可配置所需要的策略以及可选的能力。例如，选择所使用的地图、配置调度的极限值、行程预警级别、责任判定的阈值等。

### 弹性容量
出行业务存在明显的波峰和波谷，自建存在较大的浪费。SaaS 化的产品具备弹性扩容的特点，有力地满足用户业务在波峰时的需求，降低在波谷时的成本。

### 数据可视化
每个基础场景 SaaS 都配备基本的可视化数据套件，可将独立模块的数据可视化展示出来，实现数据-业务-策略的闭环，有效地提升用户使用体验，以及业务发展的可持续性。

### 数据安全
全系 API 支持 HTTPS 请求，有效地保护用户的数据，防止中间人攻击；腾讯云提供安全防护，有效地抵御 DDoS 等攻击，避免资源浪费，提升服务可用性。

### 业务多覆盖
网约车应用服务不仅适用于 To C 用户，同时也支持用户 To B 业务的开展。例如，如祺对公出行使用了调度 SaaS，有力地支撑了多个场景的调度诉求。
