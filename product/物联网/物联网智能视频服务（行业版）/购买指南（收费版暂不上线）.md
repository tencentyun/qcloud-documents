## 价格详情

物联网智能视频服务（行业版）采用灵活的计费方式，为用户提供高性价比的产品。具体如下：

- 对于设备接入支持按路数收费。
- 对于可管理的最大视频存储，支持按容量收费。

## 费用计费示例

### 示例一

某客户全国有250家连锁门店，每家门店有4路摄像头（平均400kbps码率），出于店铺运营和监管要求，需要通过 App 查看视频实时数据，并将摄像头录像数据存储10天，随时查阅。若11月该客户观看流量为 3.2 GB，则该月费用为：

- **设备接入费（按月付费）**：4 元/台/月 x 1000 台 = 4000 元/月。
- **视频存储费（按月付费）**： 41.2 GB/路/月 x 0.0048 元/GB x 1000 路 x 30 = 5932.8 元/月。
- **观看流量费（按量付费）**： 8.2 GB x 0.45元/ GB = 3.69 元。

**故总费用** = **设备接入费** + **视频存储费** + **观看流量费** = 4000 + 5932.8 + 3.69 = 9936.49元。

### 示例二

某客户全国有100间仓库，每间仓库有5路支持录制的摄像头（平均400kbps码率）和5路不支持录制的摄像头（平均400kbps码率），需要通过 App 查看视频实时数据，并将摄像头录像数据存储10天，随时查阅。若某月该客户观看流量为20GB，则该月费用为：

- **支持录制设备接入费（按月付费）**：4 元/台/月 x 500 台 = 2000 元/月。
- **不可录制设备接入费（按月计费）**：1.5 元/台/月 x 500 台 = 750 元/月。
- **视频存储费（按月付费）**： 41.2 GB/路/月 x 0.0048 元/GB x 500 路 x 30 = 2966.4 元/月。
- **观看流量费（按量付费）**： 20 GB x 0.45元/ GB = 9 元。

**故总费用** = **支持录制设备接入费** + **不可录制设备接入费** + **视频存储费** + **观看流量费** = 2000 + 750 + 2966.4 + 9 = 5725.4 元。
