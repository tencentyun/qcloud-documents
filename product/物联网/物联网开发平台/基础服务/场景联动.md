

## 操作场景

场景联动功能用于帮助用户通过可视化的规则定义，快速完成设备相关的自动化业务逻辑的开发，无需理解技术底层实现。
通过设备或时间维度的触发器定义，经过一定的执行条件过滤，最终执行输出结果。

## 操作步骤

### 新建场景

进入场景联动界面，单击**新建场景**，即可进入创建场景规则弹窗
![](https://qcloudimg.tencent-cloud.cn/raw/a0ad863a6d41cf01bf9ab1e1ea4a50c9.png)

### 新建成功

输入场景名称和场景描述，单击**保存**按钮：
![](https://qcloudimg.tencent-cloud.cn/raw/7b8bc8da655b031b43b1d37fd787b48a.png)

### 场景联动列表

保存成功后，场景联动列表会新增该条联动规则，如下图所示：
![](https://qcloudimg.tencent-cloud.cn/raw/53269ff5545cab955e3bc56fc2be8faa.png)

场景联动列表参数说明：

| 参数     | 描述                                                         |
| :------- | :----------------------------------------------------------- |
| 场景名称 | 用户可自定义该场景的中文名称；支持中文、特殊字符、英文输入，64字符。 |
| 场景描述 | 用户可用于记录该场景联动的使用场景和作用；支持中文、特殊字符、英文输入，64个字符。 |
| 状态     | 分开启、关闭两种状态，用户控制该条联动规则是否正常运行。在开启状态下，满足条件才会触发该条场景联动。在关闭状态下，场景联动规则不会触发。 |
| 创建人   | 创建该条场景联动的创建人名称。                               |
| 操作     | 查看该条场景联动规则的配置信息；日志：场景联动规则的触发日志，详细介绍见下方联动日志；删除：删除该条场景联动规则。 |

### 联动规则配置

单击**查看**按钮，进入联动规则配置界面：
![](https://qcloudimg.tencent-cloud.cn/raw/044cc9c2fcf0be5f73c5e2ee6b0af3cb.png)

操作示例：

当设备A的电量过低时，且在2022-01-01 10:35～2022-01-31 10:35期间触发时，最终执行动作，打开设备B，将灯打开，亮度设置为80，颜色为Green。
![](https://qcloudimg.tencent-cloud.cn/raw/9acaa87e27828ec3a8b8943761b25e66.png)

| 参数              | 描述                                                         |
| :---------------- | :----------------------------------------------------------- |
| 触发器（必选）    | 即触发该联动的条件，可设置为设备触发或定时触发。当设备上报的数据或当前时间满足设定的触发器时，触发执行条件判断。可以为一个规则创建多个触发器，触发器之间是或（or）关系。 |
| 过滤器 （非必选） | 即过滤条件设置。只有满足过滤器条件的数据，才能触发执行动作。可设置为设备状态或时间范围。可以为一个规则创建多个执行条件，执行条件之间是和（and）关系，必须同时满足才能够最终触发执行器。 |
| 执行器（必选）    | 即该条规则触发后最终执行的动作。用户可以设置多个动作，某一动作执行失败时，不影响其他动作的执行。 |

### 保存、启动联动规则

1. 单击**确定**按钮，保存上述配置的联动规则。
2. 进入场景联动列表，启用该条规则：
![](https://qcloudimg.tencent-cloud.cn/raw/bf7fbffd84823bc2a032861023cbbe74.png)

### 场景联动日志查看

在场景联动列表界面，单击**日志**按钮，进入查看场景联动日志界面，即可查看联动规则的执行日志：
![](https://qcloudimg.tencent-cloud.cn/raw/9b7f9675043eb8796179abe78c08db0e.png)

执行结果处表示该条联动规则是否被成功执行。
