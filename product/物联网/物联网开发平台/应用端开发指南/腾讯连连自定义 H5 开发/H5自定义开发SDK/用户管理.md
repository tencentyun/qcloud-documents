

## 获取用户信息

**接口定义**

```typescript
sdk.getUserInfo() => Promise<{
	Avatar: string,
	CountryCode: string,
	Email: string,
	NickName: string,
	PhoneNumber: string,
	UserID: string
}>
```

**返回值**

返回一个 Promise，输出参数如下。

| 参数名      | 参数描述 | 类型   |
| ----------- | -------- | ------ |
| Avatar      | 头像 URL | string |
| CountryCode | 国家代码 | string |
| Email       | 邮箱地址 | string |
| NickName    | 昵称     | string |
| PhoneNumber | 手机号码 | string |
| UserID      | 用户 ID  | string |
