
qcloud-iot-esp-wifi 面向使用乐鑫 ESP Wi-Fi 芯片/模组（例如：ESP8266）来接入腾讯云物联网服务的开发者，包括使用腾讯云 IoT AT 指令 ESP8266 定制模组固件接入，以及使用 ESP8266 RTOS 平台进行 SoC 方式开发来接入腾讯云的用户，详情可参考 [MCU+ 定制 MQTT AT模组（Wi-Fi 类）接入指引](https://cloud.tencent.com/document/product/1081/48394) 和 [MCU+ 通用 TCP AT 模组（FreeRTOS）接入指引](https://cloud.tencent.com/document/product/1081/48396)。

## SDK 获取

SDK 使用 Github 托管，可访问 Github 下载最新版本设备端 [ESP8266 SDK](https://github.com/tencentyun/qcloud-iot-esp-wifi)。
