## 指示灯说明

**电源指示灯共有如下几种亮灯方式：**

| 亮灯方式 |             用途             |
| :------- | :--------------------------: |
| 灭       |          未接通电源          |
| 绿色闪烁 |      设备开机启动过程中      |
| 绿色常亮 |         设备已经启动         |
| 黄色闪烁 | 设备启动或运行过程中出现异常 |

## 按键说明

**RST 按键共有如下几种使用方式：**

| 触发方式            |           用途            |
| :------------------ | :-----------------------: |
| 按一下（脉冲触发）  |         设备重启          |
| 长按 10s（电平触发） | 恢复原厂设置/回退到原状态 |
